Release notes 9.9.3 - Fixed detection of missing billing codes for microbiology/pathology request definitions (GLIMS_BILL-03730)

If, during tariffication, GLIMS cannot find a billing code (assignment) for a request definition of type Microbiology (or Pathology) that is marked as To charge, an error message will now be shown.

A billing code (assignment) must be found: via the request definition, the procedure or a child record (for instance: Carrier.Medium).
