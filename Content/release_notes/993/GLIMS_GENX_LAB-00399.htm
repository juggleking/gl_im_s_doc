Release notes 9.9.3 - Rich text logic support for Unix (GLIMS_GENX_LAB-00399)

An issue was detected where the rich text logic required for the Genetics module was not correctly implemented in the GLIMS package for Unix.

When:

- Printing a report for an order containing rich text

- Generating the work list via batch

The error 'Static instance failed to load' appeared in the AppServer log files.

This issue has been corrected: the code for Rich text support is now included in the GLIMS package for Unix.
