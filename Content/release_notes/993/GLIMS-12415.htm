Release notes 9.9.3 - Actions are preselected again when replacing a specimen (GLIMS-12415)

Context

The Specimen.Replace function allows to move non-finished actions from a specific specimen to another specimen of the same material in the same order. When selecting a specimen and hitting the Replace button (or double-clicking the target specimen), you will first be presented with a browser of all the unfinished actions on this specimen, with all the actions preselected.

Issue

The actions in the browser were no longer preselected.

Solution

This issue, which occurred since GLIMS 9.6.0, has been corrected: the actions are now preselected again.
