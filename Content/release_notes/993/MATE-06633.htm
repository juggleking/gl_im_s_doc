Release notes 9.9.3 - Fixed expansion of dynamic text in text editor opened from browser (MATE-06633)

The text editor that opens when using the F6 key in the I.Comm / E.Comm column of the isolation browser of the microbiology work screen has an Expand icon that allows the user to expand dynamic text. However, an issue was reported where expanding a text resulted in the previously expanded text (which was kept in memory) to be shown.

This issue has been corrected.
