Release notes 9.9.3 - Fixed issues in "Edit carrier tests" screen (GLIMS_BAC-01468)

This modification corrects a number of issues that occurred in the Edit carrier tests screen that can be opened from a microbiology action.

For instance:

  • When opening the contextual menu by right-clicking, GLIMS did not select the correct carrier, instead GLIMS selected the first carrier in the screen.
  • When the cursor was positioned in an empty cell and the user used the Insert key, an error occurred.
  • Using F6 on a carrier test (not the first) opened the editor of the first carrier.
