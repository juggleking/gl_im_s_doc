Release notes 9.9.3 - Support export to Excel for grids in the Genetics module (GLIMS_GENX_LAB-00405)

New functionality

From now on, it is possible to export any grid inside the Genetics module to Microsoft Excel. Via the key combination CTRL+E, the selected grid is exported into an Excel file.

Currently, this functionality is only available for the grids displayed in the Genetics module screens.
