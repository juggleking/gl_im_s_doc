Release notes 9.3.0 - Function gp_site.CheckUp now available for users (GLIMS-05926)

The minimal user type for the gp_site.CheckUp function has been lowered from 'System manager' to 'User' so that this function is now also available for users.

Note: when upgrading to GLIMS 9.3, a privilege will be created in GLIMS for this function for backwards compatibility.
