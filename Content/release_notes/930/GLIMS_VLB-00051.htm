Release notes 9.3.0 - Group results for export to Valab (GLIMS_VLB-00051)

Requirement

In some cases, Valab requires a group of results to perform a correct validation.

Solution

GLIMS now allows to group results before sending them to Valab. The Valab validation screen in GLIMS has a new option "Classification" which allows to select a property classification.

A "group" is defined as a property classification node with "Reportable" set to "When complete", having child nodes for properties. Of course, only properties available in the property code set of the Valab validation coding system are considered.
