Release notes 9.3.0 - Quick access to "ClearDownloadStatus" function (GLIMS-06159)

In previous versions, when trying to rerun a test while positioned in a Result context, one had to:

  1. Right-click and choose "Action outputs"
  2. Right-click and choose "Action"
  3. Right click and choose "Clear download status"

This has now been simplified. The Result table now has a new function "Clear download status" which does exactly the same as the Action-based function, but it is immediately accessible from the contextual menu/ribbon for results.
