Release notes 9.3.0 - Validity check for fax number (GLIMS-06236)

A new field Fax number validity is available in the general options (Start -> System management -> Customize -> GLIMS general). It allows to define a MISPL expression that checks the validity of fax numbers in GLIMS.

The validity check is executed when:

  • Editing and saving Correspondent configuration

  • Entering a fax number in the Add medium report screen
