Release notes 9.3.0 - Sorting services and translator in Watchdog (MATE-03201)

In previous versions, the system overview sent by the GLIMS to Watchdog daemon, always sorted the services and translators by their database creation order. As the Watchdog Dashboard has no sorting capabilities, this would result in a semi-random placement of the services and their translators within the Dashboard user interface.

Even though the Watchdog dashboard still doesn't have sorting capabilities, the application now sorts the services and translators in alphabetical order, which ensures a more logical placement of these items within the Dashboard user interface.
