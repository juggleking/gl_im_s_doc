Release notes 9.3.0 - Conversion of attachment categories (MATE_COMHL-00268)

During the upgrade to GLIMS version 9, the Attachment site attribute "Category" is converted to records in the new "AttachmentCategory" table. However, as the "Name" field of this table does not allow "spaces", these will be stripped during the conversion.

Warning: When using the Twain scanning functionality, please check the "Document type" configuration to ensure that the "attachment category" in the "Twain scan categories" corresponds to the "Name" of an actual "Attachment category" record.
