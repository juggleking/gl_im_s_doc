Release notes 9.3.0 - Flexible auto-confirmation of isolation storage (GLIMS_ISTO-00012)

Existing feature

In previous versions, when using the Isolation-based function "Storage -> Define location", it was already possible to auto-confirm the corresponding Item storage by enabling the Archive option "Auto-confirm store".

New feature

To provide more flexibility, the Isolation-based function "Storage -> Define location" itself now allows to indicate if the corresponding Item storage should be auto-confirmed. The default value of this option is synchronized with the "Auto-confirm store" setting of the selected archive.
