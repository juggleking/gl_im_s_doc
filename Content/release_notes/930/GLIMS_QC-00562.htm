Release notes 9.3.0 - Protect QC lot and QC population from being deleted (GLIMS_QC-00562)

Introduction

In the context of the ISO 15189 norm, a check has been implemented that prevents QC lots and QC populations from being deleted too soon.

New feature

A new option "QC lot validity in years" has been added to the QC setup (Start -> Quality control -> Configuration -> QC setup).

It will not be possible to delete QC lots (and their QC populations) if the QC lot is still valid according to this new setting.
