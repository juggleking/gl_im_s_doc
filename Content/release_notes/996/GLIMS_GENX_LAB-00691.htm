Release notes 9.9.6 - Fixed error that occurred during ribbon item selection after refreshing certain Genetics screens (GLIMS_GENX_LAB-00691)

An issue was reported where refreshing the genetic result screen or genetic order screen and activating the ribbon item that had been activated by the user prior to refreshing the screen, resulted in the error message "Invalid or inappropriate handle value given to RUN...IN statement. (2128)".

This issue, which occurred since GLIMS 9.9.1, has been corrected.
