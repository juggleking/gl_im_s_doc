Release notes 9.9.6 - Financial export in German "KVDT" format: support for new field FK 5010 "Chargennummer" (Q2-2021) (BILX_GKVDT-00533)

The export of financial data in the German KVDT format (using the FinancialShipment.Send function) has been updated to support the export of a "Chargennummer".

The "Chargennummer" can be entered in the Order as the result value of the property specified in the site attribute named XDT_5010Property.

The KVDT export will send the "Chargennummer" in FK 5010:

  • for all GNR (starting with) 88331, 88332, 88333, 88334, 88336Â¸88336, 883317, 88338, 88339 and 88340
  • if the result value of the property specified in the site attribute XDT_5010Property is not discontinued

For GNR 88331 etc. without a "Chargennummer", the KVDT export check will produce a warning (Regel 868).
