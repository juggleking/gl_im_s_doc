Release notes 9.9.6 - Active tab page in Levy Jennings graph no longer changes in function of the selected QC population (GLIMS_QC-01002)

When opening the Levy Jennings graph from the QC population browser, the active tab page in the graph window changed when navigating to a different QC population in the browser. GLIMS determined the active tab page in function of the selected QC population. This is now no longer the case: the selected tab page remains the active tab page when selecting a different QC population.
