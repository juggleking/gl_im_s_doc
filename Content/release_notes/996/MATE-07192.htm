Release notes 9.9.6 - Adjust tab page width in editors to avoid missing tab page titles (MATE-07192)

An issue was reported where an editor's tab pages were very small, which caused the tab page title to no longer be visible.

This issue has been corrected: if the available space is sufficient, the tab page width will now be adjusted.

The user interface themes Blue, Silver and Black do not support tab page resizing.
