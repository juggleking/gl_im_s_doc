Release notes 9.9.0 - Correction of issues with the "Delete all routine data" tool (GLIMS-11186)

This modification corrects some issues with the tool Delete all routine data:

  1. After running this tool, the assessment methods exported directly from a station were not the same as the assessment methods exported when performing an export of all the assessment methods.
  2. The data of the RouteProcedure table was deleted, though no data in this table should be deleted since it is configuration data.
  3. The tool deleted the Order table data but not the OrderTodoItem table data. As a consequence, the references to the Order to-do items in the Order table were deleted but the item records remained.
  4. Package, PackageItem, PackageAudit and PackageItemAudit data was not deleted.

These corrections make the tool more precise and performant.
