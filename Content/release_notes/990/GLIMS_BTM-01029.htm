Release notes 9.9.0 - Same screen to set blood bags as Wasted and Discontinued (GLIMS_BTM-01029)

The status of one or more blood bags can be changed by clicking-right on the blood bag(s) in the Blood bag browser > Status transition. When clicking on Wasted in previous GLIMS versions, the confirmation window that opened was different from that which opened when clicking on Discontinued. However, the Discontinued and Wasted statuses have identical parameters, such as the status date, the user who set the status and the reason for setting the status.
