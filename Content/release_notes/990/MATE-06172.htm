Release notes 9.9.0 - Fixed error which occurred when using the Database statistics tool (MATE-06172)

An issue was reported where the error Value 1002 cannot be displayed using >>9. (74) occurred when using the Database statistics tool.

This error was due to the Usr column not having been formatted to display more than 3 characters, which can be an issue in large sites with a lot of concurrent users.

This has been corrected: the width of the Usr column has been increased.
