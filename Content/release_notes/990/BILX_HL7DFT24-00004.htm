Release notes 9.9.0 - Financial export in HL7 DFT 2.4 format: include "Identifier type code" (BILX_HL7DFT24-00004)

In addition to the use of the Identification type in communication messages - functionality which is available since GLIMS 9.3 via GLIMS_PI-00298 - the Identification type will now also be used during the export of financial data in the HL7 DFT 2.4 format: the field PID-3, CX-5 "Identifier type code" will now be filled with the Mnemonic of the Default identification type of the provider (Assigned by) of an Identification.
