Release notes 9.9.0 - Financial export in "HPRIM XML" format (BILX_HPRIMXML-00043)

This modification applies to the HPRIM XML export format of the FinancialShipment.Send function (HPRIM XML GMSIH, 1.07 and 2.0):

If the price code (Mnemonic or Base value code) ends in *_G, then GLIMS will export the "actesNGAP" as

  • acteHorsNomenclature=oui
  • facturable=non
  • gratuit=oui

(whether or not the invoice item price is 0.00).

Note

When the price code's Base value code ends in "_G", the "_G" is removed from the value before it is exported in the "lettreCle".

If the price code (Mnemonic or Base value code) ends in *_Z, then GLIMS will export the "actesNGAP" as

  • acteHorsNomenclature=oui
  • facturable=non
  • gratuit=oui

and will add a new element "montantTotal" with value "0".
