Release notes 9.9.0 - HC provider import program no longer empties the Fax 2 field (GLIMS-06583)

The HC provider import program is available from the main menu via Start > System management > Database > Specific import > HCproviders. While the program does not allow to import the Fax 2 field, it was noticed that the import caused this field to become empty (i.e. no value instead of "?" which is the standard initial value). This triggered the Fax number validity check (if defined).

This has been corrected. The Fax 2 field will now have "?" as initial value after the import.
