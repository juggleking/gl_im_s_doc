Release notes 9.9.0 - Improved "Run 4GL procedure" functionality (MATE-06119)

The following features have been added to the Start > Development > 4GL > Run procedure functionality:

  • History of the last 10 procedures
  • Drag & drop
  • Browse for file
  • Clear item from history via Shift-Delete

Warning

The "Run 4GL procedure" tool requires the user type Developer and is reserved for MIPS employees.
