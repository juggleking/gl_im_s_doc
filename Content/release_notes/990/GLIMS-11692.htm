Release notes 9.9.0 - Issues with some redesigned .NET browsers if GLIMS is used on an Oracle database (GLIMS-11692)

Issue

The following issue occurred when GLIMS was used in combination with an Oracle database: fast-positioning does not always work as expected in some redesigned (.NET) browsers (due to an issue with the OpenEdge DataServer for Oracle).

Solution

For now, the classic version of the following browsers will open if GLIMS is used on an Oracle database.

  • BloodbagsByExternalId
  • BloodbagsByExternalIdAsBarCodeProductBarCode
  • BloodbagsByInternalId
  • BloodbagsByLastUpdateTime

  • EncountersOfPersonByStartTime
  • LotsByExternalization

  • ObjectEssentials
  • ObjectsByExternalization
  • ObjectsByInternalId
  • OrderidentifiersByCreationTime
  • OrderidentifiersByIdentifier
  • OrderQueryBrowser
