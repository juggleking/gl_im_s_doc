Release notes 9.9.0 - New log types for unusual blood bag work flow (GLIMS_BTM-01041)

New log types have been added for which a log entry is created each time the status of a blood bag is demoted. These log entries can be accessed via:

  • the main menu: Start > System management > Logging,
  • the contextual ribbon item Audit > Log entries on a blood bag.

BloodbagReturn Checked out Initial
