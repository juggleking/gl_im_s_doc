Release notes 9.9.0 - Purchase order browser available in .NET format (GLIMS_STCK-00504)

This modification pertains to the Stock management module. The Purchase order browser which can be accessed

  • via the main menu: Start > Stock management > Purchase orders
  • from a stock
  • from a supplier

has been redesigned.

Note

When two or more purchase orders are merged, the browser is currently not automatically refreshed.

In addition, some (synchronization) issues have been corrected in the following .NET browsers:

  • Purchase items
  • Product specifications
  • Product lots
  • Deliveries

The features of .NET browsers are described in the documentation.
