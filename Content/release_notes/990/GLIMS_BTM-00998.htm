Release notes 9.9.0 - Blood bag import in "ETS Mont-Godinne" format supports ISBT-128 codes (GLIMS_BTM-00998)

GLIMS supports importing blood bags in the ETS Mont-Godinne (Belgium) format. This blood bag import format now also allows importing blood bags identified by an ISBT-128 code.

Blood bags that are provided with an external ID of 14 or more characters will be treated as ISBT 128. The corresponding "External ID as barcode" is formed by "=" + first 13 characters of the external ID + "00". E.g.: External ID "B038718600000A" will have "=B03871860000000" as "External ID as barcode".
