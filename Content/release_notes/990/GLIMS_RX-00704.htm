Release notes 9.9.0 - Online reporting: only export most recent patient identification code (GLIMS_RX-00704)

During electronic reporting, only one identification code will now be exported for a patient per provider: the currently valid identification code with the most recent start date. Moreover, the start date and end date of the identification is now exported as well.

In previous versions, all identification codes of a patient were exported without their start date and end date.
