Release notes 9.9.0 - Parameters correctly displayed in Cybertrack transfusion reaction e-mails (GLIMS_CBT-00288)

Context

E-mails can be sent to a defined list of users when transfusion parameters and remarks have been registered in CyberTrack.

Issue and solution

In the e-mails, the same transfusion parameters were displayed several times.

This has been corrected.
