Release notes 9.9.0 - Microbiology work screen title not synchronized upon selection of order without microbiology action (GLIMS_BAC-01472)

Problem description

  1. Open the Orders by receipt time browser.
  2. Select an order with a microbiology action.
  3. Click on the contextual ribbon item Microbiology actions.
  4. From the microbiology action browser, open the microbiology work screen (displaying the information of the microbiology action contained in the selected order).
  5. If, subsequently, an order without a microbiology action is selected in the Orders by receipt time browser, the title of the microbiology work screen still refers to the microbiology action of the previously selected order.

Solution

This has been corrected. The title of the microbiology work screen will now contain "???" if an order without a microbiology action is selected.
