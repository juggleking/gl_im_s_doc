Release notes 9.9.0 - Informative error message displayed when database contains inconsistent result record (GLIMS-11056)

Issue

When the database contained an inconsistent result record, i.e. a result record whose property attribute pointed to a non-existing property, an error occurred with the property cache and GLIMS had to be restarted.

Solution

When the database contains an inconsistent result record, an informative error message is displayed and GLIMS does not need to be restarted.
