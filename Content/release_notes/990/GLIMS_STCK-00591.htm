Release notes 9.9.0 - Correction of synchronization issue in classic stock management browsers (GLIMS_STCK-00591)

In the stock management module, some classic browsers if opened from a browser where not synchronized with this parent browser. For instance, the classic purchase item browser if opened from the purchase order browser displayed all purchase items and not only the items of the purchase order which had been selected in the parent browser.

This issue, which occurred since GLIMS 9.8.0, has been corrected.
