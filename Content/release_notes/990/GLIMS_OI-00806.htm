Release notes 9.9.0 - Respect order of requests during electronic reporting of specimen details (GLIMS_OI-00806)

For an electronically created / updated order containing the following requests:

  • a request for a panel with, as member, a property, as well as
  • an explicit request for that property (which is added later on),

GLIMS would choose to report the request which was added later on as "directly requested property" and not as panel member when replying to a "ReportSpecimenDetails" message (to provide an overview of the created / updated order for the requesting application).

This issue, which occurred since GLIMS-11344, has been corrected.
