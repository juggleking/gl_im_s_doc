Release notes 9.9.0 - Correction of export of order information in "Labco DataWareHouse" format (GLIMS-11203)

Context

Since GLIMS 9.3.0, GLIMS allows to export order information in the Labco DataWareHouse format.

Issue

The invoiced amounts were not correctly distributed over the payer 1, payer 2, payer 3 and payer 4 columns in the request.csv and testorder.csv files.

Solution

This has been corrected.
