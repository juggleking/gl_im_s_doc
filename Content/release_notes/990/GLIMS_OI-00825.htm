Release notes 9.9.0 - Order import: correctly import result norm data for unsolicited result (GLIMS_OI-00825)

If the environment variable OrderImportResultNorms is set to Yes (which is the default value), the result norm data in the order import message will be imported during electronic order entry. However, an issue was reported where the result norm data was not imported if GLIMS received an order import message containing an "unsolicited" result (i.e. a new request with a result value) and its reference values.

This has been corrected.
