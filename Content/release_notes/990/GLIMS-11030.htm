Release notes 9.9.0 - Dynamic results displayed with correct object times and discriminators (GLIMS-11030)

Context

The support of Point-Of-Care connections (POC) allows GLIMS to create orders, specimens and results on the basis of result messages.

Dynamic results are results of dynamic tests, i.e. tests for the same property but executed at defined intervals.

Issue

Since GLIMS 9.6.0, all dynamic results created on the basis of POC specimens had the same object time and no time discriminator was shown. As a consequence, interpreting these results was impossible.

This issue has been solved.
