Release notes 9.9.0 - Multilingual character of "Text" field in "Text" editor indicated by flag icon (MATE-05794)

Issue

The Text field in the Text editor is a multilingual field: double-clicking on it allows opening a window in which the text can be entered in several languages. In the UI, the multilingual character of this field was not obvious.

Solution

The fact that the field is multilingual is now indicated by a gray flag. This flag is displayed in red if the text has been entered in more than one language, i.e. if translations of the original text are available.
