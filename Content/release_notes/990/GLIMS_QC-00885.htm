Release notes 9.9.0 - Unit addition to the field "Method property" on "QC population" (GLIMS_QC-00885)

Field expansion on "QC population"

Some of the fields of the QC population editor, such as target value, accuracy limit etc. always have the same unit as the linked property. This information was not directly displayed in the editor, thus the user needed additional navigation to check the unit of a property.

From now on, the Method property field of the QC population editor is expanded with the unit name of a property, if applicable. This unit applies to all corresponding fields of the QC population editor.
