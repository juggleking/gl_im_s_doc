Release notes 9.9.0 - Financial export in Belgian RIZIV format: check digit modulo 89 (GLIMS_BILL-03655)

The export of financial data in the RIZIV format has been updated to support the following changes:

  1. As of 01/04/2019, the check digit of RIZIV identifications assigned to HC providers will be calculated according to the modulo 89 method. This modification ensures that GLIMS also supports the module 89-based check digits.
  2. As of 01/07/2019, zone 47 of record type 50 will have a different format: it will no longer require a date ("datum akkoord verstrekking") but an amount ("basiswaarde laagvariabele zorg"). The new format (of which the initial value is +0000000) will be used for financial shipments with an external date >= 01/07/2019.
