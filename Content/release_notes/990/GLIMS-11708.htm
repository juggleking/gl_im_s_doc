Release notes 9.9.0 - New option to purge all person records without an object (GLIMS-11708)

Context

GLIMS contains a tool to purge old routine data. Some customers would like to have the option to only purge person records without an assigned object.

New functionality

The new check box Without object in the Objects tab of the Purge routine data tool allows the user to delete every person record without a linked object within a specific time period.

The screen logging will indicate and log the execution of the actions according to the selected configuration.
