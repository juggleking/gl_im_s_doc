Release notes 9.9.0 - Blood bag import: reading of D antigen information from Progesa file (GLIMS_BTM-01017)

When importing blood bags using the Progesa format, the field containing the antigen information D-/D+ was not read.

This has been corrected: the D antigen information provided in the Progesa file is now imported.

Warning

Customers who used a MISPL function to set the antigen D should deactivate this function when importing with the Progesa format.
