Release notes 9.9.0 - Interactively adding sub carriers to a carrier (GLIMS_BAC-01598)

In addition to adding sub carriers automatically via the MISPL function Carrier.AddSubCarrier, sub carriers can now be added interactively as well by choosing the contextual menu / ribbon item Add sub carrier from a Carrier.

Note

  • If you want to use this functionality in a tool with a parameter set, you should use the function AddSubCarrierRunSiteFunction.
  • If the specified comment does not satisfy the List of allowed values of the Medium, then the sub carriers will not be added.
