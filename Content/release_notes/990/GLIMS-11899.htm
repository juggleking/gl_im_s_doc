Release notes 9.9.0 - Specimen status visible in the order entry specimen browser (GLIMS-11899)

Issue

An issue has been reported where the specimen status was not shown in the Order entry specimen browser.

Solution

This issue has been fixed: a new column Status displays the current status of a specimen.

Note

If the specimen status is changed via the context menu on the order entry screen, the status indicated in the specimen browser is not automatically updated. This requires refreshing the order entry screen.
