Release notes 9.9.0 - Financial export in "CPOW-ITP400" format: improved error logging (BILX_CPOWISH-00094)

When errors occurred during the sending of a financial shipment in the CPOW-ITP400 (CPOWISH ITPLAT) format, the logging did not allow the user to solve these errors as it did not contain any detailed information about these errors which were mostly configuration-related. This happened if the detailed error-related information (in the form of a string) exceeded 2900 characters. In that case, the information was lost.

This has been corrected.
