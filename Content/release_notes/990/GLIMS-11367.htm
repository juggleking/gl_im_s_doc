Release notes 9.9.0 - Enhancements for tool to change the path of image results (GLIMS-11367)

Context

A tool is available to change the image path of all image results in GLIMS. It is available from the main menu via Start > System management > Tools > Update image results.

Issues

  • Image results with backslashes in their path are not found.
  • Image results with forward slashes in their path are not found if a path with backslashes is specified.
  • There's not enough space in the options screen to supply the old and the new image path.
  • Changing a result image path might trigger status changes and report rescheduling.

Solution

These issues, which occurred since GLIMS 9.8.0, have been corrected.
