Release notes 9.9.0 - Keyboard shortcuts for quick navigation between the embedded browsers in the microbiology work screen (GLIMS_BAC-01491)

From now on, the GLIMS Microbiology module supports some shortcut keyboard actions for fast navigation between the embedded browsers of the Microbiology work screen.

CTRL+1: Carrier browser

CTRL+2: Isolation browser

CTRL+3: Result browser

CTRL+4: Isolation test browser

CTRL+5: Antibiogram browser

CTRL+6: Carrier test browser (where applicable)

Warning

  • These shortcuts only work via the number row and not via the number pad of the keyboard.
  • The old browser shortcuts (Alt+P, Alt+I, Alt+R, Alt+T, Alt+A) have been disabled, as they could result in potential issues with navigation, such as accidentally triggering a menu option.
