Release notes 9.9.0 - Tool to recalculate order internal ID starting with '__' (GLIMS-10749)

Issue

If an order is created and the MISPL calculating the order's internal ID returns an error (e.g. because the internal ID already exists), the order will be created with an internal ID starting with '__'.

Solution

New tool

A tool is now available to recalculate the internal ID of the orders of which the internal ID starts with '__' on the basis of the configured MISPL expression for order internal ID calculation.

Using the tool

The procedure can be started via Start > Development > 4GL > Run procedure by a user with a role with User type set to Developer. Specify "ord_RecalculateInternalIds" as File name.
