Release notes 9.9.0 - Financial export in "HPRIM XML 1.04" format (BILX_HPRIMXML-00046)

Issue

Since the introduction of HPRIM XML 1.07, it was no longer possible to send financial shipments in the HPRIM XML 1.04 format.

Solution

This has been corrected: the HPRIM XML 1.04 format is now available again.

Note

Customers wanting to use HPRIM XML 1.04 will have to manually correct the protocol in the function parameter sets used in tools or commands.
