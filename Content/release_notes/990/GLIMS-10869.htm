Release notes 9.9.0 - OrderTodoItems browser can be opened in .Net format (GLIMS-10869)

Context

MIPS is redesigning the GLIMS browsers using the .NET format.

Issue

The OrderTodoItems browser had not been placed on the list of redesigned browsers because of issues with the Progress-Oracle dataserver. Additionally the classic version of this browser was not available either. This means that neither the .NET nor the classic OrderTodoItems browser could be opened.

Solution

The OrderTodoItems browser has been placed on the list of .Net redesigned browsers. Hence, the order to-do items can now be consulted.

Warning

The present modification does not solve the issues with the Progress-Oracle dataserver. Issues with the sorting of to-do item records could occur.
