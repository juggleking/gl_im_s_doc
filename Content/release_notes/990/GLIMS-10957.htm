Release notes 9.9.0 - Navigation issues in Actions of stations browser when cell counter is open (GLIMS-10957)

Problem description

  • Select Start > Routine > Actions > Browse actions
  • Fill in a Station and click on the OK button.
  • Open the cell counter (via the contextual menu or by double clicking on the record) for one of the records in the Actions of stations browser.
  • Issues:
    1. Using the page up and page down keys to navigate to another record in the Actions of stations browser did not work.
    2. When selecting another record in the Actions of stations browser, the data in the cell counter window was not updated accordingly.

Note

These issues occurred since GLIMS 9.6.

Solution

This has been corrected.
