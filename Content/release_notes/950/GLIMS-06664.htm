Release notes 9.5.0 - Support 'favorite' request forms (GLIMS-06664)

GLIMS offers a concept of favorites for a number of tables (Printers, Stations and Work list templates). This is now also supported for Request forms.

Definition of 'favorite' request forms

To define 'favorite' request forms:

  1. Open the Users browser and select a user.
  2. Choose the contextual menu/ribbon item Favorites -> Request form assignments

  3. Hit the Insert key and select a Request form to be added as favorite

Usage

Wherever you can select a request form (e.g. in the order entry screen), an option Show my request forms only will be available. If enabled, only the user's favorite request forms will be shown.
