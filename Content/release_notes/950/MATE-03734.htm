Release notes 9.5.0 - Extensions for marking database as test database (MATE-03734)

A new function is available that allows converting a database to a test database. It is available from the main menu via Start -> System management -> Database -> Convert this database to a test-database.

This function

  • marks the database as test-database by enabling the This is a test database option in the general settings.
  • renames service and translators and turns off their auto-start.
  • disables all cron jobs.
  • clears the Watchdog port configuration.

Note

It is no longer possible to mark a database as test database by enabling the This is a test database option via Start -> System management -> Customize -> MATE.
