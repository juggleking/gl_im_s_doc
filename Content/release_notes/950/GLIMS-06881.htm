Release notes 9.5.0 - New MISPL function GetHLAAntigen (GLIMS-06881)

A new MISPL function GetHLAAntigen(<AntigenName>) has been introduced to easily access information of a specific HLA antigen.

Example

HLAAntigen MyHLAAntigen;

String AntigenName;

AntigenName := “A*020102”;

MyHLAAntigen := GetHLAAntigen(AntigenName);

Message(IfKnownString(MyHLAAntigen.Name) + “ – “ +IfKnownString(MyHLAAntigen.Description) + “ – “ + IfKnownString(MyHLAAntigen.SortString));
