Release notes 9.5.0 - Support Report builder in generic Report tool (MATE_RB-00026)

The contextual Tools ribbon has a Report item that allows generating a custom report for the focused browser. This tool was extended with support for Report Builder.

To this end, the following options were added:

  • Format [Text/PDF/XML]
  • External template [Format PDF only]
  • XML escape texts [Format PDF/XML only]

Please check the manual for a full description.
