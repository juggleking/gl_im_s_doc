Release notes 9.5.0 - Issue with result codes (GLIMS-06592)

Problem description:

  1. In a result browser, select a result cell.
  2. Press CTRL-F.
  3. Select a Result code and press <Return>.
  4. Press <Space> and type some words.
  5. Press <Return>.
  6. The words you typed in disappear.

This has been corrected.
