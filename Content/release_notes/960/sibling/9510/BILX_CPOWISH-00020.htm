Release notes 9.6.0 - Financial export in "CPOWISH" format (BILX_CPOWISH-00020)

The following improvements have been made to the export of financial data in the CPOWISH format (ItPlat, Ihprest, IhPharm and ExtPharm):

  • For each successfully exported financial shipment, a summary of the export will now be stored in a new file. The file will have the extension .log and its file name and directory will be identical to those of the original export file.

  • The summary will also be logged in GLIMS and in the FinancialShipment.Comment field.
  • An issue was solved where an empty .rptE file (ItPlat) was generated due to a mandatory field not being filled. The first field will now be filled with the first three letters of the file name.
