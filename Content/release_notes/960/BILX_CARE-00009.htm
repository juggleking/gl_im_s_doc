Release notes 9.6.0 - New field "Is official fund group" on "Correspondent group" (BILX_CARE-00009)

New field "Is official fund group"

The field Is official fund group has been introduced on Correspondent group level. It allows to mark a Correspondent group as an official fund correspondent group.

Multiple official fund correspondent groups can be defined.

New MISPL "Fund.IsAnOfficialFundMember()"

A new (Logical) MISPL function is available to check if a fund is a member of an official fund correspondent group: Fund.IsAnOfficialFundMember().

Backward compatibility

Currently defined MISPL functions based on a configuration where correspondent groups have a Code = OFF-MUT or the site attribute _GroupOfficialFunds enabled will continue working.
