Release notes 9.6.0 - Sequence numbers of billing rule lines and inputs / outputs (GLIMS_BILL-02792)

Sequence numbers of billing rule lines or inputs / outputs longer than 3 digits were displayed as ??? in the billing rule editor (accessible via the Billing rule context function Edit lines). This has been corrected.

Note

  • For billing rule line sequence numbers, a maximum of 4 digits is allowed.
  • For billing rule line input / output sequence numbers, a maximum of 8 digits is allowed.
