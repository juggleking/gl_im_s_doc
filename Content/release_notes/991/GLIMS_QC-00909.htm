Release notes 9.9.1 - The Statistics section of the QC populations of a newly created QC lot should be empty (GLIMS_QC-00909)

The contextual functions Create new QC lot / Create new lot on QC lot allow the user to create a new QC lot from an existing one. The QC populations of the original QC lot are copied as well and linked to the new QC lot. However, the Last QC result time and Statistics calculation time of the QC populations of the new QC lot was set (copied from the original ones) while these QC populations did not have any results.

This has been corrected.
