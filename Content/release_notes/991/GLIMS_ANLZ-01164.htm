Release notes 9.9.1 - Transmit test / order cancellation with action code "C" in O.12.1.1 of internal ASTM protocol (GLIMS_ANLZ-01164)

The work order message field O.12.1.1 (Action code) of the internal ASTM protocol can now have one of the following 3 values:

  1. Q: when O.3.1.1 is referring to a QC lot.
  2. C: when O.3.1.1 is referring to a specimen and the test is cancelled.
  3. N: in any other case.

The ramifications of the new value "C" are noticeable in cases where the work order message is a reply to a ONEORDER query. Instead of 1 O-segment containing all the tests for a specimen (on the same station), GLIMS will now reply with 2 O-segments per specimen (for tests on the same station):

  1. An O-segment containing all the cancelled tests => all tests having O.12.1.1 = C
  2. An O-segment containing all the non-cancelled tests => all tests having O.12.1.1 = N
