Release notes 9.9.1 - Station.UnknownSpecimenTrigger should have access to "ManualId" in ASTM O.4.1.1 (GLIMS_ANLZ-01202)

Context: Retrieving data from the result message sent by the POC analyzer

The Unknown specimen trigger (MISPL function) of a Station has access to data in the ASTM result message received from the station's translator. The data is (temporarily) passed as a tagged list in the Specimen.InternalComment. For more information, see Point-of-care connections.

"ManualId" in ASTM O.4.1.1

An ASTM result message can now contain a "ManualId" (i.e. a specimen ID that was manually entered on the instrument) in the field O.4.1.1 of the internal ASTM protocol, the value of which can be accessed by the Unknown specimen trigger via the field tag "ManualId".
