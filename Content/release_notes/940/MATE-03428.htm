Release notes 9.4.0 - Edit correspondent starting from subtype (MATE-03428)

In previous versions, GLIMS offered a contextual menu item Correspondent -> Edit starting from a correspondent subtype such as e.g. HC Provider. This functionality was no longer available in GLIMS version 9.

These functions have been re-introduced, so users can now add them to a ribbon or menu. The inverse function is also available: e.g. AsHCProvider and AsOrganization starting from a Correspondent.
