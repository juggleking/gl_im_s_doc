Release notes 9.4.0 - Customized login screen (MATE-03476)

GLIMS now allows to customize the login screen by adding an image in its upper right corner. This can be used to e.g. distinguish between a production and test environment.

  1. Create an image of 200x40 pixels named GlimsLogin.png
  2. Put it in the GLIMS Dat folder
  3. It will be shown in the upper right corner of the login screen.
