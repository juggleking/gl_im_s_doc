Release notes 9.9.4 - Check on Hungarian social security number: use identification instead of matriculation (GLIMS-12745)

Since GLIMS 9.9.2, Hungarian customers can check the social security number of the patient during order entry by sending a request via a web service to the platform of the Hungarian National Health Insurance Fund.

To perform this check, the matriculation of the payment agreement was initially used. This is now no longer the case: the identification of the object that is issued by the national PIN provider is now used.
