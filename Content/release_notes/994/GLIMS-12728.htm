Release notes 9.9.4 - Removed unnecessary message in order entry when overriding the default billing mark (GLIMS-12728)

Context

The Correspondent editor allows the user to specify a default Billing mark that will be used for orders that are created for this issuer.

Issue and solution

If, during order entry, the user manually entered another billing mark, thus overriding the default billing mark, the message "Are you sure you want to override the issuer billing mark?" was shown.

As this message is considered unnecessary, it will now no longer be shown.
