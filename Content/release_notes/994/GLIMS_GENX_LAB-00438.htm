Release notes 9.9.4 - Fixed MISPL compilation error for VariantRetestStatus (GLIMS_GENX_LAB-00438)

In a previous modification, the enumerated status values for VariantRetestStatus were updated as follows:

  • From: Not requested,To do,Success,Failed
  • To: Not requested,To do,Confirmed,Not confirmed

This change caused a compilation error and a warning message pop-up when attempting to run a MISPL function containing the new values for VariantRetestStatus.

This issue has been fixed.
