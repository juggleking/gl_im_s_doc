Release notes 9.9.7 - Avoid error during execution of assessment method/code set conversion procedure that stops the conversion procedure (GLIMS-13555)

Issue

If a coding system is linked to multiple stations, an error occurred when the conversion procedure tried to delete a coding system record that is linked to several stations. This error prevented the conversion procedure from continuing.

Solution

This issue, which occurred since GLIMS 9.0, has been corrected.

The conversion procedure will only delete the coding system when processing the last station linked to this coding system.
