Release notes 9.9.7 - Microbiology procedures of different types linked to the same specimen were shown together in radio button list of microbiology work screen (GLIMS_BAC-01831)

When a specimen has multiple microbiology procedures of different types e.g. Germ and Virus, they were all shown together in the radio button list of the microbiology work screen.

This issue, which occurred since GLIMS 9.6, has been corrected.
