Release notes 9.9.7 - Allow Ctrl+F, double-click when modifying multiple results in incomplete results browser (GLIMS_QC-01039)

The user could no longer open the browser with choices and result codes by pressing Ctrl+F or double-clicking in the Value field of the Multi-result update screen when modifying multiple results in the incomplete results browser.

This issue, which occurred since GLIMS 9.8.16, has been corrected.
