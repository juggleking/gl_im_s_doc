Release notes 9.9.7 - Order dump function: include log entries for all tables within the order + dump order audit trail HTML file (GLIMS-13617)

This modification introduces the following changes to the Order dump function:

  • Log entries are now created for all tables within the order, not just the Order table.

    Note

    Currently, this only works for the .json.raw file.

  • The order audit trail HTML file is now dumped as well.
