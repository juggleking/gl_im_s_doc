Release notes 9.9.7 - Fixed trailing space in site attribute "_RunComparatorForKnownPIN"(GLIMS_PI-00378)

Issue

The site attribute "_RunComparatorForKnownPIN" on the SpecificSite table was introduced with a trailing space in its name (cfr. GLIMS_PI-00369), which caused the following issues:

  • The user could get "unique constraint violated" errors when doing an upgrade on an Oracle database.
  • Every functionality that depends on this site attribute being enabled did not work on Progress and Oracle databases.

Solution

This issue, which occurred since GLIMS 9.8.15, has been corrected: the script that automatically creates site attributes has been updated to ensure that the site attribute has no trailing space in its name anymore.
