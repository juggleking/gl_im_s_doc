Release notes 9.9.7 - AutoMergeByPIN should be able to handle more than two person identifications (GLIMS_PI-00388)

Issue

AutoMergeByPIN could not merge two persons with multiple person identifications (PINs) due to the following error:

"Identification commit error: Specified 'Assigned to' does not exist"

Solution

This issue has been corrected: AutoMergeByPIN can now handle more than two person identifications.
