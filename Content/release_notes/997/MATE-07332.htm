Release notes 9.9.7 - Make "Too many lines per row" message in work list clearer (MATE-07332)

When there were too many row lines in a work list, GLIMS showed an "Unable to display: too many lines per row" error message. However, this message did not contain any information on how to solve this issue. This modification makes this error message clearer:

"Unable to display: too many lines per row. Please set the value of the "Line" field of the work list template items to a smaller value."
