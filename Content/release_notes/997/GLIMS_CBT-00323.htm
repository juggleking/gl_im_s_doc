Release notes 9.9.7 - Fixed errors that occurred when accessing CyberTrack via a REST API (GLIMS_CBT-00323)

A problem was reported where an error occurred when trying to access CyberTrack via a REST API which

  • resulted in a blank screen and
  • caused the error messages Glims username and/or password combination is incorrect or LDAP Error Message: The user name or password is incorrect to appear in the REST service/AppServer log files.

This issue, which occurred since GLIMS 9.8, has been corrected.
