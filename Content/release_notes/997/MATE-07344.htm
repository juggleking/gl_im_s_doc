Release notes 9.9.7 - Introduction of bilx.pl (MATE-07344)

In order to ease maintenance when billing regulations (e.g. for the German KBV) are changing, all logic related to these regulations will be put in one bilx.pl file that can be used for several versions of GLIMS (2023.1, 10.1, 10.0, 9.9, 9.8).

From now on, an extra bilx.pl file will be delivered for ease of maintenance of the GLIMS installation.

  • The bilx.pl file is listed as a known plugin.

  • The bilx.pl file needs to be installed in the lib folder.

    Note

    For GLIMS 10.1 and 2023.1, the bilx.pl file will be part of the installer program.

    For corrective versions, the bilx.pl file will be delivered through the zip file. A manual update of the PROPATH might still be needed.
