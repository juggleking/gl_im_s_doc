Release notes 9.9.7 - Table export fields "Separator" and "File name style" were not remembered (MATE-07297)

When exporting a table, the Separator (Comma, Semicolon, Tab) and the File name style (Table name, Table short name) are normally restored from the user preferences. However, this was not done anymore and the user had to change these field values each time.

This issue has been corrected.
