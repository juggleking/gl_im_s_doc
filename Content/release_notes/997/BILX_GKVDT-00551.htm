Release notes 9.9.7 - Uniform KBV-BSNR identification provider usage (BILX_GKVDT-00551)

The KBV-BSNR provider, previously only used as identification source for BSNR for departments, is now also used as identification source for BSNR for issuers.

In the past, issuer BSNR were configured as Identification.Code with source = the HC code provider specified in the GLIMS general settings.

Note

If, in the GLIMS configuration, the KBV-BSNR provider = HC code provider, nothing changes. In the other case, the source of the existing (issuer) BSNR identification codes needs to be converted.

On the introduction of a separate KBV-BSNR provider, the Order import (LDT) BSNR provider parameter needs to be updated.
