Release notes 9.9.7 - Reintroduced status area for Cell counter (GLIMS-13716)

Issue

Since GLIMS 9, there is no longer a status area at the bottom of the main screen. This was problematic for the Cell counter, as this status area showed essential information.

Solution

This issue, which occurred since GLIMS 9, has been corrected.

The status area for the Cell counter has been reintroduced. It is now shown at the bottom of the Cell counter screen whereas in GLIMS 8 it was shown at the bottom of the main window.
