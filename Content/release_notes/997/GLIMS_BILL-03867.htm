Release notes 9.9.7 - Avoid error during invoice summary creation (Oracle database) (GLIMS_BILL-03867)

When placing a credit note on an invoice summary with the site attribute "_AutoBalanceCreditInvs' (SpecificSite) enabled, the following errors occurred on an Oracle database:

"A mandatory field did not get value. (1444)"

"** b_Balance.Debit transaction is mandatory, but has unknown (?) value. (110)"

This issue, which occurred since GLIMS 9.9.6 (GLIMS_BILL-03843), has been corrected.
