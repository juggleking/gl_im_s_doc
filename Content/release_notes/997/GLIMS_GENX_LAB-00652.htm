Release notes 9.9.7 - Adaptive XML export: improved rich text support (GLIMS_GENX_LAB-00652)

Issues

The following issues have been reported:

  • The font size was bigger when printing a rich text via Jasper
  • <custom> tags did not support rich text

Solution

These issues, which occurred since GLIMS 9.9.0, have been corrected:

  • The font size is converted in pixels for:
    • The Property classification Content MISPL
    • Custom texts

  • Custom texts containing rich texts will now be exported correctly.

  • Text containing XML unsafe characters (e.g. '<', '>') will now be escaped correctly.
