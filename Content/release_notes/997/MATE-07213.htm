Release notes 9.9.7 - Upgraded Mips.TelerikControls to v1.3.0 + Telerik controls to v2021.2.615.40 (MATE-07213)

This modification upgrades

  • Mips.TelerikControls to v1.3.0
  • Telerik controls to v2021.2.615.40

to fix several issues.

Mips.TelerikControls provides among others a Rich text editor based on the Telerik controls from Progress. This Rich text editor is used in the Genetics module.
