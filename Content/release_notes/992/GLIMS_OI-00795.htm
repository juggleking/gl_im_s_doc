Release notes 9.9.2 - Include number of containers during electronic reporting of specimen details (GLIMS_OI-00795)

When replying to a "ReportSpecimenDetails" message (to provide the requesting application with an overview of the electronically created / updated order after its scheduling in GLIMS), GLIMS will now include the number of required containers (specimens) for a material. In addition, the number of containers is included during online reporting as well (provided that the Specimen details option is enabled in the Items to export screen which can be accessed by clicking on the Report items button in the Electronic tab of the report generation screen).

The number of containers is calculated as follows: NumberOfContainers = RoundUp(SpecimenSize/Material.MaxSize).
