Release notes 9.8.0 - Correction for tool to introduce new PIN numbers based on import file (GLIMS-07829)

Introduction

GLIMS allows to introduce new PIN numbers based on an import file.

Functionality

For each Identification assigned by the "old PIN provider" and with Code = "old PIN", the tool will:

  • Set the Valid until field (if an End date is specified)
  • Create a new Identification assigned by the "new PIN provider" and with Code = "new PIN"

Correction

A correction has been implemented for this tool:

If an Identification already exists for the "new PIN provider" but with a Code <> "new PIN", the new identification with the new PIN will not be imported. Identifications rejected for this reason are logged in a .rej file.
