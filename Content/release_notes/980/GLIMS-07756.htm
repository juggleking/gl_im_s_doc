Release notes 9.8.0 - Changing specimen internal id and discriminator in order entry (GLIMS-07756)

The following issues in the specimen browser of the order entry screen have been solved:

  • When selecting a specimen and choosing the context function Change internal Id, the specified specimen id was not saved.
  • It was not possible to update a specimen discriminator directly in the specimen browser of the order entry screen.

Warning

If you choose to save your order as a pending order, all manual changes to the specimen will be ignored, since by default no specimens are created for pending orders.

Note

Problem occurred since GLIMS 9.5.
