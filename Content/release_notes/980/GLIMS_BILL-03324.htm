Release notes 9.8.0 - Fast-positioning possible in the Assigned billing codes browser (GLIMS_BILL-03324)

The fast-positioning function did not work in the .NET browser of assigned billing codes (Start -> Billing -> Configuration -> Discount -> Discount classes -> right-click on a discount class -> Show all -> Assigned billing codes).

This has been corrected.

 

Note

Note

The problem occurred since version 9.0.
