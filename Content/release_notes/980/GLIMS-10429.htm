Release notes 9.8.0 - Allow defining a print job for billing document generation via GLIMS Report Builder (GLIMS-10429)

When generating billing documents for invoices and invoice summaries using GLIMS Report Builder (via the contextual function Print), it is now possible to generate an invoice (summary) and an invoice (summary) duplicate in one action using a MISPL expression specified in the Jobs field of the Billing document printing window.

  • This also works for financial shipments and financial transactions.
