Release notes 9.8.0 - Corrections for financial export in "RIZIV" format (GLIMS_BILL-03269)

The following corrections have been implemented for exporting financial data in the RIZIV format:

  1. For additional invoices, record type 20, zone 11 ("type facturering") will now have "2" as value ("complementaire factuur").

    In previous versions, value "0" was used. As a result, additional invoices were not sent as such.
