Release notes 9.8.0 - QC severity should not be set for patient results (GLIMS_QC-00734)

Background

GLIMS allows configuring station flags for which a Severity for QC can be specified. This severity will be taken into account when calculating the QC severity of QC results marked with this flag.

Issue

However, the QC severity was also set for patient results marked with this flag. This has been corrected.
