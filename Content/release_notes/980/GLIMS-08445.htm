Release notes 9.8.0 - Addition of colour styling for LAS status values in Station and Assessment method browsers (GLIMS-08445)

Background

The Station and Assessment method browsers contain a column which indicates the LAS status of these stations and assessment methods: Connected, Disconnected and ?.

Enhancement

To make the LAS status clearer and more noticeable, it is now also displayed on a coloured background. The colour scheme is the following:

status comment
green connected
orange partially connected

Used when both

  • the LAS status of the station is Connected
  • and the LAS status of at least one of the station's assessment methods is Disconnected.
red

disconnected
no colour

other

Note

The colour coding has not been applied to the LAS status values in the browser of the Channel quality monitor.
