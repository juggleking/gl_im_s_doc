Release notes 9.8.0 - Full Microbiology Lab Automation - WASPLab (GLIMS_ANLZ-01160)

Introduction

GLIMS 9.8 has been extended to support communication with the WASPLab system for full microbiology lab automation.

Please check the documentation for a full description.

Main features

  • Data exchange for carrier grafting
  • Integration and visualization of grafting results
  • Data exchange for incubation and growth results
  • Integration and visualization of growth results
  • New MISPL functions to automatically add carriers (colony picking)
  • Integration of isolation information
  • Detailed audit trails
