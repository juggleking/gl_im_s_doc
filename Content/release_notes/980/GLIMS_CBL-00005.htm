Release notes 9.8.0 - Correction for export of prices for import in CyberLab (GLIMS_CBL-00005)

When exporting price information for import in CyberLab, the validity (start and end date) of the requestables was not taken into account so that prices for no longer valid (with end date in the past) or new (with start date in the future) requestables were also exported.

This has been corrected.

Note

Problem occurred since GLIMS 9.6.1.
