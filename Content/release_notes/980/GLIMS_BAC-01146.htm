Release notes 9.8.0 - Option to hide carrier test browser in microbiology screen (GLIMS_BAC-01146)

A new option Show carrier test browse has been added in the microbiology procedure configuration screen. If this option is disabled, the carrier test browser will not be shown in the microbiology work screen for microbiology actions derived from that microbiology procedure.

Note

If the microbiology procedure option Show carrier browse is disabled, the carrier test browser will not be shown either, even if the option Show carrier test browse is enabled.
