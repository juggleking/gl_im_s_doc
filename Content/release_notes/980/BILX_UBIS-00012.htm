Release notes 9.8.0 - Updated templates for export of transfusion data in "Ubis" format (BILX_UBIS-00012)

GLIMS allows to export data about transfused blood bags directly in the format of "Rode Kruis Vlaanderen".

GLIMS 9.8.0 ships with updated Jasper templates that can be used to convert the exported XML files to a PDF report. In order to use these templates, make sure they are specified in the site attributes on Specific site (or Department) level.
