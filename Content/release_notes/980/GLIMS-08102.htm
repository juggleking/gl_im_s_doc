Release notes 9.8.0 - Include discontinued requested codes in Order.Attribute("RequestList") (GLIMS-08102)

Introduction

The MISPL function Order.Attribute allows to return a list of requested codes / request definitions in an order by specifying one of the following attribute names:

  1. RequestList
  2. RequestListName
  3. RequestListDescription

Problem description

In GLIMS version 8, the MISPL function also returned discontinued requests. Since the introduction of "requested codes" in GLIMS v9.3, the MISPL function no longer returned discontinued requested codes.

Solution

Discontinued requested codes are now included when specifying "RequestList", "RequestListName" or "RequestListDescription" as attribute name in the Order.Attribute MISPL function.
