Release notes 9.8.0 - Scheduling of ad hoc microbiology QC tests to be executed on a given date (GLIMS_QCMB-00037)

The possibility has been added to schedule an ad hoc microbiology QC test to be executed on a given date via the new contextual menu / ribbon item Schedule Ad hoc.

Note

  1. Only microbiology QC tests for which the Frequency is set to Ad hoc will be scheduled to be executed.
  2. When the user (re)schedules an ad hoc microbiology QC test, a new QC result is created for the given date. This means that it is possible to schedule an ad hoc microbiology QC test multiple times on the same day.
