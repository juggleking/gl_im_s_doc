Release notes 9.8.0 - Error message displayed upon approval of non-selected report (GLIMS-08458)

When

  • using the contextual menu/ribbon item Approve on a report without having selected this report, and
  • the report did not refer to an HC provider, for instance,

no error message was displayed.

This issue, which occurred since 9.4.0, has been corrected: an informative error message, such as "No responsible found for this report", is now displayed.
