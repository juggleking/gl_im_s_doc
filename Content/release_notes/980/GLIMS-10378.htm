Release notes 9.8.0 - Size limitation for user-defined statistics rows and columns (GLIMS-10378)

Background

In the Statistics module, the content of raws and columns can be tailored to the user's need. Next to standard classification options (such as Days, Status and Issuers), it is possible to display user-defined classifications, using a MISPL function.

Issue and solution

The fact that the string returned by the MISPL function is limited in length was not mentioned in the GLIMS manual. As a consequence, some users reported that the content of raws or columns was sometimes truncated.

This omission has been rectified: the maximum length of the returned string and the way it is calculated are now presented in the chapter Statistics: layout screen of the manual.
