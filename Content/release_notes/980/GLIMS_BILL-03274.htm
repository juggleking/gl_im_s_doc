Release notes 9.8.0 - Tool to audit configuration for invoicing panels (GLIMS_BILL-03274)

Auditing configuration for invoicing panels

A new tool is available to audit how panel invoicing is configured at customer sites. More specifically, information is required on the use of the following options:

Start > System management > Customize > GLIMS Billing

  • Panel price check (= Completeness)

Request definition of type "Panel"

  • Accept shared members
  • Charge <> Member (use of panel specific billing codes)

Starting the tool

Click Start > Billing > Configuration > Check panel billing configuration.

A report of the configuration information is written to file. This file is previewed after the tool has run. Its location is indicated in the title bar of the preview window.
