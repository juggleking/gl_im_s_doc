Release notes 9.8.0 - Keep selected record when changing sort order of browser (MATE-04398)

When selecting a record in a redesigned (.Net) browser and changing the sort order of the records by clicking on a column header, the browser was refreshed but did not reposition to the initially selected record.

Example

  • Open the HC providers > By last name, first name browser.
  • Select a HC provider.
  • Click on the Address column header to sort the HC providers by their address.

=> The browser refreshed and another HC provider is selected.

This has been corrected.
