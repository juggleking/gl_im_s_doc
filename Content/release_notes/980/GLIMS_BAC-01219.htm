Release notes 9.8.0 - Calculate isolation severity when an isolation test is added or confirmed (GLIMS_BAC-01219)

How does it work?

A severity can be assigned to an isolation using a site function in the field Formula to calculate the severity of the micro-organism configuration. The site function is executed each time an isolation appraisal is entered or an antibiotic result's RIS report value changes (e.g. when confirming the antibiogram) for the isolation linked to the micro-organism with the severity formula.

Note

The site function is only executed if the isolation has a positive isolation appraisal.

What's new?

The site function will now also be executed each time an isolation test is added or confirmed.
