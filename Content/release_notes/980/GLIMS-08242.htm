Release notes 9.8.0 - Allow manual updates to item storages in lab archive (GLIMS-08242)

GLIMS now allows to manually edit some fields of an item storage.

  1. Select an Archive (Start -> Archive -> Configuration -> Archives).
  2. Select the contextual function Outline.
  3. Double-click one of the stored items.
  4. The Item storage window is opened. You can now manually edit the following fields:
    1. Status
    2. Reason
    3. Removal date
    4. Removal reason
