Release notes 9.9.5 - Invalid JSON file structure of Billing document template output (MATE-06925)

An issue was reported with the output of a Billing document template, where a large amount of lines (e.g. 100 000) in the output file caused a page break, resulting in a corrupted JSON file structure.

This issue has been fixed: the underlying layout builder has been adapted, removing the length limitation. Setting Page length to ? in the Billing document template will not cause a page break in the output file.
