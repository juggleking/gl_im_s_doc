Release notes 9.9.5 - Field FK 5002 'Art der Untersuchung' in German KVDT (ADT) export (BILX_GKVDT-00487)

Since GLIMS 9.9.4, the KVDT export field FK 5002 - Art der Untersuchung is filled with the external description of the billing code (and no longer with the request definition description). However, in some cases, the old behaviour is desired.

Therefore,

  • if the external description of the billing code is specified, it will be sent in FK 5002.
  • if the external description of the billing code is not specified, the request definition description (or, if empty, the mnemonic) is sent.
