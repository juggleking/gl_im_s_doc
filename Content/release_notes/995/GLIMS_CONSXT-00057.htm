Release notes 9.9.5 - Fixed error related to the size of the externalization of the previous consults (GLIMS_CONSXT-00057)

An issue was reported where consult registration resulted in the error message: "REPLACE/CONCAT may not result in data > 32000 bytes. (11678)". The error occurred if the content of the Consults about the object pane of the consult registration screen exceeded 32000 characters.

This issue, which occurred since GLIMS 9.8, has been corrected.
