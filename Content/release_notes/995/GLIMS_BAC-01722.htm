Release notes 9.9.5 - Allow more than 26 carriers for a microbiology action (GLIMS_BAC-01722)

Issue

Currently, microbiology actions only support up to 26 carriers. If a carrier is deleted, its sequence letter is not reused unless all of the carriers are deleted. This means that users could be blocked from adding carriers even when the limit of 26 carriers is not yet reached.

Solution

To allow for more carriers, the possible sequence letters have been extended from A-Z to A-ZZ.

Example

A – B – C – … – Y – Z – AA – AB – AC … AY – AZ – BA – BB – BC – … – ZY – ZZ

Note

The theoretical maximum number of carriers is now 702 (above this limit a third letter would be required). Carrier sequence letters are still not reused, meaning that the user could still be limited to a number below the theoretical limit.
