Work places

Content: Introduction - Fields

Introduction

Work places are used to group stations.  They can also help in restricting the work list views of routine users.  

Work places are available from the main menu via Start -> Configuration -> Work places.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

Code

Used where space is limited (e.g. labels).

Department

Allows grouping work places per department, completing the hierarchical structure Station -> Work place -> Department.

Cost location

Billing items for tests performed in this work place are assigned to the specified cost location.

Default stock

Please check the chapter Stock management.
