Workflow Visualization Tool

Introduction

The Workflow Visualization Tool allows to browse the core GLIMS configuration using a visual representation of the procedures.  

This tool is available from the main menu via Start -> Configuration -> Work flow visualization tool.

Warning

The Workflow Visualization Tool (GLIMS_WVT) is no longer supported.

Search screen

The first screen allows to define search criteria based on Location, Procedure input, Procedure and Procedure output.

When clicking one of the icons, more detailed search criteria can be specified.  Example for Location: Department, Lab, Station, Workplace.

Clicking an icon allows to select an item from a list.  Example: select a department.

The selected search criteria are shown at the top of window and can be removed by clicking the "X" icon:

Workflow visualization

After having specified the selection criteria, click the View all button in the top right corner to show the search results.  The Refine button takes you back to the search screen.

On the left hand side, all procedures matching your search criteria are shown.  Click one to show its details on the left hand side.
