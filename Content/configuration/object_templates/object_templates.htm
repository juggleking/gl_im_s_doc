Object templates

Contents: Introduction - Fields - Link template to object

Introduction

An object template allows to categorize an Object (type). An Object of type Animal could for instance refer to a Dog as Object template.  

Object templates can e.g. be used as validity criterion for:

  • Norms
  • Delta norm
  • Procedures

The object template is selectable at order entry.

Fields

Object templates are available from the main menu via Start -> Configuration -> Objects -> Object templates.

Name

Name to visualize the object template in GLIMS.

External id

Currently not used.

Parent

Allows to define a hierarchical structure of object templates. 

Example

The object templates "Berner Sennen" and "Golden Retriever" could have the object template "Dog" as parent.

Norms that only apply to objects with template "Dog" will also apply to objects with template "Berner Sennen" as the latter has template "Dog" as parent.

Link template to object

In the order entry screen, you can set an object template for the object of the order (in the Object page).
