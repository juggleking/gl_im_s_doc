Specimen containers definitions

Contents: Introduction - Specimen container definition - Specimen container type - Specimen container cap - Material container usage

Introduction

Specimen containers definitions are used to define the media in which a specimen is transported. The definition refers to: a specimen container type, a specimen container cap, or both.

With the material container usage table, one can specify which specimen container definitions are used for a material.

Material container usages can be used in communication with distributor systems to verify if the detected container info of the queried specimen matches with the usage of the specimen's material defined in GLIMS.

Specimen container type

A specimen container is typically a tube that holds a patient specimen and consists of a glass or plastic closed-end tube with a removable closure (container cap) on the opposite end.
