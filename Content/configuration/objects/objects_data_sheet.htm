Object / Person data sheet

Introduction

The object data sheet produces an overview with the most important data of the object.

Select an Object or Person and choose the contextual menu / ribbon item Data sheet.

Content

Text module

The content of the data sheet is defined in a Text with mnemonic DataView.

Tip

If the text module contains HTML (i.e. it starts with <HTML>), the HTML viewer will display the datasheet. 

External file inclusion

GLIMS allows to include an external file in the data sheet.

Data sheet

Pre-configuration

The Data sheet function can be pre-configured.

This implies that you can add several menu / ribbon items that open the data sheet, each with different content. The Function parameter set allows to specify a different Text.
