Object search screen

Introduction

In several places in GLIMS (e.g. consult search screen), you can use a special object search screen.  Whether or not this special search screen will be used, depends on your user preferences. 

From the main menu click Start -> Tools -> Set user preferences. The option Object search method indicates how GLIMS should lookup objects.

Features

The object search screen is based on the Person search screen (to search objects that are persons). For more information, please check the chapter Person search screen.

The Object Internal id and Externalization fields allow searching for objects other than persons (animals, lots, etc.).
