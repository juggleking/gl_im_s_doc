Introduction

This document describes where you can define printers in GLIMS and in which sequence they are used.

Overview tool

To get an overview of the currently active printers, choose Tools -> Set devices from the main menu.

This screen first shows a list of currently active printers, i.e. the printers that will be used if you print specimen or transfusion labels for instance. These fields are read-only and are only intended to provide an overview of the active printers.

The User page allows to set user-specific printers. If defined, the current user-specific printers will be shown.

The Department page shows which printers are set as default printers for the department. These fields are read-only and are only intended to provide an overview of the default department printers. To set or change the printer configuration on department level, go to the department configuration.

Standard label printer (Specimen and order labels)

During order entry/Order.PrintLabels/Specimen.PrintLabel/specimen scan

  1. Printer set in the order entry options
  2. User has set his Label printer in Tools -> Set devices.
  3. Label printer defined for the department for which the user is logged in.
  4. Label printer defined in the GLIMS general settings.

The labels generated with these programs are sorted as follows: by Material.LabelSeqNo, by Material.Mnemonic, by sampling time

Order contextual menu function "Print labels"

The Print labels screen allows indicating which labels have to be printed (order label, root specimen labels,...) and whether specimen/order labels which were already printed have to be printed again (Check if printed). This is identical to the order entry options for label printing.

When preconfigured, the user interface will not be shown.

The function Print labels has several parameters (cf. database manual, 'Order' table). The parameter PrintSpecimenLabels is the setting for the PrintRootSpecimenLabels, PrintWorkSpecimenLabels and PrintCarrierLabels.

Worklist.PrintLabels

  1. The user interface of this function allows to specify a printer. This printer will be reused when printing labels again via this function.
  2. If no printer is specified, the mechanism as described above is used (1. User-specific label printer, 2. Department label printer 3. Site-specific label printer).

Carrier label printer

When printing carrier labels via the microbiology contextual menu function Print labels or Add carrier, the label printer will be chosen as follows:

  1. User has set his Carrier label printer in Tools -> Set devices.
  2. Carrier label printer defined for the department for which the user is logged in.
  3. User has set his label printer in Tools -> Set devices.
  4. Label printer defined in the GLIMS general settings.

When printing carrier labels during order entry (or via the order series program)

  1. User has set his Carrier label printer in Tools -> Set devices.
  2. Carrier label printer defined for the department for which the user is logged in.
  3. Printer set in the order entry options
  4. User has set his label printer in Tools -> Set devices.
  5. Label printer defined in the GLIMS general settings.

When printing carrier labels from the Specimens of order browser or the Specimen archive scan program

  1. User has set his label printer in Tools -> Set devices.

Blocks/slides labels

From each pathology exam, there are two contextual menu functions Print block labels and Print slide labels which will print the label defined in the medium.

Which printer will be used:

  1. User has set his Block/slide label printer in Tools -> Set devices.
  2. Block/slide label printer defined for the department for which the user is logged in.
  3. User has set his label printer in Tools -> Set devices.
  4. Label printer defined in the GLIMS general settings.

Isolation test label printer

  1. User has set his Isolation test label printer in Tools -> Set devices.
  2. Isolation test label printer defined for the department for which the user is logged in.
  3. User has set his Carrier label printer in Tools -> Set devices.
  4. Carrier label printer defined for the department for which the user is logged in.
  5. User has set his label printer in Tools -> Set devices.
  6. Label printer defined in the GLIMS general settings.

Transfusion form printer

  1. User has set his Transfusion form printer in Tools -> Set devices.
  2. Transfusion form printer defined for the department for which the user is logged in.
  3. Blood label printer defined in the BTM general settings.
  4. Label printer defined in the GLIMS general settings.

Blood label printer

  1. User has set his Transfusion label printer in Tools -> Set devices.
  2. Transfusion label printer defined for the department for which the user is logged in.
  3. Blood label printer defined in the BTM general settings.
  4. Label printer defined in the GLIMS general settings.

Blood product label printer

  1. Blood label printer defined in the BTM general settings.
  2. Label printer defined in the GLIMS general settings.

Stock management labels

  1. User has set his Stock management label printer in Tools -> Set devices.
  2. Stock management label printer defined for the department for which the user is logged in.
  3. Label printer defined in the Stock management general settings.

Stock management order forms

  1. User has set his Stock management order form printer in Tools -> Set devices.
  2. Stock management order form printer defined for the department for which the user is logged in.
  3. Form printer defined in the Stock management general settings.
