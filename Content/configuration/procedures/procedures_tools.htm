Procedures - Tools

Edit procedures of Station

The Station based Edit procedures function provides a concise overview of the procedure configuration of a station.

Copying a procedure

The Procedure based Copy function allows to copy a procedure and all its inputs, outputs and procedure data.

Add assessment method to procedure

The Assessment method based Add to procedure function provides an easy way to add one or several assessment methods to a procedure.

Copy assessment method to another station

The Assessment method based Copy function allows to copy one or several assessment methods to another station.
