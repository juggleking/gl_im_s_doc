Edit procedures of Station

Introduction

The contextual menu / ribbon item Edit procedures on Station allows to define all the procedures (and its inputs and outputs) for that station.

Browsers

Central browser

Shows all Procedures of the selected station. Features:

  • Use the Insert key to add a procedure.
  • Use the Delete key to remove a procedure (if it has not yet been used by any scheduled action).
  • The procedure's Name, Code, Seq no and Available fields can be updated directly in the browser by clicking in the cell.
  • Use the F6 key to open a detail screen.

Left browser

Shows all Procedure inputs of the selected Procedure. Features:

  • Use the Insert key to add a procedure input.
  • Use the Delete key to remove a procedure input.
  • The procedure input's Seq no and Enabled (A) fields can be updated directly in the browser by clicking in the cell.
  • Use the F6 key to open a detail screen.

Right browser

Shows all Procedure outputs of the selected Procedure. Features:

  • Use the Insert key to add a procedure output.
  • Use the Delete key to remove a procedure output.
  • The procedure output's Seq no and Enabled (A) fields can be updated directly in the browser by clicking in the cell.
  • Use the F6 key to open a detail screen.

Tip

  • All 3 browsers can be sorted by clicking in the column headers.
  • The contextual menu matches the active browser. When the actual screen itself is focused, the contextual menu of Station is shown.
