Procedures - Copy

Introduction

Select a Procedure and choose the contextual menu / ribbon item Copy to copy the procedure and all its inputs, outputs and procedure data.

Fields

Target station

Station to which the copied procedure should be linked. When a different station is selected, the missing assessment methods are also copied.

Notes

  • The Copy function uses the standard Windows naming for copied records:

    "Copy of ..."
    "Copy (1) of ..."
    "Copy (2) of ..."
    ...

  • The name of the copied procedure is truncated at 40 characters (Oracle field limit).  This may cause a warning that a procedure already exists with that name if the original procedure name is already 40 characters long. In that case, the original procedure name should first be truncated.
