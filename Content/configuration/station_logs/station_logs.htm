Station logs

Contents: Introduction - Query - Browser - Fields

Introduction

Station logs allow registering important events regarding the station (calibration, maintenance, etc.).  These events can be shown on QC graphs.

Stations logs can be created

  • manually
  • automatically, for instance when the LAS status of an assessment method changes.

Query

Station logs are available from the main menu via Start -> Routine -> Station logs.

Station

Only logs for this station.

Minimal / Maximal event time

Time range for the event time of the station log.

Event type

Only logs with this event type.

Assessment method

Only logs for this assessment method / property.  If no Station has been selected, this field will be labeled Property.

Code

Only logs with this code.

Description

This field supports the use of wild cards.

Filter

Allows selecting a StationLog-based site function.

Browser

Station

Allows to filter on a specific station without having to reopen the query screen (using the Options button).

Access

This browser can also be accessed via the context menu / ribbon of a Station or Assessment method.

Fields

Station logs can be created manually.  In the Station log browser, use the Insert key to create a new station log.

Station

Station to which this log applies.

Assessment method

Optional reference to a specific channel.

Event time

Time at which the event took place.  Default: current time.

Event type

Describes the nature of the event.

Show on QC graph

If enabled, the event will be visible on QC graphs.

Code

Optional short code for overviews or graphs.

Short description, Long description

Free text description of the event.

Creation user

User who registered this log.
