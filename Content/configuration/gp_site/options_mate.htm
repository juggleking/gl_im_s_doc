MATE - General settings

Contents: Introduction - General - User interface - Security - Mail - Performance - Watchdog - LDAP - Report builder - Message queue

Introduction

Several features of GLIMS can be tailored to the needs of a site using general settings available under Start -> System management -> Customize -> MATE. These general settings are presented below.

Note

Changes made to these settings will only be applied after re-login.

General

Service pack

Shows the current version of GLIMS.

Service pack folder

Folder in which GLIMS service packs are located.

Printer

Default printer, can be overruled by other printer specifications.

Start tool

Tool that will be launched automatically when users login.

Tip

This can be overruled by a user-specific start function (defined in the User configuration).

Max tmp file age

How old (in days) should a tmp-file be before being purged.

Log folder

Default root folder for log files.

RC request interval

For more information, see Automatically upgrading GLIMS clients in the System management chapter of this manual.

User interface

Application title

Text to be displayed on top of the GLIMS window.

About box extra info

For more information, please check the chapter About box.

Language

The default language. For more information, please check the chapter Languages.

Align labels to the left

If set, all field labels will be left-aligned instead of right-aligned.

Home page

Default home page. Can be overruled by more specific settings.

Side bar

Default side bar. Can be overruled by more specific settings.

Ribbon

Default ribbon. Can be overruled by more specific settings.

Theme

Allows to change the user interface theme. Can e.g. be used to distinguish between a product and test environment (see below).

Security

Authorization timeout / Context timout / Connection timeout

Please check the chapter Getting started - Login.

Minimum password length / Maximum password age

Please check the chapter Users: Passwords.

Authentication extension

Please check the chapter Carte CPS - Carte Vitale.

Max login failures

Defines the maximum number of login failures after which a user will be disabled.

Notes

Note

All login failures are logged in GLIMS.

Password strength

Please check the chapter Users: Passwords.

Mail

Server, Port

In order to be able to send external emails from within GLIMS, a mail server and port must be defined.

E-mail address

E-mail address of the site, used when SITE is specified in a SendMail MISPL function or when a user who sends a mail has no e-mail address.

Use mail server authentication, User, Password

If mail server authentication is required, enable this option. When required, you also need to specify a user name and password. The password is hidden, securely encoded and never displayed.

Watchdog
