Specialisms

Contents: Introduction - Fields - Usage

Introduction

This table is used to define medical specialisms. Specialisms are available from the main menu via Start -> Correspondents -> Specialisms.

Example

Orthopedics, Trauma, Pediatrics, etc.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

Full name of the specialism.

Correspondent

When the specialism should be able to act as order agent or bill receiver, a correspondent must be specified.

Head

Health care provider who is responsible for this specialism.

Usage

A stay can refer to a specialism, which is the medical specialism currently taking care of this patient.  This is not the same as the ward of the stay: a ward can contain patients of different specialisms, and a specialism can have patients in different wards.   

The stays' specialism can be copied to the Order.Agent field when the order entry option Default agent is set to Specialism.  This allows sending a report or a bill to the specialism, or classify medical costs by specialism in statistics programs.  

Note

The specialism is NOT a type of correspondent !  When the specialism should be able to act as order agent or bill receiver, its Correspondent field should explicitly refer to a correspondent record (of any type; typically Institution or Organization).
