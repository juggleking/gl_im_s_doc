Departments

Contents: Introduction - Fields

Introduction

A lab can be subdivided into different departments. The department browser can be accessed from the main menu by selecting Start -> Correspondents -> Departments.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

The actual (long) name of the department.

Lab

Lab to which the department belongs.

Group

Department group to which this department belongs. See also Shielding.

Code

Short code that can be used e.g. to build the order internal id.

Specialty

The specific domain of this department.

  • Genetics: should be selected when using the pathology work screen for genetics.
  • Blood depot: should be selected when this department is able to stock blood bags. Please check the chapter Multiple blood bag registration.
  • Microbiology
  • HLA
  • Anatomic pathology  

Internal issuer

Issuer responsible within this department for lab-issued tests.
