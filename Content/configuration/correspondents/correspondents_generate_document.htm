Correspondents: Generate document

This function generates a document based on Text records.

Tip

By using the Correspondent function Generate document in a command / task, you could setup an automated mailing system.

Fields

Header

References a text expression to be used as header of the generated document.

Body

References a text expression to be used as body of the generated document.

Footer

References a text expression to be used as footer of the generated document.

Page skip

When several correspondents are selected, this option determines if a single document should be generated or a separate document for each correspondent.

Enabled

Separate document per selected correspondent.

Disabled

Single document for all selected correspondents.

Language type

Language when making use of multilingual texts.

User

Language of the current user. If unknown, the site language is used (see below).

Site

Language as defined in the general settings (Start -> System management -> Customize -> MATE).

Page height

Expressed in number of lines. Only applicable when a Printer is specified.

Number of copies

Number of copies to be printed. Only applicable when a Printer is specified.

Printer

Printer on which the document will be printed when clicking the Print button in the preview screen.

Preview

Allows showing a preview before printing / saving the generated document.

Output to file

If enabled, the generated document can be saved to file. When generating the document, a File specification window will be shown that allows specifying the File name and Directory.
