Correspondent data sheet

From each correspondent, you can use the contextual menu / ribbon item Data sheet to produce an HTML-based overview of the most useful data of that correspondent. 

Options

Full

If enabled, billing and report settings will also be included.

Note

If defined, the Correspondent-based site attributes Bank, Finanzamt and Teilbefunde are also displayed if this option is enabled.

Printer

Printer on which the data sheet will be printed when clicking the Print button in the preview screen.
