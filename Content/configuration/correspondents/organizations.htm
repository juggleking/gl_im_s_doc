Organization

Contents: Introduction - Fields

Introduction

The definition of an organization is important in the initial configuration of GLIMS. From the very beginning, you have to identify your laboratory as an organization. 

  1. You can create organizations by selecting Start -> Correspondents -> Organizations. Add your laboratory as organization.
  2. Now specify your organization in the general option Own organization (Start -> System management -> Customize -> GLIMS general).

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

The actual (full) name of the organization.

Main contact

Can be used on outbound documents.

Correspondent

This page contains correspondent information about this organization. Please check the chapter Correspondents for more detailed information.
