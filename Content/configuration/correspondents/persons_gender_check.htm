Person gender check

Contents: Introduction - Configuration

Introduction

GLIMS offers a built in system that warns the user if an improbable combination of first name + sex is entered for a person.  

Configuration

The Gender check table can be accessed via Start -> Correspondents -> Persons -> Gender checks and contains first names and genders.  A first name can be entered twice but with a different gender (e.g.: in Belgium "Chris" can be male or female).  First name + gender is the unique key. 

If the first name is not defined in the gender check table, you will never receive a warning when creating a new person.  If you do receive a warning, you still have the option to continue and save your configuration.

Note

  • This mechanism is not self-learning, i.e. the gender check table is not automatically updated.
  • GLIMS will only issue warning for the genders Male and Female.
