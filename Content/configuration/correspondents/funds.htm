Funds

Contents: Introduction - Fields

Introduction

Funds are organizations which can get involved in the reimbursement of an order set by means of its Financing records. Health care insurance organizations are the most common example of funds.

Note

A fund is not related with an order for other than financial reasons. Issuers, patients, etc. cannot be funds, since they are already connected to an order or its order set for other than financial reasons.

The fund browser can be accessed from the main menu by selecting Start -> Billing -> Configuration -> Funds.

Fields

Essentials

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

The actual (long) name of the fund.

Extra

Default policy name

The policy name which will be proposed during payment agreement creation or update (after the fund reference is specified).

Parent

References a higher level fund. This can be used for funds that are hierarchically organized.

Destination type

In France this field contains the Type de destinataire of the first fund.

Default alternate id assigned by, Default alternate id code, Default alternate id valid from

In Germany the "Alternate HC Code" of a fund is an IK number. A fund can have multiple IK numbers and a fund is usually identified with such an IK number.

If the fund of a patient is known but not its IK number then this Default alternate id must be used as the IK number for that fund.
