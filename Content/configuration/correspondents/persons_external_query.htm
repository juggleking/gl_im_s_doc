External person query

Introduction

In different places in GLIMS (order entry, advanced person search screen), you can lookup a patient by performing a person search in an external system.  To see how you can setup these external person queries, please check the dedicated chapters:

  • The "Object page" of the order entry screen
  • The advanced person search screen

Search result screen

Whenever an external person query returns one or more matching persons, a screen is shown with two browsers: a Patient browser and a corresponding Encounter browser. Below each browser you will find the details of the selected record.

Note

Data presented in a red color implies that its value or format is incorrect. E.g. incorrect birth date format, or unknown fund, etc. A corresponding short error message will be shown in the Error message area at the bottom of the screen.

Selecting a patient and encounter

  1. To select the correct patient: double-click the correct patient.
  2. To select the corresponding encounter: double-click the correct encounter.
  3. To deselect the encounter (no encounter will be imported): double-click the already selected encounter and it will be deselected.
  4. When the correct patient and encounter are selected, click Select to import the patient/encounter in GLIMS and possibly enter it in the order entry screen (if the query was started from within the order entry screen).

Navigation

The Next and Previous buttons can be used to browse through the next or previous patient list from the HIS. 

The Cancel button can be used to exit the screen without importing any patient nor encounter information.
