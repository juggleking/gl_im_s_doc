Companies

Contents: Introduction - Fields

Introduction

Companies are external organizations that employ people. They can be involved in commercial lab activities (industrial medicine, examination, ...). Companies should not be confused with Firms, which are used for invoicing purposes.

The company browser can be accessed from the main menu by selecting Start -> Correspondents -> Companies.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

The actual (long) name of the company.

Correspondent

This page contains correspondent information about this company. Please check the chapter Correspondents for more detailed information.
