HC Provider

Contents: Introduction - Fields

Introduction

Health care providers are employed in a medical environment. They represent the persons placing orders or taking the samples (physicians, biologists, samplers).

The HC Provider browser can be accessed from the main menu by selecting Start -> Correspondents -> HC Providers.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

First name, Last name

Full name of the physician.

Title

Example: "Dr.".

Code

Alternate code.

Initials

Can appear on outbound reports, see Validator on report.

Signature

Can appear on outbound reports, see Validator on report.

Birth date, Sex

Practice

The healthcare practice where this healthcare provider works. Also defines which results may be printed on cumulative reports addressed to one of the members of the practice.

Sampling location

Default sampling location type for orders of this issuer.

Note

If this HC provider is specified as Sampler during order entry, the Order will inherit the Sampling location specified in the HC provider record.

Distance

Default distance to the sampling location for orders of this issuer.

Note

If this HC provider is specified as Sampler during order entry, the Order will inherit the Distance specified in the HC provider record.

User account

User with which the HC provider can access CyberTrack.

Note

The link between a HC provider and a user account is also used to determine the result responsible and the report responsible.
