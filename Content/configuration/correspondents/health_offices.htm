Health office

Contents: Introduction - Fields

Introduction

In some countries, labs have to send a report to an official health office when certain micro-organisms have been discovered. 

The health office browser can be accessed from the main menu by selecting Start -> Correspondent -> Health offices.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

The actual (long) name of the health office.

Correspondent

This page contains correspondent information about this health office. Please check the chapter Correspondents for more detailed information.
