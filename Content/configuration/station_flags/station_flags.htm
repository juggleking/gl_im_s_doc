Station flags

Introduction

Analyzers can send flags along with result values to GLIMS. To store these flags, they must be known in GLIMS as Station flags.

To define a Station flag:

  • Select a Station.
  • Choose the contextual menu / ribbon item Result flags.

Fields

Station

Station that can send this flag.

Flag

Flags as sent by the analyzer.

Comment

Result-based text that will be copied to the internal comment of results marked with this flag.

Severity

This severity will be taken into account when calculating the actual severity of patient results marked with this flag.

Notes

  • Some analyzers send a part of the actual result value as a flag.  Example: result = 55, flag= <.  You will then have to use the assessment method's Conversion option to store the actual result as a combination of result and flag, e.g.: <55. 
  • Station flags can be checked in e.g. Property triggers or the procedure's Realization trigger.
  • This severity is not taken into account for QC results.  For QC results, use the option Severity for QC. 

Severity for QC

This severity will be taken into account when calculating the actual severity of QC results marked with this flag.
