Assessment methods: Copy

Contents: Introduction - Usage

Introduction

The Assessment method function Copy allows to copy one or several assessment methods to another Station.

Usage

Select one or more assessment methods and choose the contextual menu / ribbon item Copy.

Station

Station to which the selected assessment methods should be copied.

Setup

Enable this option and click the Select button to indicate which general characteristics you would like to copy from the selected assessment methods.

Limits

Enable this option and click the Select button to indicate which limit-specific characteristics you would like to copy from the selected assessment methods.

QC

Enable this option and click the Select button to indicate which QC-specific characteristics you would like to copy from the selected assessment methods.
