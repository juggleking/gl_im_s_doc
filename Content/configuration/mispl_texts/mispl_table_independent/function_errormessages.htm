Error messages

If you encounter an error that you do not know how to correct, make note of it and look up the explanation of the message in the list below.

Error message  Explanation
ERR_ARGCOVFL Too many function arguments (max allowed is n). Either the function definition or function call exceeds the maximum number of arguments (n) that can be passed to a function.
ERR_ARGTYPEMISM Data type mismatch for argument <n> of function ‘<FunctionName>’: expected ‘<DataType>‘ instead of ‘<DataType>’

The data type of the n’th argument passed to the function does not match the function definition.
ERR_BADARGC Invalid number of arguments passed to <FunctionName> (<n> expected)

The number of arguments passed to function FunctionName does not match the function definition.
ERR_BADSNSCHKSUM Scheme navigator scheme file <FileName> is damaged (checksum error)

Each scheme file contains a check digit (called the checksum), based on the arithmetic sum of all the bytes in the file. When reading a scheme file, the scheme navigator recomputes this checksum and stops with a checksum error if a mismatch occurs. This error indicates the file is corrupt or has been tampered with. To recover, restore the file from a recent backup.
ERR_BADSNSMAGIC File <FileName> is not a scheme navigator scheme file

Each scheme file contains a string constant (called the magic number) that identifies it as being a scheme file. If a magic number mismatch is detected, the file is not recognized by the scheme navigator. This error indicates the file is corrupt, has been tampered with, or is of a higher version number than your software. To recover, restore the file from a recent backup.
ERR_CONDTYPEMISM Condition data type mismatch for statement <Statement>: expected <DataType> instead of <DataType>

The expression controlling an IF, WHILE or REPEAT statement is not of data type logical.
ERR_CONV Data type conversion error

The conversion of a character string to an integer or a floating-point number failed. Check your input string for illegal characters.
ERR_DATE18TODMY Date formatting error

An attempt to extract the day, month & year from an internal date representation failed.
ERR_DATE18TOS Date formatting error

The conversion of a date to a character string failed. Check your conversion format control string for illegal directives.
ERR_DATEDMYTO18 Date conversion error (Date, format dd/mm/yyyy)

An attempt to convert a day, month & year (as indicated by Date) to an internal date representation failed.
ERR_DATESTO18 Date specification error, use <Format> format

The conversion of a character string to a date failed. Verify your date string has the appropriate format.
ERR_DBNFOUND Database <DatabaseName> was not found

An unknown database DatabaseName was referenced . Either the database name is misspelled or the database definition is not loaded.
ERR_DIVBYZERO Attempt to divide by zero

An attempt to divide an integer or floating point number by zero would cause an infinite result, outside the range of representable values.
ERR_DUPFUNC Function <FunctionName> cannot be redeclared
ERR_DUPSYM Symbol <Symbol> cannot be redeclared
ERR_DUPTYPE Data type ‘<DataType>’ cannot be redeclared
ERR_DUPVAR Variable <VariableName> cannot be redeclared
ERR_ENUMNFOUND Enumeration type ‘<EnumeratorClassName>’ was not found
ERR_FLDNFOUND Field <FieldName> of table <TableName> was not found

An unknown field TableName.FieldName was referenced . Either the field name is misspelled or the field definition is not loaded.
ERR_FOPEN Error opening file <FileName>: Errno

An error occurred while trying to open the operating system file FileName. Errno contains the operating system error. Have your system administrator check that the file exists and has the correct permission mode.
ERR_FREAD Error reading file <FileName>: Errno

An error occurred while trying to read the operating system file FileName. Errno contains the operating system error.
ERR_FUNCNOTLOADED Function '<FunctionName> currently not loaded

Your program was compiled against function FunctionName which is currently not loaded.
ERR_FUNCTUNDEF Function '<FunctionName> was not defined

An undefined function was invoked.
ERR_FWRITE Error writing to file <FileName>: Errno

An error was encountered while trying to write to the operating system file FileName. Errno contains the system error. Have your system administrator check that enough disk space is available on the target device.
ERR_IDENTUNDEF Identifier <Identifier> was not defined

The identifier Identifier was referenced, but could not be found in neither the list of active variables nor start class fields.
ERR_ILLDATATYPE Data type <DataType> is not supported
ERR_ILLENTRY Entry n is outside the range of list <List>
The enumeration value (or list entry number) is either <= 0 or greater than the number of entries in the comma separated list of enumerators.
ERR_ILLENUM <String> is not an enumerator for enumeration type <EnumeratorClassName>, expecting one of <List>
ERR_ILLINSTR Illegal instruction (opcode OpCode)

The object code interpreter is aborted because an illegal instruction is encountered. This error indicates the object file is corrupt, has been tampered with, or more likely is of a higher version number than your software.
ERR_INCOMPOP Incompatible operand(s), operation <Operator>
ERR_INTERNAL An internal error occurred in <Module>: Message

An unexpected error has occurred in module Module. Message may clarify the cause of the problem. As this information is of no use to an end-user, note down the error accurately and contact customer support for assistance.
ERR_INVARG Argument n of function call <FunctionName> is invalid
ERR_ITOS Integer formatting error
ERR_MALLOC Memory allocation error

Your application ran out of memory. To recover, try to close some other applications running on your system to free up memory.
ERR_NOHANDLE Missing code handle

Program execution was requested without specifying an object code handle.
ERR_NOSHVAR Module: Shared variable name unknown
ERR_NOSTBUF Missing start buffer for table <ClassName>

Program execution was requested without specifying an indispensable buffer name for start class ClassName.
ERR_OUTOFRANGE Numerical conversion under/overflow

A numerical conversion caused an overflow or an underflow. That is, the result of the conversion is outside the range of representable values.
ERR_READBUF Unable to read field <ClassName.FieldName>
ERR_READSHVAR Unable to read shared variable <SharedVariableName>
ERR_RECNFOUND Related <ClassName> record not available

An attempt to fetch a parent ClassName record was unsuccessful because the referred record is not available. Verify if the record is not deleted or being edited by someone else.
ERR_REFNFOUND Reference <ReferenceName> of table <ClassName> was not found

An unknown reference <ClassName.ReferenceName> was referenced . Either the reference name is misspelled or the reference definition is not loaded.
ERR_SYNTAX Syntax error
ERR_TBNFOUND Table <ClassName> of database <DatabaseName> was not found

An unknown class <DatabaseName.ClassName> was referenced . Either the class name is misspelled or the class definition is not loaded.
ERR_VARUNDEF Variable <VariableName> was not defined

Variable VariableName was referenced but not declared.
ERR_WRITEBUF Unable to write to field <BufferName.FieldName>
ERR_WRITESHVAR Unable to write to shared variable <SharedVariableName>
ERR_INTEGERTOS Integer formatting error
