Abs

This function returns the absolute value of Input.

IntegerAbs (IntegerInput)

Example:

Abs(-23) returns 23

Exp

This function raises Base to the power Exponent.

FractionalExp (FractionalBase, FractionalExponent)

Example:

Exp(10,2) returns 100
Exp(10,-2) 0.01
Exp(2.7,2) 7.29

Fabs

This function returns the absolute value of Input.

FractionalFabs (FractionalInput)

Example:

Fabs(-14.56) returns 14.56

Fmod

This function returns the modulus of fractional Dividend and Divider. (For modulus of integers : use %).

FractionalFmod (FractionalDividend, Fractional Divider)

Example:

Fmod(83.5,8) returns 3.5
Fmod(88,8) returns 0.0

Log

This function returns the natural logarithm (base e) of Input.

FractionalLog (FractionalInput)

Log10

This function returns the logarithm (base 10) of Input.

FractionalLog10 (FractionalInput)

Round

This function returns the rounded value of a given fractional value. 

FractionalRound (FractionalValue, IntegerDecimalPositions)

Example: 

Round(123.456, 2) returns  123.46

Sqrt

This function returns the square root of Input.

FractionalSqrt (FractionalInput)

Truncate

This function returns the truncated value of a given fractional value. 

FractionalTruncate (FractionalValue, IntegerDecimalPositions)

Example: 

Truncate(123.456, 2) returns  123.45
