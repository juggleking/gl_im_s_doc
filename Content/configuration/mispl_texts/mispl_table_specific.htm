Table-specific MISPL functions

There are a number of MISPL functions which are table-specific. All these MISPL functions are described per table in the GLIMS Reference manual under the heading MISPL functions.

The reference manual can be opened from the Table of Contents pane via Database documentation -> Reference manual.

Warning

In GLIMS, it might seem possible to call context functions in MISPL.  However, this is not supported by GLIMS and might not function as expected. In later versions, it will no longer be possible to even use these context functions in MISPL.
