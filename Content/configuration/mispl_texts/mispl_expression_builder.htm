MISPL and Text ® The MISPL expression builder

The MISPL expression builder is a user-friendly tool which simplifies the creation of MISPL functions. The start table will not be prompted for if it is known from the context.

 

Left window

The left window displays a conveniently arranged member list of the concerned class.  

  • Terminal members are displayed in the standard font.
  • Referring members are highlighted (similarly to a hyperlink).
  • On selection of a terminal node, the member browser becomes empty and the help window displays a dead end symbol.
  • When selecting a non-terminal method (i.e. a method that returns a reference to another class), further navigation is possible.
  • Undo & Redo buttons allow to navigate back and forth through the database.

Method parameters

To avoid member list overloading, methods are displayed without their parameter list. They are distinguished from properties by means of trailing parentheses '()'. The parameter list has been moved to the Help section, where the parameters are now displayed in the right order, along with their respective data type and reference manual section.

When selecting a method, you are guided through the parameter list by means of an inquiry that pops up for each parameter. On 'redo', the inquiry is repeated with the previously entered parameters.

Online Help

As you scroll through the member list, on-line help is displayed for the focused member, including its data type and reference manual section.
