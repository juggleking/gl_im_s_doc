Request definitions - Copy

Contents: Introduction - Fields - Logging

Introduction

To make a full copy of a request definition, select it and choose the contextual menu / ribbon item Copy.

Fields

New start date

Start date for the new request definition.

New mnemonic

Mnemonic for the new request definition.

New issuer

Allows to specify the issuer in case of an issuer-specific request definition.

Copy billing code assignments

If enabled, the billing code assignments of the original request definition will also be available for the new request definition.

Logging

Click the Do button to copy the selected request definition(s).

GLIMS will show an overview of the actions that have been executed.

Click the Review button to obtain a full logging document of the copy action.
