Property weights

For each property, you can define time-related property weights which can be used in statistical overviews (i.e. results statistics).  

Select a property and choose the contextual menu / ribbon item Statistical weights.

Value

Statistical weight of the property. If not defined (for a given period), the default statistical weight is ?.

Tip

In the result statistics module, unknown values can be filtered out by disabling the option Include unknown in the first screen.  

Attention though: when doing so, possible configuration lacks (missing values) will be 'hidden' by this.  It is advisable not to do so and instead enter at least 1 statistical weight per property, even if the weight is 0 (in that case specify '0' as value).

Start date, End date

Period during which the statistical weight is valid.
