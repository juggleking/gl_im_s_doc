Study configuration report

Introduction

When lab staff has set up a study (clinical trial) in GLIMS according to the protocol, GLIMS allows to check that the trial was set up correctly and can produce a report that lists all relevant study configuration.

License

The "Study configuration report" module is protected by a separate license named "Clinical Trial Study Collection Manual" (GLIMS_CLTRSCM).

Setup

The schema below shows the basic structure of a study setup.

Concept GLIMS definition Comments
Study Study All visits within a study are linked to that study by the Study field "Visit request code"
Set of visits Panel request definition All visits (panel request definitions) of a study are grouped in a dummy panel request definition
Visit Panel request definition For each visit, the tests are grouped in a panel request definition
Tests per visit Property / Panel Tests are defined as request definitions (property or panel)

In other words: all tests to be executed in a single visit are grouped in a panel request definition.  These panel request definitions are in turn grouped in a dummy panel request definition per study.  The link between the study and its visits is established through the Visit request code field, which refers to the dummy panel request definition.

Remarks

  1. This concept does not allow using the Study field Request code to automatically add requests when choosing a study during order entry.  Instead, you must manually enter the panel request definition linked to the appropriate visit.

Generate report

Once the trial configuration has been setup, a report can be printed using the Report Builder module.  

Template

As for other Report Builder reports, the template to be used can be specified in the general options or by department.

The default template shipped with GLIMS is "<ApplicationDirectory>/templates/examples/StudyCollectionManual/StudyCollectionManual.jasper".

Generation

To generate the report, select a study and choose the contextual menu function Study configuration report.
