Introduction

Several features of GLIMS can be tailored to the needs of a site using general settings available under Start -> System management -> Customize -> GLIMS General.

Confirm specimen on scan

Normally, when a specimen is scanned, it is automatically confirmed if this was not yet the case. However, the present setting allows limiting the automatic confirmation to specific situations. The possible values are the following ones:
  • Always: whenever a specimen is scanned, it is confirmed.
  • Per user department: the specimen will only be confirmed when it is input to at least one action for the current user's department. In other words, this option allows supporting situations in which the specimen is first scanned in a department where it is not to be processed. The specimen confirmation upon barcode scan only takes place when the specimen arrives in the department where the actual work is to be done.
  • Per user laboratory site: this value allows specimens scanned in a department A to be automatically confirmed, and thus made available, for other departments belonging to the same laboratory site. For instance, specimens scanned in a reception department can be confirmed and set as Available for all the executing departments that belong to the same laboratory site as the reception department.
