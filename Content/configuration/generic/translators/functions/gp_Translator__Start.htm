Translators - Start

To start a translator, select a Translator record and choose the contextual menu / ribbon item Start.

Notes

  • Before a translator can be started, the associated Service must be started.
  • On Windows, translators first need to be registered as a Windows service.
  • Translators can only be started from the server on which they are running, not from a client PC.
  • When logged is as a user having a role with User type set to Developer, the translator's Specific options will be displayed. 

Tip

  • GLIMS also offers a general Startup services function designed for a full start of all services and translators.
  • GLIMS also offers a Monitor services function designed for automatic restart of services and translators that unintentionally stopped.
