Services - Errors

When something goes wrong during registering, unregistering, starting or stopping of an NT service (a Service or a Translator), a message comes up with an error number. The meaning of these error numbers is explained in the table below.

Error Number Error Message Explanation
3 The system can not find the file specified Occurs when you start an NT service but the system can not find the executable.
5 Access denied Occurs when you register, unregister, start or stop an NT service on a computer where you do not have administrator privileges.
21 The device is not ready Occurs when you register an NT service and you have specified an account missing the right to access all network resources (shared drives) needed to run the Service or Translator.
1056 An instance of the service is already running Occurs when you start an NT service that has already been started.
1057 The account name is invalid or does not exist Occurs when you register an NT service and you have specified an invalid account name.
1060 The specified service does not exist as an installed service.  Occurs when you forgot to register a service or translator as NT service.
1062 The service has not been started Occurs when you stop an NT service that has not been started.
1069 The service did not start due to a logon failure. Occurs when you start an NT service that was registered an account that does not have the right to log on as a service. You can assign this right to an account with the Windows NT User Manager using the User Rights option in the Policy menu. 
1072 The specified service has been marked for deletion. Occurs when you try to register a NT service which is marked for deletion. Go to the Control Panel of your computer. Double click on the Services icon and select the specified NT Service. Close the selected NT Service. A window with a clock will appear on your screen, followed by an error message. Afterwards the NT Service will disappear from the list and you can retry to register your service.
1073 The specified service already exists. Occurs when you try to register an NT service with the same name as an existing NT service.
1326 Logon failure: unknown user name or bad password Occurs when you start an NT service that was registered an invalid account (username/password).
1722 The RPC server is unavailable. Occurs when your MA_DEFAULTSERVICEHOST variable in progress.ini or the Host field for a Service is not a Windows NT computer. It can be a not existing computer or a computer with Windows 95 or Unix as operating system.
