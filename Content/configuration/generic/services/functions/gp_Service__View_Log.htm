Services - View log

To view the log file of a service, select a Service record and choose the contextual menu / ribbon item View log. The log file will be opened with the MIPS Log Viewer application.

Note

  • For more information about the Log Viewer application, navigate to the Configuration -> Communication -> Log viewer item in the manual's table of contents.
  • If the Log Viewer application is not available for some reason, you can manually install it by double-clicking the setup.exe file in the MIPSTools folder of the GLIMS installation directory.
