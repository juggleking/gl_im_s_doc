Major new features

Genetics

  • New module entirely dedicated to the work flows of Genetics labs

 

Result validation

  • Result validation by responsibility delegation

 

Consult registration

  • Consult review by supervisors of HC providers in training
  • Consult registration for specimens
  • Overview of the consults registered for an object
  • Configurable consult statuses
  • Rich text boxes supporting formatting, dynamic text and MISPL expressions

 

Specimens

  • Possibility to indicate the sampler(s) of a specimen

 

System management

  • Support for automatically upgrading GLIMS clients

 

Communication

  • Options to select which patient data is included in the work order reply message
  • New features on behalf of specimen collection
  • Laboratory Code Set Distribution: specimen handling information
