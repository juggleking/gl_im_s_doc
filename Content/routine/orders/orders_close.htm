Order.Close

Introduction

It is possible to prevent changes to an order by closing it. Closing an order promotes it to status 'Closed'. From then on, no further modifications to the order are allowed. As this is the next status after 'Complete', this implies that only complete orders can be closed.

Fields

Allow tariffing delay

Allows closing an order although not yet completely tariffed. When the option is set, the tariffing status of the order will be ignored during completeness determination. In this way, further modifications to the order can be prevented and tariffication can take place later on.
