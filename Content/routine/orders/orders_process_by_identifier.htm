Process by identifier

The gp-site-based function Process by identifier {ProcessOrders} is available under Start. It allows processing orders similarly to the function Process by number for specimens. The "process action" consists in either opening the order entry screen or executing a tool of your choice.

Note

Note

Only one order id at a time should be entered in the Id field. In other words, it is not possible to process several orders at once.

The process action can be launched by either:

  • Scanning an id, or
  • Inserting an id manually in the Id field and pressing Enter, or
  • Inserting an id manually in the Id field and clicking on the Launch button.

Short id, Internal id and Work id can be used as search criteria. In the Seq. No. column, you can indicate whether the identifiers entered in the Id field should first be look up within Short ids, then within Internal ids and finally within Work ids, for instance.
