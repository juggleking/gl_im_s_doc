Introduction

Any record in GLIMS, and hence also orders, can be displayed in three modes: Insert, Update and View.  In case or orders:

Insert

An order is displayed in Insert mode when creating a new order (order entry).

Updated

An order is displayed in Update mode when modifying an existing order (using the F6 key or through the dedicated order update programs).

View

When viewing an order (see below for more information about opening an order in View mode).

View mode

An order is displayed in view mode when:

  • Selecting an order in an order browser and clicking the View icon in contextual ribbon.
  • You do not have Update privileges on the Order table.

When in view mode:

  • Most order fields will be visible but disabled.
  • The 'Delete' key in the embedded request and root specimen browser will have no effect.
  • The contextual menus on the request and root specimen browser will have no effect.
  • The request form: the 'Urgency' option and 'Apply urgency' button are disabled and clicking a request definition will have no effect.
