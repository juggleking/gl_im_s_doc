Order entry options: Labels

Order labels

Check if printed

If this option is enabled, GLIMS will first check whether the order labels have already been printed before (see order fields "Label1 was printed" and "Label2 was printed"). If already printed before, GLIMS will never print the order labels again.

Order label 1 - Order label 2

Determine whether or not order labels will be printed immediately upon committing the order record. If not, they can be printed at any later time upon explicit request.

Note

The layout of order labels 1 and 2 can be defined using the general settings Order label layout and Order label2 layout respectively.

Specimen labels

Check if printed

If this option is enabled, GLIMS will first check whether the specimen labels have already been printed before. More precisely, GLIMS will check if the Specimen option Label printed is enabled.

Root specimen labels

If this option is enabled, the root specimen labels will be printed immediately upon committing the order.

Work specimen labels

If this option is enabled, the labels of the work specimen (derived from the root specimen) will be printed immediately upon committing the order.

Order/Specimen label printer

Label printer

This will be the first label printer that is used when printing order labels, root / work specimen labels during order entry. Please check the chapter Set devices for more information.

Carrier labels

Carrier labels

Determines whether or nor not the carrier labels should be printed upon committing the order record. If not, they can be printed at any later time upon explicit request.

Printer

This will be the first label printer that is used when printing carrier labels during order entry. Please check the chapter Set devices for more information.
