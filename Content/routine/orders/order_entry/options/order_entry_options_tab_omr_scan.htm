Order entry options: OMR scanning

OMR translator

Represents the communication process with the external application (optical mark reader).

OMR scan mode

Manual

The user has to 'feed' the OMR manually (by pressing the Scan button).

Automatic

The order entry will automatically 'feed' the OMR (retrieve the next order) when the previous order is stored.

Continuous

The program will try to commit the scanned order immediately after having scanned it. When this succeeds, the program will automatically try to read the next order. When no errors occur, or when no questions have to be answered, this mode allows continuous scanning of a stack of forms without human intervention.
