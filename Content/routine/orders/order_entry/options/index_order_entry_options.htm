Basic routine -> Orders -> Order entry options: Introduction

The Order entry options screen can be opened by clicking the Options button in the order entry screen.

It is subdivided in six tab pages:

  • Main
  • Default
  • Sequence
  • OMR scanning
  • Scheduling
  • Labels

If you want this screen to open automatically when the Order entry program is launched, select the option Auto-activate this dialog in the Main page.

The options are saved as user-specific preferences.

The User button at the bottom of the screen allows you to set user-specific preferences (person search and correspondent search method).
