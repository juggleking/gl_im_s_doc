Basic routine -> Orders -> Order entry: introduction

General description of the order entry program

The order entry screen is available from the main menu by clicking Start -> Routine -> Orders -> Entry. It is divided into different sections:

Info window

The info window shows the most important order data.  Its content cannot be customized.

Main fields

This section contains the most important patient, issuer and order data to be entered for each order.  Using the screen customization tool, it is possible to define which fields should be shown in the Main fields section and which fields should be shown in the Detail tab pages (see below).

Click here for a detailed description of the fields that are by default located in the Main fields section.

Informational tests

This section shows all requested auto-prompt properties with their value.

Test requesting

This section allows to request the tests to be executed.

Detail tab page

This section allows to enter any additional data that is not displayed in the Main fields section.  The fields are subdivided into different tab pages:

  • Specimens
  • Objects
  • Attributes
  • Order
  • Reports
  • Billing
  • Referral
  • Extra

Buttons

Options

Opens the order entry option screen.

OMR Scan

The OMR scan button is only displayed if an "OMR translator" is specified in the Advanced page of the order entry option.  It is used to scan OMR forms and automatically create order in GLIMS.

Twain scan

GLIMS offers a solution that incorporates Twain image scanning abilities directly into GLIMS.

OK

Saves the order.

Cancel

Closes the order entry screen.
