Order entry: main fields - Animal info

These fields will only be shown when an order is created for an object of type "Animal".

Externalization

Allows to select an animal by its externalization

Owner

By first selecting the owner of the animal, you will only be able to select animals of this owner

Name

Name of the animal

External id

Birth date

Allows to register/consult the birth date of the animal. 

Age

Shows the age of an animal once its birth date is known. The age is either expressed in years, months, weeks or days, whichever is most convenient.  The reverse is also possible: specifying the age of the animal in years, months, weeks or days will automatically update the animal's birth date.

Sex

The available values are the same as for Person records. Ambiguous can be used for hermaphroditic animals.

Object template

Contains the animal's object template (e.g. horse).
