Order identifier types

Introduction

GLIMS supports the use of different order identifiers: Internal id, External id, Short id, Work id and even custom order identifiers.

Configuring custom order identifier types via a site domain

The site domain OrderIdentifierType is available in GLIMS. The first two values of the site domain's Value list are fixed and CANNOT be changed. Additional values (order identifier types) can be added according to the needs of the site.

Specifying custom order identifiers during order entry

For more information, see Order entry: Extra (Order identifiers).

Consulting the order identifiers of an order

When an order identifier is entered for an order, an order identifier record of the specified type is saved in the database. From an order, select the contextual function Show all > Order identifiers to open a browser containing the order identifiers of that order.

The order identifier in the order entry screen can be modified. If it is replaced by an empty string or "?", the order identifier record is deleted from the database. Modifications and deletions of order identifiers via the Order entry screen are logged in the Order log.

Searching for orders via their order identifier

For more information, see Order browsers (Orders by order identifier).

MISPL

The Order-based MISPL function Order.GetIdentifier can be used to return, for the chosen order, the order identifier of the specified type.
