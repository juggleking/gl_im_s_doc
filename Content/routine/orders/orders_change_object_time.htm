Order - Change object time (Order.ChangeObjectTime)

Introduction

The Change object time function allows the user to modify the object time of an order and the times of all related results, specimens, etc. Another way to change the lowest object time of an order, is by hitting the F9 key when positioned in the object date or time field of the order entry screen.

Fields

New object time

Specify the new object date and time for the order.  

Now

This button allows to set the current date and time (rounded to the minute) as new object time.

Notes

  • The times of all related results and specimens will also be updated. However, this does not apply to specimens with Sampled enabled and their related results.
  • When the object time of an order is changed using this function, the corresponding reports where Report.Medium = 'Communication' are marked with Report.NeedsChecking=YES. This will allow automatic resending of these reports (providing the NewResultCheck report generation parameter is set to 'Ignore').
