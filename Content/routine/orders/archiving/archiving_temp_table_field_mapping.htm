Temp-table definition for w_Identification

Name Datatype Format GLIMS field Remarks
pid_Patient w_Patient   N.A.  
pid_Code String   Identification.Code Repeats for each Identification valid on Order.LowestObjectTime
where Identification.Target = Order.Object.Person().Correspondent
pid_AssigningAuthority String   Identification.Source.InternalId

Note

Two fixed records are always added to this temp-table:

  1. Code = ToString(Order.Object), assigned by the application ("GLIMS")
  2. Code = Order.Object.InternalId, assigned by the own organization (SpecificSite.Organization.Mnemonic)
