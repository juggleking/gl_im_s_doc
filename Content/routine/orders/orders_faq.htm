Discontinue requests in order entry

Already stored requests can be discontinued from the order entry screen. There are three ways to do this:

  • Via delete in the embedded request browser.
  • Via the contextual menu function Discontinue in the request browser.
  • By typing '-request'.

A reason for the discontinuation will be asked (and is mandatory).

Changing the object time of an order and its contents

The contextual menu function Order.ChangeObjectTime() allows changing the object time of an order and all its results, specimens,...

For orders with multiple object times, the following should be observed:

  • The Result.ReferenceTime of all results is set to the new object time, regardless of their original value.
  • Results of 'relative' panels are converted while maintaining the difference between Result.ObjectTime and Result.ReferenceTime.
  • For Results of 'absolute' panels, only the date part of Result.ObjectTime is set to the new object date.

The ReferenceTime of a result is the time that was visible in the order entry screen at the moment the result was requested. This is also the time that is used to determine the column in which the result is reported (for 'Group by ObjectTime' reports, and to determine the time shift indication that is added on reports for dynamic results.

Unconfirm specimen availability

In order to be able to indicate that one or more specimens for a particular order are not yet available, take the following steps:

  • Use the contextual menu Requests -> Specimens on Order.  A browser will be displayed with the specimens for the selected order:

  • Select the specimens which are not yet available and use the contextual menu function Unconfirm availability to mark them as such.

Note

If the availability of the specimens has been confirmed automatically (order entry options: automatic sample login=YES), they will never be displayed on a sampling list, even after unconfirming the specimens (see above).  GLIMS does not create actions for sampling procedures when the automatic sample login is enabled.
