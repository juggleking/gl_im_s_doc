Order.Discontinue

Introduction

This function discontinues an order or a set of orders. Multi-record discontinuation asks for confirmation (identical to multi-record delete).  If the last order referring a specimen is being discontinued, the specimen serves of no more purpose and is discontinued as well.

Fields

Justification

When discontinuing an order, you must enter a justification text.

Notes

  • Discontinuing an order which already contains available, non-informational results will require an explicit confirmation.

  • When discontinuing an order, all related reports without "Last output time" will be removed as well.
