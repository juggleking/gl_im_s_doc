Order.CancelRequests

Introduction

The order contextual menu function Cancel requests (default path: Order -> Requests -> Cancel requests) allows discontinuing one or several requests in one or several orders (multi-select function).

Fields

Requests

Allows specifying a comma-separated list of requests that will be discontinued in the selected order(s). You can double click or type CTRL-F to select the requests to be discontinued from the request code browser or you click the icon to open the request form.

Reason

Mandatory reason for canceling these requests. It will be stored in the logging.

Remarks

  • GLIMS will ignore the request for cancellation for a specific request code if the request code does not appear in the order.
  • Canceling a request must be done by using the same mnemonic as the one used for creating the request in the first place. This means that when a panel was requested, canceling one item from the panel using this function is not possible. You can only cancel the entire panel, not the individual items from the panel.
