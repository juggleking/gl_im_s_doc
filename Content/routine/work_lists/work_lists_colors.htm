Customize colors in work list grid

The colors in the work list grid can be customized by setting the color parameters 60-69 in the color section of the progress.ini file.  Note that these colors will be overruled by these specified in the general color scheme program.

  • color60 = label background
  • color61 = label foreground
  • color62 = active label background
  • color63 = active label foreground
  • color64 = normal background
  • color65 = normal foreground
  • color66 = disabled background
  • color67 = disabled foreground
  • color68 = selected background
  • color69 = selected foreground
