Opening a work list

GLIMS offers different ways to open a work list screen.

Automatically after work list generation

If the option Edit in the work list generation screen (Start -> Routine -> Work lists -> Work list generation) is enabled, the work list grid will be opened automatically after the work list generation.

Using the "Edit" function on a work list

  1. Open the work list browser (Start -> Routine -> Work lists -> Work lists).
  2. Select a work list in the browser.
  3. Select the contextual function Edit.

Using the "Edit results and specimen" function from an order

Select an order and select the contextual function Edit results and specimen.

Specimens

If enabled, the input specimens will be shown in the work list grid. This allows to create a sampling list.

Results

If enabled, results will be shown in the work list grid.

Column width

Width of the grid columns, expressed in characters.

Grid type

Offers different visualization options for the grid.

Using the "Result entry" function from the main menu

From the main menu, select Start -> Routine -> Work lists -> Result entry.

Work list

Select the work list to be edited.

Creation time

Shows the creation time of the selected work list.

Entries

Shows the number of entries in the selected work list.

Advanced editor

Activate this option if you want the work list to open in the advanced format. Do not activate this option if you want the work list to open in the classic format.
