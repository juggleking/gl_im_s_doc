Introduction

In case specimens are not identified by means of a barcode, GLIMS uses an active work list to communicate with an analyzer. For each station you can specify a generated work list as active work list.  This will be the work list that is currently to be used by the station.

Active work list definition

During work list generation

During work list generation, you can enable the Activate option.  By doing so, the generated work list will automatically be set as active work list in the station configuration. 

Note

This will only work for generation via work list templates having a station reference.

Work list functions

The following contextual menu functions are available for a generated work list:

Activate

Activates the work list on the station, so that cup queries and results messages via cup number received from the station are effective on that work list

Deactivate

Deactivates the work list on the station

Cup queries

The sequence of the cup numbers of a disk will have to correspond with the sequence of the actions on the active worklist.

Example

A disk has 50 cups. The cups of disk 0 corresponds to the first 50 actions on the active work list. The tube in cup number 1 on disk 1 will correspond to action 51 on the active work list.
