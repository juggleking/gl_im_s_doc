Work list - Advanced editor

Introduction

If the option Use advanced editor was activated in the Generate work list screen when the work list was generated, the work list opens in the advanced format:

This editor offers more features than the classic editor and is particularly suited for work lists used in the context of the Genetics module.

Content of the advanced work list editor

On the left-hand side, all the actions contained in the work list are displayed. When selecting one or more actions on the left-hand side, the specimens and (locus/variant) results configured as outputs of these actions appear on the right-hand side, sorted per action. From there, the result values (numeric values, images, rich texts, etc) can be inserted, confirmed, validated or discontinued.

The specimens and results are presented in different panes: Specimens, Results, Locus results, CNVs (for retesting), Sequence variants (for retesting). These panes are only displayed if there are corresponding specimens/results to be shown. For instance, if there is no specimen output for the action(s) selected on the left-hand side, the Specimens pane is not displayed. The panes contain grids in which the records can be filtered.

Note

Contrary to the classic work list editor, the advanced work list editor cannot be customized. In other words, the layout options described in the work list template are not relevant to users of the advanced editor.

Left-hand side

Status

Lowest status of all the actions in the work list

Name

Name of the work list. It is identical to the name of the work list template, unless a Worklist name site function is defined or a name was entered manually before the generation of the work list.

Comment

Editable internal comment regarding the work list

Grid

Action list established on the basis of the chosen work list template. Press F6 on an action to open the Action editor. Use the filter icons to limit the number of actions listed.
When the tick box on the left of an action is selected, the specimen and result outputs of this action it appear on the right-hand side.

Seq. no.

Sequence number of the action (i.e. of the work list entry) in the work list. Assigned automatically on the basis of the Sort criterion field of the Work list templateeditor.

Label

If an action label function is defined, the MISPL is evaluated to populate the column with the resulting output. The label content can be customized by a site function in

  • the work list template editor > Layout > Grid > Rows > Action label
  • or in the GLIMS General settings > Main page > Externalizations > Action label

Note

If no MISPL expressions are defined for the action label, the internal ID of the input specimens will be shown instead, comma separated.

If no input specimens are available, the internal ID of the object is used.

Procedure

Code of the procedure instantiated by the action, or its name if its code is unknown.

Specimen

  • Internal id of the action's first input specimen, or
  • pool if the action has more than one input specimen, or
  • empty if the action has no specimen input.

Patient

Externalization of the order's object (i.e. patient)

U

Action's urgency. If no Sort criterion is defined on the Work list template, the actions are sorted by their urgency (most urgent first).

Sts

Action's status

Prenat

If the action's first specimen input was sampled on a foetus of the order's object, this column displays the pregnancy number, followed by the discriminator in case of twins.

Right-hand side

Note

The content of this part of the screen is similar to that of the Results screen in the Genetics module. For information specific to this module, please consult the chapter Results screen.

Header

Above the specimen and result panes, two buttons appear:

Confirm all

Click to confirm all the specimens and (locus/variant) results currently displayed on the screen and with status Available.

Validate all

Click to validate all the specimens and results currently displayed on the screen and with status Available or Confirmed.

 

When clicking on these buttons, a summary screen provides you with a list of all the specimens and results that are about to be confirmed/validated. They appear in the same order as in the editor:

Notes

  • CNVs and sequence variants with status Available and retest status To do are skipped.
  • Confirming or validating all results is not allowed if
    • this action leads to the end of the approach, and
    • a locus result is lacking a value.
  • In the summary screen, small arrows allow you to expand and collapse certain results. This is solely to improve the readability but has no effect on the validation/confirmation: collapsed results will also be confirmed/validated.
  • In the summary screen, results of activities with the same name but for different specimens are followed by the specimen ID to avoid confusion.

To confirm or validate one or more specimens or results but not all, select them and click on the contextual ribbon items Confirm or Validate.

Warning

Validation by delegation is not yet supported in the advanced work list screen!

Specimens

The Specimens pane displays the specimen outputs of the actions selected on the left-hand side.

Actn

Sequence number of the action of which the specimen is an output.

From

Internal id of the work specimen, followed by the material

To confirm the availability of one or more specimens, select these specimens and click Confirm availability in the ribbon.

Results

The Results pane displays the result outputs (except locus and variant result outputs) of the actions selected on the left-hand side.

To edit one result, select it and press F6. The Results modify screen opens. To edit several results at once, activate the check boxes on the left of the results you want to edit and press F6. The Multi-result update screen opens.

Actn

Sequence number of the action of which the result is an output.

Result mark

The second column of the grid has no header but can contain a result mark.

Root specimen

Internal id of the specimen out of which the work specimen was created

Work specimen

Internal id of the specimen on which the result was effectively measured

Property

Display name of the result property that is the output of the action whose sequence number appears in the left-most column of the row.

Value

Value of the result

Status

Status of the result

Locus results

The Locus results pane displays the locus result outputs of the actions selected on the left-hand side.

Pressing Insert on a row allows you to add locus results to the action whose number appear in the left-most cell of the row. Note that this action must be the only one selected on the left-hand side.

Actn

See above.

Specimen

Internal id of the action's specimen input, or pool if there is more than one specimen input.

For more information, please consult the chapter Results screen of the Genetics module.

CNVs (for retesting)

The CNVs pane displays the CNV results outputs of the actions selected on the left-hand side.

Pressing Insert on a row allows you to add CNVs to the action whose number appear in the left-most cell of the row. Note that this action must be the only one selected on the left-hand side.

Actn

See above.
If a variant result is linked to several actions, the lowest sequence number is shown.

Specimen

Internal id of the action's specimen input, or pool if there is more than one specimen input.

For more information, please consult the chapter Results screen of the Genetics module.

Pressing Insert on a row allows you to add sequence variant to the action whose number appear in the left-most cell of the row. Note that this action must be the only one selected on the left-hand side.

Actn

See above.
If a variant result is linked to several actions, the lowest sequence number is shown.

Specimen

Internal id of the action's specimen input, or pool if there is more than one specimen input.
