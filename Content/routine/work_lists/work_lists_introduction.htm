Basic routine -> Work lists -> What are work lists ?

Work lists offer a technical, procedure-oriented view on the work to be performed by the laboratory. Because each laboratory department has its own specialization, most often it will use its own set of work lists. A work list is a simple collection of different actions.

Work lists can be used for many purposes (e.g. sampling list), but are generally used as a way of result entry.
