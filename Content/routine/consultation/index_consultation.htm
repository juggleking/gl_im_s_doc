Consultation

The main consultation program is the order consultation program.  This query returns a list of orders.  

GLIMS also offers a number of consultation programs based on the Order table:

  • Quick report
  • Results
  • Tree view

Moreover, from the Property and Requestable table, you can generate a data sheet:

  • Data sheet

Furthermore, via the object of an order, there are two ways to obtain a result overview.

  • Browse results
  • Result overview
