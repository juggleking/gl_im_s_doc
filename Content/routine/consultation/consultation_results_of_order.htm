Results of order

Access

From each Order you can call the contextual function Results. This opens a browser which displays all results from the selected order.

Functionality

This screen offers the same functionality as the Order review program. Please see the chapter Order review for a full description.

"Confirm all" / "Validate all" buttons

The Confirm all and Validate all buttons allow confirming / validating ALL results in the appropriate status. Pre-selection has no effect in this screen. If a user is not registered as an employee (no employee assignments) or does not have confirmation / validation privileges, the confirmation / validation buttons will be disabled. The tool tip of these buttons will explain why they are disabled.
