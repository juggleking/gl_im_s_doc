Browse results of object

From the order, via object, one can select the contextual menu function Browse results. This function is Object-based and can therefore also be called directly from any object.

The Options button allows to specify further selection criteria for the result browser.

This options screen allows to query on property, property classification, object date/time, status and severity. Enabling the Hide unsolicited toggle box allows to see only the solicited results.

Note

Imported results will not be displayed if the Hide unsolicited toggle box is checked.
