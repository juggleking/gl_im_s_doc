Pre-defined menu options

Both programs (Confirm by action and Order review) can be configured as menu options with a pre-defined parameter set.

  • Click Start -> System management -> Menu -> Edit
  • Insert a new menu item for the pre-configured function.
  • Select ConfirmByAction or OrderReview as Function.

  • Double click the Function parameter set field to specify a parameter set.
  • Use the contextual menu function Setup to specify the criteria of the parameter set.

  • This pre-configuration will now be available from the main menu.
