Basic routine -> Reports -> Report language

This document explains how the report language is determined:

  • Language specified in the Report record (copied from "Default report" at creation time).
  • Language specified in the 'Report language' field in order entry. 

    An option is available to indicate that the report language defined in order entry should only be applied to reports targeted to the issuer. For all other reports of that order, the language of the target will be used (correspondent's language, see below).  To activate this option, a site attribute must be defined as follows:

    • Define a new site attribute for table "DefaultReport"
    • Name: _OrdLangForIssuerOnly
    • Label: free text
    • Data type: logical
    • Widget: toggle-box  
  • Language of the report target (specified in the Correspondent record).
  • Language of the order issuer (specified in the Correspondent record).
  • Language of the site (System management -> Customize -> Mate).
  • Language of the user.
