How to create the borders with PCL 5?

This how you can create a border around a heading with PCL 5:

Detailed explanation

&#27; => is the escape character

&#27;(s3B &#27;(s1p12v0s3b24580T

&#27;(s3B : Stroke Weight command

The stroke weight command indicates the line thickness used with a partcular font.   The default is 0 but values can range between -7 and 7. In this example we used thickness 3.

&#27;(s1p : Spacing command

Indicates the type of spacing to be used. 0 is a fixed spacing, 1 is a proportional spacing.

12V

This command specifies the height of the font (expressed in points). The default is 12 and values can range between 0.25 and 999.75.

3b

indem 3B

24580T

This command specifies the font type.

&#27;&f0S&#27;%1B

&#27;&f#S : Push/Pop cursor position command

The Push/Pop cursor command allows to save the current position and recall it later on . # can be replaced by a 0 (store cursor positie) or a 1 (recall cursor position)

&#27;%1B : Enter HP-HL2 with pen positioned on the PCL cursor position

PR;PU+1,-1;RR-500,5;

PR

This command allows the usage of relative plotting.

PU: Pen up

This command moves the pen towards the specified position without drawing a lign. In this case, 1 millimetre to the right and 1 millimetre to the bottom, i.e. 1 millimetre off the right bottom position of the last character.

RR-500,5 : Fill rectangle relative

This command allows to define a rectangle using the current position and moving toward another position. In this case, we specify -500 as x co-ordinate, in order to be sure that we can draw borders which reach the left margin. As Y position, we specify 5, which implies that the border thickness will be 5 dots (approximately 1,5 millimetre, depending on the printer dpi)

&#27;%1A&#27;&f1S

&#27;%1A : Enter PCL 5 mode

This command indicates that the printer should enter PCL mode. Possible values are 0 (previous pcl position) or 1 (current HP/GL/2 pen position).
