Printing ASCII characters above 127

In order to print ASCII characters above 127 (such as è, é, ë, ü, ö) on work lists and reports on a HP printer, you need a special initialization of your printer. With PCL-5 there are several settings needed.

Example

%-12345XE&l1X&l0S&l1T&l26A&l0O&a0P&l1L&a0L&l0E&l5D&k2G
(0N)0N(s0p16h12v0s0b16901T)s1p10h16v1s3b16901T&s1C

Situtation

  • Glims uses the character symbol-set compared to ISO 8859-1 Latin ECMA-94.
  • In PCL-5 you can select this symbol set by <esc>(0N.

Note

This is not the default of most printers. So, you need to add this in you initialization of printing.

  • Initialization can best be added into the header of the report.

For more information, please see the "The PCL 5 Printer Language Technical Reference Manual"
