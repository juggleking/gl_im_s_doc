Report query and browser

A report browser is available from the main menu. This browser uses the same query as the report generator program and thus allows showing the reports that would be printed by the report generator.  Click Start -> Routine -> Browse reports:

The report browser:

Order.Reports

From each order record you can select the contextual menu function Reports -> Reports.  This browser shows all the reports that contain at least one result of the current order, regardless of the scope of the report.  This browser allows for instance to reprint all the reports of a given order.

This is a dynamic browser that can be dynamically sorted. This implies that clicking on a column heading will change the sort order of the records (as known from other programs like Outlook, Explorer,...). An arrow indicator in the column header shows that this browser has this functionality and indicates the current sort column and order.

Note

The current sort column also becomes the column used for so-called fast-navigation (i.e. navigating to a record by typing characters in the browser).
