Report copies

Contents: Introduction - Configuration

Introduction

This chapter describes how you can indicate the number of report copies to be received by a specific correspondent.  This can be configured separately for complete and incomplete report.

Configuration

Correspondents

In the Report page, the following options are available:

Maximum interim versions

Indicates the maximum number of different interim reports the correspondent wishes to receive.  As soon as this maximum is reached, the next report will only be sent when it is complete.

This option is not taken into account during a 'reprint' query, i.e. a report query where MinimalLastOutputTime or MaximalLastOutputTime has been specified.  Moreover, this interim report limit is only taken into account for reports where Report.Medium is either 'Courier' or 'Fax' (the 'paper' reports).  This implies that this limit is ignored for e.g. electronic reports.

Setting this option to "0" indicates that this correspondent does not wish to receive any interim reports.

Interim copy count

Indicates the number of copies of an incomplete report that will be sent to this correspondent.  

NoteN

Setting this option to "0" does NOT have the effect that the interim reports will be suppressed (this option expects a minimum value of "1").  If you do not want any copy reports for a correspondent, then you should set the field "Maximum interim versions" to "0".

Complete copy count

Indicates the number of copies of a complete report that will be sent to this correspondent.

Default reports

Maximum print count 

Indicates that the report should only be generated a maximum number of times. As soon as Report.PrintVersion reaches this maximum, the report is deactivated (Report.Active=NO and Report.NeedsChecking=NO). This is helpful in those cases where the number of versions of the report sent to the report target should be limited (e.g. only send once).  

Warning

DefaultReport.MaximumPrintCount=0 will lead to the creation of inactive reports!

Note

If both the maximum print count option is specified and a "complete/interim copy count" is configured for a correspondent, the minimum of both values will be used as final printout count.

Report template

In the Main page, the following option is available:

Suppress duplicates

If enabled, the fields described above will not be taken into account, i.e. the correspondent will only receive a single report copy.  This can for instance be used to avoid sending multiple copies of fax/electronic reports.
