Basic routine -> Reports -> Printer evaluation during reporting

GLIMS allows to specify printers on different levels. Below you will find the sequence in which printers are evaluated during the report printing:

 1.  Explicit printer specified during report generation
 2.  Report.rprt_Printer
 3.  Report.Target.crsp_Printer
 4.  Report.DefaultReport.Printer
 5.  Report.ReportTemplate.Printer
 6. 'Report' printer saved in user preferences (last used explicit printer, see 1.)
 7. gp_Site.Printer

Note: sc_User.Printer is NOT used by the report program
