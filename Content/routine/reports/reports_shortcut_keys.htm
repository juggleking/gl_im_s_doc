Introduction

Each menu function in GLIMS can be called via a menu-accelerator. Check the chapter Menu customization for more information. In the reporting context, you could use these menu accelerators to quickly print a paper report, add and send a fax report, ... from any order browser, result browser (as the order menu is also available from the result menu) or even from the quick report screen (from which the order menu is available).

Example

Create a menu accelerator to add and send a fax from the quick report screen, any order browser, result browser...

  1. From the main menu, click Start -> System management -> Scheme -> Tables.
  2. Select the 'Order' table.
  3. Select the contextual menu function Menu -> Edit.

  4. In the tree view, press Insert to add a new menu option:

    Tool

    AddReport

    Label

    Label to be shown for this function in the order contextual menu.

    Windows accelerator

    Key combination that will automatically launch the function.

    Create a Function parameter set for the AddReport tool with the following options:

    Report code

    Select the report code which has been assigned to the default report used for fax reports.

    Generate now: ensures that the report will not only be added but also immediately sent.

  5. Now open a quick report or any order/result browser.
  6. Press the keys CTRL-F2.
  7. The Add report screen will be shown. Simply press 'Enter' and the fax report will be added and sent.
