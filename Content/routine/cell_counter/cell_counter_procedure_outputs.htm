Extra tools -> Cell counter: procedure outputs

For the cell count procedure, you need to define property outputs for the cells that you want to count.  Use the contextual menu function Outputs to do so.  The defined properties will be shown in the cell counter program.

ANY property output of the count procedure can be counted in the cell counter and the result will always be stored when it has been counted, regardless of the fact whether the result has been requested or the PropertyOutput.AutoPrompt setting.

However, when the result was not requested, a request for the result will only be created when PropertyOutput.AutoPrompt is enabled. Thus, such a result will be stored as 'Unsolicited' result, much the same as for analyzer result upload.
