Extra tools -> Cell counter: assessment method

You will have to configure the assessment methods of the procedure outputs for which you would like to use the cell counter.

Counter key label

This field allows you to select the key which will be used to count the property specified by the assessment method, e.g. the number '3'.

Count method

This field indicates if the counted cells have to be added to the total number of counted cells (belonging to the same formula) or if they have to be counted separately. In the latter case, the number of counted cells is stored as result value. The possible values for this field are Formula, Separate.
