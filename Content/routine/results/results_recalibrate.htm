Result: Recalibrate

Introduction

The contextual menu item Advanced > Recalibrate result on Result records allows you to recalibrate the result value by changing the value into

Fixed term + Coefficient * Old value + Square coefficient * Old value * Old value.

The fixed term, coefficient and square coefficient can be specified when selecting this menu function. You can use this function for multiple results simultaneously.

Parameters

All parameters that can be specified are used in the formula described above.  This formula is applied to all selected results.

Remarks

  • Only results for which the user has privileges, non-shielded results and non-discontinued results can be recalibrated.
  • Only numeric results, semi-numeric (type "String" but containing numbers) can be recalibrated. Results with comparators are also considered (e.g. >6 ). For other results a warning message will be shown.
  • Changes to the result values are logged. The original results are discontinued.
