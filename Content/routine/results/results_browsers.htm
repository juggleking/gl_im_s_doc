Result browsers

Introduction

There are several result browsers in GLIMS:

  1. Confirmation by action

  2. Order review

  3. Result overview of object

  4. Results of order

  5. Results of object

  6. ...

This document describes the common characteristics of these result browsers.

Window title

The title of the result browsers will contain an asterisk (*) when the Report info or Question fields of the related order contain information.

Mark columns

Request mark column

The second column of the order review screen may contain request marks.  The following request marks are supported:

English Dutch French  German Description
! ! ! ! Result with urgency level > 1
P T T T Result is to be phoned
p t t t Result has been phoned

Result mark column

The third column of the order review screen may contain result marks.  The following result marks are supported:

English Dutch French  German Description
R H R W If a result has been repeated (duplo/triplo results), an 'R' is displayed (example: R2 -> two results).  In order to select a value, use the contextual menu function 'Select value'.  For more details, see Duplo assessments.  
M G M M Results of which the value has been modified while the result status was 'Available', are marked with "M". The original value can be viewed in the result's log window.  Note that the existing 'Repeated/Replaced' ("R") indicator, which indicates that a result was modified AFTER it had been confirmed, takes precedence over the "M" indicator.

Flags

If a given result has a severity (of any kind) of at least 1, the corresponding result flag(s) will be displayed, indicating what caused the severity violation.

Note

The indicated severity is the highest of all available severities for a given result.

The following flags can be displayed:

English Dutch French  German Description
N R B N Norm severity
D D D D Delta norm severity
S S A G Station severity
M M M M Manual severity
Q Q Q Q QC severity
P T T T Phone severity

These flags are displayed in a non-proportional font so that they will always be located in the same position within the result flag column.

External system flags

Some result browsers (such as the order review program) also offers support for external expert systems.  If the specific site field Expert system is not set to ?, an extra column will be added to display the result appraisals (Result.ExpertSystemFlags) sent by the expert system.

Value

A value cell containing an asterisk (*) indicates that a result comment is available.

Severity

Severity level of the result.

Unsolicited

If a result is unsolicited, an X will be displayed in the column with the label 'U'. It is possible to create a request for such an unsolicited result by double-clicking the 'X'.

Remarks

The result browsers can detect that the displayed result value is too wide to fit in its column. This may lead to misinterpretation. Therefore, this is made clear to the user by replacing the last part of the value with the '@'-sign.  Note that such a situation may sometimes quickly be resolved by double-clicking the frame title in order to maximize the browse window.
