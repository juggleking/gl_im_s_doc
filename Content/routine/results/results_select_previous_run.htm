Result: Use value from previous run

Introduction

This function will replace the selected results with the value of the previous run (if there is such a value). This is the same operation as using the Result.SelectValue() function and selecting the column of the previous run's results to revert back to these values.

Confirmation and validation

Auto-confirmation

The new value is automatically confirmed.  If configured on the property, the confirmation trigger will also be executed.

Auto-validation

Automatic validation of the new result when applicable, in the normal way (based on result severity + assessment method threshold). When auto-validation is applied, a validation trigger will now also be executed (if configured on the property).

Usage

When used in combination with the result confirmation and validation browsers, this offers a tool to quickly revert to the previous assessments of the selected results, after which the result browser is automatically refreshed, allowing to e.g. validate the remaining and/or the newly created result values. 

The discontinuation trigger is also executed on the selected result (if configured on the property).

Remark

The value shown in the Previous run column of validation browsers is the value of the previous actual assessment.  

Example

A result has been measured three times (using the discontinue and repeat functionality).

Value 3rd measurement Value 2nd measurement Value 1st measurement
50 40 30

When applying the Use value from previous run function, the final value will be "40" and the previous run value will be "50".  When applying this function again, the final value will be "50", but the value of the previous run will also be "50", as this is the most recent actual assessment (the value "40" was not an actual assessment, but obtained by reverting the value during the first execution of Use value from previous run).
