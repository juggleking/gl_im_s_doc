Extra tools -> Specimen scan: what can it be used for ?

The specimen scan program can be used for multiple purposes:

  • See the history of the scanned specimen
  • Create a kind of manual sorting method
  • Check the work flow for a specimen
  • To print labels for the scanned specimen
  • For storage purposes

To see the history of the scanned specimen

By scanning a specimen, a history of the specimen is kept. This history can be displayed by calling the audit trail of the specimen. Open the specimen identification browser, select the specimen and use the contextual menu function Specimen > Audit trail. A browser will be displayed showing a history overview of the specimen. When a specimen is scanned, queried by a station, or receives its results from a station, a Specimen Audit record is created with event type = "Scan". You can recognize a query scan or result upload scan event by a preceding 'Q' or 'R' in the specimen audit's comment field.

Create a kind of manual sorting method

When scanning a specimen, you will immediately be able to see to which action/ procedure it is related. This means that you also know which test has to be executed on this specimen and on which station. This information allows you to manually sort the specimens and then put them on the analyzer. The scan program thus allows to create a manual sorting post.

Check the work flow for a specimen

When scanning a specimen, the work flow for this specimen will be displayed in the bottom window. If the toggle box Show details is enabled, the related actions and its status will be displayed as well as the station, the volume needed and result properties. The Show ready toggle box allows to display or hide the actions for this specimen that already obtained the status Ready. If no work is left for the scanned specimen the word Ready will be displayed on the left hand side of the work flow screen.

To print labels for the scanned specimen

The toggle boxes Root labels and Work labels allow to immediately print labels for the scanned specimen. Both the labels defined on material and on procedure input will be printed. The default label printer is specified in the specific site preference set. However, each user can select his personal printer, by selecting Start > Tools > Set devices. Each time this user logs in, this printer will be used as label printer.

For storage purposes

The specimen scan program can also be used for specimen archiving. Please check the chapter on specimen archives for more information.
