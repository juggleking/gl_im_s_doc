Specimen.Attribute

String  Attribute (String AttributeName)

Supported values for AttributeName:

"SamplingTime"

Sampling time in "DD/MM HH:MM" format.

"PropertyList"

Comma separated property mnemonic list (Property.Mnemonic) of all properties derived from the specimen.
Format: TotalCount:[Mnemonic(Count)[,Mnemonic(Count)]...]

"PropertyCodeList"

Comma separated property code list (Property.Code) of all properties derived from the specimen. Properties without a code are excluded.
Format: [Code][,Code]...

"CompletedPropertyCodeList"

Comma separated property code list (Property.Code) of all properties derived from the specimen. Properties without a code are included and the mnemonic is used instead.
Format: [Code][,Code]...

"PropertyShortNameList"

Comma separated property short name list (Property.ShortName) of all properties derived from the specimen. If the short name is not available, it is replaced by the property mnemonic (Property.Mnemonic).
Format: [ShortName|Mnemonic][,ShortName|Mnenonic]...

ProcedureCodeList"

Comma separated procedure code list (Procedure.Code) of all procedures of which the specimen is input.

"DepartmentMnemonicList"

Comma separated list of all the departments where actions for this specimen or its subspecimens are scheduled.

"LabMnemonicList"

Comma separated list of all the labs where actions for this specimen or its subspecimens are scheduled.

"RootInternalId"

Internal id of the root specimen.

"ExternalComment"

Multi-lingual specimen external comment.

"Order" or "OrderInternalId"

Searches the most recent order referring the specimen and returns its internal id.

"OrderDepartmentMnemonic"

Searches the most recent order referring the specimen and returns the mnemonic of the originating department (Order.Department).

"Issuer" or "IssuerInternalId"

Searches the most recent order referring the specimen and returns the internal id of its issuer (Order.Issuer.InternalId).

"IssuerName"

Searches the most recent order referring the specimen and returns the name of its issuer (Order.Issuer.Name).

"Agent" or "AgentInternalId"

Searches the most recent order referring the specimen and returns the internal id of its agent (Order.Agent.InternalId).

"AgentName"

Searches the most recent order referring the specimen and returns the name of its agent (Order.Agent.Name).

"DepartmentPropertyList:<dept>" and "DepartmentPropertyCodeList:<dept>"

These functions, where <dept> should be replaced by the mnemonic of a department, allow to retrieve a list of properties wich have to be assessed on this specimen in a specific department.

Example

Example of a material label:

.Object.Person().LastName
{< .InternalId} {< .Material.ShortName}
Dep.: {DepartmentMnemonic}, Size: {Size}
Work: {< .DepartmentPropertyCodeList:{DepartmentMnemonic}}

In the definition of the label, the {Size} parameter passed to this text also takes the minimal and maximal size of the material definition into account, as opposed to the Specimen.Size reference.

"StationCodeList"

Returns a comma-separated list of codes of stations for which this specimen has scheduled actions.

Example

Specimen.Attribute("StationCodeList") might return "H717,AU500".

Note

Actions with discontinued outputs are not taken into account.

"CarrierList"

Returns a comma-separated list of all carriers of all microbiology actions on the specimen, but with the restriction that each medium (type of carrier) will only be mentioned once (so the list will not contain duplicates). Optionally it is possible to specify a certain format for this list. This is done by specifying Specimen.Attribute("CarrierList: ") in which is a string that defines how each 'medium' should be shown. To do so, it should contain at least '{Mnemonic}' or '{Description}' (or both), which will be translated for each medium into the medium's mnemonic and description respectively. If is not specified (only "CarrierList"), the mnemonics are returned.

"SlideList"

This returns a comma-separated list of all carriers of type 'Slide' that can be found on the specimen.  For each carrier (slide) the list contains the complete internal id (which is the specimen's internal id + the carrier's internal id, separated by '-'), and the mnemonic of the medium.  This function can be used e.g. to produce a list of currently running pathology exams, containing for each exam the list of slides available in the lab.

Example

RETURN Replace( .Attribute("SlideList"), ",", "~n");

for specimen 'T05-12345-I' could produce:

T05-12345-I-1-1 HE
T05-12345-I-1-2 HE
T05-12345-I-1-3 HE
T05-12345-I-1-4 PAS
T05-12345-I-1-5 PAS
T05-12345-I-2-1 HE
T05-12345-I-2-2 HE
T05-12345-I-2-3 HE
T05-12345-I-2-4 PAS
T05-12345-I-2-5 PAS

"StorageList"

Returns a list of storage records for the specimen (see also "Lab archive")
Format: <ArchiveMnemonic>-<RackCode>-<Position>,...

Example

Rout-Mon-1-2,Arch-Jan-1-5

"CarrierStorageList"

Returns a list of storage records for the specimen's carriers (see also "Lab archive")
Format: <ArchiveMnemonic>-<RackCode>-<Position>,...

"IsolationStorageList"

For each storage record of each isolation on the specimen, the archive, the rack and the position in the rack is returned (see also "Lab archive")
Format: <ArchiveMnemonic>-<RackCode>-<Position>,...

"Urgency"

Returns the highest urgency of all requests linked to the specimen.
Rout-Mon-1-2,Arch-Jan-1-5
