Specimen.SetMeasuredSize

Specimen.SetMeasuredSize (PositiveFractionalSizeValue)

The SetMeasuredSize MISPL function can be used to copy the volume of a specimen entered as a result value to the Measured size field of a chosen Specimen record. To do so,

  • open the editor of the property linked to the volume result,

  • in the On confirmation field (tab Triggers), add, for instance, the following site function:

    .Result.WorkSpecimen().SetMeasuredSize(StringToFractional(.Result.Attribute("Value")));
    RETURN TRUE;
