Specimen.CollectionInfo

Specimen.CollectionInfo()

This function allows you to retrieve information provided in the Specimen info screen for a given specimen . It has no parameters and its data type is String.

Example

Example

The following MISPL allows you to retrieve the value of the Container count field for the selected specimen:

RETURN "Container count: "+ IfKnownString(IntegerToString(.CollectionInfo().ContainerCount, "%o")) + "~n";
