Specimen - Add request

The Specimen.AddRequest() contextual menu function allows to add one or more requests to a specific specimen.  This can be done either by entering one or multiple request codes, separated by a comma in the request list field, or by clicking on one or multiple request codes in the request form.

This is roughly the same as adding a request to an existing order while this specimen is selected as root specimen, but 2 important differences:

  1. The request(s) can be added for multiple specimens at once.
  2. The request(s) can be added specifically for a work specimen! During scheduling, the scheduler will make sure the action which is created has the intended specimen as input, although after scheduling, the requested result will still refer to the real 'root specimen'. For instance, if a 'Na' request is added on a Serum specimen, which in itself is derived from a Blood specimen, the Na action will be scheduled on the Serum specimen, but the root specimen (Result.Specimen) of the Na result will still be the Blood specimen.
