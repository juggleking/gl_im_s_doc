Specimen.ProcessByNumber

This function is available from the main menu through Start -> Routine -> Specimens -> Process by number.  It allows to select any function from the Specimen contextual menu and then entering a specimen id.  

To start, click the Change button and then click the function you want to use.  The selected function will appear in the window (and in the title bar). The leftmost button at the bottom will also have this function as label.

Now you can enter multiple specimen numbers in the 'specimen' field (each followed by a carriage return).  This can be done either manually or with a barcode scanner.  You can enter the specimen internal id, specimen external id or specimen identification. On each entered specimen the above selected function will be executed as soon as the ENTER key or the main button is hit.

You could for instance use this program to scan a sample and then immediately start dictating results.
