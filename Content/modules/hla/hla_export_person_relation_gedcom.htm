Export person relations to GEDCOM

Starting from a person, it is possible to export all related persons (the person's family) into a file in GEDCOM format.

GEDCOM is an acronym for GEnealogical Data Communication. It is a flexible, uniform format for exchanging computerized genealogical data. Its purpose is to foster the sharing of genealogical information and the development of a wide range of inter-operable software products.

The program is currently restricted to follow only parent, child and partner links. For each person the following data is exported:

  • last name
  • first name
  • sex
  • birth date
  • decease date
  • object attributes

The file is saved in the GLIMS session's temp-directory.  As soon as the whole family is exported into the file, the file will be opened automatically with the default Windows-registered program for files with extension ".GED". If no program is registered yet, Windows will ask the user which program should be used.

Examples of programs that could be used e.g. are:

  • Daub Ages: www.daubnet.com/english/ages.html
  • GenoPro: www.genopro.com
  • GedView: www.gedview.org/
