Anatomic pathology - User assignment

Introduction

It is now possible to assign a pathology exam to a certain user. This user can be any user who is employed in the department where the exam is executed (this means there must be an active employment for this user in this department).  The assignment of exams to users can be done manually or automatically.

Automatic assignment

As soon as an exam is created -that is during order entry- the exam can be assigned automatically by means of a MISPL function. This function must be defined on the pathology procedure, in the field "Determine technician"

The MISPL function must be based on table 'PathologyExam' and must return the 'login name' of the GLIMS user who the exam should be assigned to.

Example MISPL

---------------------------------------------------------------------------
RETURN IF .Specimen.Material.Mnemonic = "BIOPSY" THEN "Fred" ELSE "Barney";
---------------------------------------------------------------------------

This mechanism can be used e.g. to select the cut-up technician for each exam.

Manual assignment

The user who the exam is assigned to can be changed in the pathology work screen. It is displayed underneath the responsible HC provider. The combo-box only shows the users employed in the exam's department.

You can also change the user assignment for multiple examination simultaneously using the contextual menu function Change assignment on pathology examination. Glims version 8 users must manually add this menu functions using the "RunProcedure" mechanism.

Note

The user assignment as well as the responsible HC provider can no more be changed after the exam has been closed (status 'Validated-Complete')

Selection on user assignment

In the pathology exam query browser and the pathology statistics it is now also possible to make a selection on exams assigned to a certain user.
