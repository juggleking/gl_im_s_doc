Anatomic pathology

  • Introduction

The introduction will give you a general idea of the structure of the GLIMS anatomic pathology module.  It links the pathology related tables in GLIMS with the real-world pathology processes in the laboratory.  It explains how the different steps in the laboratory are represented in GLIMS.  Read this document carefully before starting the basic configuration.  

  • Basic configuration

This document lists a number of tables which should be configured before you can start and organize the actual pathology procedures. These tables contain all the basic information which will be linked together by the pathology procedures.   Each table to be configured contains a link to the reference guide where you can find detailed documentation on each field.    

  • Browsing the pathology examinations

It is possible to perform a query on the existing pathology examinations.  The query contains numerous selection options. The results of your selection will be displayed in the pathology examination browser from where you can call the pathology work screen.

  • The pathology screen

The pathology screen in GLIMS is a work screen where you can enter all the findings of the pathology examination.  It gives a general overview of the examination in all its aspects.  It allows to see the results of the different parts of the examination, change the responsible for the examination and even to view photo's if any and to enter a final conclusion for the pathology examination, etc.
