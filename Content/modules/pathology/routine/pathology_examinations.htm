Browsing the pathology examinations

  • Pathology examination query
  • Pathology examination browser

The pathology examination browser is available from the main menu via Start -> Anatomic Pathology -> pathology examinations.

Pathology examination query

Generic page:

Most of the query options are straightforward.  Many of the query fields (material, procedure, department, responsible) are displayed using a combo-box which only lists the relevant choices (e.g. only departments where pathology exams are executed, only materials on which pathology exams are performed etc.).   However, you can still select any value from the database by typing, using the up/down arrow keys, or CTRL-F or double-click.

Department

When choosing a particular department,

  • The list of 'Assigned users' that can be selected is restricted to those users that are 'employed' in that department;
  • The list of 'Responsible HC providers' that can be selected is restricted to those HC providers that are responsible for at least one 'employed user' in that department.

Status / work status

You can select a single status or work status, or a range of status or work status values.  

Note

The work status option will be hidden when you are login for a department for which the field Specialty is set to Genetics.

Cut-up The specimen is available in the lab and can be cut up into pieces.
Embedding The specimen has been cut up; the pieces should now be embedded in cassettes.
Microtomy The pieces have been embedded; these blocks should be cut with a microtome and put onto slides
Staining The slides have been produced; they should be stained now.
Reading Slides have been stained and can be evaluated under the microscope.

Material variable

This option is only available when a material has been selected.  It allows to select on the first material variable of the selected material. 

Executing department (group)

A pathology examination will be selected if:

Any of these departments:

Pathology examination -> Specimen -> Result -> Department

Matches any of the following executing departments:

The specified Executing Department.

OR any of the executing departments in the specified Executing department group.

Specific page:

These options allow choosing of which patient you want to view the examinations. 

Search for

This option allows to search for confirmed pathology examinations for which one of the results contains the specified text. The search mechanism will also consider results of specimens in the same order which are not linked to a pathology examination. Only text, integer of enumerated results from the examination will be taken into account.  

It is possible not only to specify literal words, but also word patterns. When you enter a search word that starts with a '*' character, GLIMS will consider this as a pattern to search for within the results

In patterns you can use following wildcard characters:

  • A period (.) in a particular position indicates that any single character is acceptable in that position.

  • An asterisk (*) indicates that any group of characters is acceptable, including a null group of characters.

Examples

A12.1 will only search for the exact string "A12.1" occurring anywhere in the results
*A12.1 (or *A12.1*) will search for any string matching this pattern, which means that e.g. "A1231" and "A12B1" will also be found (of course also "A12.1" will be found as well).
*A.6.1 Larynx mixes literal words / patterns and will search for all exams where the literal word 'Larynx' occurs and where the pattern 'A.6.1' also occurs.

Note

If you want to specify a period (.) or an asterisk (*) as a literal character rather than a wildcard character in the pattern, enter a tilde (~) before the character.

Example

*A..~.1 will find "A12.1" and "Abc.1" but NOT "A1231"

Pathology examinations browser

Header information

The header information in the Pathology examination browser can be customized using a text module specified in the general options (Start -> System management -> Customize -> GLIMS Anatomic pathology).  There are two fields where you can specify either a Specimen-based or a PatholoyExam-based text module.

To customize the header information in the pathology examination browser:

  1. GLIMS will check if a PathologyExam-based text module is defined in the Browser header 2 field.  
  2. If not defined, GLIMS will check if Specimen-based text module is defined in the Browser header field.
  3. If neither of them is specified, default header information will be displayed.

MISPL column

An extra column can be added of which the content is determined by a site function.  To activate this feature, a field named MISPL Column is available in the general pathology settings.  The return value of the specified site function is shown in the extra column of the pathology examination browser. The content of this column is limited to a maximum of 255 characters.

Column sorting

The pathology examination browser is a dynamic browser. This implies that clicking on a column heading will change the sort order of the records (as known from other programs like Outlook, Explorer,...). An arrow indicator in the column header shows that the browser has this functionality and indicates the current sort column and order.  

Note

The current sort column also becomes the column used for so-called fast-navigation (i.e. navigating to a record by typing characters in the browser).

Colors

The color of the conclusion indicates the status of the examination. Please see the chapter on color schemes for more information.

Tool tip

When hovering over a record, the conclusion of the examination is shown as help text. This way, it is not necessary to open the examination to see the conclusion.
