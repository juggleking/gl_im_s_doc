WASPLab asks GLIMS for isolation sequence number.

 

For each isolation, WASPLab sends a query message for the isolation number.
GLIMS creates the isolation and sends its internal sequence number.

WASPLab

For each isolation, WASPLab sends a query message for the isolation number.

GLIMS

GLIMS creates a new isolation. The micro-organism will be the default micro-organism specified in the microbiology procedure.

GLIMS sends a reply message that contains the new isolation's internal sequencer.

WASPLab sends isolation details

WASPLab sends a result message with isolation details.

WASPLab

WASPLab sends isolation details:

  • Organism
  • Colony count
  • Comment

GLIMS

The isolation details are stored in GLIMS:

  • Organism -> Micro-organism
  • Colony count -> Appraisal
  • Comment -> Comment
