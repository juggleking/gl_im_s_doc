When growth has been detected, the process of picking colonies and transferring them to new plates is supported in GLIMS by automatically adding new carriers (plates).

MISPL function to be used

The following MISPL function can be used to achieve this:

Carrier.AddSubCarrier

The added carrier will refer to the original carrier via its Parent carrier field.

Example

The MISPL function Carrier.AddSubCarrier can e.g. be used in the Medium field When carrier data changes, which is executed when the carrier's Comment, Growth status, Growth result or Grafting result changes.

Tip

GLIMS offers different MISPL functions to automatically add carriers that can be used in different situations:

  • Isolation.AddSubCarrier
  • Specimen.AddCarriers
  • MicrobiologyAction.Addcarriers
