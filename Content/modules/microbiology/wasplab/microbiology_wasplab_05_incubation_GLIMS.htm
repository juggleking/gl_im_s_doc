Consultation of growth results in GLIMS

Extended carrier browser

In the options of the microbiology work screen, make sure that the option Show the extended version of the carrier browse is checked.

Growth results

The carrier browser in the microbiology work screen shows the growth results in the Incub. column.

The growth result value is accompanied by the Time offset between brackets. This is the time lapse of the last observation since grafting.

Value of growth result

Note

Background colors are only applied when the status of the growth result is Final; i.e. when the value is displayed in uppercase (see below).

Automatic promotion of microbiology action status

When the Growth result of all carriers is Negative and Final, the status of the microbiology action will automatically be promoted to a higher status.

The exact status depends on the general microbiology setting Status action on negative growth. To access this setting, choose Start -> System management -> Customize -> GLIMS microbiology.

Manually adding growth results in GLIMS

In case of a communication failure between WASPLab and GLIMS, growth results and their status can be entered manually in GLIMS.

For one or more carriers

Use the contextual function Set growth on Carrier.

Tip

You can use this function:

  • In the Carrier browser of the microbiology work screen.
  • In the general Carrier browser available from the main menu via Start -> Microbiology -> Browse carriers.

For all carriers of one or more microbiology actions

Use the contextual function Set growth of all carriers on Microbiology action.

Tip

You can use this function in the microbiology action browser and in the microbiology work screen.
