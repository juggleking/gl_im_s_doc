Consultation of grafting results in GLIMS

Extended carrier browser

In the options of the microbiology work screen, make sure that the option Show the extended version of the carrier browse is checked.

Grafting results

The carrier browser in the microbiology work screen shows the grafting results sent by WASPLab.

Value of grafting result

Value Background color
? No color
S Green
T,F,U Red

Tip

Possible values for grafting results in different languages:

English Dutch French German
S, Success S, Succes S, Succès E, Erfolgreich
T, Temporary failure T, Tijdelijk mislukt T, Echec temporaire T, Temporärer Fehler
F, Failed M, Mislukt E, Echoué F, Fehlgeschlagen
U, Unfinished O, Onafgewerkt I, Inachevé U, Unfertig

Automatic promotion of microbiology action status

When all carriers have been successfully grafted (grafting result "S"), the status of the microbiology action will automatically be promoted from Grafting to either

  • Reading: when the microbiology action does not have a microscopy result.

    or

  • Microscopy: when the microbiology action has a microscopy result.

Manually adding grafting results in GLIMS

In case of a communication failure between WASPLab and GLIMS, grafting results can be entered manually in GLIMS.

For one or more carriers

Use the contextual function Set grafting on Carrier.

Note

When the Grafting time is not specified, the current date and time will be used.

Tip

You can use this function:

  • In the Carrier browser of the microbiology work screen.
  • In the general Carrier browser available from the main menu via Start -> Microbiology -> Browse carriers.

For all carriers of one or more microbiology actions

Use the contextual function Set grafting of all carriers on Microbiology action.

Note

When the Grafting time is not specified, the current date and time will be used.

Tip

You can use this function in the microbiology action browser and in the microbiology work screen.
