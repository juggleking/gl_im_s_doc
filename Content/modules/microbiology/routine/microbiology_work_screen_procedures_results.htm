Microbiology work screen - Procedures & Results

Procedures

In the top left area of the microbiology work screen, all microbiology procedures are shown that are scheduled on the same specimen.

You can select one individual microbiology procedure or all microbiology procedures. Only the isolations, results, etc. of the selected microbiology procedure will be displayed.

Results of specimen

In the bottom left area of the microbiology screen, related procedures are shown along with a result browser.

Procedures

The procedures shown are:

  1. The procedures linked to the Grafting, Microscopy and Processing properties specified in the Microbiology procedure configuration.

  2. Any other procedure that has outputs for the same specimen. Such procedures (that have no direct link with the microbiology procedure) are marked with an asterisk (*).

Note

  • The procedures are identified by their code. In case of several procedures with the same code, a sequence number will be added to differentiate them.

Results of specimen

The result browser displays the results of the procedure outputs of the selected procedure(s).

Property

Property mnemonic.

U (Unsolicited)

Indicates whether the property has actually been requested or not. If this column contains as asterisk, the result record does not yet exist in the database as it does not have a value yet. These procedure outputs are only displayed in the microbiology screen if they have been requested or/and if their Auto prompt field is enabled.

Status

Result status.

C (Comment)

This column contains an asterisk if an internal or external result comment is available.

Value

Result value.

Note

The results are sorted in the following order:

  1. Code of the associated procedure
  2. Sequence number of the associated procedure output.
