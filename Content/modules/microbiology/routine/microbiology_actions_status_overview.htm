Microbiology - Status overview of microbiology actions

The microbiology work screen allows changing the status of the microbiology action.

Status Code Description
Initial Ini Specimen is not yet available
Grafting Graft Specimen is available. The specimen needs to be grafted onto a carrier.
Microscopy Mic

Grafting is ready. A microscopy is required.

Tip

This step is only done if the microbiology procedure contains a Microscopy property and the option Add microscopy is enabled on in the microbiology request definition.

Reading Read

The analyst is investigating the specimen and entering isolations, isolation tests results, carrier test results, antibiotics, antibiotic results, ...

When a microbiology action is manually set to Reading, the end of incubation time is recalculated.
ReReading RRead If the status is higher than ReadConfirmed and something is changed in the microbiology screen, the action will have to be re-confirmed and re-validated, so the status goes back to ReReading (which is functionally equivalent with Reading). Another case is when the biologist thinks more investigation is required (before it can be validated).
ReadConfirmed ReadC The analyst confirmed the results entered in microbiology screen (confirmation by analyst). This allows to send out a preliminary report.
ReadValidated ReadV The biologist thinks the results in the microbiology screen are OK and validates them (confirmation by biologist). This allows to send out a preliminary report.
Ready Rdy The analyst thinks that no more investigations are required (confirmation by analyst).
Closed Cld

The biologist confirms the previous decision to move the status to Ready (confirmation by biologist).

The microbiology screen is now read-only and can only be changed by reopening the action. This can be done by selecting the context function Reopen on a microbiology action.

Note

When a microbiology action in status Closed is reopened, a justification (reason) will be asked.

A log entry containing this reason will be created. The audit trail of the microbiology action will display an extra entry with the content of this log:

Validated microbiology action modified, reason: [reason]

Discontinued Dsc The investigation has been discontinued without final result.

Notes

  • As long as the microbiology action has not reached the status ReadConfirmed, the status ReReading will not be displayed.
  • If none of the microbiology procedures shown in the microbiology screen contain a Microscopy property, the status Microscopy is not shown among the possible options in the microbiology work screen.
