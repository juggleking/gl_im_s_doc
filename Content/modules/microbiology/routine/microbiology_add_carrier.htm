Microbiology - Adding carriers

Adding a carrier to a microbiology action

The contextual function Add carrier on Microbiology action can be used to add a specific carrier to one or several selected microbiology actions. The last used settings will be remembered for each user.

Options

Medium

Only shows the Mediums that apply to the same Organism type (germ, virus, ...) as the microbiology action(s) that is/are selected.

Also print labels

If enabled, the labels for these new carriers will be printed immediately.

Immediately mark the new carrier(s) as grafted

Allows combining the addition of new carriers with the Confirm grafting function.

Also when there is already a carrier for this medium

Allows ignoring the restriction of having the same medium only once per microbiology action.

Comment

Allows entering the value of the Comment field of each new created carrier.

Tip

Another way of using the Add carrier function is by adding pre-configured menu / ribbon items: this way you can make an item that says e.g. 'Add McConkey carrier and print the labels', which does no longer ask for the medium but immediately does the job.

The function can also be used in automated batch commands.

Note

  1. You cannot add carriers to microbiology actions that are Closed or Discontinued.
  2. When used on actions that are already beyond status Reading or Re-reading, this function will lower the status of those actions automatically down to Re-reading.

Adding a sub carrier to a carrier

In addition to adding sub carriers automatically via the MISPL function Carrier.AddSubCarrier, sub carriers can be added interactively by choosing the contextual menu / ribbon item Add sub carrier from a Carrier.

If you want to use this functionality in a tool with a parameter set, you should use the function AddSubCarrierRunSiteFunction.

Options

The options that can be set are similar to those of the Add carrier function on Microbiology action (see above).

Via MISPL

GLIMS also offers different MISPL functions to automatically add carriers that can be used in different situations:

  • Carrier.AddSubCarrier

  • Isolation.AddSubCarrier

  • Specimen.AddCarriers

  • MicrobiologyAction.Addcarriers
