Microbiology work screen - Introduction

Opening the microbiology work screen

After having requested a microbiology procedure in the order entry screen, a corresponding microbiology action is created. From the microbiology action browser, the microbiology screen can be opened.

When the microbiology screen is displayed, you can use the PageUp and PageDown keys to navigate through the selected actions.

Required privileges

The microbiology screen cannot be started if the user has no privilege for the function MicrobiologyScreen on table MicrobiologyAction.

If the user has the privilege to start the microbiology screen, but does not have the Update privilege on table MicrobiologyAction, the screen will be shown read-only.

Colors

The colors in the microbiology work screen can be changed by setting the color parameters 88 and 89 in the color section of the progress.ini file:

  • color88: background color of the individual browsers in the work screen.
  • color89: line color of the non-selected browsers in the work screen.

Shortcuts

The microbiology work screen allows quick navigation between the different embedded browsers using the following shortcuts:

  • CTRL+1: Carrier browser
  • CTRL+2: Isolation browser
  • CTRL+3: Result browser
  • CTRL+4: Isolation test browser
  • CTRL+5: Antibiogram browser
  • CTRL+6: Carrier test browser (where applicable)

Note

These shortcuts only work via the number row located on top of the keyboard and not via the number pad.

Window elements

The microbiology work screen consists of the following elements:

  • General info & options
  • Carriers & carrier tests
  • Isolations & isolation tests
  • Antibiogram
  • Procedures & results
  • Comment & conclusion
