Microbiology work screen - Carriers & carrier tests

Carriers

Content

Default

By default, the browser will show a carrier for each medium linked to the microbiology procedure. For more information, see Microbiology - Medium usages.

Add carriers

You can add extra carriers manually by:

  • using the Insert key in the carrier browser,
  • using the Add carrier function on a microbiology action,
  • using the Add sub carrier function on a carrier.

Carriers can also be added automatically. For this, the following MISPL functions are available:

  • Carrier.AddSubCarrier

  • Isolation.AddSubCarrier

  • Specimen.AddCarriers

  • MicrobiologyAction.Addcarriers

Display

The carrier browser is only shown if the option Show carrier browse is enabled in the microbiology procedure.

If the microbiology work screen option Show the extended version of the carrier browse is enabled, the carrier browser will show more detailed information such as grafting results and growth results.

Note

If the carrier browser is not shown, the isolation browser will occupy the free space (providing more space for the comment fields).

Sorting

The carriers are sorted by the Seq no specified in the media. Media with the same sequence number (or both unknown) will be sorted alphabetically by Mnemonic.

Confirm grafting

The contextual menu / ribbon item Confirm grafting allows to confirm the grafting process. The confirmation triggers the automatic insertion of the grafting date.

Carrier tests

Content

Default

By default, the browser will show a carrier test for each medium test linked to the medium(s) used. For more information, see Microbiology - Medium test usages.

Add carrier tests

You can add extra carrier tests by:

  • using the Insert key in the carrier browser.
  • using the detailed carrier test window (see below).

Display

Display

The carrier test browser is only shown if the option Show carrier test browse is enabled in the microbiology procedure.

Note

If the microbiology procedure option Show carrier browse is disabled, the carrier test browser will not be shown either, even if the option Show carrier test browse is enabled.

Minimize / maximize

The carrier test window can be minimized / maximized by clicking the arrow icon in the top right corner. The isolation test window will occupy the free space. The isolation test window can also be minimized to provide more space for the carrier test window .

Enter value

Click inside the Value cell and type the value. You can also use the detailed carrier test window (see below).

Tip

Use the contextual function Add new value to enter a new value for the same carrier test.

Detailed carrier test window

The detailed carrier test window can be opened by clicking the spyglass icon in the top right corner of the carrier test browser.

First column

Sequence letter.

Second column

Carrier on which the test is executed.

Comment

Value

Each cell represents a carrier test. The cell displays the actual value.

Tip

The background color of the cell reflects the medium test. A color can be defined for each medium test.

Legend

The available medium tests are listed in the Legends box. You can add a carrier test by:

  • Double clicking a medium test
  • Selecting a medium test and clicking the Add test button.

Add test

Select a medium test in the Legend box and click the Add test button to add an extra carrier test.

Add carrier

See Microbiology - Adding a carrier to microbiology actions.

Notes

  • If there is more than 1 test for a certain carrier on the same date, multiple lines will be shown for the same carrier. They will be sorted by creation time.
  • When there are more date columns than the screen can show, you can scroll the columns (left-right). When opening the screen the most recent date will always be visible.
