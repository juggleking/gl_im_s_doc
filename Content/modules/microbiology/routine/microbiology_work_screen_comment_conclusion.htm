Microbiology work screen - Comment & conclusion

Comment

The Comment field at the bottom of the screen allows specifying comments that are intended for internal use only.

Note

GLIMS can receive a microbiology action comment (that is typically not specific to a carrier, isolation, antibiogram, etc.) from an instrument (WASPLab for instance). The received comment will be appended to the existing comment of the microbiology action, if not yet included. If the maximum length of the microbiology action comment is exceeded because of the append, the first line(s) (being the oldest ones) are trimmed and replaced with "...". This trimming is logged in a log entry with severity Warning of a log type named microbiology action comment.

Conclusion

The Conclusion field allows specifying a conclusion text that can be displayed on reports. This can be achieved by specifying a Conclusion property in the microbiology procedure configuration.

Methods to enter a conclusion text:

  • Manually type a text.
  • Use CTRL-F to lookup a result code and copy the text in the field.
  • Use the Insert key, you can select a text module. To expand the selected text module, use the Icon next to the Conclusion field. Clicking the icon a second time will revert to the short version.

Note

It can happen that several users are editing the Conclusion field simultaneously.

When saving your conclusion and there is a new version of the conclusion, you will be notified and invited to edit the conclusion further. The message shown will include the name of the last person to edit the record before you. This allows the user to contact that person and discuss the situation if necessary. In order not to loose the work you have already done the conclusion on screen will be presented as follows:

<Conclusion version currently saved> + <Your version of the conclusion>.

It is then your responsibility to merge the two conclusions so that they are not contradictory or double.

Review

Review

Allows indicating the date on which carriers, isolations and tests should be reviewed.

The review date field is shown in red if it is today.

Tip

The context function Change review date of microbiology actions can be used on one or more microbiology actions. It asks for the new date (as an absolute or relative date specification) and changes the review date of each of the selected microbiology actions.

End

This field indicates the date of the end of incubation time :

  • It is only displayed if an incubation period has been set.
  • It is read-only.
  • The date is automatically adapted when the end of incubation time is (re)calculated.
  • The date displayed in red if it corresponds to today's date (or earlier) AND the microbiology action is not closed yet.

For more information, please read the chapter Incubation period.
