Layout of tool tips in microbiology screen

When hovering over the object and specimen information fields of the microbiology screen, the content of these fields will be displayed in the tool tip. In order to obtain a clean layout, you should use a non-proportional font for the tool tips:

  1. Go to your PC desktop
  2. Select the contextual menu function Properties
  3. Select the tab page Appearance
  4. In the Item field, select Tooltip
  5. Choose a non-proportional font for your tool tips (e.g. Courier New)

How to interpret the microbiology action severity ?

The handling of "positivity" and "severity" in the microbiology context:

  • The isolation severity is 0 if there is no appraisal or if the appraisal is negative. In the case of positive isolations, the severity formula should return at least 1 (to distinguish from negative isolations).
  • The microbiology action is positive if it has an isolation with a positive appraisal.
  • The microbiology action severity is the maximum of its isolation severities.
  • The microbiology action browser also shows the severity (beside positivity); in red if >0.

Microbiology reporting: how to indicate that an antibiogram is not yet available but will follow

In the report template, you can specify a text module in the Isolation comments field.

Example 

{= IF .AntibiogramResult = ? AND .AntibioticResultIndexed(1) <> ? 
    THEN (chr(10) + "Antibiogram follows") ELSE ""
}

As a result, the preliminary report can already display the detected isolation(s) and if an antibiogram has been requested in the microbiology screen, but no value has been specified yet, the text Antibiogram follows will be printed on the report.
