Microbiology reporting - Isolation tests

GLIMS offers two ways to report isolation tests. We advise not to combine both mechanisms.

Standard mechanism

  1. In the Report template, enable the option Report isolation tests (together with isolation).
  2. For the isolation tests to be reported, enable the option Reportable in the Micro-organism test configuration screen.
  3. The isolation tests are reported between the isolation and its comments.
  4. For each isolation test, a row will be added displaying the description and value.
  5. Isolation tests with an unknown value (?) will never be reported.

Using a dynamic text

In the Report template, the field Isolation comments allows specifying an Isolation-based text module. If specified, the reports generated with this template will automatically add this text module after the appraisal of each isolation in the report.

Example

The text module below will first display the isolation's comment (if any), then the result of the Coagulase test, if performed.

{Name}\t Quantity: {=.Appraisal.Description }

{:

string Coag;

Coag := .Test("Coagulase").Value;

Coag := IF Coag <> ? THEN (chr(10) + RTrim(Coag,?))  ELSE "";
RETURN (
RTrim(FitText(  IfKnownString(chr(10) + RTrim(.ExternalComment,?)) + Coag
, 45, "", "\t", ?), ?)
);
}
