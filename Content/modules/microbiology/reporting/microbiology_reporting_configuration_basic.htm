Microbiology reporting - Basic configuration

Report property

Each microbiology procedure must have a Report property.

  • If not specified, the result of the corresponding microbiology action will not be displayed on the report!
  • The Report property is a dummy property which must have Data type set to Culture.

By including this report property in a property classification, you can determine where the microbiology results should be put on the report.

Microbiology results can be:

  • Isolations
  • A negative answer if there are no reportable isolations.

Conclusion property

The Conclusion property of a microbiology procedure represents the text entered in the Conclusion field of the Microbiology work screen. This conclusion field is used to enter comments which will be displayed on the report.
