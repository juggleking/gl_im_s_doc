Microbiology QC - Microbiology reagent

Content: Introduction - Fields

Introduction

A microbiology QC test refers to a microbiology reagent, which typically is a medium, a micro organism test or an antibiotic.

Microbiology reagents can be defined in GLIMS via Start -> Microbiology -> Configuration -> Microbiology reagents.

Fields

  • Name: free to choose name

  • Type: depending on the reagent type, you will be able to select a specific medium, micro organism test or antibiotic in the fields below.

  • Medium: can be selected if "Type" is set to "Medium"

  • Micro organism test: can be selected if "Type" is set to "Micro organism test"

  • Antibiotic: can be selected if "Type" is set to "Antibiotic"

  • Supplier: organization that supplied the microbiology reagent
