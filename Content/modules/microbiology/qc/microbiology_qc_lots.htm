Microbiology QC - Microbiology QC lot

Content: Introduction - Fields

Introduction

When performing a microbiology QC test for a specific microbiology QC material, GLIMS allows to indicate (automatically or manually) which microbiology QC lot is used. When entering microbiology QC results, GLIMS will automatically lookup a microbiology QC lot (taking into account the start and end date of the microbiology QC lot). You can also manually specify the used QC lot.

Microbiology QC lots can be defined in GLIMS via Start -> Microbiology -> Quality control -> Microbiology QC lots or via the contextual menu function "Microbiology QC lots" on "Microbiology QC material".

Microbiology QC lots can also be automatically created (as inactive) when a product lot is created in the Stock management module.

Fields

  • Material: material used in this QC lot

  • Lot number: external lot number

  • Active: is this microbiology QC lot currently active? If not enabled, microbiology QC results cannot refer to this QC lot. When creating a new microbiology QC lot, the Active field will only be enabled automatically as soon as the Start date has been specified. When clearing the Start date, the Active field will automatically be disabled.

  • Production date

  • Expiration date: if exceeded, microbiology QC results cannot have a value for this QC lot.

  • Start date/End date: only QC lots that are active during this time period will be considered when linking a microbiology QC result to a microbiology QC lot.
