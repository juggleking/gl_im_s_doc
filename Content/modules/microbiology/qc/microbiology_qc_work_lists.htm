Microbiology QC - Work list

Introduction

Once microbiology QC tests are defined and scheduled, you can now enter values for the microbiology QC results.

Microbiology QC result query

Select the main menu option Start > Quality control microbiology > Work list. Before the actual execution of the query, for each selected QC test, a result is created with a schedule date according to the test frequency, unless when such a result already exists.

  • Reagent type

  • Test category prefix

  • Test name prefix

  • QC material

  • Reagent

  • Minimal status / maximal status: this will be fixed to Initial - Available. When viewing microbiology QC results, all options will be available

  • Minimal scheduled date / maximal scheduled date

  • Minimal reading date / maximal reading date

  • Minimal availability date / maximal availability date

  • Minimal initiation time / maximal initiation time

  • Active only: only select results of active microbiology QC tests (this option is only supported when viewing microbiology QC results).

  • Not passed: only show microbiology results with a severity different from "0"

  • Filter: allows to specify a Logical MBQCResult-based site function.

Microbiology QC result browser

The microbiology QC result browser allows entering the result value in the browser itself.

Columns

  1. The first column: will contain an exclamation mark (!) when the scheduled date is lower than the reading date or current date (i.e. the result is overdue).
  2. Test
  3. QC material
  4. QC lot: can be updated in the browser itself.
  5. Reagent
  6. Reagent lot: can be updated in the browser itself.
  7. Expiration date
  8. Reagent (lot) 2
  9. Frequency
  10. Schedule date
  11. Incubation period
  12. Initiation time

  13. Value: allows entering a value directly in the browser. When the microbiology QC test is linked to a choice list, you can hit CTRL-F to select a choice as result value. When entering values for microbiology QC tests that are scheduled in the future, a confirmation will be asked. When entering a result value that causes a severity different than 0, an internal comment has to specified.

  14. Severity: this is either the range severity defined in the microbiology QC test or the severity linked to the chosen choice.

  15. Status

    • Initial: test is scheduled, but the result is not yet available

    • Available: the result is available

    • Confirmed: the result is confirmed (by selecting the contextual menu function "Confirm")

    • Discontinued: the result is discontinued (by selecting the contextual menu function "Discontinue")

  16. Reading
  17. User: user who entered the result
  18. Last 3 columns: graphical display showing the distribution of the severity of the selected results, where a result is represented by a dot and a discontinued result is represented by a cross.
  19. Comment: displays the internal comment

Other features

  • When the result is overdue, the dates are shown in red.

  • When the value has severity > 0, the value is shown in red.

  • Zooming into the result (F6) allows changing the result value, the actually used lot numbers, the reading and schedule date and the internal comment. This also allows storing the action that was undertaken when e.g. an out of bound result was found.

  • When a result is entered with a severity different from 0, an internal comment has to be specified.

  • The "Scheduled" column will have a yellow background in case of "last day" periodical tests.
  • The background colour of rows with discontinued results or results scheduled in the future is grey.

  • The contextual menu allows confirming / discontinuing the result and viewing the microbiology QC result log.

  • Most column headers can be clicked to change the browser sorting.

Microbiology QC result editor

Scheduled

Indicates the date on which the microbiology QC test is scheduled to be executed. This field is filled when the Microbiology QC result record is created (i.e. when scheduling the microbiology QC test), taking into account the microbiology QC test's Frequency or the date specified when using the contextual menu / ribbon item Schedule Ad hoc.

Initiation time

This field is filled when using the contextual menu / ribbon item Set initiation time on a Microbiology QC result and is cleared when using the contextual menu / ribbon item Reset initiation time.

It can be the date and time of grafting.

Reading

This field is filled when using the contextual menu / ribbon item Set initiation time on a Microbiology QC result and is cleared when using the contextual menu / ribbon item Reset initiation time. The Incubation period specified for the Microbiology QC test is taken into account, which means that Reading = Initiation time + Incubation period.

It is updated when the microbiology QC result's Value is entered.

Schedule end date

Indicates the date by which the microbiology QC test should be completed. Only used for cyclic microbiology QC tests.

Reports

The microbiology QC result browser and query can also be used to view or generate reports of microbiology QC results by

  • simply printing the browser

  • by using the "Report" function (in a command)
