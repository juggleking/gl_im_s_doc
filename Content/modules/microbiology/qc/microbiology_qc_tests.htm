Microbiology QC - Microbiology QC test

Content: Introduction - Fields

Introduction

A microbiology QC test contains the definition of a test. It refers to a microbiology QC material (typically the reference strain) and a microbiology reagent (typically a medium, a micro organism test or an antibiotic). It has a frequency that states when the test should be executed.

Fields

  • Name: free to choose name of the test

  • Category: allows grouping tests in a category. The microbiology QC result query allows searching on test category

  • Active: only active microbiology QC tests can be scheduled

  • QC material: the reference strain on which the test will be executed

  • Reagent: the reagent used for executing the test

  • Reagent 2: A microbiology QC test can be linked to a second microbiology reagent.

    As a consequence, a microbiology QC result may also refer to a second microbiology reagent lot. The latter is automatically determined by the first, but it also can be set or modified manually in the microbiology QC result screen.

  • Minimal / Maximal value: normal value range for results of this test. When the result value lies outside this range, the range severity will be applied

  • Range severity: severity that will be applied to results outside the minimal / maximal value range. Note that the severity has to be a number in the range 0 â€“ 3!

  • Frequency: indicates when the test should be executed.

    • Ad hoc: will not be scheduled automatically, but can be scheduled manually.

    • Daily: every day

    • Weekly: once a week, on the first day of the week

    • Monthly: once a month, on the first day of the month

    • Quarterly: once a quarter, on the first day of the quarter

    • Every weekday: each day of the week except Saturdays and Sundays

    • A specific day in the week: e.g. each Friday

    • Semiannual: once every six months

    All MB QC tests can also be scheduled on a specific date in the future.

  • Choice list: allows to limit the allowed results values for this test to the specified choice list. The severity of the selected choice will be applied to the microbiology QC result.

  • Responsible: person responsible for executing this test.

  • Execute on new reagent lot: if enabled, a new test will be scheduled whenever a new reagent lot is available (for the reagent used by this test)

  • Next execution: automatically calculated after scheduling

  • Comment: free comment text

  • Incubation period:

    In some cases, microbiology growth cannot be detected in one day. The "Incubation period" field allows indicating the number of days required before the result can be read.

    Features:

    1. For manually inserted (ad hoc) or scheduled microbiology QC results, the Reading date will be set to "Initiation time + Incubation period".

    2. In the Microbiology QC results browser via work list (Quality control microbiology > Work list), the incubation period is shown in the Incubation period column.

    3. In all microbiology QC result browsers (consultation and work list), the background color of the rows remain grey until the Reading date (initiation time + incubation period) has elapsed. When the Reading date is reached, the background color of the "Value column" will correspond with the color scheme defined for results in status "Expected". When the Reading date is elapsed, the "Value" column has a red background and the results will be automatically discontinued and rescheduled for periodic tests.
