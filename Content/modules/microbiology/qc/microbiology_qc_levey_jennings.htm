Microbiology QC - Levey-Jennings graph

A contextual menu function "Levey-Jennings graph" is available on "Microbiology QC test" and "Microbiology QC result".

The graph is only supported:

  1. for tests which have a numeric value

  2. when a minimal and maximal value is specified in the microbiology QC test;

The value used for target mean, target value and target deviation in the precision and accuracy graphs, is the mean value of the microbiology QC test's minimum and maximum.

Clicking on a data point in the graph will open the corresponding microbiology QC result record.

The same functionality is available as for regular QC population Levey-Jennings graphs.
