Microbiology quality control - Introduction

Content: Concept - Configuration - Functionality

Concept

Glims allows defining, scheduling, registering and monitoring quality control tests for microbiology. Microbiology QC is typically performed to check the quality of:

  • Carriers
  • Isolation tests (e.g. coagulase test)
  • Antibiotics

QC testing is either done on regular intervals, or when the need arises (e.g. a new reagent lot is purchased). QC for microbiology is not linked to a station, as opposed to QC for clinical chemistry.
