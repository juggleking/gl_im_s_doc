Microbiology QC - Test scheduling

Introduction

Once the microbiology QC tests have been configured, they need to be scheduled. When scheduled, microbiology QC results will be created for each microbiology reagent lot of the microbiology reagent used to perform the microbiology QC test. During this process, the start and end date of the microbiology reagent lot are checked. If the next execution of the microbiology QC test does not lie within this date range, the microbiology QC test will not be scheduled.

Note

During scheduling, GLIMS will also discontinue expired results for which no result was available yet. The internal comment of the QC result will contain the text "Autom. discontinued".

A microbiology QC result is considered to be expired when:

  1. its reading date has elapsed (the reading date is only known when the initiation time is set)
  2. the Schedule date + Incubation period has elapsed

Automatically

You can automate the microbiology QC test scheduling using a command and task.

Command definition

  • Table: MBQCTest
  • Query: Default
  • Function: Schedule

Task definition

When defining the task that decides when this command is to be executed, make sure you take the highest microbiology QC test frequency into account. E.g. when a microbiology QC test is to be executed every Thursday, the schedule command will have to run e.g. each Wednesday.

Manually

There are different ways to schedule microbiology QC tests manually:

  • In the microbiology QC work list. Before the actual execution of the query, for each selected QC test, a result is created with a schedule date according to the test frequency, unless when such a result already exists.
  • From the Microbiology QC test browser, use the contextual menu / ribbon item Schedule.
  • You can manually enter (Insert) a new microbiology QC result in the microbiology QC work list.

Ad hoc scheduling

It is possible to schedule a microbiology QC test manually for it to be executed on a given date in the future. To this effect, use the contextual menu / ribbon item Schedule Ad hoc on the MBQC test record.

Note

When the user (re)schedules an ad hoc microbiology QC test, a new QC result is created for the given date. This means that it is possible to schedule an ad hoc microbiology QC test multiple times on the same day.
