Microbiology QC - Microbiology reagent lot

Content: Introduction - Fields - Results measured with microbiology reagent lot - Microbiology reagent lots of a microbiology action

Introduction

When performing a microbiology QC test for a specific microbiology reagent, GLIMS allows to indicate (automatically or manually) which microbiology reagent lot was used. When entering microbiology QC results, GLIMS will automatically lookup a microbiology reagent lot (taking into account the start and end date of the microbiology reagent lot). You can also manually specify the used reagent lot.

Microbiology reagent lots can be defined in GLIMS via Start -> Microbiology -> Configuration -> Microbiology reagent lots or via the contextual menu function "Microbiology reagent lots" on "Microbiology reagent".

Microbiology reagent lots can also be automatically created (as inactive) when a product lot is created in the Stock management module.

Fields

  • Reagent: reagent used in this reagent lot

  • Lot number: external lot number

  • Production date

  • Active: is this microbiology reagent lot currently active ? If not enabled, microbiology QC results cannot refer to this reagent lot.

  • Expiration date: if exceeded, microbiology QC results cannot have a value for this reagent lot.

  • Start date/End date: only microbiology reagent lots that are active during this time period will be considered when linking a microbiology QC result to a microbiology reagent lot

Results measured with microbiology reagent lot

GLIMS offers a screen that shows all results measured with a specific microbiology reagent lot.

  1. Open the list of microbiology reagent lots (Microbiology -> Quality control -> Microbiology reagent lots).
  2. Select a microbiology reagent lot and choose the contextual menu function "Show all... -> Results".

Depending on the type of microbiology reagent, different types of results will be shown:

Microbiology reagent type Results
Medium List of carriers
Micro-organism test List of isolations on which the micro-organism test was performed
Antibiotic List of antibiotic results
Other No results

Remark:

When displaying the results, Glims will take the start and end date of the microbiology reagent lot into account. If these dates are not specified, the creation and expiration date of the microbiology reagent lot will be checked.

Microbiology reagent lots of a microbiology action

The Microbiology action contextual menu function "(Microbiology) reagent lots" displays all microbiology reagent lots AND regular reagent lots associated with a selected microbiology action.

Microbiology reagent lots

The microbiology reagent lots are retrieved via the

  1. carriers
  2. isolation tests
  3. antibiotic results

Note: only the microbiology reagent lots of which the start (or creation) - end (or expiration) date range contains the Carrier.GraftingTime, IsolationTest.AvailabilityTime or AntibioticResult.AvailibityTime will be included. Records without Grafting or Availability time will be ignored.

Regular reagent lots

The regular reagent lots are retrieved via the results of the specimen.

Browser columns

  1. All columns available in the standard microbiology reagent browser (lot number, active, expired, start date, end date and production date).
  2. Reagent name, reagent supplier and reagent type
  3. Measured item
    • For a carrier, this is "Medium.Mnemonic Carrier.InternalId"
    • For an isolation test, this is "Isolation.InternalSequence, MicroorganismTest.Mnemonic, IsolationTest.SeqNo"
    • For an antibiotic result, this is "Isolation.InternalSequence, Antibiotic.Mnemonic"
    • For a result, this is "Property.Mnemonic"
  4. Last known microbiology QC result
  5. All values: the microbiology QC results are sorted by reading date (if unknown, by scheduled date) and availability time. The most recent results are displayed first.

Remarks

  1. The columns "All values" and "Production date" are by default only visible when using the horizontal scroll bar or by resizing the screen.
  2. The content of the browser can be exported to MS Excel or Word using the existing CTRL-O functionality.
  3. The fact that the browser shows both microbiology reagent lots and regular reagent lots (records from different tables) entails some restrictions:
    1. The contextual menu only contains functions related to microbiology reagent lots. However, you can access the reagent lot by using the F6 key and then opening the contextual menu.
    2. CTRL-A will only select the microbiology reagent lots
