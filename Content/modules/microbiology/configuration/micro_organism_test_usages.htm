Microbiology - Micro organism test usage

Introduction

A Microorganism test can be linked to a Microorganism via a Microorganism test usage.

GLIMS allows to link a set of microorganism tests to a microorganism. When requesting isolation tests for a certain isolation in the microbiology screen, you will then no longer have to select the required isolation tests. The associated tests will immediately be displayed.

Fields

Microorganism

Microorganism on which the test is to be executed.

Test

Test that is to be executed on the microorganism.

Eligibility

Isolation-based, Logical-valued MISPL function that can be used to decide whether this usage is applicable or not.

Seq no

Determines the sequence in which the microorganism tests are displayed in the microbiology work screen.

Tip

The sequence in which the microorganism tests are shown on reports depends on the Seq no specified in the Microorganism test.
