Microbiology - Material variables

Introduction

In order to avoid an abundance of microbiology procedures (one for each material subtype), GLIMS allows to subdivide a material according to certain criteria. These are called material variables in GLIMS. Each variable can have a number of choices.

Example

It is possible that you want to make a distinction between urine that was sampled by a catheter or by a punction (sampling method). If so, you can create a material variable Method with Choices such as Catheter, Punction etc.
