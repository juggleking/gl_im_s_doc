Microbiology - Medium tests

Introduction

This table stores definitions of the tests which can be carried out on media to check if they are usable. A Medium test is linked to a Medium via a Medium test usage.

Medium tests can be accessed from the main menu via Start -> Microbiology -> Configuration -> Medium tests.

General

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Free text.

Type

Type of organism for which this test is intended.

List of allowed values

Allows to specify a choice list.

Trigger

Allows to specify a site function that will be executed whenever a carrier test gets a new value.

SOP

Standard Operation Procedure.

Background color

Is used in the carrier test browser of the microbiology work screen.

Disabled

If enabled, this medium test will not be used.

Usage

Embedded browser showing the usages of the medium test.

Assigned billing codes

Embedded browser showing the billing codes assigned to this medium test.
