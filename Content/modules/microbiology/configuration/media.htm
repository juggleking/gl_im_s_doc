Microbiology - Media

Introduction

When presence of health threatening germs is suspected, the germs are transferred to one or more carriers (media) in order to let them incubate. The nutrients used can depend of many factors (e.g. the result of the Gram-coloring, the material, the origin of the material, the patient characteristics,...).

A medium can be linked to a microbiology procedure by defining a medium usage.

Media can be accessed from the main menu via Start -> Microbiology -> Configuration -> Media.

General

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Free text.

Seq no

Carriers will be created and sorted according to this sequence number in the microbiology work screen.

List of allowed values

Allows to specify a choice list. This choice list contains the choices that are the possible values for the Comment field of the Carriers (for this type of Medium) in the microbiology screen.

If a choice list has been specified for the medium, the following features apply:

  • On double-click or CTRL-F, the list of choices will be opened, in order to select the choice you want to enter.
  • If the choice list option Allow free text is not enabled, entering other text than one of the choices will not be possible and will result in an error message.
  • If the choice list option Allow free text is enabled, entering other text than one of the choices will result in a warning message, asking for confirmation.
  • If a choice from the choice list has been entered, you can view this choice from the carrier's contextual menu/ ribbon item Chosen value.

When carrier data changes

Executed when the carrier's Comment, Growth status, Growth result or Grafting result changes.

For instance, you could use the following MISPL functions

  • Carrier.AddIsolation to automatically add a new isolation for the microbiology action.

  • MicrobiologyAction.AddCarriers so that new carriers are automatically added to the microbiology action. This can be useful for instance when the comment of a carrier turns to "P" ('positive'). Indeed, in that case, new tests need to be done, for which new carriers need to be requested.

  • Carrier.AddSubCarrier: see also WASPLab connection - Picking.

  • Isolation.AddSubCarrier
