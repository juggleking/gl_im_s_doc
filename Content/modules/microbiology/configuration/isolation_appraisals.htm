Microbiology - Isolation appraisals

Introduction

When a microorganism is detected, one can not only indicate the presence of an isolation, but also an isolation appraisal which more or less indicates the amount of microorganisms found (in the microbiology work screen).

Example

+ few
++ many
+++ in abundance
- none
Def Default appraisal
(The organism was found)

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Free text description.

Text

Multilingual text that is used on reports.

Tip

Use the placeholder {Name} to include the name of the microorganism.

Positive

Enabled

The appraisal implies the presence (growth) of the microorganism.

Disabled

The microorganism did not grow in the culture.

Note

  • For most appraisals, this should be enabled. Only for those that mean 'absence' or 'no growth', it should be disabled.
  • A microbiology action is considered to be 'positive' as soon as there is at least 1 isolation with an appraisal marked as 'positive'.

Disabled

If checked, this record can no longer be used.
