Microbiology - Antibiotic panels

Introduction

An antibiotic panel groups a number of antibiotics.

Creating an antibiotic panel

Antibiotic panels can be accessed from the main menu via Start -> Microbiology -> Configuration -> Antibiotics -> Antibiotic panels.

Name

Name to visualize the antibiotic panel in GLIMS.

Formula type

Used when entering a numeric (non-RIS) value in the microbiology work screen. The antibiotic conversion rules allow to automatically convert numeric values to RIS values. GLIMS will only convert the value for this formula type to an RIS value.

Note that you can also specify a Formula type for individual antibiotics. The Formula type specified in the Antibiotic panel takes precedence.

Disabled

If checked, the antibiotic panel can no longer be used.

Tip

We strongly recommend not to delete antibiotic panels that are in use, instead you can deactivate them by checking the Disabled field.

Adding antibiotics to an antibiotic panel

You can add antibiotics to and antibiotic panel by:

  1. Using the Insert key in the Members browser in the Antibiotic panel editor.
  2. Selecting an Antibiotic panel and choosing the contextual menu / ribbon item Members.

Panel

Panel to which the antibiotic belongs.

Antibiotic

Antibiotic that belongs to the antibiotic panel.

Seq no

Allows to sort the antibiotics within an antibiotic panel. This determines the sequence in which the antibiotics are displayed in the microbiology work screen.

Tip

For Antibiotic panel members without sequence number (or antibiotics individually added in the microbiology work screen), the Seq no of the antibiotic itself is used.

Linking an antibiotic panel to a microorganism

In many labs, fixed antibiotic panels are used for particular micro-organisms. An antibiotic panel can be linked to a micro-organism via an Antibiotic panel usage. The antibiotic panel usage determines which antibiotic panels to use for a specific microorganism and a specific material.

Tip

In the microbiology work screen, the antibiogram will automatically show all antibiotic panels that are linked to the microorganism of the isolation.

You can link an antibiotic panel to a microorganism by:

  1. Using the Insert key in the Usage browser embedded in the Antibiotic panel editor.
  2. Selecting an Antibiotic panel and choosing the contextual menu / ribbon item Usage.

Microorganism

Microorganism linked to the antibiotic panel.

Material

If specified, the antibiogram in the microbiology work screen will only show the antibiotic panel if the specified material was used.

Panel

Antibiotic panel linked to the microorganism .

Eligibility

Allows to specify a site function that defines extra criteria that should be met before the Antibiotic panel usage applies.
