Microbiology - Medium test usages

Introduction

A Medium test is linked to a Medium via a Medium test usage.

Select a Medium test and choose the contextual menu / ribbon item Usage.

Fields

Medium

The Medium on which the Medium test is to be executed.

Eligibility

Allows to specify a Carrier-based, logical-valued MISPL function. This site function can define additional criteria that should be met before the medium test usage is applied.

Seq no

Used to sort carrier tests in the microbiology work screen.
