Microbiology - Materials

Introduction

Materials are a reference to the input material, i.e. the material of the specimen used for the microbiology procedure. Materials used for microbiology examinations are stored in the Material table.

Tip

Although the same database table is used for storing materials used in clinical chemistry, we advise to use separate materials for microbiology and chemistry examinations.

You can for instance prefix all the microbiology related materials with M_.
