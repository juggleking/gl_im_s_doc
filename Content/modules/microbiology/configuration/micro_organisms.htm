Introduction

Three different types of microorganisms can be distinguished: germs, viruses and parasites. The detection of a microorganism in a specimen will be stored as an Isolation in GLIMS.

Microorganisms can be accessed from the main menu via Start -> Microbiology -> Configuration -> Microorganisms -> Browse.

General

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

Used to visualize the microorganism in the application.

Type

When entering an isolation in the microbiology work screen, the list of available microorganisms will be limited to those of which the type matches the type of the microbiology procedure.

When new isolation is inserted

Initial appraisal

Default value for the isolation appraisal when entering a new isolation in the microbiology work screen.

Mark as reportable

Should this isolation automatically be included on reports? If Mark as reportable is set to Yes and an isolation is added for this microorganism in the microbiology work screen, the isolation will be automatically marked as reportable.

Note

When the isolation's reportability is changed (manually or via an analyzer connection) after its creation and subsequently the isolation's microorganism is replaced by another microorganism, then the isolation's reportability will not be overwritten by the default Mark as reportable setting of the new microorganism.

However, if, for instance, the isolation's microorganism is changed from a microorganism with Mark as reportable set to No to a microorganism with Mark as reportable set to Yes or vice versa and the isolation's reportability was not changed prior to this, then the default Mark as reportable setting of the new microorganism is applied.

Automatically generate antibiogram

If set, an antibiogram will automatically be generated when entering a new isolation in the microbiology work screen.

Report

Description

Multilingual description for the isolation that will be used on reports.

Antibiogram property

By including the specified antibiogram property in a property classification, you can determine where the antibiotic results should be placed on the report.

Note

  • The Antibiogram property must have Data type set to Antibiogram.
  • In general, you can use a single Antibiogram property for all microorganisms.

Archiving

Default lab archive

Statistics

Is group

A microorganism group is a microorganism where the Is group option is enabled. All microorganisms which have this microorganism as parent or grand-parent etc. belong to this group.

Can be used in the classification options of microbiology statistics.

Taxonomic group

Each microorganism can be classified to a certain taxonomic group.

Usage:

  • The taxonomic group can be a criterion in the detection of doubles.
  • The taxonomic group can be used in the microbiology statistics.

Epidemiology

These field are used for the detection of isolation doubles or hospital infections.

Billing

Embedded browser showing the billing codes assigned to this micro organism.
