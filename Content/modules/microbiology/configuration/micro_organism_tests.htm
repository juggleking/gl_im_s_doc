Microbiology - Microorganism tests

Introduction

Microorganism tests can be executed in order to help and determine the true nature of a microorganism. During the actual examination, in the microbiology screen, these micro-organism tests will be represented by isolation tests.

A Microorganism test can be linked to a Microorganism via a Microorganism test usage.

Microorganism tests can be accessed from the main menu via Start -> Microbiology -> Configuration -> Microorganism tests.

Fields

Seq no

Defines the sequence in which microorganism tests appear on reports.

Tip

The sequence in which the microorganism tests are shown in the microbiology work screen depends on the Seq no specified in the Microorganism test usage.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Short name

Name as displayed in GLIMS screens.

Description

Free text description.

Organism type

Type of organism for which this test is intended.

When value entered or changed

Site function that is executed whenever the value for an isolation test is entered or changed. This can e.g. be used to add an isolation sub-test.

Example

If .Value = "POS" THEN

.Isolation.AntibioticResult("AMOX").SetReportable(NO);

ENDIF;

RETURN TRUE;

SOP

Can contain the Standard Operation Procedure.

List of allowed values

Allows to specify a choice list. This choice list defines the allowed values (choices) for the isolation test.

If a choice list is specified, users can double-click (or use CTRL-F) to select a value for the isolation test in the microbiology work screen.

Note

If a choice was entered as isolation test value, you can view this choice via the option Chosen value in the isolation test's contextual menu / ribbon.

Default value

Default value that will be assigned automatically when registering a new isolation test.

Example

This can be used for isolation tests that are only registered in GLIMS if the value is already known and positive. The value is then automatically registered. This can also be useful for billing purposes if tests are only to be billed when the value is positive.

Label to print

Allows to specify an IsolationTest-based text that can be printed as label.

Print label on creation

If enabled, the label specified Label to print will be printed automatically at the creation of an isolation test.

Reportable

Indicates whether the microorganism test should be shown on reports.

Disabled

If checked, this record cannot be used.

Assigned billing codes

Embedded browser showing the billing codes assigned to this micro organism test.
