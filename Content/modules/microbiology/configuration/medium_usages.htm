Microbiology - Medium usages

Introduction

When a microbiology procedure is requested during order entry, a microbiology action is automatically created.

As a medium can be linked to a material (via its medium usage), which itself is linked to a microbiology procedure, the carrier will be known during order entry. This allows to print carrier labels at that moment. The label can be specified in the field Label for printing of the Medium.

Select a Medium and choose the contextual menu / ribbon item Usage.

Fields

Material

Material for which the medium can be used.

Procedure

Microbiology procedure for which the medium can be used.

Parent medium

If specified, this medium usage will only be valid for parent carriers of this medium.

Eligibility

Allows to specify logical-valued, Specimen-based site function. This site function can define additional criteria that should be met before the medium usage is applied.
