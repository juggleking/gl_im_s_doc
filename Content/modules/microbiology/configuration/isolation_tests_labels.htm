Microbiology - Isolation test labels

Isolation test label definition

Isolation test labels can be defined as an IsolationTest-based text in the Label to print field of each Micro-organism test.

Isolation test label printers

Isolation test labels can be printed with the following functions:

  • In the microbiology work screen, at the top of the isolation test browser, a Print button allows printing the label for each isolation test shown.
  • The contextual menu / ribbon item Print label on Isolation test.
  • The contextual menu / ribbon item Print Isolation test labels on Microbiology action

Please check the chapter Set devices for more information.
