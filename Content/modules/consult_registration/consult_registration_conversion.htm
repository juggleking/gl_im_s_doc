Consult registration - Post upgrade conversion procedure

Consult orders that were created in an earlier GLIMS version can be converted to consults:

  1. Create a Property classification containing the properties that were used in the earlier GLIMS version for consult registration.
  2. Create a Consult type referring to this property classification and set the end date to a date in the past.
  3. Start the conversion procedure via Start > Development > Run 4GL, enter ap_ConvertOldConsults.r and click OK.
  4. Select the Consult type to be converted (i.e. the consult type created for this purpose) and click OK.
  5. The orders to be converted will be listed in a .json file, allowing the user to review them before the actual conversion.
  6. When the user confirms that these orders should be converted, the conversion will be done and Consult records will be created for all orders that were selected.
