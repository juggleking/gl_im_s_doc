Consult registration - Introduction

GLIMS allows to register consults in more detail by means of this module. Registering a consult allows you to indicate that the lab has e.g. made a call based on a result or has received a call e.g. for advice.

Note

This module requires a separate license named GLIMS_CONSXT.

This chapter describes the following topics:

  • Configuration
  • Defining a pre-configured tool
  • Registering a consult
  • Searching for consults
  • Statistics
  • MISPL
  • Post upgrade conversion procedure
  • Entity Relationship Diagram (ERD)
