Consult registration - Searching for consults

Introduction

GLIMS offers a query, browser, consult overview screen and an overview of the consults registered for an object.

Query

It is possible to search for registered consults based on a number of criteria via Start > Routine > Search consults.

From - Until

Select consults registered within the specified date/time range.

Consultant

The GLIMS user having registered the consult, i.e. the person in the lab having made a call or having received a call.

Consultant role

The role of the GLIMS user having registered the consult.

Contact person

Allows to select the correspondent with whom the lab has conferred.

Specimen

Select consults registered for the specified specimen.

Object

Select consults registered for the specified object.

Type

Allows to make a selection based on Consult type.

Logs for

Select consults having a log for the specified user.

Status from - to

Select consults with a status within the specified status range.

Filter

A Consult based site function of data type Logical allowing additional filtering.

Browser

Read-only

Double clicking a consult record opens a read only overview of the consult.

Infobox

The infobox below the browser contains the summary of the results of the selected consult record.

Search bar

The search bar below the browser and above the infobox allows searching for consults having a result value that matches the provided keyword(s).

Note

Consider a property 'smoker' with possible values yes/no; using the keyword 'smoker' will not select the consult as only the result value (yes or no) is taken into account.

Overview of a registered consult

Consult overview

Double-clicking / using F6 on a consult in the Consults browser opens the Consult overview in read-only mode.

Edit

The Edit button at the bottom of the Consult overview allows the user to edit the registered consult. However, consults can only be edited if their status allows editing. For more information, see consult statuses.

Overview of the consults registered for an object

GLIMS allows the user to open a browser that contains all consults that have been registered for an object. This browser can be accessed:

  • From an Object via the contextual menu item Show all... > Consults.
  • From a Person, Animal, Lot or QC lot via the contextual menu item Object... > Show all... > Consults.
  • From a Correspondent via the contextual menu item As Person > Object... > Show all... > Consults.
