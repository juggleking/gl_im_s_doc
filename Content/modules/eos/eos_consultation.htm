Electronic order scheduler - Consultation of pending orders

Pending orders will by default not be shown in most order browsers in order not to interfere with the routine lab tasks. There are two ways to consult pending orders:

Pending order consultation

This program can be started from the main menu via Start -> Routine -> Orders -> Pending order consultation.

  1. This program is similar to the existing Order consultation program but will by default only select pending orders (Order status option is set to Pending).
  2. This allows to consult all pending orders for a specific patient.
  3. Double-click an order to start the order activation screen.

Note

A menu option / ribbon item with a tool that has a parameter set can be defined for this function. Use the function PendingOrderConsultation in the tool configuration to this purpose.

Standard order query

The standard order query (Start -> Routine -> Orders -> Query) allows to select all pending orders by setting the Order status from/to options to Pending. This offers an overview of ALL pending orders in GLIMS.

Note

In this case the result status range in the query screen will not be taken into account (as pending orders do not yet have results). The user will be warned about this when searching for pending orders.
