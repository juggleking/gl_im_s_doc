Introduction

Product lots correspond to a supplier identification of products manufactured in one batch, so that they can be expected to be homogeneous.  When ordering products, you can indicate the requested product lot.

Fields

Product lots can be defined from the contextual menu of product specifications.

Lot number

Supplier identification

Production date

Expiration date

Date until which the lot is expected to remain within specification if stored correctly.   The function "Set validity" allows comparing this expiration date with today and will set the product lot to active or inactive.  This process can be automated by using the function in a command.

Active

Disabling a product lot will avoid specifying an expired lot number on an outbound order form.

Product lot query and browser

The Product lot query screen and browser can be opened in several ways:

  • Via Start -> Stock management,
  • Using the contextual menu/ribbon item Lots on a selected product specification,
  • By double-clicking in the Lot field of the Product query.

Menu functions

The product lot contextual menu contains some items that provide a link to related information in the GLIMS database:

Measured results

Shows all results measures with this lot. This menu function is only applicable to product lots that refer a product specification of type "Reagent" (i.e. the product type category).

Measured isolation tests, Measured antibiotic results, Used carriers

These menu functions are only applicable to product lots that refer a product specification of type "MB Reagent" (i.e. the product type category).

QC populations

Shows all QC populations for this lot. This menu function is only applicable to product lots that refer a product specification of type "QC material" (i.e. the product type category).

Creation, activation and update of product lots, (MB) QC lots and (MB) reagent lots

Creation and activation

When a product lot is created (during the configuration of product lots or during product check-in) for a product specification which has an associated

  • QC material, then no QC lot is created. QC lots are only be created upon product check-out (see the illustration in the summary table below). They are created active.
  • Reagent, then a Reagent lot is created automatically (non-microbiology reagent lots can neither be active or non-active because they do not have any Active field).
  • Microbiology reagent, then a Microbiology reagent lot created automatically but not activated. This lot is activated when a product is checked out.
  • Microbiology QC material, then a Microbiology QC lot is created automatically but not activated. This lot is activated when a product is checked out.

Note

If a product lot is created starting from an active (MB) QC lot or MB reagent lot, this (MB) QC or MB reagent lot remains active.

Update

When (MB) QC lots and (MB) reagent lots are created, they are linked to the product lots related to the product specification that refers to an (MB) QC material or an (MB) reagent. When these product lots are updated (e.g. change of product lot number or expiration date), the same changes are only applied to the linked (MB) QC lots and MB reagent lots if they are not active yet. As non-MB reagent lots do not have any Active field, they are always updated if the product lot they are linked to is updated.
