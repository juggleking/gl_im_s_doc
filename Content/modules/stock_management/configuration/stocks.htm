Stock management: Stocks

Introduction

A stock is nothing else than either a logical or physical location where products are stored. Stocks can be structured hierarchically. A stock can be a logical grouping within a laboratory, but it might as well be a building, a room, a refrigerator, a rack... It is the choice of the customer to decide to which detail the stock should be split up.

Stocks can be defined via the main menu through Start -> Stock management -> Configuration -> Stocks.

Please check the chapter Tree editors, for more information on how to work with a tree editor.

Fields

Mnemonic

Easy-to-remember identifier.

Name

Full name of the stock.

Main stock

Hierarchical reference to the main stock.

Delivery address

Reference to a correspondent. Indicates that this stock can be used as delivery address on a purchase order form.
