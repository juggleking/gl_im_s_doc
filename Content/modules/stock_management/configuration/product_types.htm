Stock management : Product types

Contents: Introduction - Fields

Introduction

Product types allow dividing stock products into different types. Examples could be reagents, instrument parts etc. Product types can be defined via the main menu through Start -> Stock management -> Configuration -> Product types.

Fields

Mnemonic

Easy-to-remember identifier.

Name

Name of this product type.

Code

Short code for use where space is limited (e.g. labels).

Category

Provides knowledge for interaction with the Reagent, QCMaterial, BloodProductSpecification, MBReagent and MBQCMaterial tables in GLIMS. This category will be used when defining product specifications. For instance, when defining a new product specification referring to a product type with category Reagent, you will be able to select a record from the Reagent table in GLIMS.
