Stock management: Suppliers

Introduction

Product suppliers must be defined as correspondents of type "Organization". Correspondents will only be considered to be suppliers when a Customer no is specified (Organization editor -> Stock management tab).

Supplier browser

The supplier browser is available from the main menu via Start -> Stock management -> Configuration -> Suppliers. This browser is also used when looking up a supplier in the purchase order entry and product check-in screens. By default, the browser will only show actual suppliers, i.e. organizations with a customer no. You can use the Options button to open the Supplier query.

Supplier query

Mnemonic prefix

Allows searching by organization mnemonic. Partial lookup is supported, i.e. you can also specify the first characters of the mnemonic.

Name pattern

Allows searching by organization name. The entered text should be surrounded by an asterisk (e.g. *bio*).

Has customer no

If enabled, only organizations with a customer no will be shown. If not enabled, ALL organizations will be shown. This allows to lookup an organization that is not yet a marked as supplier (no customer no) and possibly do so.

Customer no

Allows searching by customer no. Partial lookup is supported, i.e. you can also specify the first characters of the customer no.
