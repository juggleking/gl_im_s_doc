Purchase order invoices

Context

As a purchase order is very often delivered through multiple deliveries, the invoice and payment information of a purchase order can be entered per delivery.

Purchase order invoice

In order to support this, GLIMS contains the database table PurchaseOrderInvoice which links to both the PurchaseOrder and the Delivery table. It contains the purchase order invoices of a purchase order per delivery.

It can be accessed from a Purchase order and from a Delivery via the contextual function Show all > Purchase order invoices.

Note

Normally, it will not be necessary to access this table. However, occasional corrections to the invoice and payment related fields are possible. Only users having a role with User type set to System manager or Developer can update these fields.

Embedded browser with invoice and payment related information

An embedded Billing browser (displaying the invoice and payment information of the Purchase order invoice records) is available in the following screens:

  • Purchase order
  • Check in

Note

The Delivery query and the Check purchase order function also take into account purchase order invoices.
