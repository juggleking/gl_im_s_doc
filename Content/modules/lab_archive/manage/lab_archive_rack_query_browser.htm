Lab archive - Rack query & browser

GLIMS also offers a rack query which can be opened from the main menu via Start -> Archive -> Racks.  The following search options are available:

Type

Select only racks of this rack type

Archive (part)

Select only racks belonging to this archive (part)

Expired only

Select racks that contain only expired items.

Code pattern

Select a rack with a given code or all racks corresponding to a wildcard expression.  E.g. 21* will return all racks of which the code starts with '21'.

Usage pattern

Select a rack with a given usage or all racks corresponding to a wildcard expression.
