Lab archive - Storing carriers

The Archive, Rack and ItemStorage tables in GLIMS also allow storing carriers (as used in e.g. microbiology or anatomic pathology).

The main fields to support this are:

  • ItemStorage.Carrier, referring to the carrier being stored
  • ItemStorage.Type, containing either "Specimen", "Carrier" or "Isolation"

contextual menu functions on Carrier:

  • Storage location: opens the item storage linked to the carrier
  • Set storage location: opens a screen that allows creating, modifying or deleting an item storage record for one or multiple carriers.

MISPL functions:

  • Carrier.SetStorage(): similar to the Specimen.SetStorage MISPL function
  • Carrier.GetStorage(): similar to the Specimen.GetStorage MISPL function
  • Specimen.Attribute("CarrierStorageList"): returns a list of storage records for the specimen's carriers (similar to the Specimen.Attribute("StorageList") MISPL function).
