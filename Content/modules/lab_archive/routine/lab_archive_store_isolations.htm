Lab archive - Storing isolations

Manually storing isolations

Manually storing an isolation in a lab archive consists of different steps:

1. Mark the isolation as "To be archived"

Use the Isolation menu function/ribbon item Storage -> Mark as "to be archived".  An ItemStorage record will be created with status To do.

2. Define the location

Use the Isolation menu function Storage -> Define location to specify the Rack and Rack position of the ItemStorage record.  

  1. Its status will be set Assigned and the rack will be assigned to an archive part.  
  2. A message will be shown with the exact position that has been assigned to the isolation within the archive.

The following options are available:

Archive

Archive in which the isolation will be stored.

Auto confirm

If enabled, the status of the ItemStorage record will automatically be promoted to status Stored (see below). The default value of this option is synchronized with the Auto-confirm store setting of the selected archive.

Note:

When pre-configuring this function with a parameter set, this option offers a third value: Unknown (blue background). This value is only supported when the tool option Skip setup is enabled.  In this case, the pre-configured function will also use the default value set in the archive.

Replace all previous storages in same archive

If enabled, existing storage records for this isolation will first be deleted.

Imposed usage

Indicates that the isolation should be stored in a rack with this usage.

Code pattern

May be used to enhance the rack specification.

Reason

3. Confirm archiving

Use the Isolation menu function Storage -> Confirm archiving.  Sets the status of the ItemStorage record to Stored.

Storing isolations via MISPL

MISPL functions on Isolation

SetStorage

Similar to the "SetStorage" method on Specimen, similar to the menu function Define storage location.

GetStorageList

returns a list containing –for each storage record- the archive, the rack and the position in the rack

GetStorage

Similar to the "GetStorage" method on Specimen.  Returns the id of a certain Storage record, specified by its archive mnemonic and usage.

MISPL functions on Specimen

Attribute("IsolationStorageList")

Returns a list containing –for each storage record of each isolation on the specimen-  the archive, the rack and the position in the rack.

Printing storage labels for isolations

Allows to print an isolation storage label based on the "Label text" defined for the archive.
