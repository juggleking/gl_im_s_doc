Lab archive - Overview

Introduction

From an archive, use the menu function / ribbon item "Outline" to consult a hierarchical overview of the archive.

Archive structure

The left part of the overview screen has a tree view representing the lab archive's hierarchical structure.

  1. First level: lab archive
  2. Intermediate levels: archive sub parts
  3. Last level: racks

Features:

  1. Click an item in the tree view to see detailed information (archive, archive part or rack content)
  2. Select an item in the tree view and use the "F6" key to open the related configuration screen
  3. Right-click in the tree view to collapse/expand all levels
  4. The icons in the tree view can be customized in the Archive and Subdivision configuration.

Rack content

The top right part of the overview screen offers a visual representation of the rack and its content.  The layout is defined by the rack type.

A different icon is used for each type of storage item:

Empty position in the rack.
Stored specimen.
Stored isolation.
Stored carrier.

 Click an item to see detailed information (see below).
