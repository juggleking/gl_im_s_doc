Lab archive - Item storages

All items stored in rack

Using the Rack contextual menu function Show all -> Contained items, you can obtain a list of all items stored in the selected rack.  Zooming into this record (F6) displays the Item storage editor, which contains details about the storage of the item.

Item storage query

GLIMS also offers a general item storage query which can be opened from the main menu via Start -> Archive -> Storage query.

The query allows to specify the following search criteria:

Archiving status

The status of the archive items. Allows e.g. to lookup all items for which the storage location has already been defined but not yet confirmed.

Storage type

Only stored specimens, carriers or isolations.

Lab archive

Only items stored in this archive

Archive part

Only items stored in this archive part

Rack

Only items stored in this rack

Rack code pattern

Allows to select a rack with a given code or all racks corresponding to a wildcard expression.  

Example

21* will return all racks of which the code starts with '21'.

Rack usage pattern

Allows to select a rack with a given usage or all racks corresponding to a wildcard expression.  

Specimen, object, material, medium

Storage date range

Expired only

Reason pattern

Allows to query on text used in the Archive (sub)reason (as specified in the specimen scan program)

Removal reason pattern

Not yet supported

Filter

Allows refining the query criteria with an ItemStorage-based site function.
