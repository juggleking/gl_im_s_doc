Theoretical considerations

Example 1

Question: How many employees are there?

Method:

  • Take all employees.
  • Count them.

Result: 1 number.

123

Example 2

Question: What is the average age of the employees?

Method:

  • Take all employees.
  • For each employee, take the age.
  • Calculate the average.

Result: 1 number.

33.7

Example 3

Question: How many employees are there from each municipality?

Method:

  • Take all employees.
  • Group the employees by their municipality.
  • Count the number of employees in each group (each municipality).

Result: A (1-dimensional) table  with 1 number per municipality.

Ghent 55
Antwerp 44
Brussels 24

Example 4

Question: What is the average age of the employees, per municipality ?

Method:

  • Take all employees.
  • Group the employees by their municipality.
  • For each employee, take the age.
  • Calculate the average for each group (each municipality).

Result: A (1-dimensional) table  with 1 number per municipality.

Ghent 28.9
Antwerp 31.7
Brussels 38.1

Example 5

Question: What is the average age of the employees, per municipality and separately for male and female?

Method:

  • Take all employees.
  • Group the employees by their municipality and by male/female.
  • For each employee, take the age.
  • Calculate the average for each separate group.

Result: A 2-dimensional table with 1 number for each municipality and each sex; as soon as we have 2 dimensions, there can be empty cells as well.

  Female Male
Ghent 29.1 27.0

Note

In this example there are no male employees from Brussels, so there is no average age either.

  • Calculate the average for each separate group.

Example 7

Question: How many percent of the employees comes from each municipality?

Method:

  • Take all employees.
  • Group the employees by their municipality.
  • Vount the number of employees in each group (each municipality) and calculate the percentage.

Result: a (1-dimensional) table  with 1 number per municipality

Ghent 43%
Antwerp 38%
Brussels 19%

Example 8

Question: How many percent of the employees is male or female, for each municipality ? (case 1)  And how many percent of the employees comes from each municipality, separately for male and female ? (case 2)

Method:

  • Take all employees.
  • Group the employees by their municipality and male/female.
  • Count the number of employees in each group (each municipality) and,
    • For case 1:  calculate the percentages relative to the total for each row separately (each municipality).
    • For case 2:  calculate the percentages relative to the total for each column separately (male and female).

General principle

Step 0

We start from a certain base table in the data base; this table usually contains an extensive number of records.

Step 1

Based on certain criteria, a selection of records is made  (in the above examples: selection of employees of a company).

Step 2

Group (or classify) the selected records according to one or more specified criteria; the software allows maximum 3 criteria for classification (3 dimensions).

Step 3

For each record a certain value is important  (in the above examples e.g. the age of the employees); this is often just the occurrence of the record (when counts are wanted).

Step 4

Per group of records, a calculation is made on the values of the records; in the above examples e.g. calculating the average of the ages of the employees; often this calculation is just counting the number of records per group.

Complete overview

Conclusion

Each statistics program needs information from the user on each of these steps:

  • Which kind of records is to be treated? in technical terms: which database table is to be used ?  (in the above example a fictive table Employees was used).
  • Which records should be selected?  (usually only records of a specific time period are important).
  • How should the records be grouped?
  • Which numeric value should be taken for each record?
  • Which calculation should be performed on these numeric values?
  • What should be done with the output: should it be printed, exported, displayed on screen, mailed to someone, ...?
