Complaint statistics

Introduction

The complaint statistics module can be started

  • Interactively from the main menu under Correspondents -> Customer care -> Statistics;
  • Using a command (based on table 'gp_site') with function StatisticsCustomerCare.

All features available in the other statistics programs also apply here.  

Note

This module requires the license 'Customer Care Statistics'.

Layout screen

The complaints can be grouped by:

  • Correspondent (the issuer of the complaint)
  • Department
  • Complaint category
  • Correction status
  • Prevention status
