Reagent consumption statistics

Introduction

The stock management statistics are available from the main menu via Start -> Stock management -> Statistics.  They allow e.g. to:

  1. Analyze the reagent consumption per property/station|workplace/reagent.
  2. Analyze the reagent cost per property/station|workplace/reagent.

Layout screen

Beside the row and column classification options, the most important element is the table cell contents.  It allows to either analyze the reagent consumption or cost price.

Consumption

Select the characteristic value "Consumption".  The reagent consumption can be defined in the tab page Consumption of the assessment method.

The statistics rely upon the correct configuration of this consumption data, as well as the data in the Result reagent usage table.  For  more information please check the chapter Reagent  usage configuration.

Cost price

Two characteristic values are available:

  • Cost price (if known): will lookup the cost price based on the link from Reagent lot -> Product lot -> Purchase item.  If this link cannot be found, no price will be returned.
  • Cost price (approximate): will lookup the cost price based on the link from Reagent lot -> Product lot -> Purchase item.  If this link cannot be found, the price will be determined through Reagent lot -> Product lot -> Product specification.

The reagent consumption can be defined in the tab page Consumption of the assessment method.

The statistics rely upon the correct configuration of this consumption data, as well as the data in the Result reagent usage table.  For  more information please check the chapter Reagent  usage configuration.
