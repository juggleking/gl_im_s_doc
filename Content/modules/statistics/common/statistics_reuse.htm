Statistics -> Automatic re-use of previously calculated data in statistics programs

In all statistics programs (also epidemiology) a feature is included which will often save a lot of time. If the user has run the program and wants to re-run the program with slightly different options, the program will try to re-use the data from the previous run as much as possible (as long as the previous run was no more than 12 hours ago).

  • If the selection options are the same, the selection phase will be skipped.
  • Each classification option (rows, columns, ...) which is the same as before will not be recalculated.
  • If only options like "same rows", "same columns" have changed, or the options in the last screen (output options) have changed, both the selection and processing phase (with the progress bar) will be skipped and only the report generation will be re-executed.
