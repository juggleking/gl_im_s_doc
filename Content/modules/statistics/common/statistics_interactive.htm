Statistics -> Interactive startup

Each module has a separate menu item in the main menu. When launching the statistics program, you will always obtain the settings of the previous time you used it.

Setting up all parameters is a process which is managed by a statistics wizard, which guides you through the setup phase. This setup phase consists of 3 individual input screens which are always very similar for all modules, though differ in some detailed aspects typical to the kind of statistics required.

Each module consists of one or multiple possible base tables. Within each module it is very easy to navigate from one base table to another. To this effect, there is a list box in the upper right corners of the first and second input screen which allows switching between the base tables.
