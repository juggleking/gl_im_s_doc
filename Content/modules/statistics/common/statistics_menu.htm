Statistics -> Adding menu items with pre-configured settings 

Just as it is possible to make pre-configured batch commands, it is also possible to make menu-items for statistics with pre-configured settings. This can be useful if the same statistical query - sometimes with slightly different parameters (like e.g. the time period) is often executed manually.

To do this perform following steps:

  • Choose Start -> System management -> Menu -> Edit from the main menu. An outline editor of the main menu will be displayed. Here you can add a menu item wherever you want, using the "insert" key.

  • As function, choose StatisticsProduction", "StatisticsMedical" or "StatisticsFinancial"
