Statistics -> Microbiology statistics

Microbiology statistics give an overview on which microbiology examinations have been performed, which microorganisms have been isolated and which resistance they had against antibiotics. It consists of three parts:

  1. Examination statistics, which are based on the table of microbiology actions
  2. Isolation statistics, which are based on the table of isolations
  3. Resistance statistics, which are based on the table of antibiotic results (susceptibility tests)

They can be started from the main menu under Microbiology -> Statistics. For batch runs the function to use is StatisticsMedical. Switching between isolations/resistance/examinations is easy to do by means of the icon buttons in the upper right corner.
