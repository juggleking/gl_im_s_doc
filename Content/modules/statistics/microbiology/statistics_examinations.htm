Statistics -> Examination Statistics

Examination Statistics give an overview on what microbiology procedures have been performed in the lab. It allows to answer questions like:

  • How many actions were performed for a certain procedure, for patients of a certain ward, for a certain material or certain origin, ...?
  • How many examinations were positive, how many negative ?
  • ...

Only examinations for a specific organism type or organism.

Hospitalized

Select only (or exclude) hospitalized patients (at the date of sampling)

Ward

Select only patients that resided in specific ward(s) (at the date of sampling).  You have the choice between:

  1. entering the mnemonic of 1 single ward
  2. entering a wildcard specification for the ward mnemonic: this is a name containing one or more '*' characters.

    Example

    IC* --> all wards where mnemonic starts with 'IC'
    *01 --> all wards where mnemonic ends with '01'

  3. a comma-separated list of multiple entries of type (a) or (b). 

Example

001,IC*,Urg* --> ward '001' + all wards where name starts with 'IC' or 'URG'

It is possible to lookup a ward in the list by double-clicking in the field or hitting CTRL-F. The ward(s) selected in the lookup browser will simply be added to the existing list.
