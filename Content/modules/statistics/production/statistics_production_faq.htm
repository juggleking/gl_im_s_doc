How can I get statistics on the result assessment count

In the production statistics it is now possible to perform calculations (average, maximum,...) on the result's assessment count. That is the number of times the result has been measured. This can be useful to count also the results that were discontinue-repeated: the assessment count of the last one also takes the previous (discontinued) results into account.

So in the first screen if you do:

  • Sum - Occurrence => returns the number of non-discontinued results
  • Sum - Assessment count => returns the number of all measurements performed to get this result

The assessment count is also available as a classification option for rows, columns, ... This way you can get a table with the assessment count e.g. in the columns, like:

0   1   2   3
_____________________
Na     13 403  54   -
K       - 324  61   2
_____________________

How can I get statistics about the number of B's per ward service code?

  1. For each property the #B must be configured in the field Statistical weight in the property editor
  1. Production statistics screen 1: select a MISPL-filter to select only the results for orders issued by a ward

    RETURN  EnumeratedToString("CorrespondentType", .LastOrder().Issuer.Type)  = "Ward";

  1. Production statistics screen 2:
  • For rows select user-defined and make a MISPL-function returning the ward's service code:

  • RETURN .LastOrder().Issuer.Ward().ServiceCode;

  • For table cell contents select sum + property weight.

How can I make a selection of results on a certain issuer or on several issuers ?

In the first Production statistics screen you can enter a MISPL function in the Filter field.

Example 1

Requested by Issuer 00001 or 00002

MISPL function: RETURN   .LastOrder().Issuer.InternalId = "000001" OR .LastOrder().Issuer.InternalId = "000002" ;

Example 2

Requested by Issuers of which the Id starts with 000

MISPL function: RETURN   Matches(.LastOrder().Issuer.InternalId, "000*", NO );

Example 3

Requested by a Ward

MISPL function: RETURN   EnumeratedToString("CorrespondentType", .LastOrder().Issuer.Type) = "Ward";
