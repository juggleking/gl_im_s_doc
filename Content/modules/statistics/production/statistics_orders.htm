Statistics -> Order statistics

Introduction

Order statistics give an overview on what has been ordered by the issuers. It allows to answer questions like:

  • How many orders have been received from a certain ward, from a certain external lab, ...?
  • For how many patients were orders received ?
  • What was the average time lag between the sampling and the validation of the result ?
  • ...

Characteristic options

Amount excl. VAT / Amount incl. VAT / VAT only

Allow to calculate for each order the total amount (or VAT or both) of all invoices related to the order (except those paid by an internal lab).

Base value

Attention: if some orders are tariffed internally as well (from one internal lab to another) the total base value will be too much (some are counted double). To avoid this it is necessary to select a "tariffing lab" in the selection screen (second screen of the order statistics), where this field is available for this purpose.
