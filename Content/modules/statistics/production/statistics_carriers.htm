Introduction

Beside order, result and pathology statistics, the production statistics module also offers an option for statistics on blocks and slides.  This option will only be available for customers having a license for the 'Anatomic pathology' module.  The blocks and slides module behaves identical to all other statistics modules (See the chapter 'Common features' of the statistics manual for more information).

Material variables (1-4)

The 4 material variable options allow grouping on any (max. 4) of the configured material variables.  The new options are called 'Material variables 1', ... up to 'Material variables 4'.  This implies that:

For materials for which only 1 material variable has been configured, it is quite simple: grouping by 'Material variable 1' is done on the choices that have been made for this variable. The other options ('Material variables 2' etc.) will always return 'Unspecified'.

For materials for which more than one material variable has been configured, the system sorts (by their sequence number) the material variables of that material, and takes the first, second, third or fourth that has been configured.  If it exists, this variable is considered to classify the choices that have been made for the pathology exams.

Example

Material Biopsy

Variable 1 (seq no=10) = Location
Variable 2 (seq no=20) = Method

Classification by 'Material variables 1' will classify on 'locations';
Classification by 'Material variables 2' will classify on 'methods';
Classification by 'Material variables 3' (or 4) will always say 'unspecified'.

Note

Also variables that have been 'de-activated' are considered, so that they can still be evaluated in statistics.  You can always give these 'de-activated' variables a higher sequence number so that the active material variables have priority.
