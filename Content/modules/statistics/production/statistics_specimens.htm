Specimen statistics

Layout screen

Most classification options from the Order statistics are available.  A special option Work status is available as classification.  Possible work statuses are:

Specimen Unavailable

Result Expected

Including microbiology actions!

Result Available

ALL results at least available.

Result Confirmed

ALL results at least confirmed.

Result Validated

ALL results at least Validated.

Specific characteristic values that can be calculated in the cells:

Turnaround time (receipt)

From sampling time till specimen receipt time.

Turnaround time (avail.)

From specimen receipt time till 'all results + microbiology actions available'.

Turnaround time (conf.)

From specimen receipt time till 'all results + microbiology actions confirmed'.

Turnaround time (val.)

From specimen receipt time till 'all results + microbiology actions validated'.

For these turnaround times, you can choose whether the calculation is expressed as a textual value or as a decimal (in min.).

Selection screen

The selection screen offers both order and specimen-based query criteria.

Order-based

  • Order reception time range
  • Department
  • Study
  • Issuer / Issuer group
  • Agent / Agent group

Specimen-based

  • Material / 1st variable
  • Specimen status range
  • Minimal urgency level
  • Optional Specimen-based MISPL filter
