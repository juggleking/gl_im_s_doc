Statistics - Generic statistics

Content: Introduction - Layout screen - Selection screen - Output screen

Introduction

If the built-in statistics modules (Production, Financial, Microbiology and Blood transfusion statistics) do not allow to produce the required statistics, Glims offers a generic statistics program.  Generic statistics allow producing statistics on any table in Glims.

Warning

Generic statistics make extensive use of MISPL functionality and therefore offer less performance than the built-in statistics modules.  We therefore recommend only to use the generic statistics if the built-in modules do not meet your requirements !!

The generic statistics can be started from the main menu via Start -> Statistics -> Generic and can also be scheduled through a command and task.

Layout screen

The layout screen offers the same common functionality shared by all built-in statistics modules.  The main differences are

  1. Base table: you can choose any Glims table as base table for your statistics
  2. The Rows and Columns classifications must be specified using MISPL

Selection screen

For generic statistics, the selection screen allows selecting any existing query available on the chosen base table.  Use the Set up button the configure the parameters for this query.

If there is no (or no appropriate) query for the selected base table, it is also possible to use No query.

Warning

This will select all records from the base table, so use this option carefully !!

Additionally, a MISPL filter can be specified to perform an additional filtering on the selected records. 

Warning

Again this can yield long execution times !!

Output screen

The output screen offers the same common functionality shared by all built-in statistics modules.
