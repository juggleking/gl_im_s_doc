Statistics -> Invoice Summary Statistics

The Invoice Summary Statistics are part of the financial statistics.  Start the financial statistics program (Site -> Statistics -> Financial) and select Invoice summaries from the drop-down menu in the first screen.

Query options

The invoice summary statistics also offer two queries: Financial shipment and Standard invoice summary query.  This will influence the next selection screen (see below).

The selection screen

If query option = Standard invoice summary query

Document no from

This field can only be activated by enabling the adjacent Select option.  If this option is not set, then the selection query will not take into the document number.  If the Select option is set and the document no fields = '?', then all invoice summaries which do not yet have a document number will be selected. This implies that the '?' value in these fields has a double meaning depending on the fact whether the Select option is enabled or not.
