The selection screen

Invoice summary

This field can only be activated by enabling the adjacent 'Select' option.  If this option is not set, then the selection query will not take into account the invoice summaries.  If the 'Select' option is set and the 'Invoice summary' field = '?', then all invoice items which do not yet refer to a invoice summary will be selected. This implies that the '?' value in this field has a double meaning depending on the fact whether the 'Select' option is enabled or not.

Document no from

This field can only be activated by enabling the adjacent 'Select' option.  If this option is not set, then the selection query will not take into the document number.  If the 'Select' option is set and the 'document no' fields = '?', then all invoice items which do not yet have a document number will be selected. This implies that the '?' value in these fields has a double meaning depending on the fact whether the 'Select' option is enabled or not.

Type

Net

Take all the invoice items of invoices of type 'Debit' which have not been credited (InvoiceItem.CreditNote = ?).

Debit

All the invoice items of the 'Debit' invoices.

Credit

All the invoice items of the 'Credit' invoices.
