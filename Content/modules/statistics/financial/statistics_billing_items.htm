Statistics -> Billing item statistics

The billing item statistics module can be reached via the first screen of the invoice statistics by clicking the billing items button on the top of the screen.

The layout screen

Possible characteristics:

  • Amount
  • Base value

 

Possible classification options

All order classifications (issuers, agents, objects, receiving departments, sampling locations, order status, order urgency, encounter types, residence types)

  • Price codes
  • Billing codes
  • Reimbursement classes
  • Billing urgency
  • Receiving labs (i.e. of the order)
  • Tariffing labs
  • Executing labs
  • Dispatching labs

The selection screen

All order selection options (department, issuer(group), agent(group), order status, urgency).

Skip rejected items

When using the Base value or Occurrence option as table cell contents, rejected items (=billing items containing only credited invoice items) will not be counted if this option is enabled. In all other cases, we recommend NOT to enable this option as it might decrease the performance of the billing item statistics.

Executing department (group)

A billing item will be selected if:

Any of these departments:

  • Billing item -> Request charge -> Request -> Result -> Department

Matches any of the following executing departments:

  • The specified "Executing Department"
  • OR any of the executing departments in the specified "Executing department group"

Examples of billing item statistics

Example 1: B-secu and B-hors-nomenclature in France

In France there was a request to get the number of B-secu and B-hors-nomenclature e.g. per issuer per month...

To get this:

  • Choose characteristic Base value.
  • Classify by price code (e.g. in columns) and by issuer (in rows) etc.

Example 2: Amounts per order per panel request (or requests that have been billed together)

Row MISPL:

RETURN datetimetostring(.Order.ReceiptTime, "%x") + "  " +  .Order.InternalId + "  " + ifknownstring(.Order.Issuer.Name);

Column MISPL:

string lst;
string rqbl;
integer i; 
i := 1;
lst := "";
Repeat
  rqbl := .LinkedRequestCharge(i).Request.Requestable.mnemonic;
  if rqbl <> ? then
     if lookup( rqbl, lst, ",") = 0 then
        lst := lst + "," + rqbl;
     endif;
     i := i + 1;
  endif;
until rqbl = ?; 
lst := replace(trim(lst,","), ",", "+"); 
if lst = "" then
   lst := "code: " + .Tariff.BillingCode.code + ifknownstring("(" + .Tariff.BillingCode.ExternalDescription() + ")");
endif;
 
RETURN lst;
