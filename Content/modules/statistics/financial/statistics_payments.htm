Payment statistics

Introduction

The Invoice Payment Statistics are part of the financial statistics.  Start the financial statistics program (Site -> Statistics -> Financial) and select Payments in the top tight corner of the first screen.

Layout screen

The layout screen allows defining the row and column classification for the payment statistics.

Please check the chapter The layout screen for general functionality.

Selection screen

The selection screen allows defining which payments should be included.

Output screen

Please check the chapter The output options screen for general functionality.
