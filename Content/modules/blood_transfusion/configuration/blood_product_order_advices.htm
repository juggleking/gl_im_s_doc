BTM - Blood product order advices

Blood product order advices will be used to manage the blood product stock.

Seq no

Sequence number of the blood product order advice

Product specification

Rhesus

Used to restrict to a certain rhesus.

Department

Department of which the blood product order advice is considered

Minimum number

Indicates the minimum number of blood bags.

Minimum number function

This field returns the minimum number of blood bags. If available and the function returns a value (<> ?), it has priority over the fixed minimal amount.

Maximum number

Indicates the maximum number of blood bags.

Maximum number function

This field returns the maximum number of blood bags. If available and the function returns a value (<> ?), it has priority over the fixed maximal amount.

Added criteria

Allows the definition of additional selection criteria using a Logical, BloodBag-based MISPL expression.
Example

Example

MISPL expression:

RETURN .Antigen("Kell") = NO;

Blood product

Blood group

Used to restrict to a certain blood group.

Typing

Indicates the blood bag typing.

Until blood bag status

The rule only applies to blood bags that have a status not higher than this value. For instance, if this is Initial, reserved or crossmatched bags will not be taken into account for this rule. Usual values for this are:
    • Initial: amount of unreserved bags.
    • Checked: total amount of units in stock, including reserved and crossmatched.
