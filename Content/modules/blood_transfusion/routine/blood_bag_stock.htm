BTM - Blood bag stock management

Blood product order advices

If you have configured the blood product order advices, you can use the stock management program which is available via Start -> Transfusion management -> Order advices.

If the number of available units of a certain blood product is below the minimum number specified in the blood product order advice, the concerned blood product will be displayed in yellow.

Using MISPL

The following MISPL methods are available on the 'BloodProductOrderAdvice' table:

  • .CurrentMinimum(): returns the minimum amount that is currently valid (by field 'MinimumAmount' or overridden by MISPL function 'MinAmountFunction')
  • .CurrentMaximum(): returns the maximum amount that is currently valid (by field 'MaximumAmount' or overridden by mispl function 'MaxAmountFunction')
  • .CurrentAmount(): returns the current number of blood bags that are available in stock and that correspond to the parameters of the order advice record.

These 3 functions can be used to generate documents for ordering new blood bags. On the 'BloodProductOrderAdvice' table, make a command with function 'RunProcedure' and first parameter 'Report'. In the body text module you can use these 3 functions to display, for instance, the current amount and the desired amount.
