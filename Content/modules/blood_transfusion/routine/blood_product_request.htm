BTM - Blood request

The first step in assigning one or more blood bags to a patient is to enter a blood request in GLIMS. At order entry, you can request a blood product by means of its default request definition. You can manually type in the blood request definition or press Ctrl+F and select Blood in the Types section. All blood request definitions will then be listed in a browser.

In practice, a screening test will often immediately be requested in the same order. With the Edit results and specimens option you can then enter the results for the screening test. Check here for a practical example of how to set up a screening procedure in GLIMS.

When the order is saved, one or more blood selections with status Initial will automatically be created for the patient. The sequence number of the blood selections will also be entered. This sequence number will be used for sorting the blood selections on reports.

Note

Note

If the patient needs several blood bags, you can request the blood product request definition several times. To enhance user-friendliness you can group several requests in a panel. Each request will lead to the creation of a blood selection. A compatible blood bag will have to be selected for each blood selection.

The next step is to select a compatible blood bag for that blood selection.

Tips

Tip

In the field Action after order entry of the BTM general settings you can enter a MISPL expression that is to be executed when a blood product is requested.
