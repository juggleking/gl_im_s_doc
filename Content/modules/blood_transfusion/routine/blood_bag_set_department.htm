BTM - Moving blood bags to another department

BloodBag.SetDepartment()

The function Set department is available for blood bags. It can be accessed via the contextual menu / ribbon on a Blood bag and it allows the user to move multiple blood bags from one department (Blood depot) to another.

Tip

The function can be preconfigured so that, for instance, ribbon items can be configured to easily move blood bags to another department.

BloodBag.ChangeDepartment()

The Change department function is not preconfigurable. Its usage is therefore not recommended.
