Logistics manager - Print functionality

"Print document" button

The Prepare package and Collect packages screens contain a Print document button. This button enables you to print all the information displayed on these screens.

Configuration

The print functionality uses Report Builder. For this functionality to work, a Logistics manager printer and Jasper template must be defined.

To define a printer,

  1. go to Start -> Correspondents -> Departments -> By mnemonic
  2. select the department in which the printer will be used, i.e. in which the Prepare package and Collect packages steps will take place.
  3. open the Printers tab page and specify a printer in the Logistics management field.

To define a template,

  1. go to Start -> System management -> Customize -> GLIMS General
  2. open the Reports tab page and specify a Jasper template file in the Logistics report template file field for UNIX and/or Windows.
