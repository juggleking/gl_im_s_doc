Logistics manager - Traceability

Each step of the logistic workflow is automatically registered in the specimen audit trails and package editors.

Specimen audit trail

The audit trail of a specimen indicates when this specimen was packed, collected, delivered and unpacked. It also contains the delivery remarks entered by the courier.

To open the audit trail of a specimen, right-click on this specimen in the Specimen browser -> Log -> Audit trail).

Package editor

The editor of a package indicates when this package was prepared, collected, delivered and unpacked. It also contains the courier identification, the number of the box that was used and the current package status.

To open the editor of a package, look for this package using the Package query screen.
