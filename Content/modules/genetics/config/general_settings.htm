General settings

This chapter presents module management and colour settings with which you can customise the Genetics module.

Module management

The Genetics module can be tailored to the specific needs of a site using the general settings under Start -> System management -> Customize -> GLIMS Genetics. These settings are presented below.

Id MISPL

When a new family is created in the Genetic exam screen, GLIMS creates a unique family internal id and a family name for this family.

Family number

Allows you to insert a GeneticExam-based site function to define the format of family internal ids.

Example

Example

The following MISPL function returns an internal id based on the family creation date and time:

RETURN DateToString(Today(), "%Y%m%d") + TimeToString(Now(), "%H%M%S");

If this field is left empty, family internal ids are composed of the current year followed by a sequence number: [YY]-[SSSS].

The family internal id can be composed of 15 characters maximum.

Family name

Allows you to insert inserting a GeneticExam-based site function to define which name the new family should get.

Example

Example

The following MISPL function returns a name based on the last name of the person linked to the order:

RETURN .Order.Object.Person().LastName;

If this field is left empty, the last name of the order's object for which the family is created is used.

The automatically created internal ids and names can be modified manually in the Number and Name fields of the family screen.

Genetic exam info

Allows you to insert a GeneticExam-based site function in order to customise the content of the Genetic exam screen header.

Genetic result info

Allows you to insert a GeneticExam-based site function in order to customise the content of the Results screen header.

Other

Variant result severity

Allows you to insert a Variant-based site function (with data type Positive integer) that calculates the severity of variant results. The calculation takes place after each change of the variant result if its status is lower than Confirmed.

Webhost URL

Allows you to indicate the basis URL to access the web components through which the Pedigree drawing, Approach plan and Approach visual representations can be displayed. This URL has the following format: "https://servername.domainname:portnumber".

Example

https://glims_srv.mips.be:8443

The port number must correspond to that indicated during the installation of the components. The procedure for this installation is described in the GLIMS System management documentation.

Variant method list

Allows you to define a list of variant methods out of which users can choose in the Sequence variant and Copy number variant editors.

Locus result choice list

Allows you to define a list of values and associated severities that users can select in the Value and Sev columns of the Locus results pane.

Colour schemes

It is possible to define in which colours Approach, Genetic exam, Locus result and Variant result records should be displayed in the browsers and grids of the Genetics module. It is also possible to define colours for records of other tables. The chapter Browsers - Colors presents how to proceed.
