Attachments (configuration)

GLIMS offers the possibility to attach files to families, results, orders and genetic exams. This chapter describes the common characteristics of these attachments and what you need to configure to make use of the attachment feature.

Common characteristics

To be able to open attached .pdf files in GLIMS, a PDF plug-in, such as Adobe Acrobat Reader, should be installed for Internet Explorer.

Attachments are shielded in the same way as the records to which they are attached.

Family and Results attachments

To be able to attach files in the Family and Results screens, no specific configuration is needed.

Note

In this sense, attachments are different from (variant/locus) results, for which it is necessary to define procedure outputs.

Order and genetic exam attachments

Documents to be attached to orders and genetic exams can be of various, site-specific types (request form, consent , clinical information,...). GLIMS offers the possibility to indicate the type of each attached document in the Attachments pane of the Genetic exam screen.

To make use of this feature, the following configuration is necessary:

  1. For each document type, create a site attribute of data type Logical on the gp_Attachment table:

  2. Group these site attributes under a common site attribute category. To do so,

    • create a site attribute category named "GeneticExamAttachmentType",
    • indicate this category in the Category field of each site attribute.
