Variant codes

This chapter describes how to add codes to the Code list in variant editors.

Preliminary configuration

Before being able to add variants codes, you need to

  1. create coding systems (for NCBI and VariantWire, for instance).

  2. add a code set for the Variant table to each coding system.

    The following fields of the Code set editor should be filled in:

    • Table: Variant
    • Unique record
    • Unique value

    • URL: can contain

      • a URL of the website where information on the variant whose code belongs to the code set can be found.

        The URL must end with "{=.Value}". GLIMS replaces this {=.Value} automatically with the code stored for the variant when you double-click on a code in the Codes list of the variant editor or click on the contextual menu/ribbon item Open web page.

      • the mnemonic of a text expression created on the Code table
