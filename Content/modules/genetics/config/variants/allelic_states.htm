Allelic states

Introduction

Allelic states can be chosen out of a list in the CNV and Sequence variant grids of the Results screen. This chapter describes the configuration needed for this choice list to be available.

Configuration

For the allellic state choice list to appear in the variant grids, allelic states choices should be created. This is done in the Allelic states browser and editor, which can be opened via Start -> Genetics -> Configuration -> Variants -> Allelic states -> press Insert in the browser to open the editor.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

Free text (e.g. "Homoplasmic", "Heterozygous")

Enabled

Indicates whether the allelic state can be selected in the variant grids of the Results screen.
