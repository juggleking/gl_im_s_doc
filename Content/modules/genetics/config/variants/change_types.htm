DNA and Protein change types

This chapter presents the DNA change type and Protein change type editors.

These editors can be opened via Start -> Genetics -> Configuration -> Variants -> DNA change types / Protein change types browsers.

DNA change type

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Free text. Click the flag icon to insert translations of the description.

Usable for seq variants

Indicates whether the DNA change type can be used for sequence variants.

Examples

Non-exhaustive list of possible DNA change types for sequence variants:

MnemonicName
>substitution
del

deletion

dupduplication
insinsertion
invinversion
conconversion
fsframe shift
extextension

Usable for CNV

Indicates whether the DNA change type can be used for copy number variants.

Examples

Non-exhaustive list of DNA changes types for CNVs:

  • Complex
  • Insertion
  • Novel sequence insertion
  • Inversion
  • Deletion
  • Duplication
  • Tandem duplication
  • Gain
  • Loss
  • Gain+loss

Active

Indicates whether the DNA change type should appear in the DNA change Type field of variant editors.

Click Variants in the ribbon to open a list of all the sequence and copy number variants linked to the DNA change type.

Protein change type

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Free text. Click the flag icon to insert translations of the description.

Examples

Non-exhaustive list of possible protein changes types:

  • Deletion
  • Frameshift
  • Initiating meth
  • Insertion
  • Missense
  • Nonsense
  • Silent

Active

Indicates whether the protein change type should appear in the amino acid change Type field of sequence variant editors.

Click Sequence variants in the ribbon to open a list of all the sequence variants linked to the protein change type.
