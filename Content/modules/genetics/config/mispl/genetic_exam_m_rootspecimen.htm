GeneticExam.RootSpecimen

The GeneticExam.RootSpecimen MISPL function allows you to retrieve information on the specified root specimen in a given order for the specified material.

GeneticExam.RootSpecimen (Mnemonic MaterialMnemonic)

This can be used, for instance, to check if

  • the specimen of a specific material has been received,
  • the volume or container count meet certain criteria.

Example

RETURN .RootSpecimen("G_Blood").Status ;

Note

If the specified material

  • does not exist, an error message is displayed.
  • exists but is not present in the order, the function returns a question mark.
