Configuration for Genetic exam tariffication

To be able to tariff genetic exams, the following configuration is required:

  • Activate the To charge option on the request definitions of type Genetics.

  • Link billing codes to these request definitions via billing code assignments of type Request.

  • In the field Eligible count of these billing code assignments, you can add site functions using the MISPL functions .GetApproachCount and .GetLocusResultCount. These allow you to charge individual approaches and results a defined number of times. If you want the entire genetic exam to be billed, leave the Eligible count field empty.
