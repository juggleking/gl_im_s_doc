Disorders

Disorders can be created

  • via Start -> Genetics -> Configuration -> Disorders,
  • from a Genetic procedure editor -> double-click in the Suspected disorder field,
  • from the Disorders pane in the Family screen.

In the Disorder browser that opens, press Insert to open a new Disorder editor.

The Disorder ribbon tab provides links to

Disorder genes

Opens the list of genes related to the disorder. Press Insert to link another gene to the disorder.

Genetic procedures

Opens the list of genetic procedures that have the present disorder as Suspected disorder.

For billing and reporting purposes, coding systems and code sets can be defined for disorders.
