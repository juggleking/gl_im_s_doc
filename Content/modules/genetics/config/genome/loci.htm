Locus, locus groups and locus group types

When an HGNC file is imported, Locus records are created automatically. It is also possible to create Locus, Locus group, Locus group member and Locus group type records manually.

The records can be found via Start -> Genetics -> Configuration -> Genome -> Loci (type gene anf gene regions) / Loci (type chromosome and chromosome region).

Locus

Locus editors look as follows:

Name

Name of the locus. This name is unique and created automatically on the basis of the locus type.

Active

This option is activated if the gene's HGNC status in GLIMS is Approved.
This option can be (de)activated manually.
A locus can only be used in routine if it is active.

Type

The type can be Chromosome, Chromosome region, Gene, Gene region, Group. Depending on the type indicated, the fields of the Chromosome, Chromosome region, Gene, ... editors appear underneath.

Locus group

Loci can be grouped, to form gene panels for instance.

The locus group editor can be opened directly from the Start menu (Start -> Genetics -> Configuration -> Genome -> Locus groups). It looks as follows:

Name

Name of the locus group. Must be unique.

Type

Type of the locus group

Members

List of loci that are members of the group:
  • This list may be empty, which allows you to create a locus group for the entire exome, for instance.
  • Group members may themselves be locus groups.

Locus group member

Seq. no.

Defines the sort order of the member with respect to the other members when they are displayed in lists.

Member

Name of the locus that is member of the group

Locus group type

Name

Name of the group type: "Gene panel", "Probe", "Primer",...
