Genes, gene regions and gene transcripts

When an HGNC file is imported, the Gene and Gene transcript records are created automatically. GLIMS also allows you to

  • create Gene, Gene transcript and Gene region records,
  • link Gene records to Disorder records.

The genome-related records can be found via Start -> Genetics -> Configuration -> Genome.

Note

Information regarding gene codes is provided in another chapter.

Genes

Imported data

The HGNC file contains a "location" field that indicates where the gene is situated: chromosome numbers and regions. This field should only contain a single location.

Mnemonic

By default, the mnemonic is identical to Approved symbol. It can however be modified manually. After a re-import of the HGNC file, only the Approved symbol is updated, but not the mnemonic in GLIMS.

HGNC id

Unique id created by the HGNC for every approved symbol.

Approved symbol

Official symbol of the gene. Approved by the HGNC and publicly available.

Name

Official name of the gene. Approved by the HGNC and publicly available.

HGNC status

HGNC status of the gene symbol. Possible values:
  • Approved:
    • the gene symbol was imported for the first time and its status in the import file was "Approved", or
    • the gene symbol was re-imported and its status in the import file was different from "Entry withdrawn" and "Symbol withdrawn".
  • Entry withdrawn: the gene symbol used to exist in GLIMS with status Approved but in the last import file, its status was "Entry withdrawn".
  • Symbol withdrawn: the gene symbol used to exist in GLIMS with status Approved but in the last import file, its status was "Symbol withdrawn".

Chromosome

Chromosome number

Chromosome region

Chromosome region

Gene family

Name of the gene family or group in which the gene might have been classified.

Alias symbols

List of other symbols used to refer to the gene.
These synonymic symbols may come from the literature, other databases or represent membership in a gene family.

Previous symbols

Symbols of the gene that had been approved by the HGNC.

Date approved

Date on which the gene symbol was officially approved

Transcripts

Embedded list of the gene's transcripts

Locus name - Active

Please check the chapter on loci.

If the location in the import file has been modified since the last import:

  • the locus of the gene record with the old location is set to Inactive, and
  • a new gene record with the updated location is created

If a gene had two locations in the previous import file and now has a single location:

  • the existing gene record is updated, and
  • the locus of the gene record with the old location is set to Inactive.

Note

If the location field of the HGNC file contains more than one location, but the locations are on different chromosomes, GLIMS will create several gene records during the import.

If there is more than one location on the same chromosome for a gene, the corresponding gene record has to be created manually.

Non-imported data

The HGNC files contain other data that is not imported in GLIMS:

  • Genes located on a chromosome that does not exist in GLIMS.
  • Genes with more than one location on the same chromosome.
  • Genes that do not exist in GLIMS yet and whose status in the import file is different from "Approved".
  • Genes with status "Approved" in the import file but for which no location is specified.
  • Genes with location "Reserved" and "Unplaced".
  • Locus group, locus type, alias name and previous name.

Linking genes with disorders

It is possible to establish a link between disorders and genes. To do so, use the contextual ribbon/menu function Disorder Genes on a Gene record. Press Insert to add a disorder to the list that opens.

Gene transcripts

All the transcripts of the imported genes are imported in GLIMS.

Is default

Indicates whether the transcript is the default transcript for the gene in the laboratory.
There can only be one default gene transcript per gene.
This option is activated automatically if the gene has a single transcript in GLIMS.

Note

If a gene transcript was ever imported from a previous file but is now absent in the newest import file, it is not deleted automatically.

Once a gene transcript is marked as default, it will remain the default transcript.

Gene regions

A gene region is a section of a gene.

Gene region records can be opened by using the contextual menu/ribbon item Gene regions on a Gene record.

Region type

The possible values are
  • Exon: any part of a gene that will encode a part of the final mature RNA produced by that gene after introns have been removed by RNA splicing.
  • Intron:any nucleotide sequence within a gene that is removed by RNA splicing during maturation of the final RNA product.
  • 3'UTR: untranslated region of messenger RNA that immediately follows the translation termination codon.
  • 5'UTR: untranslated region of messenger RNA that is directly upstream from the initiation codon.
  • Amplicon: piece of DNA or RNA that is the source and/or product of amplification or replication events.

Number

Name or number of the gene region. Example: 1A.
