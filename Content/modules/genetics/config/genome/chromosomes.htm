Chromosomes and chromosome regions

Chromosome records can be imported using the CSV file provided by MIPS or created manually. Chromosome region records are created automatically when an HGNC file is imported. They can also be created manually.

The records can be found via Start -> Genetics -> Configuration -> Genome.

Chromosomes

The Chromosome editor looks as follows:

Seq. no.

Number to sort the chromosome in lists.

The sequence numbers should preferably be attributed in the following way:

  • 1-22: chromosomes 01 to 22
  • 23: chromosome X
  • 24: chromosome Y
  • 25: chromosome MT

Number

Number of the chromosome, e.g "02", "14", "X", "Y","MT". We advise you to
  • add a "0" before the number of chromosomes 1 to 9: "02", "05",...
  • use capital letters for "X", "Y","MT".

Abbreviation

Short name used to refer to the chromosome in the user interface screens: e.g. "chr 01". Maximum 20 characters.

Name

Full name used to refer to the chromosome on reports: e.g. "Chromosome 01". Maximum 50 characters.

Chromosome regions

The Chromosome region editor looks as follows:

Chromosome

Chromosome number

Region

Name of the region on the chromosome, e.g. "q11.1".
The combination 'Chromosome number + region' must be unique.
