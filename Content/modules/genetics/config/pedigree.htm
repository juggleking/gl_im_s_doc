Pedigree

In order for family pedigrees to be displayed in the Family screen, you need to make sure that

  • users have the GLIMS_GENP licence.
  • the Webhost URL is indicated.
  • if the Family screen should be opened from a genetic exam, the Support family option of the genetic procedure linked to the genetic exam is set to Yes.

  • the users' Internet Explorer version is IE 11.

    Note

    The pedigree drawing feature makes use of Internet Explorer. If the Internet Explorer version is not IE 11 yet,

    • launch the register-editor (regedit.exe),
    • go to HKEY_CURRENT_USER (or MACHINE) -> Software -> Microsoft -> Internet Explorer -> Main -> FeatureControl -> FEATURE_BROWSER_EMULATION.
    • create a new DWORD key with name "prowin32.exe" and data (decimal value) "11000" (which means IE 11).

    If the pedigree is not displayed, despite IE 11 being used, the web page might be executed in compatibility mode. To deactivate this mode, click the Settings icon -> Compatibility View settings -> deactivate the option Display intranet sites in Compatibility View:
