Reporting of genetic exam results

This chapter describes

  • the concept behind the reporting of genetic exam results
  • the status and values of report results.

The configuration needed for genetic exam results to appear on reports is described here.

Concept

Genetic result reports are generated as XML files with Report Builder. Using this XML file in combination with an iReport template file, Report Builder generates a PDF report.

Status and values of report results

Status and values of the report result:
Result statusResult value  
Initial?

No approach plan has been selected for the genetic exam for which the result has been created, so there is no approach yet.

 
Initial?One or more approach plans have been selected, so one or more approaches have been created for the genetic exam. However, none of the approaches have been started yet. Their status is Initial. 
Initial?At least one of the approaches is running but none is finished. 
Availableapproach mnemonic of the finished approachesAt least one approach is finished. The others are initial or running. 
Validatedapproach mnemonic of all the approachesAt least one approach is finished, all the others are either finished or discontinued.
