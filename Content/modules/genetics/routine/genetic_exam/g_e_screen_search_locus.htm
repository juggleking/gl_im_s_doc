Add loci

The Add loci screen makes it possible to search for loci and locus groups related to a specific chromosome, chromosome region, gene or transcript.

It can be opened from

  • the Loci to be tested pane in the Approach plan editor,
  • the approach request list in the Genetic exam screen,
  • the editors of sequence and copy number variants.

It looks as follows:

Chromosome

Select a chromosome.
The search function returns the chromosome itself as well as all the genes and chromosome regions linked to this chromosome.

Chromosome region

Select a chromosome region.
The search function returns the chromosome region itself as well as all the genes linked to this region.

Gene

Select a gene.
The search function returns the gene itself and the gene regions linked to this gene.

Transcript

Type (the first letter of) a transcript name. The search function returns all the genes that have a transcript whose name starts with the entered letter(s).
Only editable if the Gene field is correctly filled in.

Filter on gene region

This option can only be activated if the Gene field is filled in.

If this option is activated, the search function only returns gene regions (conversely, if this option is not activated, no gene regions are returned).

The returned gene regions are those linked to the specified gene. If the Filter on group option is also activated, the search function only returns locus groups that contain a gene region on the specified gene.

The following criteria allow you to filter the list further:

  • From: Mandatory field. If the field Till is

    • empty, the search function only returns the gene region corresponding to the specified number;
    • filled, the search function returns the gene regions specified in the From and Till fields as well as all the regions inbetween.
  • Till: optional
  • Type: type of the gene regions that should be returned.

Filter on group

If this option is activated, the search function only returns locus groups (conversely, if this option is not activated, no locus groups are returned).
Allows you to search for locus groups for which the other specified criteria hold.

Example

If Chromosome is set to "17", the search function returns all the locus groups of which chromosome 17 is a member.

The Type field allows you to narrow down the search to a specific locus group type.

Search results

The loci that fulfill the specified criteria are listed under the query fields. They are sorted per locus type in different tab pages.

To select a locus, either double click it or click on the green arrow in the Add column. The chosen locus now appears in the list of selected loci on the right-hand side.
