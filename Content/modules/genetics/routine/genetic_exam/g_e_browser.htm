Genetic exams browser

Introduction

The Genetic exams browser can be opened

  • via Start -> Genetics -> Genetic exam query
  • by right-clicking on an order -> Requests -> Genetic exams.

This browser is read-only. To modify a genetic exam, double-click it in the browser: this opens the Genetic exam screen.

Browser columns

Paperclip icon

A paperclip icon is displayed if there are attachments on the genetic exam.

!

An exclamation mark is displayed and the column cell is shown in red color if the urgency of the genetic exam is higher than the lowest urgency defined.

Object

Object (typically a patient) of the order from which the genetic exam has been created.

Note

Obviously, if the Object is added or modified after order creation, or merged with another Object, the Object of the Genetic exam is updated accordingly.

Order lowest object time

The lowest of all reference times of all requested codes in the order.

Desired date

 

Genetic procedure

Genetic procedure whose request in order entry lead to the creation of the genetic exam

Status

Status of the genetic exam

Ini Initial

The genetic exam has been created but it does not contain any approach (i.e. no approach plan has been selected yet to perform the genetic exam).
Run Running An approach plan has been selected to perform the genetic exam. The genetic exam has at least one approach (in status Initial, Running or Error).
Cnf Confirmed All the approaches of the genetic exam have status Finished or Discontinued.
Val Validated All the results have been validated using the Validate all button in the Results screen.
Dsc Discontinued The genetic exam itself, the requested code of the genetic exam or the order the genetic exam is part of has been discontinued (manually or by the Order.Discontinue function).

End date

End date of the genetic exam. It varies with the status of the genetic exam.

Status of genetic exam End date
Initial Empty

Running

 Latest (expected) end date of the scheduled, non-finished approaches contained in the genetic exam
Confirmed, Validated Latest end date of the (finished or discontinued) approaches contained in the genetic exam
Discontinued Date on which the genetic exam was discontinued

Responsible department

Identical to the department of the genetic procedure on which the genetic exam is based.

Severity

Severity of the genetic exam. It corresponds to the highest severity of all confirmed or validated
  • standard results (norm severities),
  • locus results, and
  • variants results

of the genetic exam. This severity indication is updated each time the status of an activity in an approach of the genetic exam turns to Finished.

Notes

  • The manual severities are taken into account.

  • The severities of

    • results that are not linked to an approach,
    • the choices defined in the List details type choice list

    are not taken into account.

  • When a result is discontinued in a finished activity, the status of the activity remains Finished. As a consequence, the severity indication is not updated. This means that, if the severity of the discontinued result was the highest severity of all the results, the genetic exam severity remains that of the discontinued result.
