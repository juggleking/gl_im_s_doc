Add variants

The Add variants screen makes it possible to search for or create copy number variants and sequence variants.

It can be opened from the

  • approach request list in the Genetic exam screen,
  • locus results pane in the Results screen.

It looks as follows:

Search criteria

First indicate whether you are searching for a sequence variant or a copy number variant using the radio buttons on the left-hand side. Then you can fill in other search criteria:

Locus

Copy number variants linked to chromosomes or chromosome regions:
  • A search for CNVs on a chromosome will also return the variants on the chromosome regions of that chromosome.
  • A search for CNVs on a chromosome region will return the variants of the chromosome regions, but not of the chromosome.

CNVs can be linked to genes or gene regions as well:

  • A search for CNVs on a gene will also return the variants on the gene regions of that gene.
  • A search for CNVs on a gene region will return the variants of the gene regions, but not of the gene.

Sequence variants can only be linked to genes or gene regions:

  • A search for sequence variants on a gene will also return the variants on the gene regions of that gene.
  • A search for sequence variants on a gene region will return the variants of the gene regions, but not of the gene.

Transcript

If Sequence variant is selected,
  • the query will return all the sequence variants that have a transcript that matches the one indicated here.
  • and if a locus (gene or gene region) is indicated, the selectable transcripts are those linked to this locus.
If Copy number variant is selected, the query will return all the copy number variants located on a gene whose default transcript matches the one indicated here.

Genomic build

Search for sequence variants or copy number variants linked to a specific genomic build.

DNA change type

Search for sequence variants or copy number variants linked to a specific DNA change type.

Clinical classification

 

Sequence variant

cDNA change

Search for sequence variants linked to a specific cDNA change.

Amino acid change

Search for sequence variants linked to a specific amino acid change.

LRG

Search for sequence variants linked to a specific locus reference genomic id.

Copy number variant

From - Till

Range of positions in which the variant's start and/or stop positions should be situated.
If the From field is not filled in, its value is considered to be "0". If the Till field is not filled in, the value is considered to be "infinite".

Search results

The variants that fulfill all the specified criteria are listed under the query fields.

You can filter the variants using the filter icons in the column headers.

To select a variant, either double click it or click on the green arrow in the Add column. The chosen variant now appears in the list of selected variants on the right-hand side.

Create a variant

If after at least one search action you have not found the variant you are looking for, you can create it by clicking on the Create button and filling in the fields of the variant editor that opens.
