Introduction

Genetic exams are orders specifically created for genetic examinations.

The creation of genetic exams starts in the GLIMS general Order entry screen. In fact, a genetic exam is automatically created when a genetic procedure is requested during order entry. In other words, a genetic exam is an instantiation of a genetic procedure requested in routine. Once a genetic exam is created, it must be elaborated further in a genetics specific screen, the Genetic exam screen.

A list of existing genetic exams can be obtained using the Genetic exam query.

The results of genetic exams are displayed in a Results screen. They can be edited in this Results screen or in the advanced Work list editor.

Notes

  • As a genetic exam is created for each requested procedure, there can be several genetic exams for a single order. However, if the same genetic procedure is requested several times in the same order, only one genetic exam is created.
  • Just like non-genetic orders, genetic exams can be discontinued (in the Genetic exams browser, select the genetic exam to be discontinued and use the ribbon/menu item Discontinue genetic exam, or discontinue the requested code linked to the genetic exam).
