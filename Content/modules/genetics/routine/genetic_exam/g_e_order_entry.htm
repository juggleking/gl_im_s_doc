Order entry

The creation of a genetic exam starts in the GLIMS general Order entry screen:

  1. Double-click in the Request field.

  2. A list of request codes appears. The button Options allows you to filter the list so that it only displays request codes of type Genetics (i.e. request codes linked to request definitions on which the Genetic procedure field is filled in).

    Select one or more request codes of type Genetics in the list of request codes.

  3. Once a request code of type Geneticshas been selected, it is added to the order and a genetic exam button appears:

    Notes

    • The genetic exam button in the order entry screen is
      • not displayed if the order does not contain any request code of type Genetics.
      • grayed out if the opened order contains a request code of type Genetics but the user does not have the required licence, Genetics Extended Laboratory (GLIMS_GENX_LAB).
    • If a request code for a genetic procedure is requested together with a request code for a specimen, the specimen will NOT be imposed on the genetic procedure. This is different from other work flows in GLIMS where the specimen is automatically linked to the procedure, so that the procedure has to be executed on this specimen.

  4. Click this button. If

    • the request codes of type Genetics added to the order are all linked to the same genetic procedure, the order is saved automatically and the Genetic exam screen opens:

    • the request codes of type Genetics added to the order are linked to different genetic procedures, the genetic exam browser opens. Select a genetic exam for which you want to insert more information:

      The mnemonic and descriptions are those of the genetic procedures associated with the genetic exams. Discontinued genetic exams are not listed.

      Once you have selected a genetic exam, the order is saved automatically and the Genetic exam screen opens.

Note

Genetic procedures can only be executed on specific materials.
