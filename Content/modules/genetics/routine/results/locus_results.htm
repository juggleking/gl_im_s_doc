Locus results

Detailed information on a locus result is available in the Locus results browser and editor: right-click on a Locus record -> Show all -> Locus results.

Most of the fields of the Locus result editor correspond to the columns of the Locus result pane of the Results screen. Additional fields are

Department

Department in which the result was entered

Object

Person on whom the locus has been tested.

Entered on

Date on which the result was entered, and therefore on which its status turned to Available.
This date is updated when a confirmed or validated result is modified.

Confirmed on

Date on which the result was confirmed.

Notes

  • When a result with status Available is directly validated, the confirmation date is set to the validation date.

  • When a confirmed or validated result is

    • discontinued, the confirmation date is not cleared.
    • updated, the confirmation time is cleared.
  • This field can be used to run a (daily) command based on confirmation time, instead of using a confirmation trigger.

Approach request

Approach request through which the locus test was requested
