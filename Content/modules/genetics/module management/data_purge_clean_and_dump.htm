Data purge, clean, and dump

The entire genetics routine data can be purged, cleaned and dumped using the following common GLIMS tools:

  • Order purge
  • Order dump

  • Purge old routine data:

    • regarding genetic exams, it is possible to limit the purge action to those that are older than a defined number of months.
    • actions linked to approaches are not purged
  • Clean routine data
