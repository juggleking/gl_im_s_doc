Overview of configuration and routine

This chapter offers an overview of

  • all the elements that should be configured to use the Genetics module,
  • the routine usage of this module and life cycle of genetic exams.

Routine usage and life cycle of genetic exams

1. Create a genetic exam

Genetic exams can be created

  • using the common order entry screen. However, the following differs from the common order entry:

    • You need to request a genetic procedure

      • using a request form, or
      • entering or selecting a request code.
    • There is a special button to open the genetic exam screen.
    • There is an additional screen that contains information regarding the collection of the specimen.
  • via electronic order entry (HL7, LDT, ...). The message must contain a request code for a genetic procedure.

2. Open the Genetic exam screen and add genetics information

  • Specimens

    • Add received specimens.

    • Add information on these specimens, using the Specimen info screen:

      • received quality and quantity, container count, container type, sampler, sampling location, etc.
      • results (concentration, ratio’s,...), material variables,
      • in prenatal contexts: information regarding the foetus
    • or choose another specimen to be reused.
  • Enter a Desired date.
  • Choose an existing family for the exam, or create a new family.

  • Open the Family screen to

    • draw the pedigree.
    • link the family members to existing or new persons in GLIMS.
    • indicate disorders, to which family members they apply and in which clinical status (via Conditions).
    • provide information about the family's history.
    • attach documents.
  • Choose an approach plan manually or automatically.
  • Add loci to be tested using the Locus search screen.

3. Use the Genetic exam query browser to search for specific genetic exams

Use the Genetic exam query browser to get a list of genetic exams meeting criteria of your choice.

4. Use the Results screen to enter results

  • Confirm specimen outputs.
  • Enter values for normal results in the Results and Interpretations panes.
  • Enter values for locus results.
  • Add new locus results or variant and loci results on existing locus results.
  • Enter values copy number variant results and sequence variant results.
  • Add variant results using the variant search screen.
  • Mark a variant result to be retested and perform the re-testing in a following activity.

5. Use the Advanced work list editor to enter results

  • Perform the same actions in the Advanced work list editor as in the Results screen.
  • Perform these actions on several specimens and results at once for different exams and patients.

6. Generate reports

Generate a PDF report for a genetic exam.

7. Bill

Tariff a genetic exam.

8. Perform statistics

Create statistics on genetic exams, locus results and variant results, with the GLIMS Statistics module.
