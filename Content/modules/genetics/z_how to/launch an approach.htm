How to launch an approach

Say we want to test if a patient has the Down syndrome. The test will take place in department DEPN and start from a blood specimen. The following example illustrates which configuration and routine steps are required to launch an approach. This approach starts with extraction of DNA from the blood received.

Configuration

There must exist

  • a material, e.g "G_Blood_AR"
  • a genetic procedure "G_DownSyndrome_AR" with
    • responsible department: "DEPN"
    • report property of type Genetics
    • materials that can be used: "G_Blood_AR"
  • an approach plan "AP_DownSyndrome"
    • where the Start activity is directly followed with an activity "Extract_DNA" of type Procedure. This activity of type Procedure has
      • department: "DEPN"
      • input material: "G_Blood_AR"
      • output material: "G_DNA_AR"
    • the approach plan usage of which
      • has department "DEPN"
      • has genetic procedure "G_DownSyndrome_AR"
      • has material "G_Blood_AR"
      • is enabled

Routine

  • Create an order (genetic exam) for the patient in which G_Blood_AR and G_DownSyndrome_AR have been requested
  • Set the status of the specimen G_Blood_AR to Available

At this point, the Genetics exam screen shows

  • the available specimen (input of the procedure G_DownSyndrome),
  • the expected specimen (output of G_DownSyndrome)
  • the running approach created on the basis of the approach plan AP_DownSyndrome.
