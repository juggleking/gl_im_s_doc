User interface

Modularity

The screens of the Genetics module are modular:

  • the screens can be moved and made smaller or larger
  • the accordion panes can be expanded and collapsed

  • the size of the frames within the screens can be modified

Layout preferences

The chosen layout is saved for each screen in the user's preferences (Context = GeneticsFamily/GeneticsOrder/...).

Features of the grids

In the Genetics module, several panes contain dynamic grids. On the following screenshot, you can see the Variants grid of the Family screen.

These grids are called dynamic grids because they allow you to choose which records they should display. You can

  • filter the results using the filter icons in the column headers.

    Note

    When a filter criterion is applied on a column, the filter icon is displayed in blue.

    To see which filter criteria are currently applied, press CTRL+F.

    To clear all the filter criteria, press CTRL+SHIFT+F.

  • move columns to a new position by dragging the column headers, except the dynamically configured columns in the Variant results and Locus results grids of the genetic results / advanced work list screens. When moving a column to a position after a dynamic column, the column is placed before the fixed columns after reopening the screen.
  • apply a change to several selected results simultaneously.
  • use the Column chooser icon for a pop-up to display or hide specific columns in the grid.

Export grids to Excel

A functionality is available to export any grid inside the Genetics module to Microsoft Excel. Via the key combination CTRL+E, the selected grid is exported into an Excel file.

Tip

Currently, this functionality is only available for the grids displayed in the Genetics module screens.
