Business Activity Monitor - TAT monitor

Introduction

As explained in the chapter property TAT, it is possible to define time frames within which results should be confirmed and validated. These time frames are called "turnaround time (TAT) limits".

The TAT monitor offers an overview of the met and exceeded TAT limits for today's orders. More precisely, the monitor shows, for each property for which TAT limits are defined, the number of results which were confirmed and validated in time or too late.

The TAT monitor is available under Start -> Business activity monitor -> TAT monitor.

Components of the TAT monitor

The TAT monitor looks as follows:

It displays a Monitor pane and a table with 6 columns: Property, Total count, AVG confirmation TAT, AVG validation TAT, Distributed confirmation TAT and Distributed validation TAT.

Monitor

The Monitor pane provides a selection list of monitor configurations for which TAT data can be displayed.

Property

List of the properties for which TAT limits are defined and which belong to the property classification specified in the chosen monitor configuration. Double-click the name of a property to consult the overdue results for this property.

The properties are displayed according to the sequence numbers they have in the property classification of the chosen monitor configuration. If the latter does not refer to any property classification, the order of the properties is based on their sequence number in the default property classification.

Total count

Number of results for specimens of which the receipt time is today. If there is no result for specimens of which the receipt time is today, a hyphen is displayed.

If the monitor configuration selected in the Monitor pane refers to

  • a department, then only the results of orders that were executed in this department are counted.
  • a property classification, then only the results whose property belongs to this classification are counted.

AVG confirmation TAT

Average time elapsed until the confirmation of a result. The value in between brackets is the Maximal time till confirmation indicated in the property editor. Results which are not in status Confirmed yet are not taken into account for the calculation of this average.

AVG validation TAT

Average time elapsed until the validation of a result. The value in between brackets is the Maximal time till validation indicated in the property editor. Results which are not in status Validated yet are not taken into account for the calculation of this average.

Distributed confirmation TAT

Number of results which have been confirmed in time or too late. If the Maximal time till confirmation is not specified in the property editor, then this progress bar is not displayed.

Distributed validation TAT

Number of results which have been validated in time or too late. If the Maximal time till validation is not specified in the property editor, then this progress bar is not displayed.
