Reagent usage

For each reagent lot, you need to define what it is used for by means of the contextual function Usage in assessment methods:

Station

Analyzer on which the reagent lot is used.

Property

The test (i.e. the assessment method) for which the reagent lot is used.

Start time / End time

Period during which the reagent lot will be used on this station and for this assessment method.

Manual

GLIMS creates "ResultReagentUsage" records for results originating from an analyzer and manually entered results. To prevent that GLIMS creates "ResultReagentUsage" records for reagent lots that are intended for manual result entry, the field Manual can be enabled.
When results are not received from an instrument but entered manually, GLIMS will create "ResultReagentUsage" records for the "manual" reagent usages currently valid for the result's assessment method.
