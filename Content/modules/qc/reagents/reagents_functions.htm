Available functions

In order to allow determining which reagent was used to obtain a specific result or which results where obtained using a specific reagent, the following functions are available:

Result.ShowReagent()

From a Result, select Log > Show reagent to open a browser with all open Result reagent usage records for the result's assessment method or station.

ReagentUsage.Results()

From a Reagent lot usage record, select Show all > Results. This function takes the selected ReagentUsage records (multi-record function!) and determines which results where obtained using the assessment method of these records (or the station in case the ReagentUsage records are method-independent). The selected results are then shown in a browser for further processing. This function allows showing all results obtained during one or more ReagentUsage periods.

ReagentLot.Results()

From a Reagent lot, select Show all > Results. Gathers all the ReagentUsage records of the selected ReagentLot records (multi-record function!) and calls the ReagentUsage.Results() function for these ReagentUsage records. This allows showing all results obtained using one or more reagent lots (station and method independent).

Notes

  1. Depending on the context from which this browser is opened, different columns will be shown.

  2. An info box at the bottom of the browser allows to configure a text named "ResultReagentUsageExternalization" based on the ResultReagentUsage table.

  3. The browser will have an automatic synchronisation of data, but maximal 1 month ago.

  4. These functions presume that only one reagent is used simultaneously per assessment method or per station. If a ReagentUsage record has no end date, then the start date of the next ReagentUsage record on the same Assessment method or Station is used to determine the end date.
