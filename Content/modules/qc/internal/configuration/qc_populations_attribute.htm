Notes:

  • The meaning of the parameters is described in the topic "QC populations".
  • All values are returned as "String"
  • Percentile25, Median, Percentile75 and all Tukey values might be outdated (the attribute "IsCalculationOutdated" will return "YES"). The function QCPopulation.Recalculate() will return updated values.
