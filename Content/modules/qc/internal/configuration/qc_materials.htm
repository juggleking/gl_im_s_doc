Quality control - QC materials

Introduction

The next step is to configure the QC materials in GLIMS via Start > Quality control > Configuration > QC materials. Each QC lot refers to a certain QC material.

Fields

Mnemonic / Description

The QC material can be identified by its mnemonic and description.

Supplier

Allows to store the supplier (Organization in GLIMS) of the QC material.

Code

The Code is a (bar)code that may be used to identify this QC material. This code can be very important to store barcode identifications originating from an analyzer. Whenever a barcode is received from a station (e.g. during result upload), GLIMS will check whether this barcode represents a QC material where this code equals the barcode. If specified, the QC lot code will take priority over the QC material code.

Tip

It is recommended to use the Code field on QC lot level instead of the Code field on QC material level. This allows you to have a different (bar)code for each QC lot so that several QC lots can be used simultaneously.

Label

The label which will be printed using the contextual function Print label on a QC lot of this material.
