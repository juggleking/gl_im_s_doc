QC lot validity in years

Please check the chapter QC Setup screen.

QC lot expiration severity

If the QC lot is expired when entering a QC result, a severity will be applied to that QC result.

By default, a severity of 999 will be applied, but this option allows to set a custom severity that is applied for a QC result when it is entered for an expired QC lot.

QC severity for non-numeric results

If a non-numeric QC result is entered for a QC population of type Quantitative or Titer, a QC severity will be applied to that QC result.

By default, a QC severity of 999 will be applied, but this option allows to set a custom QC severity.
