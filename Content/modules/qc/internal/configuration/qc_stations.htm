Quality control - Stations

On Station level, you can indicate that QC samples must be downloaded as patient samples.  When enabling the field Download QC as patient, the samples in the download message to the station will always be sent as if it were patient samples.

This option should be set for instruments that reject download messages for samples received from GLIMS as QC samples when the sample is configured on the instrument as a patient sample.

Note

The download message is mostly a reply to a sample query message received from the instrument. Example of such an instrument: Axsym.
