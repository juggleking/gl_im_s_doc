QC Setup screen

Introduction

The QC Setup screen allows to set QC specific preferences.  It is available from the main menu via "Start -> Quality control -> Configuration -> Setup".

General page

QC lot externalization

Used to visualize QC lots in GLIMS.

QC norm deviation count

Please check the chapter Reference range for QC results

Shift scheme

Please check the chapter Quality control -> Configuration: QC shifts

QC check application

Indicates the external QC check application (QCToday, other)

QC lot validity in years

This option can prevent QC lots and QC populations from being deleted too soon. It will not be possible to delete QC lots (and their QC populations) if the QC lot is still valid according to this setting.

Origins page

Target value origin

Allows defining the possible values for the "Target value origin" in the QC population configuration screen.

TEa origin

Allows defining the possible values for the "TEa origin" in the QC population configuration screen.

QC population editor page

Allows to define the sequence in which the tab pages of the QC population configuration screen should appear. This sequence is stored as a user preference.
