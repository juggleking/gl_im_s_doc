Quality control -> QC Scheduling

In case of an analyzer: via barcode

Using the contextual menu function Print label on QC lot, you can print the barcode label defined for the QC material.  This label can contain the (bar) code defined for the QC material or QC lot.

This ensures that the QC results are stored in the correct QC population (thanks to its with a QC lot and assessment method) .

Example of a QC label (defined in QC material):

{= .Material.Mnemonic + "-" + .LotNumber + "-" + IntegerToString({CopyCount}, "%03d")} 
Code {=.Material.Code}
Supplier {=.Material.Supplier.Correspondent.Name}
------------------------------------------------

In case of a manual station: via work list

Use the contextual menu Schedules on QC material in order to actually schedule QC assessments.

This function allows you to indicate for which procedure you would like to schedule QC and which work list template you would like to use. The type can be Start, Middle or End. In case of Middle QC, you will also have to specify the cycle start (indicating after how many patient samples you want to start a QC) and the cycle length (indicating after how many patients samples you want the second QC to start).

  • QcMat 3 with Cycle start 10 and Cycle length 20
  • QcMat 4 with Cycle start 20 and Cycle length 20
