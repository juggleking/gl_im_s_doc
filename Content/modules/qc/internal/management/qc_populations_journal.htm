Quality control -> QC journals

Besides the QC population print function, another overview document can be produced using the contextual menu function Journal which available from any QC population.  Selecting this function will display the following screen:

This screen allows to select a number of results over a certain period of time. The other options allow to determine the layout of the QC journal to be produced.

PDF reports are generated with the GLIMS Report Builder.
