Recalculate QC population statistics

Introduction

From each QC population, you can select the contextual menu function "Recalculate statistics".  It allows to recalculate all the statistical data about the selected population such as the total and numeric count, the calculated mean, CV and standard deviation.

Options

Start date/time

When specified, only results since this date/time are taken into account for the statistics calculation. When not specified, the last start date used for this population is reused.

End date/time

When specified, only results up to this date/time are taken into account for the statistics calculation.   

Execute checks

Allows specifying if the checks have to be re-evaluated upon processing the most recent result of the population.

Set target mean

When enabled, the method will update the population's target mean with the calculated mean value.

Set target deviation

When enabled, the QC population's target deviation will be updated with the calculated standard deviation.

Support for outlier calculation

Description

GLIMS supports the Grubbs' test for outliers.  Grubbs' test is based on the assumption of normality. That is, one should first verify that the data can be reasonably approximated by a normal distribution before applying the Grubbs' test.

The Grubbs test statistic is the largest absolute deviation from the sample mean in units of the sample standard deviation (In GLIMS, sample is a set of QC results of a QC population).

Links for more information:

  • http://en.wikipedia.org/wiki/Grubbs%27_test_for_outliers
  • http://www.itl.nist.gov/div898/handbook/eda/section3/eda35h1.htm

When configured, GLIMS will use that method when statistics are recalculated or when QC results are added (manually or by an instrument).

Configuration

The following QC population fields can be configured for activating the Grubbs' test:

Grubbs

Should be enabled if you want to check on outliers using the Grubbs' test.

Grubbs probability

Possible values: 0.100, 0.050, 0.025, 0.010, 0.005. Default is 0.100.

The value of this field indicates the probability that a value belongs to the supposed normal distribution.

Grubbs minimal values

Minimum number of QC results in the QC population before the Grubbs' test will be applied.

QC results

Is outlier

Indicates if the result was detected as an outlier and excluded from QC population statistics.

Outlier info

Contains information related to the exclusion of the result from QC population statistics.

QC result browser

The QC result browser (QC population -> Results) has an extra column indicating if a result was detected as an outlier. The outlier info is shown at the bottom of the browser (below the result comment information box).
