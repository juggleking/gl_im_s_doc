Report Builder - Chart customizer properties 

Class: be.mips.reports.chart.ElphoChartCustomizer

be.mips.reports.chart.ElphoChartCustomizer.fractionstotop (boolean, default: true)

  • when this is set to true, the fractions are rendered using a vertical line extending to the top of the chart.
  • when this is set to false, the fractions are rendered as a short vertical line extending slightly above the curve.

be.mips.reports.chart.ElphoChartCustomizer.paint (grayscale|color|custom, default: color)

  • when this is set to grayscale, the complete chart is rendered as monochrome.
  • when this is set to color, the curve and any peaks are rendered in blue.

be.mips.reports.chart.ElphoChartCustomizer.peaksasareas (boolean, default: false)

  • when this is set to true, any peaks are rendered by filling the area underneath the curve with a lighter shade of the curve colour.
  • when this is set to false, any peaks are rendered by using a rectangular area extending to the top of the chart.

be.mips.reports.chart.ElphoChartCustomizer.smooth (boolean, default: true)

  • when this is set to true, the curve will be rendered with interpolation between the points to prevent sharp edges.
  • when this is set to false, the curve will be rendered without interpolation between the points.

be.mips.reports.chart.ElphoChartCustomizer.showlabels (true|false|all|fractions|peaks|none, default: true)

  • when this is set to true, the fraction and peak labels will be displayed near the top of the chart.
  • when this is set to false, the fractions and peak labels will not be displayed on the chart.

be.mips.reports.chart.ElphoChartCustomizer.fractionlabelposition (top|belowAxis, default: top)

be.mips.reports.chart.ElphoChartCustomizer.firstfractionlabelshift (positive integer, default: 0)

be.mips.reports.chart.ElphoChartCustomizer.peaklabelposition (top|aboveArea, default: top)

be.mips.reports.chart.ElphoChartCustomizer.usecategoryaxisfonts (true|false, default: false)

be.mips.reports.chart.ElphoChartCustomizer.curvecolor (hexadecimal color or rgb triplet)

Only used if paint set to custom.

be.mips.reports.chart.ElphoChartCustomizer.peakcolor (hexadecimal color or rgb triplet)

Only used if paint set to custom.

Note

Note that color definitions take one of two forms:

  • #HHHHHH (where each H is a hexadecimal digit)
  • N,N,N (where each N is a positive integer value between 0 and 255)

Tip

To display a 'traditional' electrophoresis graph, take the following steps:

  1. Open JasperSoft Studio and modify report Result_Graph_Elpho_Chart.jrxml
  2. Select the XY Line Chart element
  3. In the property grid, find the line Properties expressions and click on the ... button
  4. A dialog box appears with a list of custom properties you can use to influence the look of the chart
  5. Apply the following values:
    • be.mips.reports.chart.ElphoChartCustomizer.fractionstotop = false
    • be.mips.reports.chart.ElphoChartCustomizer.paint = grayscale
    • be.mips.reports.chart.ElphoChartCustomizer.peaksasareas = true
    • be.mips.reports.chart.ElphoChartCustomizer.smooth = true
    • be.mips.reports.chart.ElphoChartCustomizer.showlabels = false
  6. Finally, save and recompile the .jrxml file in order to test the report
