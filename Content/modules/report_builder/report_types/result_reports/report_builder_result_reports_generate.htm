Report generation screen

Saving reports to file

Report generation screen

  1. Make sure the option "Output to file" is enabled

  2. After clicking the "Generate" button, the "File specification" window allows to specify where the report file should be saved.

  3. The "Electronic page" is not used by GLIMS report builder !
