Report Builder - Result reports: Basic configuration

Introduction

Generating result reports with GLIMS Report Builder is based on the same mechanism as the standard reports in GLIMS (default reports, report templates, report generation, ...).

Below you will find the basic configuration required for generating result reports with GLIMS reports Builder.

Default report

Create a new default report in order to generate results reports with GLIMS Report Builder.

Report template

Specify the report template which will be used for GLIMS report builder (see below).

Other

To see which other options are supported, please check the chapter Report Builder - Result reports: Default reports.

Report template

Main page

Generator

Must be set to Glims Report Builder

File type

Only used when saving the reports to file (when the option Output to file is enabled in the report generation screen)

Warning

Although the application allows it, we do not recommend generating HTML or text reports with Report builder.

Report builder was designed to generate PDF and XML reports.

Property classification

GLIMS report builder allows reusing existing PCL-5 based property classifications. PCL-5 expressions are automatically detected and filtered (unless the Report template option Remove PCL printer codes is disabled).

Layout page

Report layout template

Jasper template file that will be used to generate the reports (Windows or Unix).

Tip

GLIMS allows to perform some basic validation checks on the selected Jasper template.

Other options

To see which other options are supported, please check the chapter Report Builder - Result reports: Report templates.

Property classification (node)

No specific configuration is required. GLIMS report builder allows reusing existing PCL-5 based property classifications. PCL-5 expressions are automatically detected and filtered.

To see which options are supported, please check the chapter Report Builder - Result reports: Property classifications.

Property

No specific configuration is required.

To see which options are supported, please check the chapter "Report Builder - Result reports: Properties"

Report generation

Report query

Generator

Allows selecting all reports with GLIMS Report Builder as Generator.

Printing reports

Output to file

Must be disabled.

Printer

Make sure that the Windows or Unix path is specified for the selected printer

Saving reports to file

Output to file

Must be enabled

File specification

After clicking the Generate button, the File specification window allows to specify where the report file should be saved.

Note

The Electronic page is not used by GLIMS report builder !
