Texts page

These fields can be used to add custom fields to the GLIMS XML export. In order for this functionality to work, the MISPL must return a valid XML structure. If your data possibly contains reserved XML characters, make sure to escape them using the XMLEscaped() MISPL function.

This XML structure will be available in a <custom> element within the XML export.

Option Supported Comments
First page header Yes The returned value shall be located in the XML meta data section.
Header Yes The returned value shall be located in the XML header section.
Body Yes The returned value shall be located in every detail block of the XML.
Footer Yes The returned value shall be located in the XML footer section.
Trailer Yes The returned value shall be located in the XML trailer section.
Empty result Yes The defined text is used as result value when a value is not yet available yet (and no predecessor is found).
