Report Builder - Result reports: Features

Main features

  • Plug and play: no need to install any additional software
  • Detailed templates are shipped as of Glims version 8.5 covering all main report types
  • Single order reports: support for previous values (unlimited): previous value, date and norm severity
  • Cumulative order reports including grouping by order, specimen or object time
  • Support for genetic exam results reports (extended Genetics module).
  • Support for microbiology reports, including isolations, isolation tests, isolation comments, antibiograms, antibiotic comments
  • Support for electrophoresis graphs including fractions, fraction labels, peaks and peak indications
  • Support for result history graphs including curve and reference values
  • Support for inline and flushed comments
  • Support for norm severity indication and result marks
  • Reuse existing PCL-5 based property classifications defined in Glims (PCL-5 expressions are automatically detected and filtered)
  • Support for Property classification node field "Report style" to aid in sub-report printing conditions
  • Support for most report-related options defined in Glims such as:
    1. Property classification node's "Content" MISPL
    2. Property's "Report format"
    3. "Flush comments after"
    4. "Repeat per specimen" (Microbiology)
    5. ...

Not yet supported

  • No support for attachments / inclusions
  • No support for pathology, transfusion, epidemiology, genetics and HLA modules
