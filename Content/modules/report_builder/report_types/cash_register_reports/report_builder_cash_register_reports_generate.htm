Report Builder - Cash register reports: Generate

Configuration

No additional configuration is required to generate cash register reports using GLIMS Report Builder.

Generation

There are two ways to generate cash register reports:

From the "Cash register management" window

Go to Start –> Billing –> Cash register management and press Print.  This allows to print a report for a single cash register session.

From the "Cash register sessions by opening time" browser

Go to Start -> Billing -> Cash register sessions. The contextual menu function Print allows to print a report for multiple cash register sessions.  

This Print function can be also pre-configured.

Output options

Print as one document

When enabled, multiple sessions can be grouped in a single report.  In that case, additional total amounts are available at different levels.  Note that you need a different layout file when this option is enabled.
