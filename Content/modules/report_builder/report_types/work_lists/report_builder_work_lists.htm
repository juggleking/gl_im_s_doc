Report Builder - Outline work lists

Introduction

GLIMS allows exporting work list outline reports in an XML format which can be converted to a PDF document using Report Builder.

New output options

The work list printing screen allows to export the report to XML or PDF.  The report can be sent directly to a printer or can be saved as a file on a specified directory with a specified file name.

When "Layout type" is set to "Outline", the "Format" can be set to "XML" or "PDF":

Features

Currently only the "outline" work list reports can be generated in XML or PDF format.  The "grid" work lists can only be printed in text as before.  

Therefore, if multiple records are selected for printing in PDF format and no report type (grid/outline) is chosen, then for every work list with a layout type "Grid" specified on the work list template, the output will be Text!  If the layout type is "Outline",  the output will be PDF or XML as requested.

Layout template file

GLIMS will look for the layout template files as follows:

  1. Template file specified in the "Work list template"

  2. Template file specified in user's department

  3. Template file specified in general options (specific site)

  4. In the application directory "templates/examples/Worklist/"
