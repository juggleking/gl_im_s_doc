Report Builder - Print commands

Description

When generating Report Builder reports (result reports, work list reports, complaint reports, etc), it is possible to execute a print command on the XML output of the generated report.

Configuration

In the GLIMS printer configuration, do not specify a Windows path and specify a Print command (the Windows path has priority over the Print command).  It will always be the XML output that will be sent to the print command, even when you selected "PDF" as format in the print options window.

This allows e.g. to process the XML reports by an external application.
