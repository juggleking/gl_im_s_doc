Report Builder - Output options

Introduction

This chapter applies to:

  1. QC reports (QCAudit.Print, QCPopulation.Print, QCPopulation.Journal)
  2. Financial reports
  3. Outline work list reports
  4. Blood transfusion forms
  5. Primary Sample Collection Manual reports
  6. Cash register reports
  7. Complaint reports

Output options

The above report programs all have the same set of output options.  Some programs also have specific options that may interact with the default set of output options.

Options

Print

Must be enabled to print the report on a printer

Printer

Reference to a printer.  If no printer is selected, the report is printed on the default printer.  The default printer depends on the type of document that is printed:

  • Labels, transfusion forms, purchase order forms: see Set devices
  • Complaint export: printer defined in system management -> customize -> MATE
  • Primary sample collection manual: printer defined in system management -> customize -> MATE
  • Cash register (session) print: printer defined in system management -> customize -> MATE
  • QC reports: printer defined in system management -> customize -> MATE
  • Study configuration reports: printer defined in system management -> customize -> MATE
  • Result reports: see Printer evaluation during reporting

Output to file

If enabled, the report will be saved to file.

Directory

Directory in which the report file will be saved.  The following symbols are supported:

. Application directory
{TempDir} Temporary directory
Dynamic text Any dynamic text on the function’s specific table or gp_Site.

Name

Name of the file to which the report will be saved.  The extension is automatically added, depending on the selected "Format".  The following symbols are supported:

{Date} Current date in format “YYYYMMDD”
{Sequence} If set, the file name will be made unique by using a sequence number
Dynamic text Any dynamic text on the function’s specific table or gp_Site.

Format

The possible values depend on the report type:

Complaint reports PDF (default), XML
Cash register reports PDF (default), XML
QC reports PDF (default), XML, HTML, TEXT
Primary Sample Collection Manual PDF (default), XML, XML & PDF

Warning

Although the application allows it, we do not recommend generating HTML or text reports with Report builder.

Report builder was designed to generate PDF and XML reports.

Preview

Will show a preview of report. Previewing and printing cannot be combined.

Remarks

  1. The report options are remembered per user, unless the function is preconfigured. If a symbol (e.g. {TempDir}) is used, the non-substituted version will be stored.
