Report Builder - External templates

Contents: Introduction - Fields

Introduction

External templates allow to specify the Jasper report template to be applied when generating a report with the Report builder technology.

Example

External templates can be used to generate a browser report in PDF format.

Tip

The function Validate report template allows to check for potential problems in the Jasper template.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

External template file (Windows / Unix)

Reference to a Jasper file.
