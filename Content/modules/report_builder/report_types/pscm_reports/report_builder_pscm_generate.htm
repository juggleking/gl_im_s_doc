Report builder - Primary Sample Collection Manual

Introduction

GLIMS allows generating a primary sample collection manual in XML or PDF format for one or more selected properties. 

The content of this sample collection manual is based on the existing property data sheet but has been extended in the context of the ISO-15189 norm for clinical laboratories.

Note

This module requires a separate license (GLIMS_PSCM).

Options

Select one or more properties and choose the contextual menu / ribbon item Primary sample collection manual.

Specific options

Content type

Allows generating a full document, or an issuer / study specific document.

The Default content type determines which norms and request definitions are included. Choosing Default will generate a document which includes all of the property's norms (except for the study-specific norms) and the property's default request definition.

Attach output to property

If enabled, the generated report will be added as Attachment to the property.

Output options

Please check the chapter Report builder - Output options.

Automation with a command

It is possible to automate the Primary Sample Collection Manual export with a command. The command should be configured as follows:

  • Table: Property

  • Query: Standard

  • Query parameter set: insert a new query parameter set and choose the contextual menu / ribbon item Setup to configure the property query options.

  • Function: PrimarySampleCollectionManual

  • Function parameter set: insert a new function parameter set and choose the contextual menu / ribbon item Setup to configure the property export options.

    Tip

    To set the parameters for the actual property export, you should login with the user responsible for the command and execute the property export function interactively in GLIMS. The selected options will be saved in the user's preferences and will be reused by the command.

    The user responsible for the command is: the user of the service that executes the task of the command.
