Report builder - Utilities

A number of utilities (helper methods) are available that can be used when customizing the default Jasper templates for Report builder.

Image utilities

Add TIFF images

In order to support the use of single- and multi-page TIFF images on Report builder layouts, two helper methods are available.

  1. To determine the number of pages, use Jaspersoft Studio to add a variable to your layout, with an expression of the form:

    be.mips.util.ImageUtilities.getTIFFPageCount(the_file_path)

    where the_file_path is a java.lang.String variable

  2. To extract and use an individual page, use Jaspersoft Studio to add an image to your layout, set its expression class to 'java.awt.Image', with an image expression of the form:

    be.mips.util.ImageUtilities.getTIFFPage(the_file_path, the_page_index)

    where the_file_path is a java.lang.String variable and the_page_index is a zero-based int value.

Add embedded images

Some Report builder XML data files may contain embedded images (Base64-encoded) rather than file paths.

To include these images in the report output, a helper method is available. The report layout designer is able to include such embedded images with two simple steps:

  1. set the image expression class to 'java.io.InputStream'
  2. set the image expression to something like be.mips.util.ImageUtilities.decodeEmbeddedImage($F{FieldContainingTheBase64EncodedImage})
String utilities

String to HTML

Description

A helper function is available in Report builder to assist in the conversion of text strings to HTML for printing styled text on reports.

This function replaces reserved HTML characters with the corresponding entities, optionally replacing spaces and newlines (if preformatted is true), optionally replacing exponents (if exponents is true)

The method is called be.mips.util.StringUtilities.stringAsHtml(String content[, boolean preformatted [, boolean exponents]]):

Note

  • calling be.mips.util.StringUtilities.stringAsHtml(content)

    is the equivalent of calling be.mips.util.StringUtilities.stringAsHtml(content, false, false)

  • calling be.mips.util.StringUtilities.stringAsHtml(content, false)

    is the equivalent of calling be.mips.util.StringUtilities.stringAsHtml(content, false, false)

  • calling be.mips.util.StringUtilities.stringAsHtml(content, true)

    is the equivalent of calling be.mips.util.StringUtilities.stringAsHtml(content, true, false)

Parameters

content

Text string to be converted.

preformatted

Possible values:

  • false (replace spaces with non-breaking spaces and newlines with <br> tags)
  • true (do not replace spaces or newlines)

exponents

Possible values:

  • false (do not replace exponents with the corresponding entity)
  • true (replace exponents with the corresponding entity)

Examples

Example 1

Given a String theText with content:

This text contains
HTML characters (<>&")

Calling be.mips.util.StringUtilities.stringAsHtml(theText, false) returns:

This&nbsp;text&nbsp;contains&nbsp;HTML<br />characters&nbsp;(&lt;&gt;&amp;&quot;)

Calling be.mips.util.StringUtilities.stringAsHtml(theText, true) returns:

This text contains HTML
characters (&lt;&gt;&amp;&quot;)

Example 2

Given a String theText with content:

This text contains HTML
characters (<>&") and exponent 42^2

be.mips.util.StringUtilities.stringAsHtml(content) is the same as
be.mips.util.StringUtilities.stringAsHtml(content, false, false) and returns:

This&nbsp;text&nbsp;contains&nbsp;HTML<br />characters&nbsp;(&lt;&gt;&amp;&quot;)&nbsp;and&nbsp;exponent&nbsp;42^2

be.mips.util.StringUtilities.stringAsHtml(content, true) is the same as
be.mips.util.StringUtilities.stringAsHtml(content, true, false) and returns:

This text contains HTML
characters (&lt;&gt;&amp;&quot;) and exponent 42^2

be.mips.util.StringUtilities.stringAsHtml(content, true, true) returns:

This text contains HTML
characters (&lt;&gt;&amp;&quot;) and exponent 42<sup>2</sup>

Exponent string to HTML

Description

This function transforms text of format [NNNx10E(MMM)] or format [NNN^MMM] to [NNN<sup>MMM</sup>]

Examples

Example

be.mips.util.StringUtilities.exponentStringAsHtml("42x10E(2)") returns:

42<sup>2</sup>

be.mips.util.StringUtilities.exponentStringAsHtml("42^2") returns:

42<sup>2</sup>

Substitute list

Description

This function can be used to replace aliases in the original comma-separated list with their actual values, removing doubles, retaining relative order.

Examples

Example

be.mips.util.StringUtilities.substituteList("Trillian,Arthur Dent,Trillian", "Trillian", "Zaphod Beeblebrox") returns:

Zaphod Beeblebrox,Arthur Dent

be.mips.util.StringUtilities.substituteList("Marvin,Arthur Dent,Eddie,L.Prosser,Trillian", "Trillian", "Zaphod Beeblebrox", "Marvin", "L. Prosser", "Eddie", "L. Prosser") returns:

L.Prosser,Arthur Dent,Zaphod Beeblebrox
Date & Time utilities

Datestamp format

Description

The application exports use a standard format for datestamps, which corresponds to Java SimpleDateFormat pattern "yyyy-MM-dd"

To make it easier to use these in your .jrxml/.jasper ayout template files, a helper function is available to convert these datestamps to a standard Date object.

Examples

Example

be.mips.util.DateUtilities.convertDatestampToDate(String datestamp) returns a Date object which can then be used on your layouts with patterns that adapt to the report target locale.

Timestamp format

Description

The application exports use a standard format for timestamps, which corresponds to Java SimpleDateFormat pattern "yyyy-MM-dd'T'HH:mm:ss"

To make it easier to use these in the .jrxml/.jasper layout template files, a helper function is available to convert these timestamps to a standard Date object.

Examples

Example

be.mips.util.DateUtilities.convertTimestampToDate(String timestamp) returns a Date object which can then be used on your layouts with patterns that adapt to the report target locale.
