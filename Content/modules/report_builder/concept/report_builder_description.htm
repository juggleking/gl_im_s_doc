Report Builder - Description

Introduction

Glims Report Builder is a next-generation reporting module, combining the traditional Glims report generator system with the power of the widely popular open-source tool JasperReports. 

Main advantages

  1. Visual design of report templates (using the Jaspersoft Studio designer tool) offering the following exciting features:
     
    1. Drag-and-drop of report fields and variables onto the layout
    2. Quickly change color, font, style, alignment and other properties
    3. Title and summary sections for first page header and last page trailer
    4. Graphics primitives to add lines, rectangles and ovals
    5. Static and dynamic images to print logos and result images
    6. Sub-reports to cleanly separate different types of result lines
    7. Conditional printing to decide which items are to be printed
    8. Styled text to aid in conditional formatting of result data
       
  2. Complete separation of content and layout (technical concept):
     
    1. GLIMS exports the report data in XML format
    2. GLIMS starts a Java Virtual Machine and loads the MipsJVM ReportManager
    3. GLIMS asks the MipsJVM ReportManager to print or preview the report telling it which .XML file contains the data and which .JASPER file contains the layout
    4. JasperReports takes care of preview on screen, sending to a specific printer, etc.
       
  3. Output to PDF, RTF, HTML, XML files or directly to printer:
     
  4. Preview on screen allows the user to save the report to PDF, RTF, HTML, XLS and other formats.
     
  5. Fully integrated in the GLIMS result reporting module using the existing configuration (default reports, report templates, property classifications, ...).
