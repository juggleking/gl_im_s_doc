Report Builder - Jasper templates

Location of default templates provided by MIPS

The Jasper example templates (.jasper) provided by MIPS are located in the templates/examples directory within the GLIMS installation.  They are subdivided in several categories, i.e.:

  1. Cash register reports
  2. Complaint reports
  3. Primary Sample Collection Manual (PSCM)
  4. QC reports
  5. Result reports

Customizing the Jasper example templates provided by MIPS

  1. Never customize the Jasper example templates provided by MIPS in the "templates/examples" directory, as these may be replaced by newer versions when installing a higher GLIMS version  !!  
  2. First copy the Jasper example templates provided by MIPS to another directory (see below).  Example: for cash register reports, copy the .jasper files from "templates/examples/CashRegister" to "templates/CashRegister"  
  3. To edit or compile the layout files in Jaspersoft Studio, you need to copy the "be" and "org" directories from "<GLIMS home>/jar/cls" to the directory with the .jrxml files.

The Jasper example templates (.jasper) can be customized using the Jaspersoft Studio.

Which Jasper template will be used by GLIMS

Cash register, Complaint, Primary sample collection manual and QC reports

A full Jasper layout file name can be specified for both Unix and Windows on different levels.  GLIMS will look for a Jasper layout file in the following sequence.

1. User preferences

Currently, it is not yet possible to define the layout file names as user preferences.  This will be added in a future version.

2. Department

Fields are available on Department to specify the layout file name for each report type (Windows and Unix). 

3. General options

Fields are available on SpecificSite to specify the layout file name for each report type (Windows and Unix).

4. Other defaults

Complaint reports

<ApplicationDirectory>/templates/examples/Complaint/Complaint.jasper

Primary Sample Collection Manual

<ApplicationDirectory>/templates/examples/PrimarySampleCollectionManual/PSCM.jasper

Cash register reports

Single cash register:

<ApplicationDirectory>/templates/examples/CashRegister/CashRegister.jasper

Multiple cash registers (option "Print as one document" is enabled)

< ApplicationDirectory>/templates/examples/CashRegister/CashRegisters.jasper

QC reports

QC print (QCAudit.Print, QCPopulation.Print):

<ApplicationDirectory>/templates/examples/QCPrint/QCPrint.jasper

QC journal (QCPopulation.Journal):

<ApplicationDirectory>/templates/examples/QCJournal/QCJournal.jasper

Result reports

  1. The template file as specified in the field "Report layout template file" of the "Report template" configuration screen in GLIMS.
