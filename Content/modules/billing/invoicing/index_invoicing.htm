Invoicing process

The tariffication chapters describe how the prices are calculated, who pays what part of the costs, etc. This invoice chapter describes how order sets are created, how invoices are generated for the tariffed order sets, how these invoice are collected on invoice summaries, which can then be placed on financial shipments.  The figure below gives an overview of the invoicing process with links to detailed documentation about each step.

1. During order entry, an order set is determined for each order.
2. During tariffication, GLIMS will automatically generate one or more invoices for each order in the order set.
3. The invoices are grouped on invoice summaries.
4. These invoice summaries can be placed on financial shipments.

As shown the figure above, GLIMS knows different types of billing documents such as invoices, invoice summaries, financial shipments.  The layout of the documents is determined by billing document templates.
