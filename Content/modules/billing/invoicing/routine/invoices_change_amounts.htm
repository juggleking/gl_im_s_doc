Invoice.ChangeAmounts

This function allows manually changing the invoice item amounts. Also all other invoices (of other payers) for the same billing items can be changed in the same session. An array editor is opened showing a row for each invoice item on the invoice. The first column is the template of the billed request, the second column the tariff's nomenclature code. Then there are 2 rows for each invoice involved in this billing item: the first column shows the amount that can be changed. The second column is not editable and shows the original unchanged amount.

If any value is changed manually, the following happens when the array editor is closed:

For any fixed invoice on which an amount was changed, a corresponding credit invoice is created for the changed invoice items plus a new debit invoice with invoice items for the new amounts. A dialog box asks if a document number must be assigned to the newly created invoices. It also asks if existing credit invoices (same order, firm & payer) should be reused.

For any non-fixed invoice on which an amount was changed, the amount in the actual invoice item is changed. This causes corresponding amount changes in the invoice, invoice summary etc...
