Print price list

Contents: Introduction -  Query - General price list selection - Price list selection by payment agreement - Options

Introduction

GLIMS allows to generate an overview of a specified price list.  From the main menu, click Start -> Billing -> Print price list.

Query

General price list selection

Price list

Select any price list of which you want to generate an overview.

Issuer

If you select an issuer, the issuer specific price list will automatically be selected.

Consider as issuer's price list

If set, issuer-specific request definitions will have precedence over general request definitions; i.e. both issuer-specific and general request definitions will be included, but doubles will be avoided by only showing the issuer-specific request definition.

Only issuer-specific request definitions

If set, only issuer-specific request definitions will be included and not general request definitions.

Price list selection by payment agreement

When selecting a payment agreement by specifying a payment agreement, the general fields above will not be applied.  Make sure that you set the option "Use payment agreement".

Payer

First select a payer and then click the "Payer" button to select a payer-specific payment agreement.  The price list of that payment agreement will then be used to generate the overview.
 

General

You can also select a general payment agreement by clicking the "General" button.  The price list of that payment agreement will then be used to generate the overview.

Options

Include price list extensions

Also include tariffs of price list extensions.

Print price list details

Also include calculation details.

Validity date

Validity date for the included tariffs.

Sorted by

Indicates how the overview should be sorted.
