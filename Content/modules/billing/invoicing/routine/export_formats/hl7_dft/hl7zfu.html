Soort Tarief Indicatie

On some transactions specific rules have to be applied, e.g. a percentage is to be added if the transaction was done during nighttime. If this field contains a valid code, assigned to the transaction, the tariefsoort will be overruled by the tariefsoort given in this field through a code. (The code maps to a tariefsoort). In this field the following codes can be given:

    • NTHS
      Nachttoeslag op honorarium specialist
    • WTHS
      Weekend/feestdag toeslag op honorarium specialist
    • NWHS
      Nacht+weekend toeslag op honorarium specialist

Besides the above mentioned codes locally defined codes can be used. These local codes must start with a * symbol.

It is advised to use this mechanism carefully.

Aantal verrichtingen
