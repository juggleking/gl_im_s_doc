Invoicing: Billing document numbering and status handling

Invoice summaries and other billing documents have a document number that is calculated with the document numbering MISPL functions specified in the specific site screen.

All billing documents also have a status which indicates if the document already has an identification number (status numbered or higher), if it already has been printed / sent (typically status fixed) or if it has been fully paid (invoice summary status: balanced or higher).

Once an invoice summary or another billing document has attained the status 'Fixed', it can no longer be modified. This status can be attained after printing or sending a document or by means of the ‘Change status’ function.

This implies that it is no longer possible to add invoices to or disconnect invoices from a fixed invoice summary. Further changes are then only possible through the creation of a credit note or by manually lowering the status with the ‘Change status’ function. The latter option is not recommended if the invoice summary has already been sent.
