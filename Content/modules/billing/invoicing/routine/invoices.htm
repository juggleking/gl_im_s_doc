Invoicing: Invoices

Contents: Introduction - Invoice access - Fields

Introduction

During tariffication, GLIMS will automatically generate one or more invoices for each order in the order set; e.g. an invoice for the health insurance company and one for the patient (depending on the patient's payment agreement).  

Invoices correspond to different country-dependent documents:  

  • Belgium: 'mutualiteitsattesten', 'onderaannemingsattesten' and 'ziekenhuisattesten'
  • France: 'feuille de secu'
  • Other countries: this level has no specific meaning. The invoice contains the data of one particular order for one party.
Using the context menu function 'Invoice summary creation', invoices are grouped on invoice summaries.

Invoice access

The invoices can be accessed from the main menu via Start -> Billing -> Invoices -> Browse / Query or by using the contextual menu function Billing -> Invoices from each order record.  The invoice query can also be used in a command in combination with the Invoice.Print function.
