Invoicing: Import rejections

Introduction

GLIMS allows automatic crediting of tariffed orders. This program is available from the main menu via Start > Billing > Import rejections. The program performs an automatic crediting of tariffed orders by importing a rejection file (which is the result of processing a performance file).

Parameters

Protocol

As the layout of the import file differs, you can select the protocol. 

File name

Select the file to be imported.

Order identification type

Indicate how the order in the import file should be identified.

Credit parameter name

Is not mandatory when Only log, don't import is active.

Only log, don't import

Allows to view the rejections and log the errors without actually importing and crediting the invoices.

Functionality

When the import file is being imported: 

  • Not yet credited invoices of the order are credited
  • The corresponding rejection reason is stored in the automatically created credit note
  • The tariffication status of the order set is demoted to 'Partial' 
  • The rejection import is logged in GLIMS 

After the rejection import, you can correct the reported error and re-tariff the order set.
