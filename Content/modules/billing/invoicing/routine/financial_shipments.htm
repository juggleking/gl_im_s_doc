Invoicing: Financial shipments

Contents: Introduction - Fields

Introduction

Financial shipments are used to send financial data to external systems or to official departments (Belgian RIZIV files, German KVDT files, HL7 financial, HPRIM financial, external accounting applications etc.).

Financial shipments can be created with the function Place on financial shipment on the 'Invoice summary' (and 'Financial transaction') table. They are sent using the context menu function 'Send'.

You can use the financial shipment query in a command in combination with the 'Print' or 'Send' function.

Fields

The financial shipments can be accessed from the main menu via Start -> Billing -> Financial shipment -> By external id/ by internal id / query.

For a full description of all fields, please check the reference manual.
