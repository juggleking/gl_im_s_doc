FinancialShipment.Send

Financial shipments can then be sent to an external system or to official departments (Belgian RIZIV files, German KVDT files, HL7 financial, HPRIM financial, external accounting applications etc.). with the contextual menu function 'Send'.

Supported protocols: HL7 DFT, RIZIV, INFOHOS, German KVDT, PAD, German DTAUS 0, German DTAUS 1, Hauptverband Ostereich. Sozialvers.traege, HPRIM Facturation, SMS Cotation des actes, C Page, ASCI, VDS, Umsatzdatenbank, Vektis 9, Exact, TripleP, IRIS LB2, Acc TF, Symphonie, UZG, Europe, Xtenso-L, Reference

License checking

When exporting financial data, a specific license is required for each individual format.  When opening the export screen, GLIMS will initially only show the formats for which you have a license (or that do not require a license).  You can still see the full list by double-clicking in the Format field.
