FinancialShipment.Print

Introduction

From each financial shipment record, you can use the Print function. This allows printing all underlying invoice summaries and invoices. 

Note

You can create a command with a financial shipment query in combination with the FinancialShipment.Print function to automate this process.

Fields

Billing document template

Select a billing document template of type Financial shipment if you are printing a single financial shipment or multiple financial shipments as separate documents. You can also print multiple financial shipments as one document by selecting a billing document template of type Financial shipment list. In this case, you must first disable the option Handle records separately.

To file / Create unique file for each record

When printing multiple billing documents (invoices, invoice summaries, financial shipments) simultaneously while the option To file is enabled, you can either create a single file for all billing documents or a separate file for each billing document. The latter can be achieved by enabling the option Create unique file for each record when printing multiple billing documents (e.g. using the Print function in the financial shipment browser while multiple records are selected).

Note

Note that this option is only supported when using default file names (e.g. Directory =  {TmpDir} and Name = {Date}{##}). If this is not the case, an error message will be shown.
