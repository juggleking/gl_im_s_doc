Cash register - Overview

Concept

GLIMS allows to define cash registers that can be used by one or more users. The idea is that a user opens a Cash register session. All manual payments registered in GLIMS by this user will then be stored as Cash register movements within the open Cash register session.

Configuration

All Cash registers used in the lab need to be defined in GLIMS.

Management

The Cash register management screen allows to :

  • Open a cash register session
  • Consult the cash register movements
  • Register manual deposits / withdrawals
  • Close a cash register session
  • Print cash register session reports

Consultation

GLIMS offers differs ways to get an overview of all Cash register sessions and to print reports.
