Cash register - Configuration

Introduction

Cash registers are available from the main menu via Start -> Billing -> Configuration -> Cash registers.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Name

Full name that will be used to identify the cash register in the GLIMS user interface.

Enabled

Must be checked before you can use this cash register.

Has parent, Parent

Allows to organize the cash registers in a hierarchical structure. No specific use during cash register management.

Lab

Lab in which the cash register is located.

Department

Department within the lab in which the cash register is located (optional).

Single use

Enabled

Only the user specified in Single user can use this cash register.

Disabled

Only the users belonging to the user group specified in User group can use this cash register.

Single user

User who can use this cash register when Single use is enabled.

Warning

When inserting a new cash register, the specified user is not correctly stored. Save the cash register, reopen it and specify the user again.

User group

Users who can use this cash register when Single use is disabled.

Warning

When inserting a new cash register, the specified user group is not correctly stored. Save the cash register, reopen it and specify the user group again.
