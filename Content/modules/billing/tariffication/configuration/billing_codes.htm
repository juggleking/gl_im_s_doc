Billing codes

Billing codes group request definitions which should be tariffed in a similar way. The billing code level is an intermediate level between lab deliveries (especially properties) and price setting configuration. It avoids price setting specifications to be repeated for each specific property, material, blood product, etc.  Billing codes are linked to request definitions via billing code assignments.
