Billing configuration check

GLIMS features a tool to check the completeness and validity of your billing configuration.  This tool is available from the main menu via Start -> Billing -> Configuration -> Check billing configuration.

The program executes the following checks:

  • Does each executing class have at least one provision ?
  • Does each billing code have at least one tariff ?
  • Does each request definition that is marked as 'To charge' have at least one billing code assignment ?
  • Detection of panels with disabled members.
  • Detection of circular panel definitions, i.e. panels which has itself as panel member.
