Export list of externally executed tests

Introduction

GLIMS allows to produce a list of all tests that are executed by an external lab.

Configuration

A new item must be added manually to the main menu.

  1. Function table: gp_Site
  2. Function: RunProcedure
  3. Actual function parameter set -> Setup: select function "GenerateExternalExaminesCsv"

Options

The following options are available:

From/to

Allows to define an order receipt date range.

Lab group

Allows to select a correspondent group of executing labs.

Price list

Price list to be used for the exported price.

Output file name

Name of the output file.

Preview

If enabled, the exported data will be previewed with the Windows program associated to .csv files (typically MS Excel).

Exported data

This export program selects all requests executed in external labs (Result.Department -> Lab = where Lab.Internal=NO)  and produces a .csv file.

The following columns are exported:

  • Lab.Mnemonic
  • Order.InternalId
  • Object.Externalization
  • Issuer.Name
  • Property.Mnemonic
  • Property.Description
  • Tariff.NomenclatureCode
  • Price
