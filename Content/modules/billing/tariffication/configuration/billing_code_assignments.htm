Billing code assignments

Introduction

A billing code assignment associates one or more billing codes with a request definition, medium, medium test, micro-organism, micro-organism test, etc.:

Fields

Valid from - until

A billing code is time-dependent.  

Eligible count

MISPL expression that can be defined to determine when this billing code must be applied (extra conditions).

Billing code

Reference to the billing code that will be applied.

Executing class

The executing class that will be used.

Type

Request for chemistry labs.  All other types are used for tariffication in microbiology, pathology and blood transfusion.  Depending on the type, you will be able to select a request definition, medium, micro-organism etc.
