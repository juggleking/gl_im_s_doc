OrderSet.VerifyTariffication

From each order set you can select the contextual menu function "Verify tariffication".  This function performs the actual tariffication and gives an overview of all tariffication data.

If the order set is already tariffed, the available data will immediately be displayed.  If the order set is not yet tariffed, click the 'Verify' button to simulate the tariffication process.  If the "Show invoice items" is enabled, detailed information per invoice item will be shown.  After closing this screen, unnumbered invoices of incompletely tariffed order sets are removed.
