Who is involved in the billing process ?

It is important to make a distinction between the type of a correspondent in GLIMS and the role that will be played in the in an order. In this context, we will be speaking about the role of a correspondent. Example: an issuer (role!) does not always need to be a HC Provider (Type).

The Order set determination ERD gives an overview of all correspondents involved. Below you fill find a detailed description of this ERD.

As you can see in the figure, the different 'correspondents' are introduced at different phases of the tariffication process. Each of these 'roles' can be selected as a reimburser (via the the reimburser type specified in the policy clause).

Correspondent roles at order entry level

An order always has an object, which can be a person (patient), a lot or an animal. These three correspondent types are respectively linked to an object person, an object supplier and an object owner, which represent their role in the order.

If a patient is hospitalized, there will be an encounter which is linked to an institution.

An order also has an issuer and an agent (order agent).

Correspondent roles introduced at order set determination level

An order set is also linked to an agent: the billing agent. (this might no longer be supported in the future)

An order set can have a dispatching lab and must have a receiving lab. By means of the field Party type, it is possible to select a reimbursement party. The party type is one of the six correspondents specified at order level (object person, object supplier, object owner, institution, issuer, order agent) or can also be Specific Instance. In the latter case, the correspondent must be entered manually.

The order set party is not necessarily the one that must pay. It is the correspondent who determines how the billing will be executed by means of its payment agreement. A payment agreement can have a specific party, which will then be the same party of the order set. However, it is possible that a payment agreement does not have a party. A payment agreement without a party is a general agreement which is applicable for all parties.

Correspondent roles at payment agreement level

Once the payment agreement is selected, some more correspondent role's are introduced:

The holder of the payment agreement is another reimburser type. A payment agreement can also involve a fund. A fund is typically an organization which is not at all related to the order or order set, except that it is involved in the billing process.

A payment agreement determines the policy. At policy clause level one can still select a specific instance as reimburser type.

One can also specify a Billing Partner.

So, each of the 'correspondent roles' marked in bold, can be selected as reimburser type of a policy clause.
