Cotation: formules de prix

Les formules de prix permettent de définir combien coûte le B, le BHN etc.  A partir de chaque Code prix on peut sélectionner l'option 'Formule de prix'. 

C'est la formule de prix qui déterminera le vrai prix de l'analyse.  La relation entre le code de prix et la formule de prix peut dépendre du temps.  Cette relation indirecte permet de modifier le calcul de prix d'un groupe de tarifs par une seule manipulation au lieu de le changer pour chaque tarif individuel.

La formule de prix calcule le prix réel à partir d'une valeur de base spécifiée dans le tarif.  Chaque formule de prix a une période de validité.  Les prix réel est calculé avec un multiplicateur et un montant fixe.

Exemple:

Valeur base du tarif 10
Multiplicateur

3

 Formule de prix
Montant fixe +4
Formule: (10x3) +4  
Prix réel 34
