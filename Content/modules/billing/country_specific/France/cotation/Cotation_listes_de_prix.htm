Cotation: listes de prix

Une liste de prix détermine le tarif à appliquer pour un code de cotation. En générale, il y a une liste de prix par défaut qui est spécifiée par une organisation officielle (p. ex. INAMI en Belgique).  On pourrait par exemple créer d'autres listes de prix pour des analyses exécutées dans un contexte commercial, etc.

Les listes de prix sont accessibles par Start -> Cotation -> Configuration -> Listes de prix. 

La liste de prix à utiliser peut être spécifiée a plusieurs niveaux:

  • La liste de prix spécifiée dans la clause de police
  • La liste de prix spécifiée dans l'accord de paiement remaplce celle spécifiée dans la clause.
  • Si la liste de prix n'est pas spécifiée dans l'accord de paiement ni dans la clause de police, la liste de prix par défaut sera utilisée

Les différentes listes de prix peuvent être structurées hiérarchiquement.  Le contenuer d'une liste de prix est la liste de prix qui est située à un autre niveau de l'hiérarchie.  Si aucun tarif n'est trouvé pour un code cotation dans un certain liste de prix, le conteneur sera utilisé.
