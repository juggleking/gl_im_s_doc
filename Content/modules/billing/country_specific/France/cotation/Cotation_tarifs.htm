Cotation: tarifs

Comme expliqué dans d'autres chapitres, les codes de demande sont liés à des codes de cotation.  Ces codes de cotation sont maintenant à leur tour liés à des tarifs.  Chaque tarif appartient à une liste de prix, ce qui permet différents codes nomenclatures et prix par liste de prix 

Donc, le tarif indique le code nomenclature et le prix d'une code de cotation dans une liste de prix.  Mais le tarif ne spécifie pas le prix réel d'une analyse.  Il indique seulement une valeur de base.  Le prix réel sera basé sur cette valeur de base mais est calculée avec un code de prix et une formule de prix.

France:

Les tarifs indiquent combien quelque chose coûte en B , en BHNn etc.  A partir de chaque 'Code de Cotation Sécu' on peut sélectionner l'option 'Tarifs'.
