Feuille de sécu

This document describes how you can print the French 'Feuille de sécu'.  The 'feuille d’honoraires d’actes de biologie medicale' can be printed by means of the Print function on the fund’s invoice.

You must create a specific 'Feuille de sécu' billing document template in which you can use the OrderSet.BCData MISPL function to obtain the desired layout / contents for the 'Feuille de sécu'.

Below you will find an example text expression that can be specified in the Header field of the billing document template.  The example makes use of the OrderSet.BCData MISPL function.  

Note that we make use of the xprint program in this example.  This is an external program which merely simplifies printing of the 'Feuille de sécu' but which is by no means indispensable.  For more information on this external program, please check the Internet: http://www.4gl.fr. The xprint program can be defined as a printer in GLIMS.

Example text expression

<preview><FTimes New Roman><P10><CPI20>
<#1>
<C#1><R1><C20>fEUILLE D'HONORAIRES D'ACTES DE BIOLOGIE MEDICALE <C#1><R1><C120>No de facture
<C#1><R2><C120>{=.Invoice.DocNo}
<C#1><R4><C20>RENSEIGNEMENTS CONCERNANT L'ASSURE(E) ET LE MALADIE (1)
<C#1><R4><C0><FROM><R5><C150><RECT>
<C#1><R5><C10><I>no matricule</I>{=.Invoice.PaymentAgreement.Matriculation}
<C#1><R6><C10><FROM><R9><C80><RECT>
<C#1><R6><C10><C10>{=.Invoice.PaymentAgreement.Party.Name}

<C10>{=IfKnownString(.Invoice.PaymentAgreement.Party.AddressLine1)}
<C10>{=IfKnownString(.Invoice.PaymentAgreement.Party.Municipality.Country.Code) + "-" + IfKnownString(.Invoice.PaymentAgreement.Party.Municipality.PostalCode) + " " + IfKnownString(.Invoice.PaymentAgreement.Party.Municipality.Name)}

<C#1><R10><C10><I>Si le malade n'est pas l'asssure</I>:
<C#1><R11><C10><FROM><R14><C80><RECT>
<C#1><R11><C10>{=IF EnumeratedToString("RelationType", .Invoice.PaymentAgreement.Relation) <> "Holder"
THEN .Invoice.Order.Object.Person().LastName ELSE ""}

<C30>{=IF EnumeratedToString("RelationType", .Invoice.PaymentAgreement.Relation) <> "Holder" THEN DateToString(.Invoice.Order.Object.Person().BirthDate, "%d %m %y") ELSE ""}

<C#1><R5><C80>CODE ORGANISME AFFILIATION
<C#1><R6><C90><FROM><R8><C150><RECT>
<C#1><R6><C90>{=.Invoice.Payer.InternalId + " " + IfKnownString(.Invoice.Payer.Identification("RIZIV"))}

<C90>{=IfKnownString(.Invoice.Payer.Name)}
<C90>{=IfKnownString(.Invoice.Payer.AddressLine1)}
<C90>{=IfKnownString(.Invoice.Payer.Municipality.Country.Code) +
"-" + IfKnownString(.Invoice.Payer.Municipality.PostalCode) + " " + IfKnownString(.Invoice.Payer.Municipality.Name)}

<C#1><R10><C80>CODE ORGANISME COMPLEMENTAIRE
<C#1><R11><C90><FROM><R14><C150><RECT>
<C#1><R11><C90>{=.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.InternalId + " " + IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.Identification("RIZIV"))}

<C90>{=IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.Name)}
<C90>{=IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.AddressLine1)}
<C90>{=IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.Municipality.Country.Code) +
"-" + IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.Municipality.PostalCode) + " " + IfKnownString(.Invoice.OrderSet.LinkedFinance(2).PaymentAgreement.Fund.Correspondent.Municipality.Name)}

<C#1><R16><C40>PARTIE RESERVE AU LABORATOIRE
<C#1><R16><C0><FROM><R17><C150><RECT>
<C#1><R18><C90><I>Prescripteur</I>{=IfKnownString(.Invoice.Order.Issuer.Name) + " " +IfKnownString(.Invoice.Order.Issuer.Identification("RIZIV"))}
<C#1><R30><C20>ACTES DE PRELEVEMENTS <C90>FRAIS ACCESSOIRES
<C#1><R30><C0><FROM><R31><C150><RECT>

<R33><C10>(prelev 5)=> {= .Invoice.OrderSet.BCData(?,?,"France_Prelev",?,"GrossAmountTotal")}
<R33><C35>(IF,IFD 6)=> {= .Invoice.OrderSet.BCData(?,?,?,"IF,IFD","GrossAmountTotal")}
<R33><C60>(ID 7)=> {= .Invoice.OrderSet.BCData(?,?,?,"ID","GrossAmountTotal")}
<R33><C85>(IK 8)=> {= .Invoice.OrderSet.BCData(?,?,?,"IK","Quantity")}
<R33><C110>(IK 9)=> {= .Invoice.OrderSet.BCData(?,?,?,"IK","GrossAmountTotal")}

<C#1><R35><C40>ACTES DE BIOLOGIE MEDICALE
<C#1><R35><C0><FROM><R36><C150><RECT>
<C#1><R35><C0><FROM><R50><C150><RECT>

<R36><C2><I>Actes effectues <C40><100% <C50>100% <C60>HN <C80>Montant des actes y compris supplements</I>
<R38><C2>{= .Invoice.OrderSet.BCData("ClasseFrN",?,?,?,"NomenclatureCodesAndBValues")} <C40>{= .Invoice.OrderSet.BCData("ClasseFrN",?,?,?,"GrossAmountTotal")}

<C2>{= .Invoice.OrderSet.BCData("ClasseFr100%",?,?,?,"NomenclatureCodesAndBValues")} <C50>{= .Invoice.OrderSet.BCData("ClasseFr100%",?,?,?,"GrossAmountTotal")}
<C2>{= .Invoice.OrderSet.BCData("ClasseFrHN",?,?,?,"NomenclatureCodesAndBValues")} <C60>{= .Invoice.OrderSet.BCData("ClasseFrHN",?,?,?,"GrossAmountTotal")}

<R49><C2><I>Total des coefficients</I> <C40>{= .Invoice.OrderSet.BCData("ClasseFrN",?,?,?,"GrossAmountTotal")} <C50>{= .Invoice.OrderSet.BCData("ClasseFr100%",?,?,?,"GrossAmountTotal")} <C60>{= .Invoice.OrderSet.BCData("ClasseFrHN",?,?,?,"GrossAmountTotal")}

<C#1><R38><C80>{=.Invoice.OrderSet.BCData(?,?,?,?,"GrossAmountTotal")}

<C80><I>Part organisme obligatoire:</I>
<C80>{=.Invoice.OrderSet.BCData(?,"Fund",?,?,"GrossAmountTotal")}
<C80><I>Part organisme complementaire:</I>
<C80>{=.Invoice.OrderSet.BCData(?,"Fund 2",?,?,"GrossAmountTotal")}

<R48><C80><I>Date de la prescription: <C110>execution</I>
<R49><C80>{=DateTimeToString(.Invoice.Order.PrescriptionTime, "%d/%m/%Y")}
<R49><C110>{=DateTimeToString(.Invoice.OrderSet.TariffTime, "%d/%m/%Y")}

<C#1><R55><C5>LABORATOIRE .....

<C5><B>REF No {=.Invoice.Order.Object.Person().Correspondent.InternalId} {=DateTimeToString(.Invoice.Order.ReceiptTime, "%d/%m/%Y")} - {=DateTimeToString(.Invoice.Order.ClosureTime, "%d/%m/%Y")}</B>
<C5>Prelevements {=.Invoice.OrderSet.BCData(?,?,"France_Prelev",?,"GrossAmountTotal")}
<C5>Deplacement {=.Invoice.OrderSet.BCData(?,?,?,"IK","GrossAmountTotal")}
<C5>total analyses {=.Invoice.OrderSet.BCData(?,?,?,?,"GrossAmountTotal")}
<C5>part non rembourser {= .Invoice.OrderSet.BCData(?,"Object person",?,?,"GrossAmountTotal")}
<C5>part a regler exterier ???
<C5><B>PART A REGLER {= .Invoice.OrderSet.BCData(?,?,?,?,"GrossAmountTotal")}</B>
