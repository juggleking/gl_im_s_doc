Facturation: polices d'assurances

Les polices d'assurances permettent de définir les différents mécanismes de remboursement.  Ils sont accessibles par Start-> Cotation -> Configuration -> Polices d’assurance -> Polices d’assurance.

 

  • Liste de suppléments: Expression MISPL qui permet de compter le B3 « Frais de dossier (sang) » ainsi que les compléments ‘<B20’.
Type : STRING
STRING CodeSupplement;
Fractional BtotalSang ;
BTotalSang := StringToFractional(.BCIData(?,?,"Sang",?,?,?,YES,"BaseValueTotal"));
IF BTotalSang < 20
THEN 
  CodeSupplement := RangeLabel((20 - BTotalSang), "5:9905,10:9910,15:9915"); 
ELSE
  CodeSupplement := "";
ENDIF;
CodeSupplement := CodeSupplement + ",9003";
RETURN TRIM(CodeSupplement,",");

‘Sang’ est le mnémonique du ‘Groupe Code Cotation sécu’ à configurer de la manière suivante :  Start -> Cotation -> Configuration -> Groupe Code Cotation ss.
