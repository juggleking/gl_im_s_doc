KV Specifika files

The KV-Specifika files are not loaded in GLIMS tables, but are loaded by processes in temporary tables on a need-to-know basis. The directory were the KV-Specifika files are placed must be specified in the environment variable GLIMSBILLDAT. This directory must be accessible by all clients!! So not only the server machine, but also all PC clients. If no shared directory is available for this purpose, then the KV-Specifika files must be copied to each PC.

When GLIMS needs a KV-Specifika file for an OKV and a quarter QJJ:

  • it will look in the GLIMSBILLDAT directory for a file names kvsp<OKV>_<version>.QJJ. If there are multiple, it will take the one with the highest <version> number.
  • If there are none it will first try to find a file with extension PRF: this was only important for the KV-DT Pr√ºfung.
  • If there is still no file found, it will continue its search with the previous quarter (and further if needed).
