Epidemiology - Isolation discriminator

Contents: Introduction - Configuration - Usage

Introduction

In order to better distinguish different strains of the same micro-organism, a field is available on isolations, labeled "Discriminator". Note that discussions might arise about the correct terminology, but in absence of consensus this term was chosen, and it does not change anything to the functionality.  The "discriminator" is considered as a text that represents some more detailed information about the micro-organism that was selected for that isolation.

Configuration

The discriminator text will be automatically calculated by a MISPL function and this calculation can be based on all information known in GLIMS about the isolation, including isolation tests and antibiogram.

In the micro-organism configuration the field "Function to discriminate similar isolations" can be used to enter a site-function, based on table "Isolation", which calculates a text that will be used as the discriminator of the isolation.

Usage

  1. When the mechanism for the isolation doubles detection is run (either when validating a microbiology action or in batch during the night), GLIMS will calculate the discriminator of the isolations and store it in the "discriminator" field on the Isolation record.
      
  2. When the mechanism for the isolation doubles detection also creates
     
    • New isolation series
    • New cross infections
    • New hospital infections

    the discriminator is also copied to these new records. 

  3. When GLIMS tries to decide if 2 isolations on the same patient are identical, it will also compare their discriminator fields and if those fields are different then the isolations will be considered as different.
     
  4. When GLIMS tries to decide if 2 isolation series on 2 different patients are identical and should be linked by a cross infection, it will also compare their discriminator fields and if those fields are different then they will be considered as different and no cross infection will be created between these isolation series.
