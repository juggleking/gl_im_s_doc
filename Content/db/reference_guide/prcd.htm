Procedure data prcd glims yes yes Normal

For each procedure defined in GLIMS, you can add procedure data. Procedure data allows to show extra information in other GLIMS programs such as work list grids, the cell counter or send extra information in analyzer connections.

General > History index

Can be specified if a historical value is needed. In case of a history index of "0", the program searches for a result of the given property, object and object time. When there are 2 orders for the same object time, each containing this property, preference will be given to the result of the same order as the original action.

General > Show

This field allows to define whether a ProcedureData item (e.g. NA(-1) to see the previous result of NA) is to be shown on the work lists if the corresponding property is not an output of this action (i.e. is not requested today). The allowed values are:

- Always: this is actually the original behaviour. The previous result is shown even when no request was made today.
- When Requested Today: the previous result will not be shown if the current action has no output for the same property.

Mandatory ShowProcedureData Show

One of: Always, When requested today
