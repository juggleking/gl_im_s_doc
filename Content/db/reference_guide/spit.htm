Description of specimen identification type

Mandatory String Description

Early receipt days

This field allows finding specimen identification codes (barcodes) that arrive in the lab earlier than the validity start date of the identification code.
This is useful when orders are planned for the future and it might happen that a specimen arrives earlier in the lab.

The 'EarlyReceiptDays' will have effect on the specimen receipt program and on station instruments that are querying or sending results for a barcode.
Take for instance a specimen that has been planned for 3 days further than today. When setting 'EarlyReceiptDays' to '2', Glims will 'recognize' the barcode from tommorow on.
