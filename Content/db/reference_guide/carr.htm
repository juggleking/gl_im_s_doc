AddIsolation 

Logical AddIsolation (Mnemonic OrganismMnemonic)

Mnemonic OrganismMnemonic
Optional: mnemonic of the organism to be used for the new isolation.
If not specified, the 'initial organism' of the microbiology procedure will be used. If this one is not specified either, an error message will appear.

AddIsolationWithAppraisal 

Logical AddIsolationWithAppraisal (Mnemonic OrganismMnemonic, Mnemonic Appraisal)

Mnemonic OrganismMnemonic
Optional: mnemonic of the organism to be used for the new isolation.
If not specified, the 'initial organism' of the microbiology procedure will be used. If this one is not specified either, an error message will appear.
Mnemonic Appraisal
Optional: Mnemonic of the isolation appraisal to be used for the new isolation.
If not specified, no appraisal will be given to the isolation created.
