Genetic request definition gntr glims yes yes Normal

This tabel stores the genetic sub type of the request definition.
A genetic request definition is linked to a request definition having type ‘Genetics’.
It refers to a certain ‘GeneticProcedure’. Requesting this means that the genetic procedure will be requested and a genetic exam for that genetic procedure will be created.
