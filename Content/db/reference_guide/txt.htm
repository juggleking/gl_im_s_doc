The table of texts contains records for reusable text modules which may include variable components, such as symbolic references to other Text records or MISPL code.

Table

Is an optional reference to a table where the text is scoped to. A text must be scoped to a table if its Text field contains dynamic text (embedded MISPL expressions or references to other table-scoped Text records). For example, if a text is scoped to the User table, it can contain a variable reference to the user login name, as in "The user's login name is {=.LoginName}".

Mandatory rt_Table Table
