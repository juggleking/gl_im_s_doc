Trigg check tgc glims yes yes None

This table stores the actual parameters of the QC Trigg checks. The Trigg Tracking Signal can be carried out on the results belonging to a specific quality control population. This check has a start time: only results after this time are processed. This later allows to recalculate the TrackingSignal, using StartTime and the initial values. The check is violated if the absolute value of TTS (TrackingSignal) exceeds Template.Limit.
