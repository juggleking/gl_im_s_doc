Introduction

The Entity Relationship Diagrams help to explain the relationship between the different tables in GLIMS. They give a deeper insight into the relational database structure of GLIMS. Each entity refers to a table described in the reference guide. These diagrams can be very useful when building MISPL functions.  You can click each entity to go to the corresponding page of the reference guide.
