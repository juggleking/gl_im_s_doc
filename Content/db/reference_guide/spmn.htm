AddCarriers 

This MISPL function allows adding one or more carrier(s) to one or more microbiology action(s) on the current specimen.

Caution:
- Every parameter may have the value "?" except the second one: at least one medium mnemonic should be specified.
- Unfilled logical parameters are by default interpreted as NO.

Logical AddCarriers (String MicrobiologyMnemonic, String MediumMnemonicList, Logical Print, Logical MarkAsGrafted, Logical DoubleAllowed, String Comment)

String MicrobiologyMnemonic
Allows to specify the mnemonic of a microbiology procedure.

* If a mnemonic is specified, the carrier(s) will be added to the microbiology action derived from that microbiology procedure.
* If no mnemonic is specified (value is "?"), the carrier(s) will be added to all the microbiology actions of the current specimen.
String MediumMnemonicList
Comma-separated list of the mnemonics of the media to be automatically added. It should contain at least one mnemonic. It should not contain any white space.
Logical Print
If the value is YES, the label(s) of the added carrier(s) (if defined) will be printed.
Logical MarkAsGrafted
If the value is YES, the added carrier(s) will be grafted and the grafting time will be set to the current date and time.
Logical DoubleAllowed
Usually two carriers with the same medium cannot be added to one and the same microbiology action. However, if the value is YES, it will be possible to add two carriers with the same medium.
String Comment
The comment for the added carrier(s).

SpecimenReviewObjective Objective
One of: Review, Confirm, Validate, Confirm + validate
Pull-down selection list with 4 choices: 'Review', 'Confirm', 'Validate', 'Confirm + Validate'. The 'Review' option is intended for a kind of 'work status' query, allowing to query for specimens which are not yet available, or for which there are no or lacking results. The effect of the other options is obvious. They will cause the appropriate specimens to be selected, and the eligible results to be pre-selected for confirmation, validation or both.
