QCDeviationAccuracyUsage Help | Fields

Value Enumerator Abbreviation
1 Ignore Ignore
2 Calculate limits CalcLimits

Options on how to interpret and apply the incoming Deviation-field (M.5.1.1) of an ASTM QC result record (M.3.1.1 = QC) when used for the accuracy of a qc population.

- Ignore: indicates that GLIMS should ignore this value if it comes in
- Calculate limits: indicates that GLIMS should use it to calculte the accuracy limits. The deviation is used as a multiple of the "qc norm deviation count" (set on the assessment method or as GLIMS QC setting, default is "3") to calculate the accuracy low- and high-limit of a qc population as follows:
Accuracy low limit = TargetValue - <QC Deviation count> x SD
Accuracy high limit = TargetValue + <QC Deviation count> x SD
(<QC Deviation Count> = QCNormDeviationCount on assessment method OR QCNormDeviationCount on Specific site OR 3)
