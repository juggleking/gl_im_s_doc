Westgard check template wgct glims yes yes Normal

This table defines a Westgard (1) type check that can be carried out on the results belonging to quality control populations.

1. Trend detection in Control Data: Optimization and Interpretation of Trigg's Technique for Trend Analysis", Clinical Chemistry, Vol 21, No 10, 1975.

The Deviation count is the number of standard deviations (also called Sigma) for violation limit.

Mandatory PositiveFractional DeviationCount

Primary key. Automatically generated identifier whose value uniquely identifies each record.

Mandatory Unique Automatic WestgardCheckTemplate Id

gp_Text InternalComment

Last update time

Automatic DateTime LastUpdateTime

Last update user

Name

The Name identifies the record of the Westgard check template.

Mandatory Unique String Name

One result per shift

Indicates that only one result per shift (i.e. the first of the shift) should be taken into account for this check. Also, this implies that all other results are not checked.

Mandatory Logical OneResultPerDay

Placing

Mirrored: when a control measurement (result) in a population exceeds the mean plus and another the mean minus control limit. Example Westgard rule R4s.

Gathered: when a control measurement exceeds a mean plus or mean minus but falling on one side of the mean. Example Westgard rule 6x.

Summed: when the sum of two or more control measurements exceeds the rule.

N/A: no specific placing is applied

Mandatory WestgardPlacing Placing

One of: N/A, Mirrored, Gathered, Summed

Population count

The Population count specifies the number of populations involved in this check (minimal 1).

Mandatory PositiveInteger PopulationCount

Severity

The Severity specifies the severity of the result when the check is violated

Mandatory PositiveInteger Severity

Value count

The Value count indicates the number of results to be taken into account per QC population.

Mandatory PositiveInteger ValueCount

Version

Record version number. Automatically incremented each time the contents of a record changes.

Automatic PositiveInteger Version
