Price formula prcf glims yes yes Normal

Each price formula record stands for a formula to determine an effective price. It consists of a fixed and proportional term. Every price formula record has a limited validity period.

Valid until

This field indicates the validity period's end date. The price formula is valid up to and including the specified end date. If no end date is specified, then the validity period is implicitly ended (excluding) at the successing record's start date. The successor is the next price formula record for the same price code in order (ascending) of start date. If there is no successor, the validity period is considered as not ending.

Multiplier

This field contains the base value multiplier for the price calculation. Suppose 'B' is the corresponding base value:
Price = B * Multiplier + FixedAmount.
Note that a fixed price system can be configured by setting the multiplier to 0.
