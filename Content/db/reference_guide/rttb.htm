This table is accessible at run-time and contains
all entries for tables accessible at run-time.

Fields

Id

Primary key. Automatically generated identifier whose value uniquely identifies each record.

Mandatory Unique Automatic rt_Table Id

Name

Mandatory Unique Automatic String Name

Short name

Temporary fix for oracle : shortname max 5 chars

Mandatory Unique Automatic String ShortName

Type

'Permanent' for database tables,'Temporary' for temp-tables

Mandatory Automatic TableStorage Storage

One of: Permanent, Temporary

Version

Record version number. Automatically incremented each time the contents of a record changes.

Automatic PositiveInteger Version
