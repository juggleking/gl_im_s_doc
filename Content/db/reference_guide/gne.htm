Gene gne glims yes yes Normal

List of genes (gene symbols). This is data that is usually imported/updated from official organizations (like HGNC).

Fields

Gene > Alias symbols

List of other symbols used to refer to this same gene.
Synonyms may be from the literature, from other databases or may be added to represent membership of a gene family.

String AliasSymbols

Gene > Chromosome

Reference to the chromosome where the gene is located.

Mandatory Chromosome Chromosome

Gene > Chromosome region

Reference to the chromosome region where the gene is located.

ChromosomeRegion ChromosomeRegion

Creation time

Automatic DateTime CreationTime

Creation user

Automatic sc_User CreationUser

Gene > Date approved

Date that the gene was approved officially.

Date DateApproved

Gene > Gene family

Name given to a gene family or group the gene has been assigned to.

String GeneFamily

Gene > HGNC id

A unique identifier created by the HGNC for every approved symbol.

String HgncId

Gene > HGNC status

Status of this gene symbol in the HGNC table.
It can have following values:
- “Approved” : the symbol is HGNC-approved
- “Entry withdrawn”: the symbol was previously approved, but is no longer thought to exist.
- “Symbol withdrawn”: the symbol was previously approved, but has been merged into (replaced by) another symbol.

HgncStatus HgncStatus

One of: Approved, Entry withdrawn, Symbol withdrawn

Gene > Previous symbols

Symbols previously approved by the HGNC for this gene.
Many genes will have no data in this field as the symbol or name will never have been changed.

String PreviousSymbols

Gene > Approved symbol

The official gene symbol that has been approved by the HGNC and is publicly available.
Symbols are approved based on specific HGNC nomenclature guidelines.

Mandatory String Symbol
