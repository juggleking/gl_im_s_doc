Approach plan aprp glims yes yes Normal

Defines a plan containing multiple activities which can be executed in the context of a genetic exam.
In an approach plan the geneticist defines the full workflow of the laboratory including the analytical strategy, according to
• the technical actions to be performed
• genes (loci) to analyze,
• the clinical context
• the patient status.

Details > Expected TAT (in days)

Expected duration (in days) between start date (date of scheduling the approach plan) and end of executing it.
This field is manually entered and is an estimation by the configurating user. It can be worst case, best case, average or any meaning.
The real meaning is left open to the configurator.
