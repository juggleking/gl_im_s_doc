Panel request definition pnrb glims yes yes Normal

Panels are request definitions grouping other request definitions. All request definitions in a Panel can be requested simultaneously by specifying the panel mnemonic.

Fields

Accept shared members

This field is evaluated during the panel completeness control (during the tariffication). When a panel is requested during order entry, all its members are instantiated if the corresponding results (microbiology actions) are not yet present in the order. If one of these members is already present, can the panel then be considered as complete or not? This may be important because only complete panels can be tariffed.

Mandatory Logical AcceptSharedMembers

Charge

Indicate how the panel should be billed:

Members only: only the panel member requestables will be evaluated for tariffication
Panel only: only the panel requestable will be evaluated for tariffication
Panel and members: both the panel requestable and the panel member requestables will be evaluated for tariffication
Members or panel: the panel requestable will only be tariffed if all panel member requestables are not to be tariffed because they have not been requested with a billing mark that is included in the "Tariff member billing marks" field of the panel requestable.

PanelCharge Charge

One of: Members only, Panel only, Panel and members, Members or panel

Confirm

When a panel is requested, and this option is enabled, the expanded panel is first shown to the user in a separate window using an updateable browser. In this browser, the user can click on each panel member to add it to or remove it from the panel. The user can also modify the panel member's discriminator (an object time) using the standard syntax. Whether a panel member is included in the panel by default can be decided by the field PanelMember.Optional, which has a default value of FALSE, indicating that it is included in the panel by default. The selected panel members are not requested until the user commits the panel preview browser.

Mandatory Logical Confirm
