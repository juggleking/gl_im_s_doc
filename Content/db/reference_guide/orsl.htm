Sub id

* Type 'Label'
- Incrementing label sequence number
* Type 'Result'
- Incrementing result sequence number
* Type 'Isolation'
- ? : when the microbiology research is negative (.Value will contain the reason why) or the Isolation data can not yet be reported (.Value will contain the 'empty result text').
E.g. the Results status is lower then the Report.MinimalResultStatus,
- >0 when an Isolation is reported (= incrementing isolation sequence number)
* Type 'Antibiotic'
- ? : antibiogram cannot be reported yet, status too low (.Value will contain the 'empty result text').
- >0 when an Abtibiotic is reported (= incrementing antibiotic sequence number)
- 1 : Start of an antibiogram and the current record will be the first antibiotic.

PositiveInteger SubId

Type

Type of result (normal, isolation or antibiotic).

Please note that the type is set to 'Result' when the Generator in the report template is set to HL7, HPRIM, EDI or EDIMDVI. For those generators, the report template option 'Structured microbiology' is not supported. The data type of the linked property is therefore not taken into account.

Mandatory ObservationalResultType Type

One of: Label, Result, Isolation, Antibiotic
