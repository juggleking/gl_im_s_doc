Genetic exam gnte glims yes yes Normal

Request of type ‘Genetic’ will cause a ‘Genetic Exam’ to be created.
The genetic exam is an instance of a certain GeneticProcedure.

LocusResult GetLocusResult (String LocusName, LocusResultStatus MinimalStatus, Integer MinimalSeverity, Integer Index)

String LocusName
Only LocusResults will be selected where LocusResult.Locus.Name = LocusName if <> ?, otherwise the LocusName is not taken into account
LocusResultStatus MinimalStatus
One of: Expected, Available, Confirmed, Validated, Discontinued
Only LocusResults will be selected where LocusResult.Status is at least this status; if not specified, LocusResultStatus["Expected"] is assumed. Previously this was LocusResultStatus["Validated"]. This means that users that depend on the old behavior by specifying ? must explicitly state LocusResultStatus["Validated"] in their current MISPL.
Integer MinimalSeverity
Only LocusResults will be selected where LocusResult.Severity >= minimal Severity
Integer Index
This is the only mandatory parameter which must be <> 0. This is a loop counter variable:
- when 1, the first locus result is returned (in sequence of field availability time)
- when 2, second result etc
- ... until the return value becomes unknown.
- when -1, the last locus result is returned (in sequence of field availability time)
- when -2, the one but last result is returned etc
- ... until the return value becomes unknown.
