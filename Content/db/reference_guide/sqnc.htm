Transcript

Reference sequence; more specific location on the gene

When a variant is created and a gene is chosen (or changed to another gene), this field will be filled automatically with the Default Transcript string of the chosen Gene record.
The user can modify this directly if he wants into one of the other transcripts of the gene, or into a totally different transcript not registered for the gene. (of course this will not change the transcript value of the gene itself).
