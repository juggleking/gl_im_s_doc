This table stores which material subtypes can be output by a given procedure.

Fields

Auto login

If set, the specimen status will automatically be promoted from 'Expected' to 'Available'.

Mandatory Logical AutoLogin

Behavior in genetics approach plan > Number of specimens to be auto requested

[used for genetics]
Number of specimens to be produced in an activity of an approach plan
Default is empty; which means that the material is not auto-requested.
If not empty (1 or higher), this indicates that this material must be added automatically when the procedure must be added for an activity in an approach plan.
If the value is 1, the new action that is created will have a specimen of this material as output.
If the specified value is higher than 1, it indicates how many identical specimens will be produced for that material. This means that –when the procedure must be added for genetics- glims should add this number of specimens (of this material) to the order, as output of an new action for this procedure.
