Work lists offer a technical, procedure-oriented view on the work to be performed by the laboratory. Because each laboratory department has its own specialization, most often it will use its own set of work lists.

Work lists are typically created with the work list generation function, which is available from the main menu.

Edit 

The Worklist spinner allows specification of the work list to be edited, by name.

Upon committing the dialog, the selected work list is visualized in Grid view for result editing

The rows in the work list grid correspond to actions. An action is the realization of a procedure. It appears on a work list because the procedure appears in the work list template. The columns in the work list grid correspond to action outputs, which are typically properties. Results occur in cells at row/column intersections.

The row labels are obtained by evaluating, on the associated action, the action label expression stored in the work list template. If it isn't specified there, the specification in the GLIMS customization screen is used. If it isn't specified there either, GLIMS still builds a default label based on action inputs or object references.

Result cells are enabled for input until the result is confirmed. A result still to be confirmed has a white cell background, whereas a confirmed result cell is grey. Rows and columns can be selected together to speed up result confirmation, for instance. The Refresh button updates the active work list. If a work list still expects results from an analyzer, refresh the work list before your close it, to see if they already appear on the work list.

Results with severity greater than zero are automatically highlighted. Results which haven't been requested are prefixed with a '@'-character. Results having internal or external comments are prefixed with a '*'-character. On windows clients, part of the comment will be visible via the 'tooltip' by holding the mouse cursor over a result cell for a few seconds. Results which have been discontinued are enclosed with opening and closing brackets.
