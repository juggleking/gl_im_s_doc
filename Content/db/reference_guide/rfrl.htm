This table stores amongst others the data of the German 'Überweisungsschein' that has no corresponding place in other Glims tables. Also specific data for other 'Schein' types like "Notfall" are in this table.
Note that a referral sent to a subcontractor ('UnterÜberweisungschein') is not a GLIMS referral, but only a printed form of a worklist!

Referral to

'Uberweisung an' field from the 'überweisungsschein' in Germany. If this field is not filled in, it will automatically default to the value "Laborarzt" on the KV-DT tape.

String ReferralTo

Referral sub group

From the many 'Scheinuntergruppen' that are available only two are typically used by labs. Type 21 (Order or 'Auftragsleistungen') was the type to be used before 1/7/99. Starting from 1/7/99 a new type of 'Überweisungsschein' (muster 10) is to be used; this is type 27. Also two notfall types are supported.

Mandatory ReferralSubGroup SubGroup

One of: Ordered, Referral, EmergencyDefault, Emergency, LG Direktabrechung, Konsiliar, Co-/Aftertreatment

Referral type

In Germany this is the 'Kurativ/Präventiv/Sonstige hilfen/...' indication from the 'überweisingsschein'. This is an important field in combination with the EBM-laborreform from 1/7/99!

Mandatory ReferralType Type

One of: Curative, Preventive, ESS, Hospital
