Blood transfusion > Irregular antibody screening > Latest screening on

This field is meant to store the date-time of the last time the patient was screened for irregular antibodies.
It can be set from within MISPL functions by calling Person.SetMedicalRecord("LastScreeningTime", <date-time>). How this field is actually used (in which cases it is or it isn't changed) is actually the choice of the lab. It is meant to be used by e.g. the 'Before selection' trigger, or e.g. the eligibility MISPL on the 'Antigen-antibody check' (in the blood product).
