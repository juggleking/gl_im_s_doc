Fields

Archive where stored

The lab archive in which this rack is curently stored. This does not automatically imply that the rack is currently stored in a part of this archive.

In combination with the code, the archive will be used to determine the correct rack in the specimen archive scan program. This reference allows a direct connection between a rack and its archive, without the need to assign the rack to an archive part. Whenever the rack is put in another archive part, this field will be synchronized. It can however also be filled manually, stating that the rack is intended for a specific archive.
