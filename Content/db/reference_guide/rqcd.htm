Request code rqcd glims yes yes Normal

Request codes are codes that can be used to request something in the order entry module of GLIMS. What is actually requested is determined by the request definitions that are linked to the request code.

Electronically explicitly requestable

Determines whether or not the request definition can be explicitly requested or not during electronic order entry. This feature can be used, for example, to disable the explicit requesting of panel members.
Contrary to Explicit, which is about explicit requestability during manual order entry, this field is about whether or not a requestable can be explicitly requested electronically.

Explicitly requestable

Determines whether or not the request definition can be used explicitly during manual order entry. This feature can be used to disable the explicit requesting of panel members, for example.
INFO: A non-explicit request can always be added via MISPL-methods like Order.AddRequest()
