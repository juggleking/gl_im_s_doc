Max Exceeded

A billing rule line may have several outputs (billing codes). Which of those outputs are applied, depends on the fact whether the billing rule's maximum translations is yet exceeded or not. If that number is exceeded, only the outputs for which the 'Max exceeded' will be applied, and vice versa. Billing rule outputs for which this field is '?' are applicable in both cases.
