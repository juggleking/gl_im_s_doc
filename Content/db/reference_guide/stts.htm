This table is used as base table for text modules which are used as document header or table header in statistics reports.

Fields

Contents

This field contains a description of the options that were chosen in the first input screen (the layout screen):
- the rows/columns/multi-table classification
- the cell contents (characteristic value + calculation selection)

Mandatory String Contents

Period begin

This field contains the beginning of the period which has been selected in the statistics run. It is a date plus an optional time indication (if the time was specified).

Mandatory String PeriodBegin

Period end

This field contains the end of the period which has been selected in the statistics run.
It is a date plus an optional time indication (if the time was specified).
If no end date or time was selected, the indication '(no end date)' will appear.

Mandatory String PeriodEnd

Selection

This field contains a description of all criteria that have been selected to filter the records when doing the statistics run. (in the second input screen)

Mandatory String Selection

Table subject

This field only contains a value for the table headers, when using the multi-table option.
It contains the mnemonic/name/description of the item which is specific for the separate table. It conbtains the same value as would appear in the rows of columns for the same classification type.
Example: if multi-table = "properties" then a separate table per proprty will be displayed and this field will contain the property name each time the table header is to be shown.

Mandatory String TableSubject
