Choice list chcl glims yes yes None

A choice list contains a number of choice records which can be chosen during a particular operation. For instance, when a property of data type 'enumerated' refers to a choice list, the list conatins a acceptable result values for the property.

Merge into (replace by) another choice list 

This function can be used to merge 2 choice lists together.
You should select the old list first, then call this method and select the new list, which will be used from now on instead of the old list.
All properties, isolation tests etc. that referred to the old list will now refer to the new list.

Void Merge (ChoiceList TargetList, Logical ReplaceReferences)

ChoiceList TargetList
The choicelist in which the old list will be merged
Logical ReplaceReferences
Indicates if the configuration records referring to the original choice list should refer to the new choice list after the operation.
Examples:
Property.ChoiceList, Medium.ChoiceList, MediumTest.ChoiceList, MicroOrganismTest.ChoiceList.

Note that SpecificSite.ExplicitNegAnswerList, StudySubject.ObjectRoleChoiceList, StudySubject.ConclusionChoiceList and Choice.ChoiceList will NOT be redirected, regardless of this parameter.
