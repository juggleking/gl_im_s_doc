WorkListType Help | Fields

Value Enumerator Abbreviation
1 Complete Compl
2 Only new New
3 Continuation Cont
4 Continuation-reset ContReset

Determines which actions are put on a new work list version:
- 'Complete': all actions are placed on the work list
- 'Only new': only actions that are not already on a work list (of the same template) are placed on the new work list
- 'Continuation': only actions that are not on a work list of today are placed on the new work list. The numbering of the new work list starts with the last number of the previous work list of today + 1.
- 'Continuation-Reset': same as previous but numbering starts from 1.

For continuation work lists (with continued numbering), any unfinished QC actions at the end of the last worklist of the same day are deleted. Next, the QC schedule of the new work list is added as if the new entries where really added to the previous work list, i.e. there is no start QC, and the QC cycles continue at the same position where the previous work list ended.
