StationSpecimenVariables Help | Fields

Value Enumerator Abbreviation
1 Only source body site Only body site
2 All All

Determines which specimen variables should be downloaded to the station.

The options are:

1) Only source body site:
Only the specimen variable known as the specimen source body site (E.g.: left
arm, left hand, right lung) is downloaded.
The source body site variable is identified by the material variable for which
the name equals Station.stn_SourceBodySiteName. In case Station.stn_SourceBodySiteName is not specified, Glims will assume that the body site variable is
the one with the lowest sequence number (backwards compatible).

2) All:
All specimen variables are downloaded.
If the specimen source body site is identified with
Station.stn_SourceBodySiteName, Glims will download this specimen variable as
such. All other specimen variables are download as non specimen source body
site variables.
If the Station.stn_SourceBodySiteName is not specified, Glims assumes there
is no source body site and will download all specimen variables as non specimen
source body site variables.
