Text reporting > Postamble

The name of a file that will be printed, using the print command, after the actual print job. The print program will search for the postamble file in the PROPATH variable. If the file cannot be found, a message will be displayed and printing will continue, but without postamble.

String PostAmble

Text reporting > Preamble

The name of a file that will be printed, using the print command, before the actual print job. The print program will search for the preamble file in the PROPATH variable. If the file cannot be found, a message will be displayed and printing will continue, but without preamble.

String PreAmble
