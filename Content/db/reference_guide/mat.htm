Is root material

Using the member Material.IsRootMaterial indicates to the system that this material can be received by the lab. During order entry, GLIMS allows to explicitly request a material on which the tests should be executed. This will only be possible for root materials.

Logical IsRootMaterial

Size > Maximal size

Specifies the maximal filling for a container type, expressed in the size unit. During order entry, GLIMS will determine the required volume of the materials in order to be able to execute all the tests. If this exceeds the maximal size of a material, additional labels will be printed for all the required tubes.

PositiveFractional MaximalSize

A period, expressed in days, during which a specimen can be reused. In the order entry options, one can indicate that GLIMS is allowed to reuse the specimens of a patient.

A sampling-related short code, mainly used for billing purposes.
