The Report Scope can be one of the following options: Order, Encounter, Day, Period, Object, Year, Study, Ward. It imposes criteria on the results to be included in the same report.

Values:

- Order: All results of a particular order will be displayed on the report. This is the standard report type.

- Specimen: All results of the same root specimen are put on the same report.
Results which are not linked to a specimen (e.g weight of the patient) wil be included on each report

- Encounter: All results wihtin the same encounter.

- Day: all results of the same day.

- Year: all results obtained for a patient during an entire year.

- Object: all results for the same patient.

- Study: all results of orders related to the same study. Each order is printed on its own report page.

- Ward: all results for the same ward and the same day

Remarks:

For Encounter, Year, and Object reports, a single Report Group contains all the report pages releated to the same encounter, year or object. Each page holds as many report columns as defined in the report template.

For Order, Specimen, Day and Study reports, a single Report Group contains a separate page for each order or specimen.

The column count defined in the report template determines how many cumulative orders or specimens can be added to the original columns (when DefaultReport.PreviousResults <> None).
