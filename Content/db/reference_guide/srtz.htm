A sorting zone defines the physical location on a sorting (output) area of a LAS. Typically this is a square area on which trays are placed holding specimen carriers.
The codes of a sorting zone can be used in communication with LAS systems, enabling Glims to control the position were specimen should be sorted or aliquotted, and storing information about the specimen's sorting result.

Type

The type of sorting zone indicates what the usage of the zone is.

Most of the zones will be used for routine work, type 'routine'

Others are:
error - in case sorting failed
archive - work for specimen is finished and can be stored in an archive
external - for work to be carried out outside the lab
urgent - for urgent specimens
