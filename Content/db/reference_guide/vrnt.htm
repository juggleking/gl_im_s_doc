Found internally

Means that the variant has already been found through a specimen managed by the own lab. Note this can be Glims, but also a predecessor lab system.
So this is not really an attribute that can always be calculated on the fly.
When the variant has been detected using Glims, there will be a GeneticExam where this variant is detected and stored as a result.

So as soon as the variant appears as ‘present/detected’ in at least 1 GeneticExam, this field will be automatically set to TRUE, and it can no more be set to FALSE by the user (so it should be come insensitive).

Mandatory Logical FoundInternally
