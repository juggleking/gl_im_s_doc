Specimen sorting target sst glims yes yes None

This table links procedures with sorting zones of a distributor. It allows to define the possible targets on which a specimen can be sorted, depending on: the definitions of the procedure itself (input and outputs), the urgency of specimen, the need of archiving a specimen,...
