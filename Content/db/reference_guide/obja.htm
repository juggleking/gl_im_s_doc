Object attribute obja glims yes yes Normal

Object attributes specify all the possible attributes for an object (patient). By means of object attributes, extra diagnostic information can be stored and taken into account during result validation, norm interpretation, tariffication. In the order entry page, the object's active attributes can be shown and modified. The attributes can either be activated by typing in a time period the attribute is already active, or by typing in a start (and possible end) date.
