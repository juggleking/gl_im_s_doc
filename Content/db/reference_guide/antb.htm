Temp table w_AntibioticComment:
This temp table contains fiels antb_Antibiotic and antb_Specimen. The goal of the temp table is to run a mispl on it for configuring antibiotic comments on the antibiotic result. Users can, in the mispl, retrieve information about the antibiotic and the specimen, to make the antibiotic comment dependent of this information.

Fields

Antibiotic

This field indicates the antibiotic. Users can, via mispl, retrieve information about the antibiotic.
