Conversion > Formula type

This is the default formula type for converting numeric results to RIS, (default MIC)

The abrs_RISRawValue is calculated using the Ab_Formula mispl, in function of the abrs_FormulaType (MIC, Agar Difusion, E-test). The default abrs_FormulaType is abp_FormulaType. If the antibiotic is not selected via a panel or abp_FormulaType =? then use the ab_FormulaType.
On entry of a numeric value (old way), the type is displayed in which it will be stored (MIC, E-test, AD).

Mandatory AntibioticFormulaType FormulaType

One of: MIC, Agar diffusion, E-test

Conversion > Obsolete > MIC high pivot

This is the MIC value to be used as the pivot value between MIC values considered "Resistant" (higher than this limit) and MIC values considered "Intermediary" (lower).
The real decision is made in the Antibiotic Formula function, which can make use of this field.
See also Lower MIC limit.
