Mnemonic

Mnemonic code for isolation appraisal

Mandatory Unique Mnemonic Mnemonic

Positive

This toggle box is set if the appraisal implies the presence (growth) of the microorganism.
If not set (OFF), it means that the microorganims did not grow in the culture.
So for most appraisals this should be set ON. Only for those that mean 'absence' or 'no growth', it should be off.
A microbiology action is considered to be 'positive' as soon as there is at least 1 isolation with an appraisal marked as 'positive'.

Mandatory Logical Positive
