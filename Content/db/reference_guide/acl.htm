Profile

Mandatory AccessProfile Profile

Read (see record existence)

The access right 'Read’ means that the user has the right to see the record in lists (grids/browsers), i.e. see that the record exists.
It does not mean that the user has the right to see the full contents.
If the user has 'Read' access but no 'View' access to a certain Result record, the user will not be able to see following fields of the result:
RawValue, RawUnit, Comparator, Mantissa, Exponent, Choice, Severity, NormSeverity, DeltaNormSeverity, ManualSeverity, PhoneListSeverity, ExpertSystemFlags, ExternalComment, InternalComment, Log.

Mandatory Logical Read

Automatic PositiveInteger Version

View (see all fields)

The access right 'View’ means that the user has the right to see the record and all its fields (the full contents).
For the result table, it means that if the user has 'View' access to a certain Result record, the user will be able to see all fields of the result, including the value, severity, comments, etc.

Mandatory Logical View
