BloodSelectionDiscontinuation 

This function discontinues and repeats the blood selection for which the current result is the compatibility test. The blood selection is also REPEATED: a new blood selection is created with status 'Initial', so that the process of selecting a blood bag etc. can re-start.

Some background info:
Although the compatibility result can be available, its value
can indicate that the blood bag is incompatibe. Hence, the blood selection must be discontinued.
Example:
RETURN (
if .result.attribute("VALUE") = "neg" then
.result.BloodSelectionPromotion()
else if .result.attribute("VALUE") = "pos" then
.result.BloodSelectionDiscontinuation()
else if .result.attribute("VALUE") <> ? then
Message (" Invalid result for Compatibilit check ")
else ?
);

will discontinue the blood selection as soon as the compatibilityresult is positive.

This function allows to retrieve a related result of the same object and object time.

When 2 or more results exist for the same property and object time, the Result.RelatedResult() method gives precedence to results of the same specimen or the same order. The evaluation sequence of all criteria is:

1. solicited;
2. same specimen;
