Vertragsarztnummer sdav glims yes yes Normal

This table is used in Germany only: hence the German names for the table and its fields. The table is used to store the official physician identifications. Each quarter this file is populated with the 'Arztverzeichnis-Stammdatei' using a dedicated import program. The financial shipment method that creates the 'KVDT' file in Germany generates warnings if the order issuer is not in this table.
