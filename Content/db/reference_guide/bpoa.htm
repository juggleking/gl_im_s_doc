Maximum number function

This field returns the maximum number of blood bags. If available and the function returns a value (<> ?), it has priority over the fixed maximal amount.

gp_SiteFunction MaxAmountFunction

Until blood bag status

The rule only applies to blood bags that have a status not higher than this value.
E.g. if this is 'Initial', reserved or crossmatched bags will not be taken into account for this rule.
Usual values for this are:
- 'Initial' : amount of unreserved bags.
- 'Checked' : total amount of units in stock, including reserved and crossmatched.

Mandatory BloodBagStatus MaximalStatus

One of: Verification, Initial, Reserved, Checked, Checked out, Administered, Expired, Discontinued, Wasted, Undelivered
