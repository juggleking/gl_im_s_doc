Discount code dscc glims yes yes Normal

This table contains a record for each possible origin for granting a billing code based discount (e.g. The discount for regular veterinarians, the discount for large commercial issuers). A discount code hides a series of effective consecutive discount rates for possibly several discount classes.
This type of discounts can be granted based on payment agreements or billing rules. The discount amount is determined by the discount formulas.
IMPORTANT: the discounting mechanism based on the DiscountClass and DiscountCode table is distinct from the one based on the Discount and DiscountGroup* tables.
