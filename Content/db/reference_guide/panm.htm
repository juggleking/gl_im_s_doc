Billing marks

When specified, then this panel member will only be requested when the request billing mark appears in this list.
This allows defining panels with a billing-specific content.

Discriminator

A discriminator can be a relative time (e.g. +30, +02:00, -15), an absolute time (e.g. 08:00, 20:00+1d), or a string discriminator (e.g. 'Before dialysis').
Its primary purpose is to distinguish between specimens of the same materials, test requests of the same property, ....
