Quality control actions for a given station need to be scheduled on a worklist. This table states: For a given worklist, on some positions in the beginning, middle or end of a generated worklist, some procedures on QC materials must be added.

Cycle length

The Cycle length field is only applicable for 'Middle' types. The cycle length indicates after how many patient samples you want the second QC to start). Example:

Two middle QC's:

QcMat 3 with Cycle start 10 and Cycle length 20
QcMat 4 with Cycle start 20 and Cycle length 20
would result in:

10 patients, QcMat3 , 10 patients, QcMat4, 10 patients, QcMat3, 10 patients, QcMat4, 

PositiveInteger CycleLength

Cycle Start

Indicates after how many patient samples you want to start a QC. Allows to define a QC schedule where multiple sets of QC materials are scheduled with a different cycle. This allows schedules like:

QCMat1, QCMat2, 10 patients, QCMat 3, QCMat4, 10 patients, QCMat 5, 10 patients, Qcmat3, QCMat4,...

Note that in cases where different cycles coincide, the QCSchedule.SeqNo member decides the order in which the materials are scheduled.
