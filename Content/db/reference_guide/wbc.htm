Bar code

The scanned bar code

String BarCode

Context

Contains the complete context of the scan action, in the form of a 'tagged list'.
Currently supporter tags are:
- Purpose
- Station
- Worklist
The contents of this field can be explore by using the ExtractTag(Tag, List, "\") MISPL function.
Most of the context is also readily available in specific context fields (like w_BarCode.Purpose, w_BarCode.StationMnemonic, w_BarCode.WorkListMnemonic)

Mandatory String Context

Purpose

Indicates for which purpose the bar code was scanned. This may help in analyzing the bar code.

Examples of supported values:

Results: Station result download
Query: Station query
SpecimenScan: call from the specimen scan progran
SpecimensOfOrder: call from the 'Specimens of order' browser
WorkListScan: call from the work list generation program
Reception: call from the specimen reception scan program
SpecimenReplace: call from the standalone specimen replace program

Mandatory String Purpose
