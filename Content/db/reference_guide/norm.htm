Method

It is possible to define Norms for a specific assessment method. If at least one active norm for the actual assessment method is known, then only the norms for the specific assessment method are taken into account. It is also possible to specify a station for a norm.

AssessmentMethod AssessmentMethod

Attribute time limit

The Attribute time limit is a number representing the attribute's time limit and expressed in the Attribute time unit field alongside. Make sure that the day unit is filled in the Specific site preference set editor. If the object attribute is for example pregnancy, the attribute limit could be set to 2 and unit to month. Then, norms only apply to women pregnant for two months.

Discriminator

If specified, this norm will only be applied when the property was requested with the specified discriminator. When Glims evaluates which norms are applicable, the following logic is applied: as soon as 1 norm for the result's discriminator is found, only norms matching the result discriminator will be considered. Other norms will no longer be taken into account.

String Discriminator

Eligibility

The Eligible spinner references a Result-based, Logical-valued expression. It is evaluated on the result to be checked, to see whether or not the norm is applicable. When executing the referenced MISPL function, the updated result record is not yet written to the database. This implies that start table methods like .NumericValue() will NOT work! (Since the result method dispatcher will re-read the outdated record from the database). The only Result fields that can be relied upon in these MISPL functions are .RawValue and .AvailabilityUser

gp_SiteFunction Eligible

Enabled

The Enabled toggle-box can be used to disable the norm without deleting it. Not deleting a norm allows it to be included with values assessed earlier, when the norm was still enabled.
