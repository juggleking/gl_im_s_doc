Print consent form 

This function is used any time one wants to print a consent form automatically. The printer used by GLIMS will be the printer that was last used by the logged in user during manual printing of a consent form starting from an order or person lab relation (using the contextual menu function 'Print consent form').

MultiRecord Logical PrintConsentForm (rp_Printer Printer)

rp_Printer Printer
Printer used to print the consent form. If not specified the printer used the last time will be used, or the site printer.
