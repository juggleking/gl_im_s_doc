User interface - Home page

About home pages

The home page is shown immediately after login and is accessible at any time via the Home tab page.

Configuration levels

Home pages can be configured at different levels:

  1. Site (Start -> System management -> Customize -> Mate)
  2. Department
  3. Role
  4. User

If no home page is configured at a specific level, a fall back to a higher level is applied.

Home pages

Home pages can be defined via the main menu: Start -> System management -> Home pages.

Fields

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Multilingual description for the home page.

Background color

Double-click to open the color picker.

Background image / position

Full path to an image which will be used as background for the home page.  The position defines how the image is displayed:

None

The image is left-aligned at the top

Tile

The image is tiled across the tab page

Center

The image is centered within the tab page

Stretch

The image is stretched across the tab page

Zoom

The image is enlarged within the tab page

Home page sections

A home page consists of different home page sections which are accessible via the contextual ribbon item Sections.

Fields

Home page

Home page to which this section belongs.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Multilingual description.

Label

Multilingual label that will be shown as title of the section.

Left / Right / Top / Bottom

Expressed in pixels.  Defines the position and size of the section.  GLIMS also allows to 'anchor' the home page sections by entering negative values.

Examples

  • Left=10, Right=100 means that the section will be anchored left, ten pixels from the edge; and the width is fixed at 90 pixels
  • Left=10, Right=-10 means that the section will be anchored left and right, ten pixels from both edges; the width varies as the window is resized.
  • Left=-100, Right=-10 means that the section will be anchored right, ten pixels from the edge; and the width is fixed at 90 pixels.
  • Left=-100, Right=200 is an invalid combination.

The same goes for Top and Bottom.

Section type

Shortcut

The section will contain shortcuts to tools. Use the contextual ribbon/menu item Shortcuts to add the shortcuts for this section.  A shortcut refers to a tool and can have its proper label and tool tip.  The sequence number allows to arrange the shortcuts within the section.

Date time

Allows to add a section that displays the current date and/or time.

Web browser

Allows to define a section that will contain a web browser.  Use the URL field to define what will be shown initially.
