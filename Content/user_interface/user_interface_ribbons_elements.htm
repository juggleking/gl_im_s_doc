Introduction

Ribbons are an alternative to the classic start menu and contextual (right-mouse click) menus.  

Tip

To minimize the ribbon, right-click in the ribbon and choose Minimize the Ribbon.

Ribbon elements

A Ribbon consists of a set of Ribbon tabs where each Ribbon tab consists of several Ribbon groups. Each ribbon group is populated with Ribbon items. A ribbon item is an element that triggers some functionality of the application.
