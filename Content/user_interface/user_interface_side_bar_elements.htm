Introduction

The side bar pane is located on the left of the screen and allows to group a number of frequently used functions.  It is an alternative navigation method next to the classic start menu, the ribbon or the home page.

To show the side bar, click the Side bar button on the left:

By default, the side bar will auto-hide.  You can also pin the side bar:

Side bar elements

A side bar consists of several side bar groups, each with a number of side bar items, which in turn refer to a tool.
