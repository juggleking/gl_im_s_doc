User interface - Ribbon tab customization

Introduction

To ease the configuration of a ribbon, a Ribbon tab customization tool is available. It allows to configure ribbon tabs, ribbon groups and ribbon items from a single screen.

Warning

GLIMS 9.4 offers a pre-release of the ribbon customization tool.

  • It is only available to users having a role with User type set to Developer.
  • We strongly recommend to validate this new tool in a test environment and not to use it in production systems !

To start the tool:

  1. Select a Ribbon tab (starting either from a Table or Ribbon context).

  2. Choose the contextual menu / ribbon item Ribbon Customization Tool.

Window elements

Features

Ribbon groups

Create a new group

  1. In the right pane, right-click the Ribbon tab and choose Add New Group.

  2. Enter a name for the new group.

Move a group

  1. In the right pane, select a Ribbon group and
    1. Right-click and choose Move up / Move down or
    2. Drag & drop it to its new location or
    3. Use the Arrow buttons at the right hand side

Rename a group

  1. In the right pane, right-click a Ribbon group and choose Modify

Delete a group

  1. In the right pane, select a Ribbon group and
    1. Right-click and choose Remove or
    2. Press the Delete key

Ribbon items

Create a new item

  1. In the left pane, select a Tool and drag it to the right pane (or use the arrow button in between the left and right pane).

  2. You can drop the tool when red arrow icons are shown.

Tip

To create a split button, drop the Tool exactly on an existing ribbon item (not below and not above).

Move an item

  1. In the right pane, select a Ribbon item and
    1. Right-click and choose Move up / Move down or
    2. Drag & drop it to its new location or
    3. Use the Arrow buttons at the right hand side

Rename an item

  1. In the right pane, right-click a Ribbon item and choose Modify

Delete an item

  1. In the right pane, select a Ribbon item and
    1. Right-click and choose Remove or
    2. Press the Delete key

Reset your configuration

If you want to undo all your changes:

  1. In the right pane, right-click the Ribbon tab and choose Reset Tab.

Save your configuration

To save all your changes, press the OK button.
