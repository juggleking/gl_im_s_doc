User interface - User preferences

Most screens in GLIMS can be moved and resized. Some of these screens also contain modular elements, such as ribbons and info boxes, which can be hidden or resized.

The dimensions and positions a user chooses for a given screen and its content are saved under Start -> System management -> Security -> Users -> select the user -> Preferences -> Context = OrderEntry/ResultsOfOrder/...). They are then applied when the screen is opened again, both within the same session and in a new session.
