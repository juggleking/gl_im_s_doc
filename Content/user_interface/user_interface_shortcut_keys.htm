Shortcut keys

Short cut key Action
ALT + underlined letter Open the corresponding menu
ALT + F4 Close the application
Tab Go to the next field
SHIFT + Tab Go back to previous field
CTRL + C Copy the selection
CTRL + V Paste the selection
CTRL + X Remove and copy the selection
SHIFT + arrow keys Select a zone
ESC Leave screen
INSERT Create new record
F1 Open the GLIMS documentation
F2 Confirm and go to next record
F4

Calls the contextual menu (equivalent to right mouse-click) in non-.NET interfaces

"Menu" key

or

Shift+F10

Calls the contextual menu (equivalent to right mouse-click) in .NET interfaces
F5 Refresh current browser: the query is immediately reopened using the current query parameters.
SHIFT + F5 Refresh current browser: the user is allowed to update the query parameters screen and the query is reopened.
ALT + F5 Activate the auto refresh (every 60 seconds) functionality for the current browser.
Page Up This will reposition to the previous record (in editors) or the previous set of records (in browsers).
Page Down This will reposition to the next record (in editors) or the next set of records (in browsers).
F6 Zoom, open
F7 Go to the first empty field
F10 Open the main menu
Space bar Select / Deselect
CTRL + E Deletes the text after the cursor
CTRL + F Find
CTRL + R Undo/Recall function (e.g. puts deleted text back)
CTRL + O Export to Word or Excel
