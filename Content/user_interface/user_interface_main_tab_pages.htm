User interface - Main tab pages

Introduction

Whenever a function is started from the ribbon, side bar, home page or start menu, a screen is opened in a new main tab page.  This allows opening several work flows simultaneously and switch between them by clicking the tab page.

When opening a related window (via the contextual ribbon, F6, ...), the new window opens in the same main tab page.  When closing a tab page or switching to another tab page, all screens within the main tab page are closed/hidden.

Example

Customization

Change sequence of tab pages within one tab group

By drag-and-drop, you can change the sequence in which the tab pages are displayed.

Multiple tab groups

By default, GLIMS only shows the content of one single tab page.  It is possible to create horizontal/vertical tab groups that display several tab pages simultaneously beside / below one another.  

  1. In a tab page header, right-click and choose New Horizontal Tab Group or New Vertical Tab Group


  2. The selected tab page will be moved to a new tab page group


  3. Tab groups can be resized.

Moving a tab to another group

Tabs can be moved to another group by:

  1. Drag-and-drop (click a tab page header and drop it onto a tab page in another tab group)
  2. Right-click and select Move to Previous Tab Group or Move to Next Tab Group
