Internal mail

Users can send and receive internal messages in GLIMS. GLIMS also allows sending mails from within MISPL. For more information, please check the chapter Sending mail from within MISPL.

Sending mail

Select the main menu item Start -> Tools -> Mail -> Send.

Recipient, Group

Internal messages can be sent to individual users or to a user group.

Tip

When selecting a recipient or a group of recipients, always hit Enter so that the recipients are listed in the browser (as shown in the example above).

Subject, Content, Priority

Specify the subject, the actual message and its priority.

Reading mail

Select the main menu item Start -> Tools -> Mail -> Read.

Recipients are continuously notified of any incoming mail. They can read, delete or reply to incoming mail messages. Deleting incoming mail deletes only the association between the message and the recipient. The message itself is kept, to allow any other recipients to read it.

Notes

Note

  • The message itself is stored only once, whatever the number of recipients.  
  • In the message browser, the messages are sorted with the most recent one on top.

Calculator

A built-in calculator can be activated from the main menu via Start -> Tools -> Calculator. You can copy the results of your calculation to the clipboard and subsequently paste them into a GLIMS field.

Calendar

The calendar can be opened from the main menu via Start -> Tools -> Calendar or from a date selection field in GLIMS.
