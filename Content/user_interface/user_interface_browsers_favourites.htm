Browsers: Favorites

Content: Introduction - Usage - Management

Introduction

GLIMS offers a concept of favorites for a number of tables (currently Printers, Stations, Work list templates and Request forms).  The browsers of these tables have an option and the bottom left which allows limiting the displayed records to the user's favorites.

Usage

When a record is selected, it is automatically added to the list of favorites.  

Example

  1. Open the work list generation screen
  2. In the tab page Station, select a station
  3. This station will now automatically be added to your favorites
  4. The next time you select a station and you enable the option Show my stations only, you will only see your favorite stations.

The same goes for work list templates and printers.  Wherever you select a station, work list template or printer, you can enable the option Show my stations/work list templates/printers only to limit the records to your favorites.

Note

In case of request forms, the favorites must be defined manually.

  1. Open the Users browser and select a user.
  2. Choose the contextual menu/ribbon item Favorites -> Request form assignments

  3. Hit the Insert key and select a Request form to be added as favorite

Management

System managers have direct access to a user's favorites by using the contextual menu/ribbon item Favorites -> Printers / Stations / Work list templates / Request forms from a User record.  

These browsers allow manually adding or removing favorites.
