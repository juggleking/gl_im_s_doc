User interface - Expert settings

An Expert settings menu item is available in the top right corner of the GLIMS window:

  1. Error stack: can be used when MIPS staff requires detailed information about an error.
  2. Debug alert
  3. Client log manager
  4. Local execution
  5. Dynamic objects
  6. Monitor db access
  7. Execution log

Note

This functionality is for system managers and developers only. Exceptions are Debug alert and Clientlog manager which are also available for regular users.
