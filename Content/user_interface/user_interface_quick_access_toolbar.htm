User interface - Quick access toolbar

Introduction

The quick access toolbar is a set of tools located in the upper left corner of the GLIMS workspace.  The quick access toolbar is saved per user and is tied to the ribbon being displayed.

Customization

Add tools

To add a tool from the ribbon/start menu to the quick access toolbar, right-click a tool in the ribbon/start menu and select Add to Quick Access Toolbar.

You can also add an entire ribbon group to the quick access toolbar.

Remove tools

To remove a tool from the quick access toolbar, right-click a tool in the quick access toolbar and select Remove from Quick Access Toolbar.

Location

The quick access toolbar can be shown below or above the ribbon.
