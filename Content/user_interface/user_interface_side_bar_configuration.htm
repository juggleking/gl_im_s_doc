Side bars

Site-based vs role-based

Identical to the ribbon concept, one can setup a single side bar for all users or a different side bar for each individual role.

The side bar to be used as general side bar for all GLIMS users is specified in the general settings (Start -> System management -> Customize -> MATE).

The role-based side bars are specified for each role:

Fields

Side bars can be defined via Start -> System management -> menu -> Side bars.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Description

Multilingual description of the side bar.

Side bar groups

Side bar groups can be defined via the contextual menu or ribbon tab of a side bar.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Sequence

Defines the sequence of this group within the side bar.

Label

Multilingual text that will be shown in the side bar.

Tool tip

Multilingual text that will be shown when hovering over this group in the side bar.

Small / Large icon

Currently only the large icon is used.

Side bar

Read-only reference to the side bar to which this group belongs.

Side bar items

Side bar items can be defined via the contextual menu or ribbon tab of a side bar group.

Sequence

Sequence of the side bar item within the side bar group.

Tool

Reference to the tool associated with the ribbon item. Only tools with base table "gp_Site" are allowed as sidebar item.

Label

Label that allows to overrule the label of the associated tool.

Tool tip

Tool tip that allows to overrule the tool tip of the associated tool.
