Regenerate a ribbon

If you have customized a GLIMS ribbon but would like to return to the default ribbon shipped with GLIMS, you can regenerate that ribbon.

  1. Click Start -> System management -> Scheme -> Tables
  2. Select the table(s) for which you want to regenerate the ribbon

  3. Choose the contextual menu / ribbon item Regenerate ribbon

    Ribbon

    Select the ribbon that you want to regenerate.

    Remove tools already present in ribbon

    If disabled, the default ribbon will be applied, but your custom configuration will be kept.  Your personal configuration will not be lost but will be complemented with the default configuration.

    If enabled, your custom configuration will be removed and the ribbon will be completely reset to the default ribbon.
