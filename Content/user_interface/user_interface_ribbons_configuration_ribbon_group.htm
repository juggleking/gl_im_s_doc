Fields

Ribbon groups gather related functions.  From a ribbon tab context, choose the contextual menu/ribbon item Ribbon groups.

Mnemonic

Short, unique identifier. Cannot contain spaces, quotes nor backward slashes.

Tab table

Read-only, automatically filled-in with the associated ribbon tab table name.

Tab mnemonic

Read-only, automatically filled-in with the associated ribbon tab mnemonic.

Sequence

Defines the position of the ribbon group within the ribbon tab.

Label

Multilingual description used for displaying the ribbon group.

Collapsed icon

Can contain a path to an icon file to be used when the ribbon group is collapsed (resized screen) or added to the quick access toolbar.

Dialog tool

Link to the tool to use when the ribbon group dialog button is clicked.
