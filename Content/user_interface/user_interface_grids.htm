User interface: grid options

GLIMS offers several grid screens (matrix, array).  Examples: editing a work list, entering results, etc. The ribbon offers the following grid-specific items:

Refresh

Refreshes the screen once.

Edit by row

If enabled, the cells will be visited by row instead of by column during navigation (tab, arrow down/up).

Prefix/Suffix

Allows to define a textual prefix/suffix that will be automatically added to the value specified in the cells.

Skip disabled fields

If enabled, disabled (grayed out) cells will be skipped when navigating through the grid (tab, arrow down/up).

Skip non-empty fields

If enabled, cells with a value will be skipped when navigating through the grid (tab, arrow down/up).
