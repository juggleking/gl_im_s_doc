User interface - Languages

Contents: Language definition - Choosing the user interface language - Changing the user interface language

Language definition

Before you can choose the language in which the user interface and menus should be shown, you must first define the required languages in the Language table.  

Click Start -> System management -> Languages.

Code

Specify a code for the language, e.g. E for English

Name

Enter a name for the language (will be shown when a language must me chosen).  E.g. English

System language

Select the system language that corresponds to the language that you are creating.  GLIMS currently supports the following languages: English, French, Dutch, German, Spanish, Czech, Slovak and Hungarian.

Choosing the user interface language

Once the languages have been defined, the user can now choose in which language he wants to work.  There are several ways to indicate the user interface language:

  1. Select a language in the Login screen
  2. If no language is selected in the login screen, GLIMS will check the language defined in the User configuration
  3. If no language is selected in the login screen and no language is defined for a user, GLIMS will use the default language specified via Start -> System management -> Customize -> MATE.

Note

The language of the login screen itself is determined by the default language specified via Start -> System management -> Customize -> MATE.

Changing the user interface language

GLIMS offers a tool to change the user interface language without having to change any configuration. This tool requires the user type Developer. 

Click Start -> Tools -> User interface language.  

Language

Can only be specified when Use default is disabled. This language will overrule any other language specification and will remain valid until you disable it using this function.

Use default

If enabled

GLIMS will search the user interface language to be used as follows:

  1. Language selected in the Login screen
  2. If no language is selected in the login screen, the language defined in the User configuration
  3. If no language is selected in the login screen and no language is defined for a user, GLIMS will use the default language specified via Start -> System management -> Customize -> MATE.

If disabled

Specify another language in the Language field.
