User interface - Status area

The status area at the bottom of the screen shows the following information:

Left side

Information of the currently opened window.  E.g. the number of selected records in a browser.

Right side

  • User name
  • Department of the logged in user
  • If a user is logged in via a remote desktop connection, then GLIMS displays both the 'Device' and 'Terminal' name. This text is of the form '<Device> (via <Terminal>)' - e.g. 'JohnDoeClient (via CitrixServer)'.
  • Role of the logged in user
  • User interface language
