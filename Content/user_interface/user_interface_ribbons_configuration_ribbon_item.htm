Fields

Starting from the ribbon group browser, it is possible to list all ribbon items linked to a specific ribbon group.  Note that a ribbon needs to contain at least one ribbon item before the ribbon will be visualized.

Ribbon group mnemonic

Read-only, automatically filled-in with the associated ribbon group mnemonic

Ribbon group ribbon tab mnemonic

Read-only, automatically filled-in with the associated ribbon group ribbon tab mnemonic

Ribbon group ribbon tab table

Read-only, automatically filled-in with the associated ribbon group ribbon tab table mnemonic

Sequence

Defines the position of the ribbon item within the ribbon group

Preferred size

Defines the button and icon size to be used (see Tools). When resizing the screen (or when not enough space is available), the ribbon will automatically resize its items to fit the available space.

Normal

Medium button with image and text (tool's small icon is used).

Image only

Small button with image but without text (tool's small icon is used).

Large

Large button with image and text (tool's large icon is used).

Tool

Link to the tool associated with the ribbon item

Label

Multilingual field, containing the label for the associated ribbon item. If no label is provided, the label of the associated tool is used.

Parent

In case of a split button (see below), the ribbon item in the drop-down box must refer to the sequence number of the parent ribbon item.

About split buttons

A split button consists of a main button that can be clicked to start a tool but which also has a drop-down menu with extra items.  

  • Both the main button and the drop-down items are configured as ribbon items.  
  • The drop-down items are ribbon items that have a parent ribbon item (i.e. the main button)

A practical example would be to have the standard order query as main button and a number of pre-configured order queries as drop-down items.
