Customize the menu structure

The structure of the start and contextual menus can be customized. 

Main menu

Click Start -> System management -> Menu -> Edit.  The tree editor that appears allows to customize the structure of the main menu.

contextual menus

Click Start -> System management -> Scheme -> Tables.  Select the table of which you want to change the menu and choose the contextual ribbon/menu item Edit.

Tree editor

Please check the chapter Tree editors, for more information on how to work with a tree editor.

Adding new, pre-configured menu options

Both in the main menu and in the contextual menus, you can add new menu options with pre-defined parameters sets. In the tree view, click Insert to add a new item:

Table

Automatically set.

  • When editing the main menu, this will automatically be set to gp_Site.  
  • When editing a contextual menu, this will automatically be set to the base table of the menu.

Tool

Tool that will be executed when choosing this menu item.  The tool configuration determines which function and function parameter set should be used and offers an option Skip setup.

Label

Multilingual label. If no label is provided, the label of the associated tool is used.

TTY/Windows accelerator

Allows defining an accelerator key that will execute the defined menu option. This is the list of valid menu-accelerators on Windows:

  • F5, F8, F11
  • SHIFT-F1 through SHIFT-F12 (except SHIFT-F4 and SHIFT-F10)
  • SHIFT-CTRL-F1 through SHIFT-CTRL-F12 (except SHIFT-CTRL-F4 and SHIFT-CTRL-F10)
  • ALT-F1 through ALT-F12
  • SHIFT-ALT-F1 through SHIFT-ALT-F12
  • CTRL-F1 through CTRL-F12 (except CTRL-F4 and CTRL-F10)
  • CTRL-ALT-F1 through CTRL-ALT-F12 (except SHIFT-CTRL-F4 and SHIFT-CTRL-F10)
  • CTRL-SHIFT-ALT-F1 through CTRL-SHIFT-ALT-F12
  • ALT-CTRL-A through ALT-CTRL-Z

Tip

To obtain an overview of the currently defined accelerator keys, click Start -> System management -> Menu -> Accelerator keys.

Add horizontal line

A menu item with no function specified, whose label contains only "-", "_" or " " characters, will be interpreted as a "ruler" in the menu, i.e. a slim horizontal line in between two menu-items. This can be helpful to design menus which are easier to interpret.

Regenerate the menu

The start menu item Start -> System management -> Menu -> Regenerate allows regenerating the main menu and table menus, thus restoring the default menu structure provided by MIPS.  

Replace existing

If enabled

All menu customizations will be lost and replaced by the default menu options.  

If disabled

Only the missing default menu options will be added and any menu customization will be kept.

Translate options

Allows to translate all the menu options in all supported languages.  

Note

If you have specified customized option labels, you can manually enter translations as the label field is multilingual.

Log directory, Log file name

Allows to log all changes in the specified directory and file name.

Regenerate keys

Each menu function automatically receives a shortcut key (underlined). If you have changed the labels of the menu functions, you can use the start menu item Start -> System management -> Menu -> Regenerate keys to regenerate the shortcut keys.

Translate from glossary

The start menu item Start -> System management -> Menu -> Translate from glossary must no longer be used.  You can translate menu options using the Regenerate function.
