User interface: Tree views

Introduction

Tree views allow visualizing and editing hierarchically structured data. They can be used in two modes:

View mode

In view mode, the tree view is only used to show information, and nodes in the tree cannot be moved, added or deleted.  Examples are the order, specimen and microbiology tree views.

Update mode

In update mode, the user is allowed to move, add and delete nodes in the tree. Examples are the menu editor, the request form editor, the property classification editor.

Note

Tree views check the required privileges before you can make updates. When updating is not allowed because of lacking privileges, the promote/demote ribbon items are disabled, and drag/drop is not possible.

Screen layout

Left window

Shows the actual tree view.

More information can be displayed by enabling the Detailed tree ribbon item.

Right window

By default, the most important fields (and values) are shown for the selected node.  

To view all the fields of the record, enable the Show all fields ribbon item.

Tip

The colors used in the right window are determined by the color manager (System management->Customize->Colors) and are used for orders and results.

Ribbon

The following ribbon items are specific to tree views. For more information on the other ribbon items, see Ribbons: contextual tools.

View

Detailed tree

Show more details in the tree view.

Show all fields

Switch between showing all fields and showing only the browser fields with known values.

Move

Move up

Move the selected node one position up (disabled in view mode).

Move down

Move the selected node one position down (disabled in view mode).

Move left

Move the selected node one level up (promote) (disabled in view mode).

Move right

Move the selected node one level down (demote) (disabled in view mode).

Expand

1 - 6

Show all nodes of level 1-6.

All

Show all nodes.

Working with the mouse

Left click on a node Select the node and optionally update the detailed view
Double click on a node Open the node editor for the selected node
Drag and drop a node Move the node to a new place
Right click on a node If possible the context menu of the selected node is shown
Left click on a column of the detailed view Sort the column
Left click on a white arrow pointing right Expand a node
Left click on a black arrow pointing down Collapse a node
Move the separator bar between the tree view and the list view Change the size of the tree view and the list view

Working with the keyboard

Up and Down arrow Walk through the nodes
Left and Right arrow Expand or collapse node
F6 or ENTER Open the node editor
Insert Add a new node
Delete Delete selected node
CTRL-left arrow Promote node (move one level up)
CTRL-right arrow Demote node (move one level down)
CTRL-up arrow Move node up
CTRL-down arrow Move node down
CTRL-C Copy selected node
CTRL-X Cut selected node
CTRL-V Paste selected node
CTRL-F Search a node
