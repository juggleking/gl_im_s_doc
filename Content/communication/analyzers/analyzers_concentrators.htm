Station configuration for concentrators

Content: Introduction - Configuration - Concentrator translators

Introduction

This document describes how to configure station and translators in case a "concentrator" is used (connection to a remote PC which in turn is connected to a number of analyzers).

Configuration

For each analyzer connected to the concentrator PC, a station with corresponding procedures must be defined in GLIMS. The external id of the station should contain the name of the analyzer e.g.

<station 1>
<station 2>
<station n>

Each station (1-n) has its own remote id. The inbound and outbound id's are to be defined in these station (1-n). There is no need to define a station for the concentrator. For all station (1-n), the SAME translator must be specified; e.g. "labt" (labtrain).

How does the connection work ?

Order download

GLIMS detects that several stations are linked to the same translator and will send all data for these stations.

Result upload

The concentrator sends the station's external id along with the result. GLIMS looks for a station with a corresponding external id and translator.

Concentrator translators

Examples of translators that can be used for concentrator connections: pcd, imma, abln, corn, maes, ...
