Distributor and concentrator channel codes

Introduction

Via the assessment method codes of the concentrator / distributor station, you can map client station assessment methods with codes that are valid for the concentrator / distributor system.

Usage

The Station contextual menu function Assessment method codes shows a browser with all information of the client assessment methods and the code values.

Main content

The browser shows all assessment methods of the distributor / concentrator's client stations.

Columns

Station

Mnemonic of the client station.

Type

Type of the client station.

Group

Download group of the client station.

Property

Inbound ID

Outbound ID

Code in

Allows to enter a Code in used for result upload.

Code out

Allows to enter a Code out used for work order download.

Enabled

Allows to activate the code mapping (code in and code out). Codes can also be enabled / disabled through the contextual menu.

Popup menu

The contextual menu allows copying values from property mnemonic, inbound id, outbound id to the "in" and "out" codes. The contextual menu option Overwrite allows overwriting existing code values. 

Storage of "code in" and "code out"

The code in and code out are stored in the assessment method code table which has the following fields:     

Station

Refers  the concentrator or distributor station on which the codes are defined.

Assessment method

This is the assessment method of a station related to the concentrator or distributor to which the Code in and Code out are defined.

Code in

Code to be recognized in an incoming message received from the concentrator / distributor system.

Code out

Code used in an outgoing message transmitted to the concentrator / distributor system.

Enabled

Allows to deactivate the code mapping (both Code in and Code out).

Header 

The browser header shows the client station of the assessment method, and the relation of that station with the concentrator / distributor. The header also shows which procedures and material are used with the assessment method.

Footer

The browser footer shows details about the property.

Restriction

Distributor and concentrator systems should use the same code for work order download as for result upload

Uploading results without station remote id specified in the result message

By defining assessment method codes on a distributor or concentrator, GLIMS allows upload of results for a given channel without having the knowledge for which instrument (concentrator client) the result was meant.
