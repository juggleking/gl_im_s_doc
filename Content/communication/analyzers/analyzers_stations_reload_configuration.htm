Reload station configuration

Contents: Introduction - Details

Introduction

From each station, you can select the contextual menu function "Reload configuration".  This function will release configuration data previously loaded in memory by the internal service.  The internal service uses the data for communication with the station and the data is loaded upon receipt of the first result message (upload) or query message (download) from the station.
