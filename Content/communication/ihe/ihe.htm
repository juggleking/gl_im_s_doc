IHE communication - Basic configuration

Introduction

GLIMS supports using the IHE LTW (Laboratory Testing Workflow) profile for communicating with external systems.

Basic configuration

As of GLIMS 9.3, an External application needs to be configured via Start > System management > External applications:

Mnemonic

Must be set to IHE-LTW.

Supported IHE profiles

Allows to indicate which LTW transactions should be used.

  • LAB 1: Placer Order Management

  • LAB 2: Filler Order Management

  • LAB 3: Order Results Management

    Notes

    LAB 3 requires the configuration of a default report with Scope = Order.

Notes

The site attribute LTW context (Start > System management > Customize > GLIMS general), used prior to GLIMS 9.3, is no longer taken into account.
