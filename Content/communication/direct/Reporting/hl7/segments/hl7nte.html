3. Comment

  • HL7 Message type: ORU
    • Order based OBR
      • NTE after OBR segment
        • OD : Order Description
      • NTE after OBX segment
        • TL: Test Classification Label (if available)
        • TD: Test Description (if containing newlines) - OBX.3.2 will contain the same value BUT with newlines replaced by a space char
        • RC: Result ExternalComment
        • XT:Extended Result Value
    • Specimen based OBR
      • NTE after OBR segment
        • OD : Order Description
        • SC : Specimen ExternalComment
      • NTE after OBX segment
        • TL : Test Classification Label (if available)
        • TD : Test Description (if containing newlines) - OBX.3.2 will contain the same value BUT with newlines replaced by a space char
        • RC : Result ExternalComment
        • XT:Extended Result Value
    • Request based OBR
      • NTE after OBR segment
        • OD : Order Description
        • SC : Specimen ExternalComment
        • QC : Request ExternalComment
      • NTE after OBX segment
        • TL : Test Classification Label (if available)
        • TD : Test Description (if containing newlines) - OBX.3.2 will contain the same value BUT with newlines replaced by a space char
        • RC : Result ExternalComment
        • XT:Extended Result Value

4. Comment type 

Only supported for HL7 v2.4 !

"OD"

Order Description: applicable for OBR segments Order/specimen/request based

"SC"

Specimen External Comment: applicable for OBR segments specimen/request based

"SD"

Specimen Discriminator: applicable for OBR segments specimen/request based

"QC"

Request External Comment: applicable for OBX segments

"RC"

Result External Comment: applicable for OBX segments

"TD"

Extended property/material/microbiology description: applicable for OBX segments
"XT"

Extended Result Value: applicable for OBX segments

"TL" 

 Property Classification Label: applicable for OBX segments
