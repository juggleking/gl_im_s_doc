Report generation

You must now create a command to automate the report generation to HexaFlux.

Report query settings

Report generation settings

  • Output to file: disabled
  • Translator: not to be specified
  • Destination: directory
  • Path: directory where the report files are saved
  • Names: {###}hrpim,{###}text
  • Extensions: hpt,txt
  • Pin Provider: PIN provider to be used
  • Receiver: HEXAFLUX
