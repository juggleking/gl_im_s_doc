9.12 Action code

The HPRIM translator will verify the value of the field 'Action code' to determine the kind of order operation.  the 'Action code' has the following possible values:

A Add request to existing order Translated to an order create URL
C Cancel order Translated to an order cancel URL
N Create new order Translated to an order create URL
L New order, sampling to be done by executor Translated to an order create URL
R Update order Two options:
  1. The option ORMModification=CancelAndReCreate is specified: the order will be cancelled and recreated. Be aware that this option only works if the corresponding broker has RenameOrderOnDiscontinue=Yes.  Check the driver documentation for more info on this option (e.g. smph.exe)
  2. If this option is not set: order create URL
