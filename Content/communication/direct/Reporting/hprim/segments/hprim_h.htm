7.7 Message type

Depends on the program creating the message.  Can be one of ADM, FAC, ORM or ORU

7.13 HPRIM version

Depends on the program creating the message.  Examples:

  • HPRIM reporting (ORU) supports version 1.2 and 2.1.  The version is defined in a site attribute 'HPRIMVersion' on ReportTemplate.
  • HPRIM facturation (FAC) support version 2.1
