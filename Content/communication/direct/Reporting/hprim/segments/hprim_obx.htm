10.3. Value Type

  • Results Not Validated : "ST"
  • Results Validated
    • Result ExternalComment present : "TX"
    • No Result ExternalComment : "ST"

10.4. Observation Identifier

  • Property.prop_Mnemonic
  • Property.prop_Description
  • "L"
  • Property. PropertyIdentification.prid_Code
  • Property.prop_Description
  • "E"

10.6. Result Value = Result.rslt_RawValue

  • Results Not Validated : Empty
  • Results Validated
    • Result ExternalComment present : Result Value + "\n" + ExternalComment
    • No Result ExternalComment : Result Value

Caution ! If there’s no Result Value, the OBX segment only contains :

    • 10.1. : OBX sequence number
    • 10.2. : Result Value Type
    • 10.3. : Observation Identifier
    • 10.6. : Result Units
    • 10.7. : Result References Range
    • 10.11. : Result Status
    • 10.14. : Result ObjectTime

10.7. Units

    • Property.prop_Unit.Unit_Name
    • Empty
    • "L"
10.8. References Range : Low-High Limit

Calculated field (STRING or ST)

Syntax: LowLimit + LimitSeperator + HighLimit

  • LowLimit : Norm.norm_LowLimit
  • LimitSeperator: "<", ">" or "-"
  • HighLimit : Norm.norm_HighLimit
10.9. Observation Flags
    • Result Severity 0 : Flags = "N"
    • Result Severity <> 0 :
      • Result DeltaNormSeverity > 0 : Flags = "U"
      • Result DeltaNormSeverity < 0 : Flags = "D"
    • Result PropertyTypeNumeric
      • Numeric Result present :
        • NormSeverity > 0 :
          • Flags empty : Flags = "H"
          • Flags not empty : Flags = Flags + "H" (as 2nd repeat)
        • NormSeverity < 0 :
          • Flags empty : Flags = "L"
          • Flags not empty : Flags = Flags + "L" (as 2nd repeat)
      • Conversion to numeric failed :
        • Flags empty : Flags = "A"
        • Flags not empty : Flags = Flags + "A" (as 2nd repeat)

10.12. Observation Result Status

    • Validated : "F"
    • Expected : "I"
    • Confirmed : "P"
    • Available : "R"
    • Discontinued : "X"
    • Initial : "I"

10.15. Result ObjectTime

Caution ! If there is no Result ObjectTime, no message will be constructed nor send !

10.16 Executing department

Identification code of Result.Department. If the not available, the correspondent internal id of Result.Department.

10.17 Code of validation user

Result.rslt_ValidationUser. usr_LoginName
