Hyperlinks

Contents: Introduction - Syntax - Examples

Introduction

The "Action" tab page of the contextual ribbon tab "Tools" contains an option "Hyperlink". Activating this option will cause the focused field to be scanned for Hyperlink information. If any hyperlink information is found, the hyperlink will be followed.

Syntax

The hyperlink information must comply with the following syntax:

Hyperlink Link = (RecordLink | DocumentLink)
RecordLink (Class | C) =ClassName (( Externalization | X) =Externalization) | (ID= RecordId)
DocumentLink (Method | M) = (WindowsApplication | WA) (Application | A) = (ApplicationName) (Document | D) = URL
  • The texts is red must be typed in literally
  • The texts in green are variables that must be substituted
  • The ' l ' sign is a synonym for 'or'.
  • The "(" and ")" signs indicate that you must choose one of the values separated by the "|" sign
  • ClassName: name of the table to which the record belongs
  • Externalization: please check the chapter 'Indexes' in each reference manual page in order to see how each table is externalized.
  • RecordId: each record in GLIMS has an ID.  The id of a record by checked using a MISPL expression
  • ApplicationName: GLIMS supports two fixed names: "Word97" and "InternetExplorer".  If you specify another name or no name at all, the file specified in the URL will be opened with the application that is linked to the file extension (as determined by your operating system).
  • URL: the full path name of the file.

Examples

RecordLink with Externalization

  • Link=(C=Order X=19980801-00001)
     
    Will open the order editor for the order with externalization "19980801-00001"

RecordLink with RecordId

  • Link=(C=Person ID=11)
     
    Will open the person editor for the person with id "11"

DocumentLink with ApplicationName

  • Link=(M=WA A=Word97 D=C:\Standard Operating procedures\SOP0001.doc)
      
    Will open the document "SOP0001.doc" with MS Word 97

DocumentLink without ApplicationName

  • Link=(M=WA D=C:\Standard Operating procedures\SOP0001.gif)
     
    Will open the file "SOP0001.gif" with the application linked to files with extension ".gif"
