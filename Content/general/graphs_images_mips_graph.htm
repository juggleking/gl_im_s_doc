MIPSGraph

The MIPSGraph ActiveX Control is a control that can be inserted in a Windows application and used to show all kinds of graphs.

Screen layout

     

 

1. Print

Shows a print dialog box box where the user can choose a printer, set some print options and eventually print the graph.

2. Zoom in

Double the zoom factor of the graph.

3. Zoom out

Divide the zoom factor of the graph by 2.

4. Reset zoom

Restore the zoom factor to 1:1.

5. Free zoom

The mouse cursor changes in a magnifying glass, if the user now clicks somewhere in the drawing area, the screen will be zoomed around that  point.

6. Zoom window

The mouse cursor changes into a drawing cross and the user can draw a rectangle on the screen, when the mouse is released the graph is zoomed to that rectangle.

7. Group/split the datasets

If there is more than one dataset, the graph can be shown with all datasets drawn under each other, with a common X axis. This button toggles the grouping or splitting.

8. X options

Shows a dialog box where the user can set all X axis options.

9. Y options

Shows a dialog box where the user can set all Y axis options.

10. General options

Shows a dialog box where the user can set all general options.

11. Show/Hide the legend

Toggles the display of the legends.

12. About

Shows the About box.

13. Dataset legend

Shows the names of all the datasets.  Left clicking on the legend item will mark the dataset, right clicking will show a contextual menu where the user can choose to show or hide the dataset.

14. Area legend

Shows the names of all the areas.

General Options

Show dots on data point

Toggles the drawing of the little circles around the graph points.

Show coordinates

Toggles the drawing of the mouse coordinates on the screen.

Group datasets on one graph

Same as toolbar button 7.

Smoothed

The graph lines will be smoothed but still go through all the graph points.

Show trend lines

Trend lines are similar to the smoothed graph lines but they don't necessarily go through the graph points.

Legend

Same as toolbar button 11, but here the legend is divided in three parts : lines, areas and datasets.

X Options

Mark Distance

Distance between two consecutive markers (the short lines under the X axis).

Label Distance

Distance between two consecutive labels.

Show Grid

Toggles the drawing of the X grid (vertical gray lines), the distance between two grid lines is the same as the mark distance.

Position

Position of the X axis, possible values : "Top", "Bottom" and "on Y=0".

Y Options

The same options are available as for the X options.

contextual menu

If the user clicks the right mouse button somewhere on the graph, a contextual menu is shown:

Print

Same as toolbar button 1.

Options

Same as toolbar button 10.

Zoom

Shows a dialog box where the user can set the zoom factor for the X and the Y axis.

Free Zoom

Same as toolbar button 5.

Show Coordinates

Toggles the drawing of the mouse coordinates on the screen.
