About the GLIMS manual

Accessing Help

From the ribbon

In the upper right corner of the GLIMS screen, click the Help icon an choose "Help". This will display the main help page.

Using [F1]

From any application window, press [F1]. Pressing F1 opens the topic that is directly related to the application window that you are using. From this topic, you can still access other parts of the help system.

Note

The F1 key will only work if at least one internet browser is set as default browser.

Finding information

The help system provides 2 ways to find information: via Contents and Search (upper right corner).

  • Contents - The Contents tab provides the structure of the help system. Click a book to open the underlying chapters and pages. Topics are grouped according to the main categories of the application.

  • Search - The Search field in the upper right corner provides full-text search. Type a word in the search box and press Enter to list all topics containing that word.

    Tip

    If you want to find a specific combination of words that are always next to each other in the same order (e.g., property classification node), you can enter the search keywords within quotation marks (e.g., "property classification node").

Database manual

The GLIMS database manual is part of the GLIMS help system and can be found in the Contents tab, in the book named Database documentation -> Reference guide. This database manual also displays automatically when you use the MISPL builder.
