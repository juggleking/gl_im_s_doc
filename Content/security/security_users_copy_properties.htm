Copying user properties

GLIMS allows to copy a number of properties from another user to the selected user. Select a User and choose the contextual menu / ribbon item Copy properties from.

Template user

User from which the properties should be copied to the selected user.

Privileges

Add

Add missing privileges, i.e. add privileges from template user that do not exist for the selected user.  Choose None, All or Some to select which privileges you want to copy.

Delete

Delete superfluous privileges, i.e. privileges that are not defined for the template user will be deleted for the selected user.

Group assignments

Add

Add missing group assignments, i.e. add group assignments from template user that do not exist for the selected user.  Choose None, All or Some to select which assignments you want to copy.

Delete

Delete superfluous group assignments, i.e. group assignments that are not defined for the template user will be deleted for the selected user.

Roles

Add

Add missing roles, i.e. add roles from template user that do not exist for the selected user.  Choose None, All or Some to select which roles you want to copy.

Delete

Delete superfluous roles, i.e. roles that are not defined for the template user will be deleted for the selected user.

Preferences

Add

Add missing preferences, i.e. add preferences from template user that do not exist for the selected user.  Choose None, All or Some to select which preferences you want to copy.

Delete

Delete superfluous preferences, i.e. preferences that are not defined for the template user will be deleted for the selected user.

Replace

Update common preferences, i.e. in case of common preferences, update with the settings of the template user.
