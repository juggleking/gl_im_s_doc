User groups

Introduction

GLIMS allows creating user groups. This can be useful if you want send a mail to a group of users.

Creating a user group

To create a User group:

  1. Click Start -> System management -> Security  -> User groups.  
  2. Hit the Insert key to create a new user group and enter a name. Once the new group has been created, there are two ways to add members
    1. Use the contextual menu / ribbon item Members.  Hit the Insert key to select a user.
    2. Double click (or use F6) the user group and hit the Insert key in the embedded browser of users.
