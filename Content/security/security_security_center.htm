Introduction

The Security Center is the central tool for managing users, roles and privileges.  It is specifically designed to:

  1. Create privileges
  2. Assign tables, functions and fields to a privilege
  3. Create roles
  4. Assign privileges to roles
  5. Assign users to roles
  6. Create users
  7. Assign privileges to users
  8. Assign roles to users
  9. ...

The Security Center can be started from the main menu via Start -> System management -> Security -> Security center.
