Export configuration data for CyberLab - Billing rules

To export billing rules, the option Export billing rules of the CyberLab export window should be activated.

The following sections describe which data is exported and which not.

Exported data

The following data is exported:

  • The data of the billing rule tables:

    • billing rule data
    • billing rule line
    • billing rule line input
    • billing rule line output

    Some billing rule fields are not mandatory:

    • minimumCount
    • minimumBaseValue
    • minimumAmountValue
    • maximumTranslations

  • Billing rules without billing rule line outputs:

    • seqNo
    • billingCode
    • ifMaxExceeded: optional
    Tip

    Tip

    Exporting billing rules without billing rule line outputs is a way to let billing codes disappear so that they are not billed.

  • Reimbursement class.

    The reimbursement class is indicated in the BillingRuleLineInput section of the xml file.

Non-exported data

The following data is not exported:

  • billing rules of which the end date has passed
  • billing rules of which the start date lies in the future
  • billing rules without billing rules lines
  • billing rules with billing rule lines without inputs
  • billing rule scope
  • MISPL functions
