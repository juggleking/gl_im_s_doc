Export configuration data for CyberLab - Issuers and group practices

To export issuers and group practices, the option Export issuersof the CyberLab export window should be activated.

If the HC code provider is indicated in the CyberLab export window, GLIMS looks for the HC provider identification:

  • if a valid identification is found, it is added to the export file.
  • if a non-valid identification is found, none of the request definition data is exported.

If the HC code provider field is left empty, the HC provider mnemonic is exported instead of the identification.

 

Output file: "Issuers.xml". This file contains:

  • a list of group practices, i.e organisations linked to HC Providers through Members of practice ("HCProvider.Practice" field) in GLIMS.
  • a list of issuers
