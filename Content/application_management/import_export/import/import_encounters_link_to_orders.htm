Connect orders to encounters via import file

Introduction

The program enct_iui allows to import a file where each line contains an order internal id and an encounter external id. When both the order end the encounter exist and belong to the same patient, the order will be connected with this encounter.

Import format

<OrderInternalId> <EncounterExternalId>
<OrderInternalId> <EncounterExternalId>
...

or

"<OrderInternalId>" "<EncounterExternalId>"
"<OrderInternalId>" "<EncounterExternalId>"
...

Starting the program

To execute this program:

  1. Click Start -> Development -> 4GL -> Run procedure
  2. Enter "enct_iui", do not check "Pass handle"
