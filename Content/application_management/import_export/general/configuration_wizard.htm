System manual - Configuration wizard

The configuration wizard can be started from the main menu via Start -> System management -> Customize -> Configuration wizard.

Load printers

This option can only be used if no printers have been defined yet in GLIMS. Moreover, it can only be used on Windows systems. If these criteria are met, all printers defined in Windows will be imported in GLIMS.

Assign default reports

This option requires that an Own organization be defined in the GLIMS General settings. If so all default reports that are not yet assigned to a correspondent will be assigned to the Own organization.

Create fund identifications

This option requires that an HC code provider be specified in the GLIMS General settings. For each fund that does not yet have an identification for the health care code provider (specified in SpecificSite), an identification will be created (with Source=HealthCareCodeProvider and Target=Fund).

Populate default property classification

If no default property classification is available, a new one will be created. The newly created or existing default property classification will be populated/extended with all properties that are not yet used in this classification.

Set EURO support

A default country (country of the Own organization) must be available. This country must use the Euro as currency. If so, the following Billing General settings fields will be set:
  • EuroSupport=Yes
  • EuroCurrency is set
  • LocalEuroValue is set

Set billing partner

A default country (country of the Own organization) must be available and this country must be The Netherlands. If so, GLIMS will check if there is an institution with mnemonic 'ComPartner' and name 'Communication partner'. If not, it will be created. This institution will be set as Billing partner in the Billing General settings.

Set health care code providers

Depending on the default country and the site language, GLIMS checks the existence of the following correspondents:
  • Belgium: RIZIV (NE) or INAMI (FR)
  • Germany: KBV (or IK as alternate HC Code Provider)
  • France: SECURITESOCIAL
  • The Netherlands: VEKTIS

If available, it will be set as HC code provider in the GLIMS General settings. It will not be created automatically.

Set firm

This option requires that an Own organization be defined the GLIMS General settings. If so, a firm will be searched with the same mnemonic as that organization. If not available, it will be created. Moreover, the option will make sure that this firm has the same Name, Municipality and Address as the Own organization.

Create employee assignment

If the logged in user is not yet assigned to any department:

  • An internal lab is searched. If not available, a new internal lab will be created.

  • If there is a firm of which the name matches the name of the Own organization, this firm will be linked to the internal lab (Default firm field).

  • A department within the internal lab is searched. If not available, a department with name "Default" is created.

  • An employee assignment is created linking the logged in user with the department.
