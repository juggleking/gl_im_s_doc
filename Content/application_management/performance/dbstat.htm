Database statistics

Access

This tool is available from the main menu: Start -> System management -> Database -> Statistics. For more information on each field, please check the Progress manuals.

Warning

This tool has been developed for MIPS employees only. The function has minimal user type Developer and should therefore not be executed without the assistance of a MIPS employee! 

Locks

If the Show option is set to Locks, two buttons Analyze and dump and Generate pro traces are available.

Analyze and dump

Allows to dump lock information to a text file.

Generate pro traces

Allows to generate protrace files for locked/locking processes (for local processes only).

Performance

If the Show option is set to Performance, a Diagnose button allows to verify the following items (PROGRESS databases only, not for ORACLE !):

Database buffer cache hit ratio >= 93 %
Checkpoints are at least 90 seconds apart
Buffers flushed should be less than 100 /day
Record waits should be less than 3 /hour
Page writer queued writes < 25 % of total apw writes
No APW should be active on GENRO database
Minimum 1 APW's for genrw
Minimum 2 APW's for glims
Empty bi buffer waits should be less than 0.1% of # bi writes
writes by BIW % should be >= 60; (or 30 when AIW also running)
Partial bi writes < 50% total bi writes
check if db need to be extend (writing in overlow extend)
warning if db block size < 4 K
warning if bi block size < 4 K
Lock table high water mark < 80% of lock table
-Ma <n> should be -Ma 4 at most (-Ma 6 for genro allowed).

A warning and/or advice will produced if one of the above items is not met.
