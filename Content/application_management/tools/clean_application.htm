Application cleanup

GLIMS offers a tool that allows to

  1. Disable the Needs checking field of reports
  2. Set the Phone status of orders to No

for report/orders dating from a specific point in time (x months old).

  1. For reports the value of Report.LastOutputTime and Report.EndTime is checked.
  2. For orders, the value of Order.LastUpdateTime is checked.

The tool can optionally be run in test-mode. In that case, no changes will be done to the database, but the log will show how many records satisfy the specified criteria.
