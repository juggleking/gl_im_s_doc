Introduction

The About box provides some general information on the application and its environment. It can be accessed by clicking on the Help icon (question mark) in the upper right corner of the GLIMS screen and choosing About GLIMS.

Application page

Name

Is the name of the application.

Version

Shows the major (structural) and minor (functional) version number.

AppServer version

Shows the application version on the AppServer (or ? if no AppServer is connected). This should always show the same version as the client, if not, an upgrade of one of them needs to be executed.

Progress product

Shows which version of the underlying Progress RDBMS you're using.

Terminal type

TTY (UNIX) or WIN (Windows).

Current directory

Is the folder you started the application from.

Temporary directory

Server identifier

Identifies your server system. It is required to obtain software module activation codes.

Databases page

Logical name

Allows you to select the database about which you want information on the rest of the page.

System page

Operating system / OS Release

Identify the version of the operating system you are currently using.

Libraries

Displays all third party libraries used in GLIMS. For each library, it shows the name, version number and license.

Extra info

This customizable tab page contains the content as configured in the MATE general settings:

  1. Choose Start > System management > Customize > MATE.
  2. Open the User interface tab page.

  3. Specify a dynamic text in the field About box extra info. If no dynamic text is specified, the tab page will not be visible.

    Note

    The dynamic text can contain HTML for the visualization of images, tables...
