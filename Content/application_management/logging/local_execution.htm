Local execution

In certain situations, it is necessary to review client logs collected from code that is usually executed on AppServer. However, collecting client logs from code executed on AppServer is cumbersome. The Expert settings option Local execution, makes it possible to collect client logs without using AppServer.

When this option is activated, the session does not hop to AppServer anymore.

Note

When GLIMS starts, this option is inactive, even if it had been activated in the previous session.
