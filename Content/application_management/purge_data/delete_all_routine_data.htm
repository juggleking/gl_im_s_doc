Delete all routine data

Description

In addition to the tool which allows to purge OLD routine data, GLIMS also offers a way to delete ALL routine data. It goes without saying that this program should be used with extreme care. Its primary use is to clean the database from all test data in order to prepare a new installation for routine start.

The program will clean routine data from the database (Orders, results, billing data, work lists,...). The deletion of patient data (Person, Medical Record, ...), blood bag data, QC results and object data (Objects, object attribute history, ...) is optional.

Access

This tool is accessible to users of type System manager and Developer.

Warning

Given the risks involved, this tool is by default not added to the menu / ribbon.
