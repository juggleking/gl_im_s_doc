Directory cleanup

Here you can download a zipped Visual Basic script which allows to delete files from a given directory (and subdirectories).This could be used to delete temporary files from a directory on your GLIMS server/client.

Command:cscript obsolete.vbs ## folder [-s] [-l]

  • ##: the number of days to keep
  • folder: folder (current folder by default) to do the deletes in
  • -s: does the deletes on all sub-folders
  • -l: list files to delete without actually deleting them.

Examples

1) cscript obsolete.vbs 5 d:\idmn

Will delete all files older then 5 days on the directory d:\idmn

2) cscript obsolete.vbs 5 d:\idmn -l

Will list all files older then 5 days on the directory d:\idmn but will not delete them

3) cscript obsolete.vbs 5 d:\idmn -l -s

Will list all files older then 5 days on the directory d:\idmn and subdirectories but will not delete them.
