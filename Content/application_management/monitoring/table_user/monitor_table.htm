Table monitor

The Table Monitor tool (accessible from the main menu via Start -> System management -> Activity -> Table monitor) shows figures about table access/create/deletes.  Note that this tool is only supported for Progress databases !

Which tables can be monitored by the table monitor depends on a Progress database startup parameter.  Use Table Range Size (-tablerangesize) to specify the number of tables for which you want to collect statistics.   (for database "glims" specify it in "glimssv.pf").  Example: -tablerangesize 800
