Current logins

GLIMS allows the consultation of ongoing login sessions via Start -> System management -> Security -> Current logins.

The browser shows all current database connections.

Shutdown

This button is only be available on Progress systems. Clicking it will attempt to shutdown all databases.

Disconnect

This button is only be available on Oracle systems. It is used to disconnect a selected session from the database. Only user sessions can be disconnected and you cannot disconnect your own session.
