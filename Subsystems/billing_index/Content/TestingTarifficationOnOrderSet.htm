Testing tariffication on Order set

It is possible to do a tariffication test on an Order set. This can be useful to check the tariffication after adjustments were made in the billing configuration or to check why certain Billing codes are/aren't tariffed.

When executing a tariffication test, the tariffication process will be executed for the Order set as if it were a real tariffication.

However, GLIMS will create interim Billing items and Invoices for that order. Depending on the logging settings of the tariffication, GLIMS will provide a log file where the application of the Billing rules throughout the tariffication process of the Order set are logged.

Executing a tariffication test

By following the print screens listed below, a tariffication test can be executed on a specific Order set.

In order to obtain a tariffication test, all toggle boxes on the screen should be checked off and the Logging level 'Billing rule details' should be chosen.

At this point, the Billing items and the Invoices on the Order set can be checked. After reviewing them and closing the screen, they will disappear.

Checking the Billing rule details

When choosing the Logging level 'Billing rule details', GLIMS will provide a logging that displays how the different Billing codes are chosen during the tariffication process.

Via Start - System management - Logging - Logs, the logging can be viewed.

The logging gives the different steps that have been done during the tariffication process.

  • The Billing codes that were initially picked up (1)
  • Billing rules that were applied, including overview of the before and after (2)
  • Billing codes that will actually be tariffed (3)

In the example below, originally the code 127035 was picked op twice as can be seen in the yellow part (1).

Once because of the HT Request and once because of the RBC Request.

In the orange part (2), a Billing rule is applied (Max127035) that allows this specific Billing code to be tariffed only once.

In the final red part, an overview is given of those Billing codes that will actually be tariffed. Here, Billing code 127035 is shown only once, so it will only be tariffed once.
