Release notes v6.0.2 - Default watchdog.dashboard.properties file (WatchDog-00319)

An issue was reported where the Watchdog Dashboard properties file (conf/Watchdog.dashboard.properties) was not properly converted from Watchdog 5 to Watchdog 6.

In addition, the example file contained old settings - some of which were no longer used or revised.

This has been corrected:

  • the example properties file now contains the correct entries
  • old properties files are now interpreted correctly and converted to the new format when the user hits the OK button in the Preferences window.
