Release notes v6.0.2 - Correction for Watchdog daemon monitoring (WatchDog-00321)

An issue was detected where Dashboard no longer detected that the backing WatchdogDaemon had timed out (i.e. no message was received in the last two minutes).

This has been corrected. Dashboard now detects that the Daemon has timed out, displays a proper error message to the user, and closes itself - as in earlier versions.
