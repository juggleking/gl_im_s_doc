Release notes v6.0.1 - Display Filter description above grid (WatchDog-00286)

Each of the following tabs in the Dashboard have a filter panel:

  • Communication
  • Tasks and Reports
  • System Status
  • Message Log

It was remarked that the user may not realize that a filter is active when the filter side panel is collapsed.

Hence it was decided to display a filter description above the grid in each of the aforementioned tabs.
