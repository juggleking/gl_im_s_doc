About the release notes

Sorting and filtering

Modification ID Allows to search for a modification by typing any part of its ID (e.g. "ANLZ").
Version Allows to filter and sort on the version in which the modification has been implemented.
Category Allows to filter and sort on the category to which the modification belongs.
Title Allows to search for a modification by typing any part of its title (e.g. "report builder").
Type Allows to filter and sort on the type of modification (only new functionality, only corrections or both)
Also in / by Indicates if a modification has already been implemented in a corrective version of the previous functional version, and by which modification.

Export to MS Excel

The Excel icon allows to export the entire release notes table to an MS Excel sheet. 

Full PDF document

The PDF icon offers a full overview of all release notes (including the detailed description) in a single PDF file.

Window resizing

You can resize the window containing the release note table by dragging the grey horizontal bar.
