About Watchdog

  • Watchdog is a background program that monitors the connection status of services and translators, available disk space, etc.

  • It is installed on the server running the application you monitor.

  • This information is further broadcasted to a dashboard for visualization on client workstations.

  • Whenever a problem occurs, Watchdog can alert every dashboard and every user of with a Role of type System manager (see Watchdog notifications).

  • Watchdog can also be integrated with Nagios.

Note

Watchdog is an optional program component and requires a separate license.
