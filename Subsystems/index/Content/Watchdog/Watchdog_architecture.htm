Watchdog architecture

Overview

Message flows 

At regular intervals, services report their status (and the status of the connected translators) to the Watchdog daemon.
The Watchdog daemon transmits this information via the Dashboards, Alerts or a Nagios agent.
When starting / stopping a service/translator from within the Dashboard, this request is sent to the Watchdog daemon.  The Watchdog daemon then asks the appropriate XML control service to perform the start/stop action.
