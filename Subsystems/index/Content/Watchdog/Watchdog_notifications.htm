Watchdog notifications

Notification info

Watchdog can send notifications about:

  • The status of services

  • The status of translators

  • The status of task schedulers (if enabled in the Start Watchdog window)

  • Quality control issues (if enabled in the Start Watchdog window)

  • The databases

  • The FTP server

  • Server disk space (if enabled in the Start Watchdog window)

  • The status of AppServer

Notification methods

Notifications are sent to all system managers (users having a Role with Type set to System manager) and can be sent as an e-mail or an internal application mail.

Method Description Activation
Windows Messenger message A Windows Messenger message will only be received by the system manager, if his login name in the application you monitor is the same as his Windows login name. Can be activated in the Start Watchdog window.

E-mail

An e-mail is only sent to the system manager if

  • an e-mail address

  • a mail server and

  • a mail port

have been defined in the Options (System management -> Options -> Mail tab).

Can be activated in the Start Watchdog window.

 

Internal application mail

Notifications will be sent as internal application mail to all 'System managers'.

Can be activated in the Start Watchdog window.
