Dashboard - Message Log area

Filter panel

Collapse / Expand

The Filter panel can be collapsed / expanded with the Arrow icon.

Search

The Text Search filter is applied to all items in the Main panel.

Main panel

The Main panel provides an overview of all monitoring messages received by Dashboard.

Notes

  • This overview only shows message logs, not application logs.
  • In the Setup area, a message expiration period can be specified, for each message type, after which the messages should be automatically removed.

Details panel

Detailed info

Select one of the items in the Main panel to view its details in the Details panel.
