Dashboard - Communication area

Filter panel

Collapse / Expand

The Filter panel can be collapsed / expanded with the Arrow icon.

Search

The Text Search filter is applied to all items in the Main panel.

View Active Only

If enabled, inactive items will be hidden in the Main panel.

Filter by Group

The Filter by Group option is applied to the Translator Servers, Message Queues and Communication Engines.

Tip

To view a subset of items:

  1. Disable the check box All Groups
  2. Enable the check boxes for which you want to view items.

The different filter criteria are:

All Groups

Show / Hide all items.

Instrument Group

Show / Hide items for all or specific instrument groups.

Communication Engine

Show / Hide items for all or specific communication engines.

Service Group

Show / Hide items for all or specific service groups.

Note

When changing configuration in the application being monitored, you have to stop and start Watchdog before the changes are reflected in the filter panel.

Main panel

Translator services

The first column shows the Service (of type Translator Server). The second column shows the Translators associated with the service of the first column.

Appearance of services

Regular Hover Selected Status Description
Inactive Defined in database, but not started.
Active Sent activity confirmation less than one minute earlier
Disconnected

Did not send an activity confirmation since more than one minute.  

This can be caused either by the service being inactive or by the service being busy.

Note

Watchdog generates an Alert if a service remains in status Undetermined for more than 50 seconds.

Appearance of translators

Regular Hover Selected Status Description
Inactive Defined in database, but not started.
Connected Started and connected to the application you monitor.
Idle

Some Translators will close their external connection during 'idle' state, and report their state as Disconnected - leading people to think that something is wrong with the Translator when it is in fact still running. A status 'Idle' was introduced to prevent false alarms.

Translators will show an hourglass icon.
Disconnected

Some translators will close their external connection during 'idle' state, and report their state as Disconnected - leading people to think that something is wrong with the Translator when it is in fact still running.

Translators will show a warning icon.
Alert Only applies to Java translators: indicates that the Java Communication Engine is disconnected

Notes

  • A translator using FTP as low-layer protocol does not have the notion of being connected or disconnected. It is either Inactive or Active.
  • Watchdog generates an Alert if a translator remains in status Disconnected for more than 10 seconds.

Message Queues

The first column shows the Service (of type Message queue). The second column shows the Queue processors associated with the service of the first column.

Appearance of message queues

Regular Hover Selected Status Description
Inactive Defined in database, but not started.
Active Sent activity confirmation less than one minute earlier
Disconnected

Did not send an activity confirmation since more than one minute.  

This can be caused either by the message queue being inactive or by the service being busy.

Appearance of queue processors

The appearance of queue processors is identical to that of Translators.

Communication Engines

The first column shows the Service (of type Java communication engine). The second column shows the last Communication engine message.

Appearance

Regular Hover Selected Status Description
Inactive Defined in database, but not started.
Active Sent activity confirmation less than one minute earlier
Disconnected

Did not send an activity confirmation since more than one minute.  

This can be caused either by the communication engine being inactive or by the service being busy.

Note

If a Communication engine does not send CommunicationEngineAlive messages for 2 minutes, its status is automatically changed to Disconnected.

In turn, the status of all associated Translators is automatically changed to Alert. The Watchdog Dashboard will display these in Red to signal that these Translators need special help to start working again...

Details panel

Start / Stop

The selected service or translator can be started / stopped using the icons in the top right corner.

Note

The user must have sufficient privileges to start / stop services and translators.

Detailed info

Select one of the items in the Main panel to view its details in the Details panel.

Message area

Below the detailed info, a message area is available that can show a history of messages for the selected item.
