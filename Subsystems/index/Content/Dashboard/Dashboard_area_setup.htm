Dashboard - Preferences

Introduction

The Preferences window allows to set preferences for the Dashboard application.

It is available from the Hamburger menu.

Note

The preferences are stored in the watchdog.dashboard.properties file located in Watchdog's conf directory.

Message Expiration

For each type of message / warning, you can specify the period after which such messages / warnings should be automatically removed from the Message Log.

The trashcan icons allow to instantly remove all current messages / warnings.

Notification Windows

About Dashboard notifications

In addition to Watchdog notification, the Dashboard itself also offers notification windows.

Dashboard distinguishes between the following notification types:

Information

Warning

Alert

The Alert type is meant for situations where the system administrator should investigate as soon as possible but there is no immediate technical error.

Error

You can define:

  • if these notifications should be displayed at all.

  • how long they should be displayed before being closed automatically .

  • where they should be positioned.
