Dashboard - Window elements

Hamburger menu

The hamburger menu is located in the top left corner of the screen.

The user can pick the following items:

Preferences

Allows to set preferences for the Dashboard application.

About Dashboard

Shows information about the Dashboard application.

Exit Dashboard

Allows to exit the application.

Main Tabs

Items

  • Communication

  • Tasks and Reports

  • System Status

  • Message Log

Main area

The content of the main area depends on the selected Main Tab .
