Starting Dashboard

From the application

Access

The Dashboard can be opened from within the application:

  • from the application's Main menu, via System management -> Services -> Watchdog -> Dashboard.

Application to be monitored

When opening the Dashboard from within the application, it goes without saying that Dashboard will monitor that same application.

Single Sign-On

When opening the Dashboard from within the application, the user will not have to enter user name and password to connect to the Dashboard. The user credentials and language are inherited from the application.

Standalone

Access

To start the Dashboard from outside the application:

  • Double-click the Dashboard icon on the desktop.
  • Select the Dashboard menu item in the Windows start menu.

Tip

The Dashboard command-line interface accepts following named arguments:

--name="<Daemon service display name>"

--host="<Daemon service host name>"

--port="<Daemon service port number>"

--user="<Login user name>"

This allows system administrators to setup customized dashboard.bat scripts to skip the Service Discovery mechanism and override the Operating System user name.

Example script:

start "" "c:\samplenet\jre8\bin\java.exe"

-Dwatchdog.home="c:\samplenet\watchdog"

-jar ./lib/watchdog-javafx-dashboard-6.0.0.jar

--name="Your MIPS application"

--host="SampleNetSvc"

--port="3500"

--user="JaneDoe"

The example is broken up for legibility. The actual script would have above on a single line.

Application to be monitored

When starting the Dashboard from outside the application, a Connection dialog appears.

Select one of the services below

The Service Discovery mechanism will detect all Watchdog daemon processes. By selecting one of these Watchdog daemons, the user determines which application will be monitored by the Dashboard.

Specify the host and port of the service

Manually specify the Host and Port of the Watchdog daemon.

Login

User name, Password

The supplied User name and Password are verified by the application to be monitored.
