Main panel

Disks

Watchdog monitors all disks on the application server if the option Monitor server disks is enabled when starting the Watchdog.

Tip

Watchdog checks the usable disk space on all server disks (except read-only and removable drives) and sends warning or alert notifications when the usable disk space drops below the thresholds specified when starting Watchdog (Disk space warning threshold and Disk space alert threshold).

Appearance

Regular Hover Selected Status Description
OK Usable disk space is above the disk space thresholds specified when starting Watchdog.
Warning Usable disk space has dropped below the Disk space warning threshold specified when starting Watchdog.
Alert

Usable disk space has dropped below the Disk space alert threshold specified when starting Watchdog.

FTP Servers

Watchdog monitors FTP servers used for uploading images to the LIS, not the FTP server used for handling images received from instruments.

The FTP connection settings (Host and Port) are defined in the LIS channel.

Appearance

Regular Hover Selected Status Description
Inactive

The FTP host and FTP port are specified for a LIS channel, but the Outbound translator of that LIS channel is not active.
Active

The FTP host and FTP port are specified for a LIS channel.

The Outbound translator of that LIS channel is active.

Watchdog is able to communicate with the FTP server
Disconnected

The FTP host and FTP port are specified for a LIS channel.

The Outbound translator of that LIS channel is active.

Watchdog is not able to communicate with the FTP server

Databases

Watchdog monitors the server databases.

Appearance

Regular Hover Selected Status Description
Active

The database is running.
Disconnected

WatchdogDaemon was unable to check the database.

Application Servers

Watchdog monitors the OpenEdge Application Servers.

Appearance

Regular Hover Selected Status Description
Active

The application server is running.
Disconnected

The application server is disconnected.

Service Controllers

Watchdog monitors the application's Service Controllers.

Appearance

Regular Hover Selected Status Description
Inactive Defined in database, but not started.
Active Sent activity confirmation less than one minute earlier
Disconnected

Did not send an activity confirmation since more than one minute.  

This can be caused either by the service being inactive or by the service being busy.

Note

Unlike other types of services, you cannot start or stop services of type Service controller from the Watchdog Dashboard, to prevent that they are accidentally stopped and no longer available to handle requests to start or stop other application supportive processes on their host server.
