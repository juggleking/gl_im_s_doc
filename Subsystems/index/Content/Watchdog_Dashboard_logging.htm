Watchdog / Dashboard logging

Watchdog daemon

Logging options are specified when starting Watchdog

When starting the Watchdog daemon, the settings specified in the Logging tab page will be applied.

Logging options are not specified when starting Watchdog

When no logging settings are specified when starting the Watchdog daemon, the Log4J logging system will be used.

Dashboard application

For the Dashboard application, the Log4J logging system will be used.

Log4J logging system

Introduction

Dashboard uses Log4J, a feature-rich and configurable logging system developed and maintained by the Apache community.

Configuration files

The configuration files for application logging can be found in Watchdog's "conf" directory.

A separate configuration file is available for the logging of the Watchdog daemon, the Dashboard application and the Nagios agent.

Note

The configuration file for the Watchdog daemon is only used when no logging settings are specified when starting the Watchdog daemon. The latter take precedence.
