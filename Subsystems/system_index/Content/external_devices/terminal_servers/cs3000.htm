Setting up the hardware

Configuring ports

Analyzer communication

For each analyzer to be connected to the terminal server, you will have to configure a port for it.

  1. Connect the analyzer using the appropriate cable (Standard db-25 connector). Plug in the cable in any port of the terminal server, and connect it to the serial interface of the analyzer.
  2. Start telnet from any workstation in the network to the terminal server. You can do this by going to the Start-menu -> Run..., and type telnet + OK.
  3. Go to the Connect -> Remote System menu item
  4. Fill in the following parameters and hit the Connect button
    • Host Name: The IP number of your terminal server
    • Port: telnet
    • TermType: vt100
  5. Hit enter once
  6. enter SET PRIvilege = NetMgr
  7. setd -tcpappl tcpbp=3000  (for raw tcp)
  8. setd -tcpappl tcpnp = disabled 
  9. setd !'potrnr' -term BA = 9600 (set baud rate)
  10. setd !'portnr' -term PAR = none (set parity)
  11. setd !'portnr' -term SB = 1 (set stop bits)
  12. setd !'portnr' -term DV = Host
  13. setd !'protnr' -term FCF = none
  14. setd !'portnr' -term HED = off
  15. setd !'portnr' -term LFI = none (LFInsertion)
  16. setd !'portnr' -term XOFF = Disabled
  17. setd !'portnr' -term XON = Disabled
  18. setd !'portnr' -term MO = Transparent
  19. setd !'portnr' -term ECHOD = off
  20. setd !'portnr' -term 
  21. enter LOGout on the prompt
  22. Close telnet.
