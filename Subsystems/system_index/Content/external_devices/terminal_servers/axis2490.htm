Terminal servers: Axis 2490 Serial Server

Supplier

  • Axis
Note: On the Axis website, the Axis 2490 Serial Server is stated as a discontinued product. It will be supported until 15/06/2009.

Setting up the hardware

When you set up the terminal server for the first time, it does not have an IP address yet.

  1. Connect the terminal server to the network, by plugging in a standard network cable into the network connector.
  2. Connect the axis power supply.
  3. Acquire an unused Internet Address from your Network Administrator and choose a unique name for your AXIS 2490
  4. Note the serial number (Ethernet address) on the underside of the unit. You need to know this to set the IP address.
  5. Using an appropriate method for your operating system, assign your product with a unique IP Address from a computer on your network, as follows:
    • Windows, type the following commands:
      • arp -s <AXISserver IP address> <Ethernet Address> <my pc IP address>
      • ping -t <AXISserver IP address>
    • Unix, type the following commands:
      • arp -s <AXISserver IP address> <Ethernet Address>
      • ping -t <AXISserver IP address>
Note: In some Unix systems, the arp command can be located in a directory that is not on the command path; e.g: /usr/sbin/arp.

Configuring ports

Analyzer Communication

For the analyzer to be connected to the terminal server, you will have to configure a port for it.

  1. Enter the Internet adress or the host name of the AXIS as a URL in your Web browser as follows:
    • http://'ip-address' or
    • http://'host'
  2. First time setup, use the Installation Wizard:
    • Create a new user: Mips\gl1ms1
    • Fill in the required IP addresses
  3. Configuration of the ports
    • Choose Admin
    • Choose the required port and fill in the settings of the port.

Physical connection

RS-232, you need to cross pin 2 and 3 between the terminal server and the analyzer (Same as configuration as the Axis 5600)
