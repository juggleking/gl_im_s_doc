Jetstream - Installation notes

Connection to terminal server

telnet 'server-name' or telnet 'server-ip-adr'
e.g. # telnet js1

After this, a login prompt will be displayed, type 'admin' and press <return>. At the password prompt, just press <return>. This is the default admin user password.

login:admin
passwd: (none)

The command line prompt (#) will be displayed. Now you are logged in as the system administrator. To use Full Screen mode (menus), you must first set your terminal type. Use the 'set line' command as follows:

# set line . termtype vt100

Now to enter Full screen mode, type screen and press <return>. The main menu will be displayed.

# screen

Main Menu

Select the option 'Server Configuration'

  • Server name : the name of your server e.g. mts1

  • Internet Address : e.g. 195.49.144.4

  • Broadcast Address: e.g. 195.49.144.255

  • Subnet Mask : 255.255.255.0

  • Domain Name

After completion, press <return> to exit. The Quit form will be  displayed to save the form.

Select the option 'Network Configuration Menu'

  • Select the Host table (hostnames and internet addresses of the hosts)

  • Select the Change Host to have an overview of the already known hosts.

  • Select Add Host to add a new one.

Select the option 'Line Configuration'

Select the option 'Line Settings' 

  • Select 'none' as Flow

  • Select 'dumb' as terminal type

  • Select 'off' as Modem

  • Select the corresponding Speed, Bits, Parity

  • Hostname : the name of the host

  • Type : rev raw: this line type enables TCP/IP hosts to establish connections on external machines via JetStream ports. The JetStream is effectively transparent. Reverse raw connections are typically used to access printers or dial-out modems.

  • Host Port: 23

  • TS Port : the socket port number (E.g.: 10001)

Press <enter> and accept settings.  Note: You can also copy line settings to other lines.

GO BACK TO LINE COMMAND

Press escape until you reach the main menu, the select line command (enter)
You should return to the prompt

REBOOT THE SERVER

Whenever you set or change the Server Configuration, you must reboot the server. This will broadcast the information across the network.

# reboot server

CHECK OUT CONNECTION

Issue command from the host:

telnet terminal-server port 

and see if you get connected on the port.
