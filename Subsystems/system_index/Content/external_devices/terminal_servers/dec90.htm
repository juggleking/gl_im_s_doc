Terminal servers: Dec 90m

Installation

  • telnet `tcp-ip address of the terminal server`
  • access (default password at the prompt, to log in)
  • c (connect)
  • set priv (to make sure that you can change the settings)
  • system (default password)

To see the settings of a port :

  • show port `port-number` (port-number is the number of the port between 1-16)

Do not forget to change the following options for the ports

  • change port `port-number` dedicated none
  • change port `port-number` access remote
  • change port `port-number` default protocol telnet
  • change port `port-number` telnet listener `tcp-port` (take a number for "tcp-port" between 2001 – 2016).
  • change telnet listener `tcp-port` type raw tcp
  • change telnet listener `tcp-port` port `port-number` enabled
  • change telnet listener `tcp-port` connections enabled

Normally we try not to have any settings by the option "ENABLED CHARACTERISTICS"

  • change port `port-number` lock disabled
  • …

and so on until you don't have any options after "ENABLED CHARACTERISTICS".

If all options are set type

  • lo `port-number`

To exit the telnet session type

  • logout

How to change the IP address on a DEC 90M

In dos:

telnet 128.18.30.200
<return>
# <return>
# access <return>
#<return>
Enter Username > mips <return>
Local >
Local > set priv <return>
password system <return>
Local > <return>
Local > define int ad 10.x.x.x <return>
Local > define int mask 255.x.x.x <return>
Local > logout<return>

Reboot the server ( remove power cable and re-activate).

The dec 90m now has a new IP address
