OMR translator generated errors

  • Translator generated errors:
    • OM__ERR_CATEGORY_AMBIGUOUSGFORM
      More then one form of the selected category meets the sheet read by the reader. This is probably a form parametrization error.
    • OM__ERR_CATEGORY_NOVALIDIDENTIFIER
      Category has no valid identifier.
      Invalid sheet data was read. Probably, a sheet of an incorrect type was read.
    • OM__ERR_FORM_INTERNAL
      Internal form processing error. Contact your software provider.
    • OM__ERR_FORM_PARSEERROR
      A sheet of unknown type was received. Probably the omr reader uses a different protocol then the protocol expected.
    • OM__ERR_FORM_UNEXPECTED
      The translator unexpectedly received a sheet from the reader.
      Check the translator external interface timeout, it may be too short.
    • OM__ERR_FORM_UNMATCHEDCLOCKS
      The sheet being read does not contain the expected ammount of clocks. Probably, a sheet of an incorrect type is being read.
    • OM__ERR_CONTROL_INTERNAL
      Internal control processing error. Contact your software provider.
    • OM__ERR_CONTROL_NOCHILD
      In the parametrization, a non-existing child control is being calculated from within a control. Only direct child controls can be calculated.
    • OM__ERR_CONTROL_MISPL
      Internal MISPL processing error. Contact your software provider.
    • OM__ERR_CONTROL_TOMANYFIRES
      Some marks or barcodes wich may not be available were set on the sheet. Correct the sheet.
    • OM__ERR_CONTROL_TOFEWFIRES
      Some marks or barcodes wich need to be set on the sheet were not available. Correct the sheet.
    • OM__ERR_CONTAINER_MANDATORY
      A container control node was marked mandatory in the parametrization, but was not available on the sheet read. Correct the sheet.
    • OM__ERR_MARK_MANDATORY
      A mark control node was marked mandatory in the parametrization, but was not available on the sheet read. Correct the sheet.
    • OM__ERR_BARCODE_MANDATORY
      A barcode control node was marked mandatory in the parametrization, but was not available on the sheet read. Correct the sheet.
    • OMR__ERR_BLOCK
      Reader blocks (technical failure, paper jam, ...). Check the OMR.
    • OMR__ERR_INTERNAL
      Internal reader communication error. Check the OMR.
    • OMR__ERR_NO_CLOCKS
      No clocks on sheet read by the OMR.
      The sheet scanned by the OMR is not a OMR form.
    • OMR__ERR_NO_DATA
      No data nor errors returned from sheet parser. Probably, the database form parametrization is still empty (unconfigured).
    • OMR__ERR_NO_FORM
      No sheet was available on the OMR.
      Place a sheet in the OMR input tray.
    • OMR__ERR_UNKNOWN
      Unknown message format received from the OMR.
      An incorrect sheet was placed on the OMR.
  • MISPL generated errors
    Set in MISPL by the configuring user using a dedicated set error MISPL function. Here, any string can be added to the error log.
