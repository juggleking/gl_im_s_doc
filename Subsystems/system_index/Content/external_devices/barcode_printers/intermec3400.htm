Example of a label

This example applies to a barcode-label on the Intermec-3400 printer; the used label is 4.45 cm wide and 3.45 cm high. The intermec-3400 printer uses templates to print labels. First we define a template, then we define the data to fill that template. Please note that all text in the following example is meant literally; the string <STX> is 5 characters and should not be translated before sending it to the printer!

First we initialize the printer:

<STX><SI>D0<ETX>

 
<STX><SI>T1<ETX>

 
<STX><SI>S60<ETX>

 
<STX><SI>I2<ETX>

 
<STX><SI>D0<ETX> Don't push the labels out after printing
<STX><SI>T1<ETX> Type labels = 'continuous'
<STX><SI>S60<ETX> Print speed = 60 (max)
<STX><SI>I2<ETX> 2 Image bands (see handbook, this affects the speed)

Template definition:

<STX><ESC>P<ETX>
<STX>E4;F4;<ETX>
<STX>H0;o120,10;f3;c25;h12;w9;d3, ;<ETX>
<STX>B1;o230,50;f3;c2,0;h105;w2;i0;d0,20;<ETX>
<STX>H2;o130,10;f3;c25;h12;w9;d0,30;<ETX>
<STX>H3;o105,10;f3;c26;h12;w10;d0,32;<ETX>
<STX>H4;o80,10;f3;c25;h12;w12;d0,2;<ETX>
<STX>H5;o80,30;f3;c25;h12;w12;d0,20;<ETX>
<STX>H6;o90,180;f3;c26;h18;w24;d0,20;<ETX>
<STX>H7;o130,220;f3;c25;h12;w9;d0,20;<ETX>
<STX>H8;o55,10;f3;c25;h12;w9;d0,40;<ETX>
<STX>H9;o50,220;f3;c25;h20;w15;d0,6;<ETX>
<STX>H10;o5,320;f4;c25;h12;w12;d0,25;<ETX>
<STX>H11;o25,10;f3;c25;h10;w9;d0,40;<ETX>
<STX>R;<ETX>

It is a template with 11 fields including 10 of type H (Human Readable) and 1 of type B (Barcode).

o = origin
f = rotation
c = font type
h = font height
w = font width
d = data type (d0 = data entered during print mode, 40 = max length)

<STX>R;<ETX> we're done defining template(s), start print mode

<STX><ESC>E4<ETX>
<STX><CAN><ETX>
<STX>1120063311<CR><ETX>
<STX>200112-00633-11<CR><ETX>
<STX>TEST TESTKIND92<CR><ETX>
<STX>M<CR><ETX>
<STX>01-01-1992<CR><ETX>
<STX>CITO<CR><ETX>
<STX> 09:30<CR><ETX>
<STX>li hepbl centr<CR><ETX>
<STX><CR><ETX>
<STX>cx7<CR><ETX>
<STX>na,k<CR><ETX>
<STX><ETB><ETX>

<STX><ESC>E4<ETX> access template E4
<STX><CAN><ETX> clear all host entered data for this template

Then the fields follow. Normally, such a field should contain a MISPL-expression, as e.g. {=.Order.InternalId}.

<STX><ETB><ETX> This template is ready, print the label.
