GLIMS - Upgrade on a client with an Oracle database

Introduction

This document describes the upgrade of a GLIMS 8 to GLIMS 9 client for use with an Oracle database.

Installation

Extract the installation files you have downloaded from the MIPS website in an installation folder of your choice.

Start the installshield setup by running setup.exe from the installation folder.

 

Wait for the initialization to complete.

 

A welcome dialog is shown. Click Next to proceed.

 

The setup will show a warning screen when the system requirements are not fully met.
Warning messages are not blocking the setup.
Fatal messages can stop the installer from continuing.

 

Optionally you can click in the "Open logfile" button to show more details.

 

Please accept the License Agreement.

 

The installer will detect that a GLIMS 8 client is installed.

 

Verify the location of the current GLIMS 8 installation.
E.g.: C:\glims8

 

Verify the location of the current GLIMS 8 progress.ini file
E.g.: C:\glims8\bin\progress.ini

 

Choose the folder where GLIMS 9 will be installed.
E.g.: C:\glims9

 

Most setups can be performed with the Typical option.

 

However, if you want to install rather rare components like the Java Communcation Engine, the Carte Vitale drivers or the German billing data files, you can choose for a Custom install. In this case the following screen will be shown.

 

The temp folder is typically located under the GLIMS root folder.
E.g.: C:\glims9\tmp

 

A list of supported Oracle versions found on the system is shown. Select the version to be used with GLIMS.

 

Enter the Oracle Net Service Name (TNS Name) for GLIMS.
In case of a client upgrade, it is likely that there exists already such an entry in tnsnames.ora.

 

Enter the servername or IP address of the GLIMS application server.

 

Specificy the name server listening port as chosen on the GLIMS server. In most cases, the proposed default value is being used.

 

Click Install to begin the installation.

 

Wait until the setup is finished.

 

Choose Finish when the setup is ready.

 

When starting the GLIMS Client for the first time an automatic version update might take place. This merely depends on the versions being installed.

 

Next the GLIMS login screen is shown.
