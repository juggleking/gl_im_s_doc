GLIMS - Corrective upgrade on a client

Upgrade procedure

  1. If you are not on GLIMS 9.9 yet, please switch to the minor upgrade procedure.
  2. If you are already on a GLIMS 9.9 version, take the following steps:
    1. Make sure there are no running GLIMS clients anymore on the system
    2. Run the corrective version self extractor (e.g. Glims9.8.2Progress116_pid3.exe)
    3. Make sure that the "Extract to:" path points to your GLIMS installation folder
    4. Click the "Extract" button and wait for the extraction to complete (e.g. c:\glims9)



    5. You are ready to start the client again, but make sure that the server is upgraded to the new minor version first. 
      The client and the server versions must always match.

Do not copy individual files (e.g. corr.pl) from one system to another. Always extract the full service pack into the GLIMS folder. Some files are dependent on other files in the service pack.
