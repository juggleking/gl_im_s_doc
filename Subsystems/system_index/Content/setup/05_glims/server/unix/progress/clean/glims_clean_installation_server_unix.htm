GLIMS - Clean installation on a Unix server

 

 

 

To start the installation:

Go to the correct path, e.g.: cd /mips/install/glims930pro/install

Copy glimsinst_pro.cfg, e.g.: cp glimsinst_pro.cfg glimsinstpro.cfg

Edit glimsinstpro.cfg with the desired settings (e.g. with vi), e.g.: vi glimsinstpro.cfg

Start the install/upgrade and follow the steps in the setup screens: ./pinstall -c ./glimsinstpro.cfg

Info:

Glims must be installed with the a non-root user (e.g. mips)

The setup must be started in the root of the install folder (e.g. /mips/install/glims930pro/install)

 

 

Edit $GLIMS/.glims.rc to set the following parameters on the desired values, e.g.:

  • DLC=/mips/dlc113
  • GLIMS=${GLIMS:-/mips/glims9pro}
  • GLIMSDBF=/mips/glims9prodbf

The other parameters should be OK by default.

 

Databases maintenance

Check the status of the databases: mam glims status

Info:

  • To start the databases: mam glims start
  • To stop the databases: mam glims stop
  • To show all options: mam glims

 

Start a GLIMS session: mam glims session
