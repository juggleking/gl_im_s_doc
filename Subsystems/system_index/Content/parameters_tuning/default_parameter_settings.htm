System manual -> Default system parameter settings

Configuration of server parameters is not always evident.  Glims therefore attempts to implement an auto-update of server parameters. The following parameters will be forced for each database:

In <dbname>sv.pf:

  • -semsets 3 (unix only)
  • -bibufs 25

In <dbname>db.pf

  • -bi 8192 -biblocksize 8  (on Windows -biblocksize 4)
  • (-bi 4096 for genrw)

On Unix systems, we also check whether background writers are set. (BI, before image writer, APW asynchronous page writers (minimum 1 for genrw, 3 glims database), watchdog).

We recommend not to change these default parameters.  However,  if a site attribute on gp_site is defined with a name <dbname>SVRPARAMS (for <dbname>sv.pf) or <dbname>DBPARAMS (for <dbname>db.pf), then the database server parameters for database <dbname> will be set to the value of this site attribute.
