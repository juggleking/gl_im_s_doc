System manual -> Oracle DB options

!! Warning: this functionality should only be used by MIPS employees !!

By defining a site attribute on gp_Site called 'OracleOptions', it is possible to alter some session parameter settings. This makes it possible to fine-tune Glims customers using an ORACLE database.  This site attribute can contain any valid ALTER SESSION statement like:

alter session set hash_join_enabled = true ;
alter session set hash_area_size = 1000000 ;

rem the lower optimizer_index_cost_adj, the more nested loops (default 100) ;
rem not supported with 8.0.4;
rem alter session set optimizer_index_cost_adj = 100 ; 

rem the higher optimizer_index_caching, the more nested loops (default 0) ;
rem not supported with 8.0.4;
rem alter session set optimizer_index_caching = 0 ; ;
