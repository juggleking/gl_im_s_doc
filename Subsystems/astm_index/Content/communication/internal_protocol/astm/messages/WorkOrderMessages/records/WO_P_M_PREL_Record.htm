Work order message - Person relation record (M)

Usage

This record contains data for a person related to the patient identified in the preceding patient information record.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'prel'
M.3.1.2 Record type name Fixed: 'PersonRelation'
M.4.1.1 Person ID Person ID
M.4.1.2 Relation type ID Relation type ID
M.4.1.3 Relation type name Relation type name
M.4.1.4 Person externalization text Person externalization text
M.5.1.1 Relation start date Relation start date (date value)
M.5.1.2 Relation end date Relation end date (date value)
M.6.1.1 Relation comment Relation comment

Element Descriptions

Person ID (M.4.1.1)

This field contains the ID of patient's related person. The station patient identification setting determines the content of the field, except that the setting 'Object internal id' is not supported. See the patient ID field in the patient information record for more details about the station setting.

Relation type ID (M.4.1.2)

This field contains the person relation type ID. The ID's are the integer values of the Glims enumeration type. See the Glims database reference manual for the possible values.

Relation type name (M.4.1.3)

This field contains the person relation type name. The names are the (untranslatable) names of the Glims enumeration type. See the Glims database reference manual for the possible names.

Person externalization text (M.4.1.4)

This field contains the externalization of patient's related person.

Relation start date (M.5.1.1)

This field contains the start date of the person relation.

Relation end date (M.5.1.2)

This field contains the end date of the person relation.

Relation comment (M.6.1.1)

This field contains the person relation comment
