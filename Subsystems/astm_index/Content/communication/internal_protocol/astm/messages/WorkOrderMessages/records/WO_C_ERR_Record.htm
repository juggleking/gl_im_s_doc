Work order message - Error record (C)

Element Layout

Reference Name Description

C.5.1.1 Comment type Fixed: 'G'

Element Descriptions

Error code (C.4.1.1)

This field contains an error code defined by Glims.

All codes have a prefix identifying the table or context for which the error occurred, followed by a dollar sign ($) and the actual code.

Example: STN$IdNotFound

Error message (C.4.1.2)

This field contains the message corresponding with the error code in C.4.1.1. The message is reported in the language of the service batch user.

Example:

The message corresponding with the error code 'STN$IdNotfound' is:

"Specimen nor QC lot '<id>' found",

in which <id> is the value of field Q.3.1.2 in the query message.
