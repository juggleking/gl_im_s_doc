Work order message - Test order comment record - test related (C)

Usage

This record is available when the query message field Q.14.1.3 contains 'AddComment' and the result request has an internal comment.

The record is omitted when tests are grouped in one test order record (see query message field Q.4.1.1).

Element Layout

Reference Name Description
C.4.1.1 Comment code Request internal comment
C.5.1.1 Comment type Fixed: 'T'

Element Descriptions

Request internal comment (C.4.1.1)

If the query message field Q.14.1.3 contains 'AddComment', this field is contains the result request internal comment.
