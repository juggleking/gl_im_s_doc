Work order message - Synchronized procedure data result record (R)

Usage

This record contains synchronized procedure data result information. It is sent with a work order result synchronization message.

Element Layout

Reference Name Description
R.3.1.4 Procedure data keyword Fixed: 'PRCD'
R.3.1.5 History index Procedure data history index
R.3.1.6 Sequence number Procedure data sequence number
R.3.1.7 Test ID Procedure data test ID
R.4.1.1 Result value Procedure data test result value
R.5.1.1 Result unit Procedure data result unit
R.9.1.1 Result status Procedure data result status
R.12.1.1 Start time Procedure data result status start time (date time value)
R.13.1.1 End time Procedure data result status end time (date time value)

Element Descriptions

Procedure data history index (R.3.1.5)

This field contains the procedure data history index value of the synchronized procedure data result.

Procedure data sequence number (R.3.1.6)

This field contains the procedure data sequence number value of the synchronized procedure data result.

Procedure data result test ID (R.3.1.4)

This field contains outbound ID of the synchronized result's procedure data. If the outbound ID is not available, the ID is the result's property mnemonic.

Procedure data result value (R.4.1.1)

This field contains the result value of a synchronized procedure data result.

Procedure data result unit (R.5.1.1)

This field contains the unit name of procedure data property.

Procedure data result status (R.9.1.1)

This field contains the status of the synchronized procedure data result. The table below show the possible codes according to the statuses in Glims:

Test status code

Result status in Glims
X Discontinued
CT Modified and confirmed
CV Modified and validated


Procedure data result status start time (R.12.1.1)

This field contains the object time of the synchronized procedure data result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the availability time of the result.

Procedure data status end time (R.13.1.1)

This field contains the availability time of the synchronized procedure data result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the confirmation time of the result.
