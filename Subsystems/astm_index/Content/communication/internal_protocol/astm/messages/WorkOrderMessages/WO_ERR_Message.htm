ASTM Work order error message

Usage

This message is a reply to a query message for which Glims detected an error.

The work order error message is returned when:

  • the query message explicitly requested for it by means of field  Q.14.1.10.
  • the query message is requesting sorting information for a specimen by means of field Q.14.1.9.

Message terminator record (L)

Error reply for which search for data failed

For a single specimen or patient query, this means the search for the specimen or patient was not successful.

For a global query (like a specimen ALL query), this means no single instance (specimen or patient) was found that met the criteria of the query.
