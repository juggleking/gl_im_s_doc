Work order message - Carrier status record (M)

Usage

This record contains carrier information related to the microbiology carrier identified in the preceding carrier record. The status itself is available in M.4.1.3 of the carrier record.

The records are only available if the query message is requesting it (query specific tag 'StatusInfo')

Element Descriptions

Carrier creation time (M.4.1.1)

This field contains the time the carrier was created.

Carrier creation user (M.4.1.1)

This field contains the login name of the user responsible for creating the carrier.

Carrier status time (M.4.1.1)

This field contains the time the carrier has reached the status. It is either the carrier grafting time when it is known, or the carrier creation time when the grafting time is unknown.

Carrier status user (M.4.1.1)

This field contains the login name of the user responsible for reaching the status. It is either the carrier grafting user when it is known, or the carrier creation time when the grafting user is unknown.
