Work order message - Pathology carrier test record (M)

Usage

This record contains a pathology carrier test related to the pathology carrier identified in the preceding pathology carrier record.

Element Descriptions

Pathology carrier test sequence number (M.4.1.1)

This field contains the pathology carrier test sequence number value.

Medium test code (M.4.1.2)

This field contains the medium test mnemonic.

If the inquired station has a coding system with a code set on table medium test, then the value for this field is retrieved from the code value. If no code value is defined, the carrier test record is omitted

Medium test type (M.4.1.3)

This field represents the type of medium test used with the pathology carrier. Possible values are 'Block' and 'Slide'.

Carrier test status (M.4.1.4)

This field represents the status of the carrier test. Available codes are listed in the following table:

Test status code

Status in Glims Pathology carrier test sent earlier
I Pathology carrier test value is not available yet No
A Pathology carrier test value is available No
CI Pathology carrier test value is not available yet Yes
CA Pathology carrier test value is available Yes

Carrier test value (M.5.1.1)

The field contains the value of the pathology carrier test.
