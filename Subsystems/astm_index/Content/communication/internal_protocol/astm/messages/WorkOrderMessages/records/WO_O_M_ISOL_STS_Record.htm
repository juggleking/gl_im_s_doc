Work order message - Isolation status record (M)

Usage

This record contains status information related to the isolation identified in the preceding isolation result record. The status itself is available in R.9.1.1 of the isolation record.

The records are only available if the query message is requesting it (query specific tag 'StatusInfo')

Element Descriptions

Isolation creation time (M.4.1.1)

This field contains the time the isolation was created.

Isolation creation user (M.4.1.1)

This field contains the login name of the user responsible for creating the isolation.

Isolation status time (M.4.1.1)

This field contains the time the isolation has reached the status.

Isolation status user (M.4.1.1)

This field contains the login name of the user responsible for reaching the status.
