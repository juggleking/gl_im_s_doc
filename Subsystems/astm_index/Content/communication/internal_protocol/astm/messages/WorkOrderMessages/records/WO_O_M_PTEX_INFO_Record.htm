Work order message - Pathology examination info record (M)

Usage

This record contains information related to the pathology examination test identified in the preceding test order record. The pathology examination test itself is available in O.5.1.4 of the test order record.

Element Descriptions

Pathology examination responsible ID (M.4.1.1)

This field contains the internal ID of correspondent responsible for the pathology examination test.

Pathology examination responsible name (M.4.1.1)

This field contains the name of correspondent responsible for the pathology examination test.

Pathology examination assignment name (M.4.1.1)

his field contains the login name of the user to whom the pathology examination test is assigned to.

Pathology examination department name (M.4.1.1)

This field contains the name of department where the pathology examination test is executed.

Pathology examination status time (M.4.1.1)

This field contains the time the pathology examination test has reached the status (Confirmation, validation of discontinuation time).

Pathology examination status user name (M.4.1.1)

This field contains the login name of the user responsible for reaching the status.
