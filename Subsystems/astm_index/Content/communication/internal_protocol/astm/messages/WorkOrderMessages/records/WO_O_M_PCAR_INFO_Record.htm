Element Descriptions

Pathology carrier processing time (M.4.1.1)

This field contains the time the pathology carrier was processed.

Pathology carrier processing user name (M.4.1.1)

This field contains the login name of the user responsible for processing the pathology carrier.

Pathology carrier comment (M.4.1.1)

This field contains the pathology carrier comment text.

Pathology carrier creation time (M.4.1.1)

This field contains the time the pathology carrier was created in Glims.

Pathology carrier creation user name (M.4.1.1)

This field contains the login name of the user responsible for creating the pathology carrier.

Pathology carrier status time (M.4.1.1)

This field contains the time the pathology carrier has reached the status sent in M.4.1.3 of the pathology carrier record.

Pathology carrier status user name (M.4.1.1)

This field contains the login name of the user responsible for the pathology carrier reaching the status, sent in M.4.1.3 of the pathology carrier record.
