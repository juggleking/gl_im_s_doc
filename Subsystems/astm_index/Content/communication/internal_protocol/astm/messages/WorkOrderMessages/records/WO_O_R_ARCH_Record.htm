Work order message - Archive result record (R)

Usage

This record is available when the query specific tag 'ArchiveData' is available in the query message and the result is validated and from an 'archive' assessment method.

Element Layout

Reference Name Description
R.3.1.4 Test ID Archive result test ID
R.4.1.1 Result value Validated archive result value
R.5.1.1 Result unit Archive result unit
R.12.1.1 Start time Archive result start time (date time value)
R.13.1.1 End time Archive result end time (date time value)

Element Descriptions

Archive result test ID (R.3.1.4)

The outbound ID of the archive result's assessment method.

Glims allows to configure alternate codes when dealing with concentrator and distributor stations. See Glims manual for more details.

Validated archive result value (R.4.1.1)

The validated result value from an 'archive' assessment method.

Archive result unit (R.5.1.1)

The unit name of the 'archive' assessment method, or the property's unit name if the latter is not specified.

Archive result start time (R.12.1.1)

The object time of the 'archive' result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the availability time of the result.

Archive result end time (R.13.1.1)

The availability time of the 'archive' result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the confirmation time of the result.
