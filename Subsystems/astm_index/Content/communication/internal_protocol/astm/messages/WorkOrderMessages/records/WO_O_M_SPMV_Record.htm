Work order message - Specimen variable record (M)

Usage

This record specimen variable information related to the specimen identified in the preceding test order record.

The fields 'specimen variable' and 'source body site name' of the inquired station have influence on the availability and content of these records. See the Glims manual for more details.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'spmv'
M.3.1.2 Record type name Fixed: 'SpecimenVariable'
M.4.1.1 Specimen variable name Specimen variable name
M.4.1.2 Source body site indication Source body site indication
M.5.1.1 Specimen variable choice Specimen variable choice
M.5.1.2 Specimen variable choice value Specimen variable choice value
M.6.1.1 Specimen variable extra info Specimen variable extra info

Element Descriptions

Specimen variable name (M.4.1.1)

This field contains the material variable name of the specimen variable.

Source body site indication (M.4.1.2)

This field contains the value 'SSBS' when the specimen variable is known as the source body site variable.

Specimen variable choice (M.5.1.1)

This field contains the material variable choice mnemonic of the specimen variable.

Specimen variable choice value (M.5.1.2)

Specimen variable extra info (M.6.1.1)

This field contains the material variable choice value of the specimen variable. The value is reported in the language of the service's batch user.
