Work order message - Historical result record (R)

Usage

This record contains information about historical results of the test referred in O.5.1.4.

The assessment method's field 'send previous results' in Glims determines if sending historical result is applicable for the test and the integer value in Q.14.1.4 indicates how many results Glims should retrieve.

Element Layout

Reference Name Description
R.3.1.4 Test ID Result test ID
R.4.1.1 Result value Result value
R.5.1.1 Result unit Result unit
R.12.1.1 Start time Result start time (date time value)
R.13.1.1 End time Result end time (date time value)

Element Descriptions

Result test ID (R.3.1.4)

The outbound ID of the historical result's assessment method.

Glims allows to configure alternate codes when dealing with concentrator and distributor stations. See Glims manual for more details.

Result value (R.4.1.1)

The confirmed or validated value of the historical result.

Result unit (R.5.1.1)

The unit name of historical result's assessment method, or the property's unit name if the latter is not specified.

Result start time (R.12.1.1)

The object time of the historical result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the availability time of the result.

Result end time (R.13.1.1)

The availability time of the historical result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the confirmation time of the result.
