Work order message - Donor info record (M)

Usage

These records contain donor information related to the blood bag ID identified in the preceding test order record.

Element Descriptions

Blood bag product code (M.5.1.1)

This field contains the donor's blood bag product mnemonic.

If the inquired station has a coding system with a code set on table blood product, the value for this field is retrieved from the code value. If no code value is defined, the record is omitted.

Blood bag product specification code (M.5.1.1)

This field contains the donor's blood bag product specification mnemonic.

If the inquired station has a coding system with a code set on table blood product specification, the value for this field is retrieved from the code value. If no code value is defined, the record is omitted.

Blood bag product specification barcode (M.5.1.2)

This field contains the donor's blood bag product specification barcode.

Donor's blood group (M.5.1.1)

This field contains the donor's blood group. See Glims manual for a list of blood group enumerated values.

Donor's rhesus (M.5.1.2)

This field contains the donor's rhesus. See Glims manual for a list of rhesus enumerated values.

Donor's antigens (M.5.1.1)

This field contains the donor's antigens as a summary list. The list contains the mnemonic of each antigen followed by a '+' or a '-' and a blank.

If the inquired station has a coding system with a code set on table antigen, the antigen is represented by the code value. If the code is not defined, the antigen is not present in the summary list. If non of the antigens in the list is coded, the summary list is empty and the record is omitted.
