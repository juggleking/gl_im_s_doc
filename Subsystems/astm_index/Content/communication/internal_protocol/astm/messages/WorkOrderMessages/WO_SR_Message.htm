ASTM Work order message for result synchronization

Usage

This work order message is used for result synchronization. See the query specific tag 'SynchronizeResults'.

Message Layout

Message header record (H)
{
Patient information record (P)
[
Patient information comment record (C)
]
{
Person relation record (M)
}
{
Test order record (O) - for each synchronized non procedure data result
[
Test order comment record - test related (C)
]
{
Synchronized result record (R)
{
Result status info record (M)
}
}
}
{
Test order record (O) - for each synchronized procedure data result
{
Synchronized procedure data result record (R)
{
Result status info record (M)
}
}
}
{
Microbiology test order block
}
}
Message terminator record (L)
