Work order message - Isolation test result record (R)

Usage

This record is available when microbiology is requested in the query message (see field the Q.14.1.6) and when Glims is configured for sending isolation tests in a work order message.

Element Layout

Reference Name Description
R.3.1.4 Isolation test ID Isolation test ID
R.3.1.5 Material type Material type
R.3.1.8 Isolation test record mark Fixed: 'BCT'
R.9.1.1 Isolation test status Isolation test status code

Element Descriptions

Isolation test ID (R.3.1.4)

This field contains the outbound id of the micro organism test method.

Material type (R.3.1.5)

This field contains the remote material id of the micro organism test method.

Isolation test status code (R.9.1.1)

This field represents the status of the isolation test. Available codes are listed in the following table:

Test status code

Status in Glims Carrier sent earlier Comment
I Isolation test availability time is unknown No  
A Isolation test availability time is filled in No  
O Isolation test has been removed No Glims sends removed isolation tests when microbiology synchronization is active
CI Isolation test availability time is unknown Yes  
CA Isolation test availability time is filled in Yes  
CO Isolation test has been removed Yes Glims sends removed isolation tests when microbiology synchronization is active
