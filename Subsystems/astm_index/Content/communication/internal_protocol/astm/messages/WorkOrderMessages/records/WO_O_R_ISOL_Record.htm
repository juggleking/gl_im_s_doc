Work order message - Isolation result record (R)

Usage

This record is available when microbiology is requested in the query message (see field the Q.14.1.6) and when Glims is configured for sending isolations in a work order message.

Element Layout

Reference Name Description
R.3.1.4 Isolation record keyword Fixed: 'ORG'
R.3.1.5 Isolation sequence Isolation sequence number
R.4.1.1 Organism code Organism code
R.4.1.4 Isolation carrier sequence number Isolation carrier sequence number
R.9.1.1 Isolation status code Isolation status code

Element Descriptions

Isolation sequence number (R.3.1.5)

By default, this field contains the internal sequence number of the isolation.

When the query message is requesting for entries from a report work list (see field Q.14.1.7), the number is either the internal or the external sequence number. The field isolation numbering on the inquired station determines which sequence number is used.

Organism code (R.4.1.1)

This field contains the isolation's organism mnemonic.

If the inquired station has a coding system with a code set on table micro organism, the value for this field is retrieved from the code value. If no code value is defined, the field remains empty.

Isolation carrier sequence number (R.4.1.4)

This field contains the sequence number of the carrier from which the isolation originates from.
