Work order message - Synchronized result record (R)

Usage

This record contains synchronized result information. It is sent with a work order result synchronization message.

Element Layout

Reference Name Description
R.3.1.4 Test ID Result test ID
R.4.1.1 Result value Synchronized test result value
R.5.1.1 Result unit Synchronized result unit
R.9.1.1 Result status Synchronized result status
R.12.1.1 Start time Synchronized result status start time (date time value)
R.13.1.1 End time Synchronized result status end time (date time value)

Element Descriptions

Result test ID (R.3.1.4)

This field contains the outbound ID of the synchronized result's assessment method.

Synchronized test result value (R.4.1.1)

This field contains the result value of a synchronized result.

Synchronized result unit (R.5.1.1)

This field contains the unit name of the assessment method, or the property's unit name if the latter is not specified.

Synchronized result status (R.9.1.1)

This field contains the status of the synchronized result. The table below show the possible codes according to the statuses in Glims:

Test status code

Result status in Glims
C Modified but not confirmed yet
X Discontinued
CT Modified and confirmed
CV Modified and validated


Synchronized result status start time (R.12.1.1)

This field contains the object time of the synchronized result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the availability time of the result.

Synchronized result status end time (R.13.1.1)

This field contains the availability time of the syncrhonized result.

If the query specific tag 'OldResultTimeMode' is available in the query message, the start time will be the confirmation time of the result.
