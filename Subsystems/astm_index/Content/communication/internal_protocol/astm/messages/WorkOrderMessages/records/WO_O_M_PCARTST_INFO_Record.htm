Element Descriptions

Pathology carrier test availability time (M.4.1.1)

This field contains the time the pathology carrier test value was available.

Pathology carrier test availability user name (M.4.1.1)

This field contains the login name of the user responsible for the availability of the pathology carrier test value.

Pathology carrier test creation time (M.4.1.1)

This field contains the time the pathology carrier test was created in Glims.

Pathology carrier test creation user name (M.4.1.1)

This field contains the login name of the user responsible for creating the pathology carrier test.

Pathology carrier test status time (M.4.1.1)

This field contains the time the pathology carrier test has reached the status sent in M.4.1.3 of the pathology carrier test record.

Pathology carrier test status user name (M.4.1.1)

This field contains the login name of the user responsible for the pathology carrier test reaching the status, sent in M.4.1.3 of the pathology carrier test record.
