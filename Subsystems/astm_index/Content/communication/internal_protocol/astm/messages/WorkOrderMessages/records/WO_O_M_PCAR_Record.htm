Work order message - Pathology carrier record (M)

Usage

This record contains a pathology carrier related to the pathology examination identified in the preceding test order record.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'PCar'
M.3.1.2 Record type name Fixed: 'PathologyCarrier'
M.4.1.1 Pathology carrier sequence number Pathology carrier sequence number
M.4.1.2 Pathology carrier ID code Pathology carrier ID code
M.4.1.3 Pathology carrier status code Pathology carrier status code
M.4.1.4 Pathology carrier medium type Pathology carrier medium type
M.5.1.1 Pathology carrier medium code Pathology carrier medium code

Element Descriptions

Pathology carrier sequence number (M.4.1.1)

This field contains the pathology carrier sequence number value.

Pathology carrier ID code (M.4.1.2)

This field contains the pathology carrier internal Id.

Pathology carrier status code (M.4.1.3)

This field represents the status of the carrier. Available codes are listed in the following table:

Test status code

Status in Glims Pathology carrier sent earlier
I Pathology carrier is not processed yet No
P Pathology carrier is has been processed No
CI Pathology carrier is not processed yet Yes
CP Pathology carrier is has been processed Yes

Pathology carrier medium type (M.4.1.4)

This field represents the type of medium used with the pathology carrier. Possible values are 'Block' and 'Slide'

Pathology carrier medium code (M.5.1.1)

This field contains the pathology carrier medium mnemonic.

If the inquired station has a coding system with a code set on table medium, the value for this field is retrieved from the code value. If no code value is defined, the carrier record is omitted
