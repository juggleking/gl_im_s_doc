ASTM Work order message

Usage

This message is the reply to a specimen or QC lot related query message.

Message Layout

Message header record (H)
{
Patient information record (P)
[
Patient information comment record (C)
]
{
Person relation record (M)
}
{
Ordinary test order block | Microbiology test order block | Cross match test order block | Pathology test order block
}
}
Message terminator record (L)

Ordinary test order block

Test order record (O)
{
Specimen variable record (M)
}
{
Specimen label record (M)
}
{
Test order comment record - not test related (C)
}
{
Procedure data result record (R)
}
[
Test order comment record - test related (C)
]
{
Historical result record (R)
}
{
Archive result record (R)
}

Microbiology test order block

Test order record (O)
{
Microbiology test status info record (M)
}
{

}
{
Specimen label record (M)
}
{
Carrier record (M)
{
Carrier status info record (M)
}
}
{
Test order comment record - not test related (C)
}
{
Procedure data result record (R)
}
[
Test order comment record - test related (C)
]
{
Isolation result record (R)
{
Isolation status record (M)
}
{
Carrier record (M) (1)
{
Carrier status info record (M)(1)
}
}
{
Isolation test record (R)
{
Isolation test status info record (M)
}
}
}

(1) Carrier records and carrier status info records of an preceding isolation record, are only available when requested in the query message (query specific tag 'CarrierWithIsolation')
