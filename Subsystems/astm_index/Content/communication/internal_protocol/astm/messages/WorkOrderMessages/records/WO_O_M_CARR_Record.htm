Work order message - Carrier record (M)

Usage

This record contains carrier information related to the specimen identified in the preceding test order record, or related to an isolation in a preceding isolation result record.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'carr'
M.3.1.2 Record type name Fixed: 'carrier'
M.4.1.1 Carrier sequence number Carrier sequence number
M.4.1.2 Carrier external ID Carrier external ID
M.4.1.3 Status code Carrier status code
M.4.1.4 Parent carrier number Parent carrier sequence number
M.4.1.5 Parent carrier external id Parent carrier external id
M.5.1.1 Carrier medium code Carrier medium code

Element Descriptions

Carrier sequence number (M.4.1.1)

This field contains the carrier sequence number value.

Carrier external ID (M.4.1.2)

This field contains the carrier external Id.

Carrier status code (M.4.1.3)

This field represents the status of the carrier. Available codes are listed in the following table:

Test status code

Status in Glims Carrier sent earlier Comment
I Carrier grafting time is unknown No  
G Carrier grafting time is filled in No  
O Carrier has been removed No Glims sends removed carriers when microbiology synchronization is active
CI Carrier grafting time is unknown No  
CG Carrier grafting time is filled in Yes  
CO Carrier has been removed Yes Glims sends removed carriers when microbiology synchronization is active

Parent carrier sequence number (M.4.1.4)

This field contains the sequence number of the carrier of which the current carrier is derived (analoguous to M.4.1.1).

Parent carrier external ID (M.4.1.5)

This field contains the external identifier string of the carrier of which the current carrier is derived (analoguous to M.4.1.2).

Carrier medium code (M.5.1.1)

This field contains the carrier's medium mnemonic.

If the inquired station has a coding system with a code set on table medium, the value for this field is retrieved from the code value. If no code value is defined, the carrier record is omitted
