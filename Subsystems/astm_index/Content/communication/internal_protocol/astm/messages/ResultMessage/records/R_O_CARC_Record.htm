Result message - Specimen carrier comment record (M)

Usage

This record contains carrier comment and follows a preceding specimen carrier record. The comment is related to that carrier.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'carc'
M.4.1.2 Carrier comment text Carrier comment text
M.4.1.3 Action code Action code ('R', 'A')

Element Descriptions

Carrier comment text (M.4.1.2)

This field contains the carrier comment text. Glims will store the text in the comment field of the carrier.

Action code (M.4.1.3)

This field contains the action code. The supported values are 'R' (replace) or 'A' (add).

When the action code is a 'R', Glims will try to replace the carrier comment. When the action code is a 'A', Glims will add the comment to the one already available.
