Result message - Microbiology test record (M)

Usage

A microbiology test record contains information about the microbiology examination carried out on the specimen.

This record is an alternate solution for fields received with the test order record: O.5.1.6 and O.5.1.7, since field O.5.1.6 is also used as the 'dilution factor' transmitted to Centralink/SampleNet.

When present, then the data in M.4.1.1 and/or M.4.1.2 overrules any data in O.5.1.6 and O.5.1.7.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'MTst'
M.4.1.1 Microbiology procedure Microbiology procedure
M.4.1.2 Microbiology test Microbiology test
M.5.1.1 Negative comment code Negative comment code
M.6.1.1 Microbiology test comment Microbiology test comment

Element Descriptions

Microbiology procedure (M.4.1.1)

This field contains the microbiology procedure name. Glims uses this field to find the microbiology action of the specimen being uploaded.

Microbiology test (M.4.1.2)

This field contains a test ID representing the microbiology procedure. Glims uses this field to find a grafting or processing property which is related to the microbiology action of the specimen being uploaded.

Negative comment code (M.5.1.1)

A code value representing the explicit negative answer choice of the microbiology action for the microbiology procedure that is being identified via the provided M.4.1.1 field or M.4.1.2 field.
The provided code is the mnemonic of a choice in the explicit negative answer choice list that is set in the GLIMS microbiology module. However, the code can also be mapped onto one of the choices from the mentioned choice list via a coding system on the station in GLIMS.

Microbiology test comment (M.6.1.1)

A comment for the microbiology action.
The provided comment will be appended to the existing microbiology action comment when not yet included. When the result of the append yields a comment that is too large, then the first line(s) will be replaced with "...".
