Result message - Message header record (H)

Element Layout

Besides the ASTM encoding characters, Glims recognizes the following fields:

Reference Name Description
H.5.1.1 Sender Name or ID Sender
H.9.1.1 Characteristics of Sender Translator name
H.14.1.1 Date and Time of Message Time (date time value)

Element Descriptions

Sender (H.5.1.1)

This field contains the message sender name or ID.  It is accessible by the unknown specimen trigger.

Translator name (H.9.1.1)

Glims uses this field to identify the translator record in the database. Field is required.

Time (H.14.1.1)

This field contains message date and time. It is accessible by the unknown specimen trigger.
