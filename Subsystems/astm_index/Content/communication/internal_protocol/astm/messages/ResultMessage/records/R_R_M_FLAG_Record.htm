Result message - Result flag record (M)

Usage

This record is an alternate manner to upload result flags to Glims. Is is not recommended but still supported for compatibility reasons.

More common is the upload via result comment flag records.

Element Layout

Reference Name Description
M.10.1.1 Result flag mode Flag treatment code ('FLAG', 'FLAG_APPEND')
M.11.1.1 Result flag Result flag

Element Descriptions

Flag treatment code (M.10.1.1)

This field determines how the result flag in the field below should be treated.

When the value is set to 'FLAG', the result flag in the field R.7.1.1 of the result record is replaced by the one in this record.

When the value is set to 'FLAG_APPEND', the flag of this record is treated as a next result flag.

If this field is empty or has another value, the record is omitted.

Result flag (M.11.1.1)

This field contains a result flag, known in Glims as the flag field of the station flag table.
