Result message - Entering device record (M)

Usage

The message record contains information about the device (computer) on which the data was entered.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'EnteringDeviceId'
M.4.1.1 Entering device ID Entering device ID

Element Descriptions

Entering device ID (M.4.1.1)

This field contains the identification of the entering device, usually the name of the computer.
