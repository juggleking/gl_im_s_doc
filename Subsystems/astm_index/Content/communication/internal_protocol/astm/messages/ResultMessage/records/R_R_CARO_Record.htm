Result message - Isolation carrier observation record (M)

Usage

The carrier observation information in this record is related to the isolation carrier record preceding it.
Grafting or checking a carrier for any growth during incubation are considered possible state changes on a carrier that can be observed and for with the outcome can be sent in a result message.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'caro'
M.4.1.1 Growth result Growth result ('N','P','C','U')
M.4.1.2 Status Status ('P','F')
M.4.1.3 Offset Time-offset "marker" to the grafting time
M.4.1.4 Start time Start time of the "action" that lead to this observation.
M.4.1.5 Completion time Completion time
M.6.1.1 Grafting result Grafting reuslt('S','T','F','U')

Element Descriptions

Growth result (M.4.1.1)

This field contains a growth result. Hereby indicating that the observation done was to inspect the carrier for growth.
The possible value are:

  • N: negative
  • P: positive
  • C: contaminated
  • U: unclassified

Status (M.4.1.2)

This field contains the status of the observation.
The possible value are:

  • P: preliminary. Indicates that this can still change in the future.
  • F: final. Indicates that the provided observation data will (and should) not change anymore.

Offset (M.4.1.3)

This field contains a marker (or label) to indicate a time offset of the last observation since grafting.

For example, '8h' or '16h' could be valid markers to indicate that the observation is 8 hours (or 16 hours) after the carrier was grafted.
Consider it way to indicate a certain phase in the incubation process.

Start time (M.4.1.4)

Optional datetime field to indicate at what time the observation started.

Completion time(M.4.1.5)

A datetime to indicate the actual completion time of the observation.
In case of a grafting result, this one is considered to be the grafting time of the carrier.

Grafting reuslt (M.6.1.1)

This field contains a grafting result. Hereby indicating that the observation is the observed outcome of the grafting of the carrier.
The possible value are:

  • S: success
  • T: temporary failure
  • F: failed
  • U: unfinished
