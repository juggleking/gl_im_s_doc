Result message - Sort result record (M)

Usage

A sort result record contains information about the result of the specimen's sorting process on the sorting system.

The sort result records are processed by Glims when the sample location type in field O.4.1.5 of the preceding test order record contains the values 'RS' or 'RSA'.

Element Descriptions

Date time of sort process (M.4.1.2)

This field contains the date and time a specimen was processed. 

The processing time is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Sorting zone code (M.4.1.2)

This field contains the zone code where the specimen was sorted to. Glims adds the information to the specimen audit trail.

Remaining volume after sort process (M.4.1.2)

This field contains the remaining volume for this specimen after it has been aliquotted.

In Glims, the value is stored in the specimen remaining size field, on condition that field M.4.1.3 contains a valid unit code.

The remaining volume is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Remaining volume unit code (M.4.1.3)

This field contains the remaining volume unit for this specimen.

Glims searches in a unit with a name that matches the code. If no unit is found, the remaining volume is ignored.

The unit is used to convert the volume in M.4.1.2 to the unit of the specimen's material. The converted value is stored in the specimen remaining size field.

Glims allows to map the unit code by defining a code set for unit table on the station's coding system. If the code set is available but the unit code is not defined, the remaining volume is ignored.

The remaining volume unit is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Aliquoted flag (M.4.1.1)

This field has a fixed value 'Aliquoted' and indicates a specimen has been aliquoted into sub specimens. Glims adds the information to the specimen audit trail.

The flag is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Decapped flag (M.4.1.1)

This field has a fixed value 'Decapped' and indicates a specimen has been decapped.

The flag is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Recapped flag (M.4.1.1)

This field has a fixed value 'Recapped' and indicates a specimen has been recapped.

The flag is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Centrifuged flag (M.4.1.1)

This field has a fixed value 'Centrifuged' and indicates a specimen has been centrifuged.

The flag is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Dilution factor code (M.4.1.2)

This field contains the dilution factor when the specimen has been diluted.

Dilution condition code (M.4.1.3)

This field contains the dilution condition when the specimen has been diluted.
