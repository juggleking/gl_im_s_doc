Element Descriptions

Comment code (C.4.1.1)

This field contains the result comment represented as a code.

The Glims station fields 'comment storage', 'comment as text reference' and 'multiple comment lines' have influence on the treatment of this field. See the Glims manual for more details.

Comment text (C.4.1.2)

This field contains the result comment represented as a text.

The Glims station fields 'comment storage' and 'multiple comment lines' have influence on the treatment of this field. See the Glims manual for more details.

Comment target (C.5.1.2)

This field indicates if a comment code and/or text is to be treated as 'internal' or 'external' comment.

Glims uses this field to decide if the comment value(s) are to be stored in the result internal or external comment.
