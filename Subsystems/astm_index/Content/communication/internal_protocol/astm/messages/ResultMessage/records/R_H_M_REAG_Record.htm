Result message - Global reagent information record (M)

Usage

This record contains reagent information related to the station receiving the result message. See the Glims manual for more details about reagent information storage.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'REAG'
M.4.1.1 Reagent lot number Reagent lot number
M.5.1.1 Reagent serial number Reagent serial number
M.7.1.1 Test ID Test ID
M.8.1.1 Instrument ID Instrument ID

Element Descriptions

Reagent lot number (M.4.1.1)

This field contains the reagent lot number.

Reagent serial number (M.5.1.1)

This field contains the reagent serial number.

Test ID (M.7.1.1)

This field contains the instrument test ID for which the reagent is used.

Instrument ID (M.8.1.1)

This field contains the instrument for which the reagent is used.

This field is meaningful if the sending application serves multiple instruments. Glims will search for a concentrator (or distributor) client station with a matching remote id. The reagent information will be linked to that station.
