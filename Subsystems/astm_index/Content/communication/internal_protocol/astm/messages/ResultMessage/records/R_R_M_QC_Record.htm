Result message - QC record (M)

Usage

This record contains QC data accompanying the (qc) result.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'QC'
M.4.1.1 Target value Value (decimal) used to set target parameters of the QCPopulation
M.5.1.1 Deviation value Value (decimal) used to set deviation parameters of the QCPopuplation

Element Descriptions

Target value (M.4.1.1)

This field contains a value (decimal) that will be used to set a target (precision and/or accuracy) for the QCPopulation of the QCResult in the R-record.

The interpretation of this field is specified on the station-level in GLIMS.

NOTE: No conversion is done in GLIMS on this value. It should be in the correct unit of the property.

Deviation value (M.5.1.1)

This field contains a value (decimal) that will be used as an allowed deviation for the QCPopulation of the QCResult in the R-record.

The interpretation of this field is specified on the station-level in GLIMS.

NOTE: No conversion is done in GLIMS on this value. It should be in the correct unit of the property.
