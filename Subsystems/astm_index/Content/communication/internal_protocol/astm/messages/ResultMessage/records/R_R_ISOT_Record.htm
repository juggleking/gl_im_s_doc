Result message - Isolation test result record (R)

Usage

This record contains an isolation test result value for the isolation identified by the preceding isolation result record.

Element Layout

Reference Name Description
R.3.1.4 Isolation test ID Isolation test ID
R.3.1.5 Isolation test result keyword Fixed: 'BCT'
R.4.1.1 Isolation test result value Isolation test result value
R.11.1.1 Operator Operator

Element Descriptions

Isolation test ID (R.3.1.4)

This field contains the ID of the isolation test. Glims uses the micro organism test method inbound id to identify test ID's.

Isolation test result value (R.4.1.1)

This field contains the isolation test result value. Glims stores it in the value field of the isolation test.

Operator (R.11.1.1)

This field contains the ID of the operator who performed the isolation test.

When the field 'store operator ID' is enabled on the result receiving station, Glims will search for the user with a login name or initials matching the operator ID. When no match is found, the isolation test record is rejected.
