Element Descriptions

Patient ID (P.3.1.1)

This field contains the patient ID. It is accessible by the unknown specimen trigger.

Second patient ID (P.4.1.1)

This field contains the second patient ID. It is accessible by the unknown specimen trigger.

Third patient ID (P.5.1.1)

This field contains the third patient ID. It is accessible by the unknown specimen trigger.

Physician (P.14.1.1)

This field contains the physician ID. It is accessible by the unknown specimen trigger.

Admission date (P.24.1.1)

This field contains the patient's admission date, expected format is CCYYMMDD. It is accessible by the unknown specimen trigger.

Discharged date (P.24.1.2)

This field contains the patient's discharged date, expected format is CCYYMMDD. It is accessible by the unknown specimen trigger.

Admission status (P.25.1.1)

This field contains the patient's admission status. It is accessible by the unknown specimen trigger.

Location ward (P.26.1.1)

This field contains the patient's location ward name or code. It is accessible by the unknown specimen trigger.

Location room (P.26.1.2)

This field contains the patient's room number. It is accessible by the unknown specimen trigger.

Hospital service (P.33.1.1)

This field contains the hospital service currently assigned to the patient. It is accessible by the unknown specimen trigger.
