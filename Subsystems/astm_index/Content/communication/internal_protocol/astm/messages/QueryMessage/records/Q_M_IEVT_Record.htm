Query message - Instrument event record (M)

Usage

An instrument event record contains information about an event that occurred when a specimen has been scanned (and inspected) on the sorting system. Although multiple records may follow the request information record, Glims will only process the last one.

Element Layout

Reference Name Description
M.3.1.1 Record type code Fixed: 'IEVT'
M.4.1.1 Event code Event code
M.4.1.2 Event date time Event date time (date time value)
M.4.1.3 Event text Event text

Element Descriptions

Event code (M.4.1.1)

This field contains the code of the event. In Glims this field is known as the code field of the station event code table.

When Glims finds the station event code, it will:

  1. execute the station event evaluation function (when defined). The return value of the function may influence the zone Glims will sort to.
  2. log the event in the specimen audit trail

Event date time (M.4.1.2)

This field contains the date and time the event occurred.

The event date time is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.

Event text (M.4.1.3)

This field describes the event.

The event text is accessible in Glims by the event evaluation function on the station. See the Glims manual for more details.
