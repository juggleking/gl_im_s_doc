Introduction

The log viewer is a tool to view and analyze log files. There are two types of log viewer: the Dual log viewer and the Multi log viewer.

The Dual log viewer allows you to open two log files. You can view both a translator/driver log and its corresponding internal service log. The Multi log viewer allows you to open an unlimited amount of log files.

Screens

Both the screen of the Dual as of the Multi log viewer consist of the same parts: an upper part which contains icons to execute actions, the log file itself and a lower part with extra information on the log file.

Log file

The log file part contains the following columns:

Date and time

Indicates when and at what time the log entry was registered.

Source

Indicates which module of the code generated the message.

Action

This field can either be empty or it mentions Xmt (send message) or Rdl (read message).

Channel

Indicates the communication channel.

Message

Indicates the message of the log entry.
  The following in the tab page of the Multi log viewer indicates that entries have been added to the log file and that they have not been viewed yet.

Lower part

Last modified

Indicates at what time the log file was last modified.

Log file size

Indicates the actual size of the log file in bytes.


Location

Button which indicates the full path where the log file is saved. When clicking this button, you will be taken to the location where the log file is saved.

Total

Indicates the total number of lines in the log file.

Selected

Indicates the number of lines that are currently part of the selection. The counter is updated as lines are added to the log file.

Current

Indicates the sequence number of the selected line. Please note that this number is not a counter of the number of displayed lines.
